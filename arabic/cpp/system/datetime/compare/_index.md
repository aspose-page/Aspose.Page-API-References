## DateTime::Compare method


تقارن قيمتين ممثلتين بواسطة مثيلات محددة من فئة [DateTime](../) وتُرجع القيمة التي تشير إلى المواقع النسبية للقيم على خط الزمن.

```cpp
static constexpr int System::DateTime::Compare(DateTime t1, DateTime t2)
```


| Parameter | Type | الوصف |
| --- | --- | --- |
| t1 | DateTime | المقارنة الأولى |
| t2 | DateTime | المقارنة الثانية |

### ReturnValue

قيمة أقل من 0 إذا كان **t1** أسبق من **t2**؛ 0 إذا كان **t1** هو نفسه **t2**؛ قيمة أكبر من 0 إذا كان **t1** لاحقًا على **t2**
