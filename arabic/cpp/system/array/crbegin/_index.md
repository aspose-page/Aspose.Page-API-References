## Array::crbegin method


تُرجع مُكرِّرًا عكسيًا إلى العنصر الأول في الحاوية المعكوسة. وهو يُطابق العنصر الأخير في الحاوية غير المعكوسة. إذا كانت الحاوية فارغة، فإن المُكرِّر المُرجع يساوي [crend()](../crend/).

```cpp
const_reverse_iterator System::Array<T>::crbegin() const noexcept
```


### ReturnValue

مُكرِّر يشير إلى العنصر الأخير المؤهل كـ const في الحاوية.
