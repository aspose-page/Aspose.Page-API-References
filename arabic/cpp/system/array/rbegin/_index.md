## Array::rbegin() const method


تُرجِع مُؤَشِّرًا عكسيًا إلى العنصر الأول في الحاوية المعكوسة. وهو يطابق العنصر الأخير في الحاوية غير المعكوسة. إذا كانت الحاوية فارغة، يكون المُؤَشِّر المُرجَع مساوٍ لـ [rend()](../rend/).

```cpp
const_reverse_iterator System::Array<T>::rbegin() const noexcept
```


### ReturnValue

مُؤَشِّر يُشير إلى العنصر الأخير في الحاوية المؤهَّلة بـ const.
