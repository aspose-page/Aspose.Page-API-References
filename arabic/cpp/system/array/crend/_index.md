## Array::crend method


يعيد مُكرِّرًا عكسيًا إلى العنصر الذي يلي العنصر الأخير في الحاوية المعكوسة. وهو يتطابق مع العنصر الذي يسبق العنصر الأول في الحاوية غير المعكوسة. هذا العنصر يعمل كعنصر نائب، ومحاولة الوصول إليه تؤدي إلى سلوك غير معرف.

```cpp
const_reverse_iterator System::Array<T>::crend() const noexcept
```


### ReturnValue

مكرِّر يشير إلى العنصر النظري المؤهَّل بـ const الذي يسبق العنصر الأول في الحاوية.
