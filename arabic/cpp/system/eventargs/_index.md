## EventArgs class


الفئة الأساسية للفئات التي تمثل سياقًا يُمرَّر إلى المشتركين في الحدث عندما يُطلق الحدث. يجب تخصيص كائنات هذه الفئة فقط باستخدام الدالة [System::MakeObject()](../makeobject/). لا تقم أبدًا بإنشاء نسخة من هذا النوع على المكدس أو باستخدام المشغل new، لأن ذلك سيسبّب أخطاء وقت التشغيل و/أو أخطاء التأكيد. احْط دائمًا هذه الفئة بمؤشر [System::SmartPtr](../smartptr/) واستخدم هذا المؤشر لتمريره إلى الدوال كمعامل.

```cpp
class EventArgs : public System::Object
```
