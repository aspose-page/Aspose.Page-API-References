## System::CheckedCast method

```cpp
template<typename TTo,typename TFrom> TTo System::CheckedCast(TFrom value)
```


| Parameter | الوصف |
| --- | --- |
| TTo | النوع الذي يجب تحويل القيمة المحددة إليه |
| TFrom | نوع القيمة المحددة |

| Parameter | Type | الوصف |
| --- | --- | --- |
| value | TFrom | القيمة المراد تحويلها |
