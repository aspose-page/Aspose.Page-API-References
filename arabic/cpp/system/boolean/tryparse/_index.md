## Boolean::TryParse method


يحوّل السلسلة المحددة إلى قيمة من نوع bool.

```cpp
static bool System::Boolean::TryParse(const String &value, bool &result)
```


| Parameter | Type | الوصف |
| --- | --- | --- |
| value | const String\& | السلسلة للتحويل. |
| النتيجة | bool\& | المرجع إلى متغيّر bool حيث تُوضع نتيجة التحويل؛ تكون النتيجة true إذا كانت السلسلة المحددة مساوية لـ "True" و false إذا كانت السلسلة المحددة مساوية لـ "False". |

### ReturnValue

True إذا كانت السلسلة المحددة مساوية إما لـ "True" أو "False"، وإلا - false.
