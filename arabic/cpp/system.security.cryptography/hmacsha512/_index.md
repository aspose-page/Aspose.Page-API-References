## HMACSHA512 class


رمز توثيق الرسائل القائم على التجزئة [Authentication](../../system.security.authentication/) يستخدم دالة التجزئة [SHA512](../sha512/). تم تنفيذها جزئياً. يجب إنشاء كائنات هذه الفئة فقط باستخدام الدالة [System::MakeObject()](../../system/makeobject/). لا تقم بإنشاء مثال من هذا النوع على المكدس أو باستخدام المشغل new، لأن ذلك سيؤدي إلى أخطاء وقت التشغيل و/أو أعطال التأكيد. احرص دائماً على تغليف هذه الفئة في مؤشر [System::SmartPtr](../../system/smartptr/) واستخدام هذا المؤشر لتمريره إلى الدوال كمعامل.

```cpp
class HMACSHA512 : public System::Security::Cryptography::HashAlgorithm
```
