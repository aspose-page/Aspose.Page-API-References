## HMAC class


يجب أن ترث جميع تطبيقات رمز المصادقة القائم على التجزئة للرسائل [Authentication](../../system.security.authentication/) ([HMAC](./)) هذه الفئة المجردة. يجب تخصيص كائنات هذه الفئة فقط باستخدام الدالة [System::MakeObject()](../../system/makeobject/). لا تقم بإنشاء نسخة من هذا النوع على المكدس أو باستخدام المشغل new، لأن ذلك سيؤدي إلى أخطاء وقت التشغيل أو أخطاء تأكيد. احرص دائمًا على تغليف هذه الفئة في مؤشر [System::SmartPtr](../../system/smartptr/) واستخدام هذا المؤشر لتمريره إلى الدوال كمعامل.

```cpp
class HMAC : public System::Security::Cryptography::HashAlgorithm
```
