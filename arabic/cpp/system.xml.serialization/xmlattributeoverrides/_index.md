## XmlAttributeOverrides class


يسمح بتجاوز السمات عندما يتم استخدام [XmlSerializer](../xmlserializer/) لتسلسل أو إلغاء تسلسل كائن. يجب تخصيص كائنات هذه الفئة فقط باستخدام الدالة [System::MakeObject()](../../system/makeobject/). لا تقم بإنشاء مثيل من هذا النوع على المكدس أو باستخدام المشغل new، لأن ذلك سيسبب أخطاء تشغيلية و/أو أخطاء تأكيد. احرص دائمًا على تغليف هذه الفئة في مؤشر [System::SmartPtr](../../system/smartptr/) واستخدام هذا المؤشر لتمريره إلى الدوال كمعامل.

```cpp
class XmlAttributeOverrides : public System::Object
```
