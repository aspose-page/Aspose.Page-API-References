## NamespaceHandling enum


يحدد ما إذا كان يجب إزالة إعلانات مساحة الاسم المتكررة في [XmlWriter](../xmlwriter/).

```cpp
enum class NamespaceHandling
```

### القيم

| الاسم | القيمة | الوصف |
| --- | --- | --- |
| Default | 0 | يحدد أن إعلانات مساحة الاسم المتكررة لن تُزال. |
| OmitDuplicates | 1 | يحدد أن إعلانات مساحة الاسم المتكررة ستُزال. لكي تُزال مساحة الاسم المتكررة، يجب أن يتطابق البادئة ومساحة الاسم. |
