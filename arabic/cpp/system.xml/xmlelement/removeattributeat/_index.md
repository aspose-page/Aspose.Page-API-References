## XmlElement::RemoveAttributeAt method


يزيل عقدة السمة ذات الفهرس المحدد من العنصر. (إذا كانت السمة التي أزيلت لها قيمة افتراضية، يتم استبدالها فورًا).

```cpp
virtual SharedPtr<XmlNode> System::Xml::XmlElement::RemoveAttributeAt(int32_t i)
```


| Parameter | Type | الوصف |
| --- | --- | --- |
| i | int32_t | فهرس العقدة التي سيتم إزالتها. العقدة الأولى لها الفهرس 0. |

### ReturnValue

عقدة السمة التي تم إزالتها أو **nullptr** إذا لم توجد عقدة في الفهرس المحدد.
