## HttpContext class


فئة تجريبية تجعل ربط الشيفرة المترجمة مع مراجع [HttpContext](./) ممكنًا، لكنها لا تُنفّذ. لا تحتوي على أي أعضاء مُنفّذة بشكل صحيح. يجب إنشاء كائنات هذه الفئة فقط باستخدام الدالة [System::MakeObject()](../../system/makeobject/) function. لا تقم أبدًا بإنشاء نسخة من هذا النوع على المكدس أو باستخدام العامل new، لأن ذلك سيتسبب في أخطاء وقت التشغيل أو أخطاء تأكيد. احرص دائمًا على تغليف هذه الفئة في مؤشر [System::SmartPtr](../../system/smartptr/) واستخدام هذا المؤشر لتمريره إلى الدوال كمعامل.

```cpp
class HttpContext : public System::Object
```
