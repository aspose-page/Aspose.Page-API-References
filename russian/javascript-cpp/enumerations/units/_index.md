## Units enumeration

Указывает тип размера.

```js
enum Units
```

### Значения

| Имя | Значение | Описание |
| ---        | ---   | --- |
| Points | `0` | Точки (1/72 дюйма). |
| Inches | `1` | Дюймы. |
| Millimeters | `2` | Миллиметры. |
| Centimeters | `3` | Сантиметры. |
| Percents | `4` | Проценты существующего размера. |
