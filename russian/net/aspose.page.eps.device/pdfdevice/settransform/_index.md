## PdfDevice.SetTransform method

Определяет текущее преобразование. Поскольку большинство форматов вывода не реализуют эту функцию, обратное преобразование currentTransform вычисляется и умножается на заданное преобразование . Затем результат пересылается вызовом call в writeTransform(Transform).

```csharp
public override void SetTransform(Matrix transform)
```

| Параметр | Тип | Описание |
| --- | --- | --- |
| transform | Matrix | Преобразование, которое необходимо применить. |
