## DocumentCoverFront.CoverFrontOption.PrintBoth field

Указывает, что обложка, указанная параметром «CoverFrontSource», может быть напечатана на любой стороне титульного листа. Если[`DocumentCoverFrontSource`](../../documentcoverfrontsource/)[`ParameterInit`](../../parameterinit/) Элемент не указан, эту опцию следует игнорировать.

```csharp
public static CoverFrontOption PrintBoth;
```
