## PageDeviceColorSpaceProfileURI class

Указывает относительный URI корневого пакета пакета для профиля ICC, содержащегося в документе XPS. Обработка этого параметра зависит от настройки функции PageDeviceColorSpaceUsage. Предполагается, что все элементы, использующие этот профиль, уже окрашены в соответствующий цвет устройства space, и не будет управлять цветом в драйвере или устройстве. https://docs.microsoft.com/en-us/windows/win32/printdocs/pagedevicecolorspaceprofileuri

public sealed class PageDeviceColorSpaceProfileURI : StringParameterInit, IDocumentPrintTicketItem,
