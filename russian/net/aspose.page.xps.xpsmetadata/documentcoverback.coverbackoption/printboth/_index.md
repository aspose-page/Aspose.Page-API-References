## DocumentCoverBack.CoverBackOption.PrintBoth field

Указывает, что обложка, указанная параметром «CoverBackSource», может быть напечатана на обеих сторонах титульного листа. Если[`DocumentCoverBackSource`](../../documentcoverbacksource/)[`ParameterInit`](../../parameterinit/) Элемент не указан, эту опцию следует игнорировать.

```csharp
public static CoverBackOption PrintBoth;
```
