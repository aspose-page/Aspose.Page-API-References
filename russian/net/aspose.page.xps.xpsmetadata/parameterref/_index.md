## ParameterRef class

Класс, реализующий общую ссылку на параметр PrintTicket. A элемент определяет ссылку на[`ParameterInit`](../parameterinit)элемент. А[`ScoredProperty`](../scoredproperty) элемент, содержащий элемент ParameterRef, не имеет явно установленного[`Value`](../value) элемент. Вместо этого[`ScoredProperty`](../scoredproperty) element получает свое значение из[`ParameterInit`](../parameterinit) элемент, на который ссылается a элемент. https://docs.microsoft.com/en-us/windows/win32/printdocs/parameterref
