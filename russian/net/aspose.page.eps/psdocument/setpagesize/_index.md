## PsDocument.SetPageSize method

Устанавливает размер страницы. Для создания страниц разного размера в одном документе используйте[`SetPageDevice`](../setpagedevice/) сразу после этого метода.

```csharp
public void SetPageSize(float width, float height)
```

| Параметр | Тип | Описание |
| --- | --- | --- |
| width | Single | Ширина страницы в результирующем файле PostScript. |
| height | Single | Высота страницы в результирующем файле PostScript. |
