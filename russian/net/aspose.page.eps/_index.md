**Aspose.Page.EPS** является корневым пространством имен для всех классов, работающих с файлами PS/EPS.

## Классы

| Учебный класс | Описание |
| --- | --- |
| [FontConstants](./fontconstants/) | Этот класс определяет набор констант для сохранения шрифтов. |
| [GradientBrush](./gradientbrush/) | Этот класс используется для инкапсуляции LinearGradientBrush и PathGradientBrush с возможностью установки режима обтекания на зажим. |
| [LoadOptions](./loadoptions/) | Базовый класс для параметров загрузки документов. |
| [PageConstants](./pageconstants/) | Этот класс определяет набор констант, описывающих страницу. Удобные объекты предоставляются для различных полей, ориентаций, масштабирования и стандартных размеров страницы. |
| [PsConverterException](./psconverterexception/) | Этот класс содержит информацию об ошибке, которая возникает при преобразовании файла PS в документ PDF. |
| [PsDocument](./psdocument/) | Этот класс инкапсулирует документы PS/EPS. |
| [PsDocumentException](./psdocumentexception/) | Этот класс содержит информацию об ошибке, которая возникает при создании, редактировании и сохранении файла PostScript. |
| [PsLoadOptions](./psloadoptions/) | Параметры загрузки документа PS. |
