## PdfSaveOptions.OutlineTreeExpansionLevel property

Указывает, до какого уровня следует расширять структуру документа при просмотре файла PDF. 1 - показаны только элементы структуры первого уровня, 2 - показаны элементы структуры только первого и второго уровня, и так далее. По умолчанию 1.

public int OutlineTreeExpansionLevel { get; set; }
