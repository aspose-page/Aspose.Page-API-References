## PdfSaveOptions.OutlineTreeHeight property

Определяет высоту дерева структуры документа для сохранения. 0 - дерево структуры не будет преобразовано, 1 - будут преобразованы только элементы структуры первого уровня, и т. д. По умолчанию 10.

```csharp
public int OutlineTreeHeight { get; set; }
```
