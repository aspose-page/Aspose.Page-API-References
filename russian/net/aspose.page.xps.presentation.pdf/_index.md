**Aspose.Page.Xps.Presentation.Pdf** пространство имен предоставляет классы для преобразования документа XPS в документ PDF.

## Классы

| Учебный класс | Описание |
| --- | --- |
| [PdfDevice](./pdfdevice/) | Класс, инкапсулирующий устройство для создания изображений. |
| [PdfEncryptionDetails](./pdfencryptiondetails/) | Содержит детали для шифрования PDF. |
| [PdfSaveOptions](./pdfsaveoptions/) | Класс для параметров сохранения XPS-as-PDF. |
## перечисление

| перечисление | Описание |
| --- | --- |
| [PdfEncryptionAlgorithm](./pdfencryptionalgorithm/) | Перечисление режима шифрования. Опишите с помощью алгоритма и длины ключа. Это перечисление расширено, чтобы можно было еще больше расширить функциональность. Это перечисление реализует шаблон "Base-to-Core". |
| [PdfImageCompression](./pdfimagecompression/) | Указывает тип сжатия, применяемый к изображениям в файле PDF. |
| [PdfTextCompression](./pdftextcompression/) | Определяет тип сжатия, применяемый ко всему содержимому файла PDF, кроме изображений. |
