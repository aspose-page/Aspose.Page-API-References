## SaveOptions.AdditionalFontsFolders property

Указывает дополнительные папки, в которых конвертер должен найти шрифты для входного документа. Папка по умолчанию — стандартная папка шрифтов, в которой ОС находит шрифты для внутренних нужд.

```csharp
public string[] AdditionalFontsFolders { get; set; }
```
