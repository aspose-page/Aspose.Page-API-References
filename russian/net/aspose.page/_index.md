**Aspose.Page**является корневым пространством имен для всех классов библиотеки Aspose.Page, которые находятся либо непосредственно в ней, например **Устройство** или косвенно через несколько подпространств имен.

## Классы

| Учебный класс | Описание |
| --- | --- |
| [BuildVersionInfo](./buildversioninfo/) | Этот класс предоставляет информацию о текущей сборке продукта. |
| [Device](./device/) | Этот класс инкапсулирует рендеринг документа на абстрактное устройство. Рендеринг документа выполняется страница за страницей. |
| [Document](./document/) | Суперкласс для всех инкапсулированных документов. |
| [ExternalFontCache](./externalfontcache/) | Используйте этот класс для получения инкапсуляции шрифта в форме, приемлемой для[`Device`](../aspose.page/device/) . |
| [License](./license/) | Предоставляет методы лицензирования компонента. |
| [Margins](./margins/) | Этот класс инкапсулирует верхнее, левое, нижнее и правое поля. |
| [Metered](./metered/) | Предоставляет методы для установки измеренного ключа. |
| [SaveOptions](./saveoptions/) | Этот класс содержит параметры, необходимые для управления процессом преобразования. |
| [UserProperties](./userproperties/) | Специальный класс свойств, который позволяет устанавливать типизированные свойства и возвращать . Это также позволяет искать соединение двух объектов свойства по умолчанию , если этот объект свойства не содержит свойства. |
