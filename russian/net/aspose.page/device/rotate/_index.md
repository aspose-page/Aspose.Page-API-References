## Rotate(double) {#rotate}

Повернуть текущую матрицу преобразования. Вызывает writeTransform(Transform). Вращение с положительным углом тета поворачивает точки на положительной оси x по направлению к положительной оси y.

```csharp
public virtual void Rotate(double theta)
```

| Параметр | Тип | Описание |
| --- | --- | --- |
| theta | Double | Угол в радианах, на который нужно повернуть. |
