## Device.IsDirectRGB property

Указывает, использует ли устройство прямой режим RGB, то есть RGB.

```csharp
public virtual bool IsDirectRGB { get; }
```

### Возвращаемое значение

True, если используется прямой режим RGB, и false в противном случае, то есть BGR.
