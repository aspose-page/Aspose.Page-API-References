## OpenPage(string) {#openpage_1}

Выполняет необходимую подготовку устройства перед рендерингом страницы.

```csharp
public bool OpenPage(string title)
```

| Параметр | Тип | Описание |
| --- | --- | --- |
| title | String | Заголовок страницы. |

### Возвращаемое значение

Истина, если страница из запрошенного диапазона, в противном случае ложь. Используется в устройствах, которые могут отображать указанный массив номеров страниц.

## OpenPage(float, float) {#openpage}

Делает необходимую подготовку устройства перед рендерингом каждой страницы.

```csharp
public bool OpenPage(float width, float height)
```

| Параметр | Тип | Описание |
| --- | --- | --- |
| width | Single | Ширина страницы. |
| height | Single | Высота страницы. |

### Возвращаемое значение

Возвращает true, если номер открытой страницы попадает в диапазон выбранных номеров страниц, и false в противном случае.
