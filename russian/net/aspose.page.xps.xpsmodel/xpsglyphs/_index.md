## XpsGlyphs class

Класс, инкапсулирующий функции элемента Glyphs. Этот элемент представляет собой серию однородно отформатированного текста из одного шрифта. Он предоставляет информацию, необходимую для точного рендеринга, и поддерживает функции поиска и выбора при просмотре потребителей.

```csharp
public sealed class XpsGlyphs : XpsContentElement
```
