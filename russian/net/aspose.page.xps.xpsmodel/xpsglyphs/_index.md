## XpsGlyphs class

Класс, инкапсулирующий функции элемента Glyphs. Этот элемент представляет собой серию однородно отформатированного текста из одного шрифта. Предоставляет информацию, необходимую для точного рендеринга, и поддерживает функции поиска и выбора при просмотре потребителей.
