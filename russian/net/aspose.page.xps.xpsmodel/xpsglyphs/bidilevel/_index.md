## XpsGlyphs.BidiLevel property

Возвращает/устанавливает значение, определяющее уровень двунаправленной вложенности алгоритма Unicode. Четные значения подразумевают расположение слева направо, нечетные значения подразумевают расположение справа налево. Компоновка справа налево помещает начало прогона с правой стороны первого глифа, с положительной шириной продвижения (представляющей продвижение влево), размещая последующие глифы слева от предыдущего глифа.

public int BidiLevel { get; set; }
