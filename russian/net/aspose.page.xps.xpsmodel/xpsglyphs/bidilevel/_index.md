## XpsGlyphs.BidiLevel property

Возвращает/задает значение, указывающее уровень вложенности двунаправленного алгоритма Unicode. Четные значения подразумевают расположение слева направо, нечетные значения подразумевают размещение справа налево. первого глифа, с положительной шириной продвижения (представляющей продвижение влево), размещение последующих глифов слева от предыдущего глифа.

```csharp
public int BidiLevel { get; set; }
```
