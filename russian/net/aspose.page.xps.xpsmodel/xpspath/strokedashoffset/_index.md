## XpsPath.StrokeDashOffset property

Возвращает/устанавливает начальную точку для повторения шаблона массива штрихов. Если это значение опущено, массив штрихов выравнивается с исходной точкой штриха.

```csharp
public float StrokeDashOffset { get; set; }
```
