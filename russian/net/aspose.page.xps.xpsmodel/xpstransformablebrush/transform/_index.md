## XpsTransformableBrush.Transform property

Возвращает/задает матричное преобразование, примененное к координатному пространству кисти. Свойство Transform объединяется с текущим эффективным преобразованием рендеринга для получения эффективного преобразования рендеринга, локального для кисти. Окно просмотра для brush преобразуется с использованием локального эффективного преобразования рендеринга.

```csharp
public XpsMatrix Transform { get; set; }
```
