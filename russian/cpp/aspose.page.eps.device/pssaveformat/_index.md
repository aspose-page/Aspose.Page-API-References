## PsSaveFormat enum


Это перечисление содержит доступные варианты формата сохранения. Он может быть PS или [EPS](../../aspose.page.eps/). [EPS](../../aspose.page.eps/) используется только для одностраничных документов, тогда как файл PS может содержать любое количество страниц.

```cpp
enum class PsSaveFormat
```

### Значения

| Имя | Значение | Описание |
| --- | --- | --- |
| PS | 0 | Этот параметр указывает, что результирующий документ должен быть файлом PostScript (PS) или Encapsulated PostScript ([EPS](../../aspose.page.eps/)). |
| EPS | 1 | Этот параметр указывает, что результирующий документ должен быть файлом Encapsulated PostScript ([EPS](../../aspose.page.eps/)). Он используется только для одностраничных документов;. |
