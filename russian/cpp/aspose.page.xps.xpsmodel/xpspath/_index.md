## XpsPath class


Класс, инкапсулирующий свойства элемента Path. Этот элемент является единственным способом добавления векторной графики и изображений на фиксированную страницу. Он определяет одну векторную графику, которая будет отрисована на странице.

```cpp
class XpsPath : public Aspose::Page::XPS::XpsModel::XpsContentElement
```
