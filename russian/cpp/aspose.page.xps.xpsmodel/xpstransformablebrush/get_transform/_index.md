## XpsTransformableBrush::get_Transform method


Возвращает/устанавливает матричное преобразование, применяемое к координатному пространству кисти. Свойство Transform конкатенируется с текущим эффективным преобразованием рендеринга, чтобы получить эффективное преобразование рендеринга, локальное для кисти. Область просмотра кисти преобразуется с использованием локального эффективного преобразования рендеринга.
