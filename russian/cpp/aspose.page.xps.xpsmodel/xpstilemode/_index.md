## XpsTileMode enum


Допустимые значения свойства TileMode плиточных кистей.

```cpp
enum class XpsTileMode
```

### Значения

| Имя | Значение | Описание |
| --- | --- | --- |
| None | 0 | В этом режиме рисуется только один базовый тайл. Оставшаяся область остаётся прозрачной. |
| Tile | 1 | В этом режиме базовый тайл рисуется, а оставшаяся область заполняется повторением базового тайла так, чтобы правый край каждого тайла прилегал к левому краю следующего, а нижний край каждого тайла прилегал к верхнему краю следующего. |
| FlipX | 2 | Расположение тайлов похоже на режим Tile, но чередующиеся столбцы тайлов отражаются по горизонтали. Базовый тайл позиционируется согласно указанному вьюпорту. Тайлы в столбцах слева и справа от этого тайла отражаются по горизонтали. |
| FlipY | 3 | Расположение тайлов похоже на режим Tile, но чередующиеся строки тайлов отражаются по вертикали. Базовый тайл позиционируется согласно указанному вьюпорту. Строки выше и ниже отражаются по вертикали. |
| FlipXY | 4 | Расположение плиток аналогично режиму Tile tile, но чередующиеся столбцы плиток отражаются по горизонтали, а чередующиеся строки плиток — по вертикали. Базовая плитка позиционируется согласно указаниям области просмотра. |
