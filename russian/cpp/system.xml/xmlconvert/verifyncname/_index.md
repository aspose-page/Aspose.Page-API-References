## XmlConvert::VerifyNCName method


Проверяет, что имя является допустимым **NCName** согласно рекомендации W3C Extended Markup Language. **NCName** — это имя, которое не может содержать двоеточие.

```cpp
static String System::Xml::XmlConvert::VerifyNCName(const String &name)
```


| Параметр | Тип | Описание |
| --- | --- | --- |
| name | const String\& | Имя для проверки. |

### ReturnValue

Имя, если оно является допустимым NCName.
