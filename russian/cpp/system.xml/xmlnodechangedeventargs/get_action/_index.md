## XmlNodeChangedEventArgs::get_Action method


Возвращает значение, указывающее, какой тип события изменения узла происходит.

```cpp
XmlNodeChangedAction System::Xml::XmlNodeChangedEventArgs::get_Action()
```


### ReturnValue

Одно из значений перечисления, описывающих событие изменения узла. Значение **[Action](../../../system/action/)** не различает, когда событие произошло (до или после). Вы можете создать отдельные обработчики событий для обработки обоих случаев.
