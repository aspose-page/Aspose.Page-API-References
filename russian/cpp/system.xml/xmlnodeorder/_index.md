## XmlNodeOrder enum


Описывает порядок узлов в документе относительно второго узла.

```cpp
enum class XmlNodeOrder
```

### Значения

| Имя | Значение | Описание |
| --- | --- | --- |
| Before | 0 | Текущий узел этого навигатора находится перед текущим узлом предоставленного навигатора. |
| После | 1 | Текущий узел этого навигатора находится после текущего узла предоставленного навигатора. |
| То же | 2 | Оба навигатора находятся на одном и том же узле. |
| Неизвестно | 3 | Позиции узлов нельзя определить в порядке документа относительно друг друга. Это может произойти, если два узла находятся в разных деревьях. |
