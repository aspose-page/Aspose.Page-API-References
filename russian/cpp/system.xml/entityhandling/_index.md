## EntityHandling enum


Указывает, как [XmlTextReader](../xmltextreader/) или [XmlValidatingReader](../xmlvalidatingreader/) обрабатывают сущности.

```cpp
enum class EntityHandling
```

### Значения

| Имя | Значение | Описание |
| --- | --- | --- |
| ExpandEntities | 1 | Разворачивает все сущности и возвращает развернутые узлы. |
| ExpandCharEntities | 2 | Разворачивает символьные сущности и возвращает общие сущности в виде узлов [XmlNodeType::EntityReference](../xmlnodetype/). |
