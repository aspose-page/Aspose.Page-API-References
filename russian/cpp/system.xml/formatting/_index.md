## Formatting enum


Указывает параметры форматирования для [XmlTextWriter](../xmltextwriter/).

```cpp
enum class Formatting
```

### Значения

| Имя | Значение | Описание |
| --- | --- | --- |
| None | 0 | Специальное форматирование не применяется. Это значение по умолчанию. |
| Indented | 1 | Заставляет дочерние элементы отступать в соответствии с настройками [XmlTextWriter::set_Indentation](../xmltextwriter/set_indentation/) и [XmlTextWriter::set_IndentChar](../xmltextwriter/set_indentchar/). |
