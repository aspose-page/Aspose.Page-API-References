## WhitespaceHandling enum


Указывает, как обрабатываются пробельные символы.

```cpp
enum class WhitespaceHandling
```

### Значения

| Имя | Значение | Описание |
| --- | --- | --- |
| All | 0 | Возвращает узлы **Whitespace** и **SignificantWhitespace**. Это значение по умолчанию. |
| Significant | 1 | Возвращает только узлы **SignificantWhitespace**. |
| None | 2 | Не возвращает узлы **Whitespace** и **SignificantWhitespace**. |
