## EmfType enum


Указывает типы записей, размещаемых в файле EMF.

```cpp
enum class EmfType
```

### Значения

| Имя | Значение | Описание |
| --- | --- | --- |
| EmfOnly | n/a | [Windows](../../system.windows/) расширенный метафайл. Содержит команды GDI. Метапо файлы этого типа называют файлом EMF. |
| EmfPlusOnly | n/a | [Windows](../../system.windows/) расширенный метафайл плюс. Содержит команды GDI+. Метапо файлы этого типа называют файлом EMF+. |
| EmfPlusDual | n/a | Двойной [Windows](../../system.windows/) расширенный метафайл. Содержит эквивалентные команды GDI и GDI+. Метапо файлы этого типа называют файлом EMF+. |
