## ImageLockMode enum


Указывает свойства области изображения, которая блокируется.

```cpp
enum class ImageLockMode
```

### Значения

| Имя | Значение | Описание |
| --- | --- | --- |
| ReadOnly | 1 | Регион заблокирован только для чтения. |
| WriteOnly | 2 | Регион заблокирован только для записи. |
| ReadWrite | n/a | Регион заблокирован как для чтения, так и для записи. |
| UserInputBuffer | 4 | Изображение находится в буфере ввода пользователя, доступ к которому контролируется пользователем. |
