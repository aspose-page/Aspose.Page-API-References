## MetafileType enum


Представляет тип графического метафайла.

```cpp
enum class MetafileType
```

### Значения

| Имя | Значение | Описание |
| --- | --- | --- |
| Недействительный | 0 | Недопустимый метафайл. |
| Wmf | 1 | Стандартный WMF. |
| WmfPlaceable | 2 | Формат размещаемого [Metafile](../metafile/). |
| Emf | 3 | EMF (не EMF+). |
| EmfPlusOnly | 4 | EMF+ без двойных, записей более низкого уровня. |
| EmfPlusDual | 5 | EMF+ с двойными, записями более низкого уровня. |
