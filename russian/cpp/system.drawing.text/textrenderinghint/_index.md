## TextRenderingHint enum


Указывает качество рендеринга текста.

```cpp
enum class TextRenderingHint
```

### Значения

| Имя | Значение | Описание |
| --- | --- | --- |
| SystemDefault | 0 | Глиф с системным режимом рендеринга по умолчанию. |
| SingleBitPerPixelGridFit | 1 | Битовая карта глифа с хинтингом. |
| SingleBitPerPixel | 2 | Битовая карта глифа без хинтинга. |
| AntiAliasGridFit | 3 | Сглаживание с хинтингом. |
| AntiAlias | 4 | Битовая карта глифа со сглаживанием без хинтинга. |
| ClearTypeGridFit | 5 | Битовая карта глифа CT с хинтингом. |
