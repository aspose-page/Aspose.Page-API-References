```cpp
class IXpsTextConversionOptions : public virtual System::Object
```

## Методы

| Метод | Описание |
| --- | --- |
| virtual [get_PreserveText](./get_preservetext/)() | В [XPS](../../aspose.page.xps/) некоторые текстовые элементы могут содержать ссылки на альтернативные формы глифов, которые не соответствуют ни одному коду символа в шрифте. Если этот флаг установлен в true, текст из таких элементов [XPS](../../aspose.page.xps/) преобразуется в графические фигуры. Затем сам текст отображается прозрачным поверх них. Это делает текст таких элементов выделяемым. Но побочным эффектом является то, что выходной файл может быть значительно больше оригинала. Если этот флаг установлен в false, символы, которые должны отображаться как альтернативные формы, заменяются другими символами, которые сопоставляются с альтернативными глифами. Поэтому текст, хотя и остаётся выделяемым, будет изменён и, вероятно, станет нечитаемым. |
