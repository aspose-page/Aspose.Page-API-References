## IXpsTextConversionOptions::get_PreserveText method


В [XPS](../../../aspose.page.xps/) некоторые текстовые элементы могут содержать ссылки на альтернативные формы глифов, которые не соответствуют ни одному коду символа в шрифте. Если этот флаг установлен в true, текст из таких [XPS](../../../aspose.page.xps/) элементов преобразуется в графические фигуры. Затем сам текст отображается прозрачным поверх них. Это делает текст таких элементов выделяемым. Однако побочный эффект — выходной файл может быть значительно больше исходного. Если флаг установлен в false, символы, которые должны отображаться как альтернативные формы, заменяются другими символами, которые сопоставляются с альтернативными глифами. Поэтому текст, хотя и остаётся выделяемым, будет изменён и, скорее всего, станет нечитаемым.

virtual bool Aspose::Page::XPS::Presentation::IXpsTextConversionOptions::get_PreserveText()=0
