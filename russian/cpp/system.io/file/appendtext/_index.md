## File::AppendText method


Создает объект [StreamWriter](../../streamwriter/), который добавляет текст в указанный файл, используя кодировку UTF-8. Если указанный файл не существует, он создается.

```cpp
static StreamWriterPtr System::IO::File::AppendText(const String &path)
```


| Параметр | Тип | Описание |
| --- | --- | --- |
| путь | const String\& | Путь к файлу для открытия или создания |

### ReturnValue

Указатель shared pointer на созданный объект [StreamWriter](../../streamwriter/), связанный с указанным файлом
