## FileAttributes enum

### Значения

| Имя | Значение | Описание |
| --- | --- | --- |
| Archive | 32 | Элемент файловой системы является кандидатом для удаления или резервного копирования. |
| Compressed | 2048 | Элемент файловой системы сжат. |
| Device | 64 | Not used. |
| Directory | 16 | Элемент файловой системы является каталогом. |
| Encrypted | 16384 | Элемент файловой системы зашифрован. |
| Скрытый | 2 | Элемент файловой системы скрыт. |
| IntegrityStream | 32768 | Элемент файловой системы поддерживает целостность данных. |

| NotContentIndexed | 8192 | Элемент файловой системы не будет индексироваться службой индексации содержимого. |
| NoScrubData | 131072 | Элемент файловой системы не включён в сканирование целостности данных. |
| Offline | 4096 | Элемент файловой системы находится в офлайн-режиме, и его данные недоступны сразу. |

| ReparsePoint | 1024 | Элемент файловой системы — файл, содержащий точку повторного разбора. |

| Временный | 256 | Объект файловой системы является временным файлом. |
| Недействительный | 0 | Недопустимое значение. |
