## DashCap enum

```cpp
enum class DashCap
```

### Значения

| Имя | Значение | Описание |
| --- | --- | --- |
| Плоский | 0 | Квадратная заглушка, которая делает квадратными оба конца штриха. |
| Округление | 2 | Круглая заглушка, которая скругляет оба конца штриха. |
| Треугольный | 3 | Треугольная заглушка, образующая концы штриха в виде треугольников. |
