## CombineMode enum


Указывает, как комбинируются области отсечения.

```cpp
enum class CombineMode
```

### Значения

| Имя | Значение | Описание |
| --- | --- | --- |
| Заменить | 0 | Одна область отсечения заменяется другой. |
| Пересечение | 1 | Две области отсечения объединяются путем взятия их пересечения. |
| Объединение | 2 | Две области отсечения объединяются путем взятия их объединения. |
| Xor | 3 | Две области отсечения объединяются, беря только площадь, заключённую в одну из областей, но не в обе одновременно. |
| Исключить | 4 | Две области отсечения объединяются, беря площадь первой области, не пересекающуюся со второй. |
| Дополнение | 5 | Две области отсечения объединяются, беря площадь второй области, не пересекающуюся с первой. |
