## LineCap enum


Указывает доступные стили окончаний, которыми объект [Pen](../../system.drawing/pen/) может завершать линию.

```cpp
enum class LineCap
```

### Значения

| Имя | Значение | Описание |
| --- | --- | --- |
| Плоский | 0 | Плоская кромка линии. |
| Квадратный | 1 | Квадратная кромка линии. |
| Округление | 2 | Круглая кромка линии. |
| Треугольный | 3 | Треугольное окончание линии. |
| NoAnchor | 16 | Без привязки. |
| SquareAnchor | 17 | Квадратный якорный конец линии. |
| RoundAnchor | 18 | Круглый якорный конец линии. |
| DiamondAnchor | 19 | Алмазный якорный конец линии. |
| ArrowAnchor | 20 | Стрелочный якорный конец линии. |
| AnchorMask | 240 | Маска, используемая для проверки, является ли конец линии якорным. |
| Custom | 255 | Пользовательский конец. |
