## LineJoin enum


Указывает, как соединяются последовательные линии или кривые.

```cpp
enum class LineJoin
```

### Значения

| Имя | Значение | Описание |
| --- | --- | --- |
| Miter | 0 | Mitered join, которое образует острый или обрезанный угол в зависимости от того, превышает ли длина среза предел среза. |
| Bevel | 1 | Bevel join, который образует диагональный угол. |
| Округление | 2 | Round join, который образует плавную круглую дугу между линиями. |
| MiterClipped | 3 | Mitered join, который образует острый или фасочный угол в зависимости от того, превышает ли длина среза предел среза. |
