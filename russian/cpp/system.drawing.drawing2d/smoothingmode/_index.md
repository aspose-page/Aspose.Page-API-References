## SmoothingMode enum


Указывает, какой тип сглаживания (антиалиасинга) применяется к линиям, кривым и краям заполненных областей.

```cpp
enum class SmoothingMode
```

### Значения

| Имя | Значение | Описание |
| --- | --- | --- |
| Недействительный | n/a | Недопустимое значение. |
| Default | n/a | Режим по умолчанию, без сглаживания. |
| Высокая скорость | n/a | Без сглаживания, высокая скорость. |
| HighQuality | n/a | Сглаженный рендеринг, низкая скорость. |
| None | n/a | Без сглаживания. |
| AntiAlias | n/a | Сглаженный рендеринг. |
