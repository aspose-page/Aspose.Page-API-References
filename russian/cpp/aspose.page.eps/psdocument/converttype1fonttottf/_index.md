## PsDocument::ConvertType1FontToTTF method


Преобразует шрифт Type 1 в **TrueType**. Имя преобразованного TTF‑шрифта будет таким же, как у входного шрифта Type 1, с расширением ".ttf". Файл TTF будет сохранён в указанную выходную директорию.

```cpp
void Aspose::Page::EPS::PsDocument::ConvertType1FontToTTF(System::String type1FontFilePath, System::String outputDir)
```


| Параметр | Тип | Описание |
| --- | --- | --- |
| type1FontFilePath | System::String | Путь к файлу шрифта Type 1. |
| outputDir | System::String | Каталог вывода, в котором сохраняется полученный шрифт **TrueType**. |
