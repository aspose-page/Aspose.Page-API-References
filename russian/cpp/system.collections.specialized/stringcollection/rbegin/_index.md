## StringCollection::rbegin() const method


Возвращает обратный итератор к первому элементу перевёрнутого контейнера. Он соответствует последнему элементу неперевёрнутого контейнера. Если контейнер пуст, возвращаемый итератор равен [rend()](../rend/).

```cpp
const_reverse_iterator System::Collections::Specialized::StringCollection::rbegin() const noexcept
```


### ReturnValue

Итератор, указывающий на последний элемент контейнера с const‑квалификатором.
