## SaveOptions::set_JpegQualityLevel method


Категория Quality указывает уровень сжатия изображения. Доступные значения от 0 до 100. Чем меньше указанное число, тем выше степень сжатия и, соответственно, ниже качество изображения. Значение 0 дает изображение наихудшего качества, а 100 — наилучшего.
