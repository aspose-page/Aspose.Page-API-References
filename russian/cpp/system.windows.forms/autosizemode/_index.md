## AutoSizeMode enum


Определяет, как работает автоматическое изменение размера.

```cpp
enum class AutoSizeMode
```

### Значения

| Имя | Значение | Описание |
| --- | --- | --- |
| GrowAndShrink | 0 | [Control](../control/) растет и сжимается по необходимости. Его нельзя изменять вручную. |
| GrowOnly | 1 | [Control](../control/) растет по необходимости, но не может сжиматься ниже минимального размера. Его можно изменять вручную. |
