## FlatStyle enum


Определяет, будет ли элемент управления отрисован плоским или трёхмерным.

```cpp
enum class FlatStyle
```

### Значения

| Имя | Значение | Описание |
| --- | --- | --- |
| Flat | 0 | [Control](../control/) рисуется плоским. |
| Popup | 1 | [Control](../control/) рисуется плоским, если только мышь не перемещается над ним, когда он рисуется в 3D-стиле. |
| Standard | 2 | [Control](../control/) рисуется в 3D-стиле. |
| System | 3 | Стиль управляется ОС. |
