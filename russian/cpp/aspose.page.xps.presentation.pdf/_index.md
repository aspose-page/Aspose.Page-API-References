Пространство имен **Aspose.Page.Xps.Presentation.Pdf** предоставляет классы для рендеринга документа [XPS](../aspose.page.xps/) в документ PDF.

## Классы

| Класс | Описание |
| --- | --- |
| [PdfDigitalSignatureDetails](./pdfdigitalsignaturedetails/) | Содержит сведения о цифровой подписи PDF. |
| [PdfEncryptionDetails](./pdfencryptiondetails/) | Содержит сведения о шифровании PDF. |
| [PdfSaveOptions](./pdfsaveoptions/) | Класс параметров сохранения XPS как PDF. |
