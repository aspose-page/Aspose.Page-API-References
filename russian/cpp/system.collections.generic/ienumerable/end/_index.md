## IEnumerable::end() method


Возвращает итератор, указывающий сразу после последнего элемента (если он существует) коллекции. Этот итератор нельзя использовать для изменения ссылочного объекта, потому что [GetEnumerator()](../getenumerator/) возвращает копию‑объекта T.

```cpp
iterator System::Collections::Generic::IEnumerable<T>::end()
```
