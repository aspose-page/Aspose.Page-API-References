## SmartPtr::const_pointer_cast method


Преобразует указатель к другому типу с помощью const_cast над указываемым объектом.

```cpp
template<class Y> SmartPtr<Y> System::SmartPtr<T>::const_pointer_cast() const
```


| Параметр | Описание |
| --- | --- |
| Y | Целевой тип указного объекта. |

### ReturnValue

Указатель изменённого типа, который всегда находится в режиме shared.
