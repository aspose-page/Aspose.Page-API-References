## SmartPtr::get_shared_count method


Получает количество совместных указателей, существующих для ссылочного объекта, включая текущий. Проверяет, что текущий указатель находится в режиме совместного доступа.

```cpp
int System::SmartPtr<T>::get_shared_count() const
```


### ReturnValue

Количество существующих shared‑указателей на ссылочный объект, если таковые есть. Если указатель равен null, возвращает 0.
