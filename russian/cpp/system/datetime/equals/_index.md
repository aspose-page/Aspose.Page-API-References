## DateTime::Equals(DateTime) const method


Определяет, представляет ли указанный экземпляр класса [DateTime](../) то же значение даты и времени, что и текущий объект.

```cpp
bool System::DateTime::Equals(DateTime other) const
```


| Параметр | Тип | Описание |
| --- | --- | --- |
| other | DateTime | Экземпляр класса [DateTime](../), с которым сравнивается текущий объект. |

### ReturnValue

Истина, если оба **other** и текущий объект представляют одно и то же значение, иначе — ложь.
