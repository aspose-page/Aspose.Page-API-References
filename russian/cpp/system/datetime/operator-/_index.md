## DateTime::operator-(DateTime) const method

```cpp
TimeSpan System::DateTime::operator-(DateTime value) const
```


| Параметр | Тип | Описание |
| --- | --- | --- |
| value | DateTime | Экземпляр класса [DateTime](../) , обозначающий один конец вычисляемого интервала |

### ReturnValue

Экземпляр класса [TimeSpan](../../timespan/) , представляющий интервал времени между значениями даты и времени, представленными текущим объектом и **value**.

```cpp
DateTime System::DateTime::operator-(TimeSpan value) const
```


| Параметр | Тип | Описание |
| --- | --- | --- |
| value | TimeSpan | Интервал времени для вычитания |

### ReturnValue

Новый экземпляр класса [DateTime](../), представляющий значение даты и времени, полученное в результате вычитания **value** из значения, представленного текущим объектом.
