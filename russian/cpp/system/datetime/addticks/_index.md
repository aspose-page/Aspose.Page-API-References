## DateTime::AddTicks method


Возвращает новый экземпляр класса [DateTime](../), представляющий значение даты и времени, которое является суммой значения, представленного текущим объектом, и указанного количества интервалов в 100 наносекунд.

```cpp
DateTime System::DateTime::AddTicks(int64_t value) const
```


| Параметр | Тип | Описание |
| --- | --- | --- |
| value | int64_t | Количество интервалов в 100 наносекунд для добавления; значение может быть положительным или отрицательным. |

### ReturnValue

Новый экземпляр класса [DateTime](../), представляющий значение даты и времени, которое является суммой значения, представленного текущим объектом, и **value** интервалов в 100 наносекунд.
