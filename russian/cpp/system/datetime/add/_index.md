## DateTime::Add method


Возвращает новый экземпляр класса [DateTime](../), представляющий значение даты и времени, полученное в результате добавления указанного временного интервала к значению даты и времени, представленного текущим объектом.

```cpp
DateTime System::DateTime::Add(TimeSpan value) const
```


| Параметр | Тип | Описание |
| --- | --- | --- |
| value | TimeSpan | Временной интервал для добавления |

### ReturnValue

Новый экземпляр класса [DateTime](../), представляющий значение даты и времени, полученное в результате добавления временного интервала **tm** к значению даты и времени, представленному текущим объектом
