## DateTime::AddMilliseconds method


Возвращает новый экземпляр класса [DateTime](../), представляющий значение даты и времени, которое является суммой значения, представленного текущим объектом, и указанного количества миллисекунд.

```cpp
DateTime System::DateTime::AddMilliseconds(double value) const
```


| Параметр | Тип | Описание |
| --- | --- | --- |
| value | double | Количество миллисекунд для добавления; значение может быть положительным или отрицательным |

### ReturnValue

Новый экземпляр класса [DateTime](../), представляющий значение даты и времени, которое является суммой значения, представленного текущим объектом, и **value** миллисекунд
