## DateTime::Compare method


Сравнивает два значения, представленные указанными экземплярами класса [DateTime](../), и возвращает значение, указывающее относительные положения значений на временной шкале.

```cpp
static constexpr int System::DateTime::Compare(DateTime t1, DateTime t2)
```


| Параметр | Тип | Описание |
| --- | --- | --- |
| t1 | DateTime | Первый операнд сравнения |
| t2 | DateTime | Второй операнд сравнения |

### ReturnValue

Значение, меньшее 0, если **t1** раньше **t2**; 0, если **t1** равен **t2**; значение, большее 0, если **t1** позже **t2**
