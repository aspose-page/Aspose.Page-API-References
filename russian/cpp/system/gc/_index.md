## GC class


Представляет эмулированную сборку мусора, которая больше похожа на заглушку и фактически ничего не делает. Это статический тип без экземплярных сервисов. Вы никогда не должны создавать его экземпляры никакими способами.

```cpp
class GC : public System::Object
```
