## ObjectExt::UnboxStringSafe method


Разупаковывает строку из упакованного значения.

```cpp
static String System::ObjectExt::UnboxStringSafe(const SmartPtr<Object> &obj)
```


| Параметр | Тип | Описание |
| --- | --- | --- |
| obj | const SmartPtr\<Object\>\& | Упакованное строковое значение. |

### ReturnValue

Если **obj** является упакованной строкой, возвращает распакованное значение, иначе возвращает пустую строку.
