## Array::rend() const method


Возвращает обратный итератор на элемент, следующий за последним элементом обратного контейнера. Он соответствует элементу, предшествующему первому элементу обычного контейнера. Этот элемент служит заполнительным, попытка доступа к нему приводит к неопределённому поведению.

```cpp
const_reverse_iterator System::Array<T>::rend() const noexcept
```

Итератор, указывающий на теоретический элемент, предшествующий первому элементу контейнера с const‑квалификатором.
