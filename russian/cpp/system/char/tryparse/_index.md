## Char::TryParse method


Пытается преобразовать строку, состоящую из одного символа, в символ UTF-16. Функция успешно завершается только тогда, когда входная строка не равна null и имеет длину ровно один символ.

```cpp
static bool System::Char::TryParse(const System::String &s, char_t &result)
```


| Параметр | Тип | Описание |
| --- | --- | --- |
| s | const System::String\& | [String](../../string/) для преобразования |
| результат | char_t\& | Выходная переменная, которая будет содержать результат преобразования, если преобразование успешно |

### ReturnValue

Истина, если преобразование успешно, иначе - ложь
