## Char::IsPunctuation(char_t) method


Определяет, классифицируется ли указанный символ как символ пунктуации.

```cpp

```


| Параметр | Тип | Описание |
| --- | --- | --- |
| c | char_t | Символ для проверки |

### ReturnValue

True если указанный символ является символом пунктуации, иначе — false
