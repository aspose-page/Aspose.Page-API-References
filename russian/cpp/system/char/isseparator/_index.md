## Char::IsSeparator(char_t) method


Определяет, классифицирован ли указанный символ как разделительный символ.

```cpp
static bool System::Char::IsSeparator(char_t c)
```


| Параметр | Тип | Описание |
| --- | --- | --- |
| c | char_t | Символ для проверки |

### ReturnValue

True, если указанный символ является разделительным символом, иначе — false
