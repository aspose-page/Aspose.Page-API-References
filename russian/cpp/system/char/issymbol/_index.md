## Char::IsSymbol(char_t) method


Определяет, классифицирован ли указанный символ как символ.

```cpp
static bool System::Char::IsSymbol(char_t c)
```


| Параметр | Тип | Описание |
| --- | --- | --- |
| c | char_t | Символ для проверки |

### ReturnValue

True если указанный символ является символом, иначе - false
