## Object::SetTemplateWeakPtr method


Устанавливает n‑й аргумент шаблона как слабый указатель (вместо shared). Позволяет переключать указатели в контейнерах в режим weak.

```cpp
virtual void System::Object::SetTemplateWeakPtr(uint32_t argument)
```


| Параметр | Тип | Описание |
| --- | --- | --- |
| аргумент | uint32_t | Индекс шаблонного аргумента, который следует рассматривать как слабый указатель. |
