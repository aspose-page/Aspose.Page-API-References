## MarshalByRefObject class


Обеспечивает доступ к объектам за пределами границ домена приложений в приложениях с включённым удалённым вызовом. Объекты этого класса должны создаваться только с помощью функции [System::MakeObject()](../makeobject/). Никогда не создавайте экземпляр этого типа в стеке или с использованием оператора new, так как это приведёт к ошибкам времени выполнения и/или сбоям утверждений. Всегда оборачивайте этот класс в указатель [System::SmartPtr](../smartptr/) и используйте этот указатель для передачи его в функции в качестве аргумента.

```cpp
class MarshalByRefObject : public virtual System::Object
```
