## Boolean::Parse method


Преобразует указанную строку в значение типа bool.

```cpp
static bool System::Boolean::Parse(const String &value)
```


| Параметр | Тип | Описание |
| --- | --- | --- |
| value | const String\& | Строка для преобразования. |

### ReturnValue

Переменная типа bool, в которую помещается результат преобразования; результат равен true, если указанная строка равна "True", и false, если указанная строка равна "False".
