## TypeInfo::IsInstanceOfType method


Определяет, является ли указанный объект экземпляром текущего типа.

```cpp
bool System::TypeInfo::IsInstanceOfType(const SharedPtr<Object> &obj) const
```


| Параметр | Тип | Описание |
| --- | --- | --- |
| obj | const SharedPtr\<Object\>\& | Объект для сравнения с текущим типом |

### ReturnValue

true, если текущий тип находится в иерархии наследования объекта, представленного переменной obj
