## System::MemberwiseClone method


Выполняет поэлементное клонирование с использованием конструктора копирования.

```cpp
template<typename T> SmartPtr<Object> System::MemberwiseClone(T *ptr)
```


| Параметр | Описание |
| --- | --- |
| T | Класс для копирующего конструирования. Информация о подклассе теряется. |

| Параметр | Тип | Описание |
| --- | --- | --- |
| ptr | T * | Указатель на объект для клонирования. |

### ReturnValue

Указатель на клонированный объект.
