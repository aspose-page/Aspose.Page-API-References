## String::rend method


Возвращает обратный итератор к позиции перед первым символом (если есть) фактического буфера строки.

```cpp
reverse_iterator System::String::rend() const
```


### ReturnValue

Обратный итератор к элементу, предшествующему первому символу фактического буфера строки (если он существует), либо к элементу непосредственно перед выделенным буфером.
