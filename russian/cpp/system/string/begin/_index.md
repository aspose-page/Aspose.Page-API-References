## String::begin method


Возвращает указатель на начало фактического буфера строки. Никогда не переallocирует ничего. Не гарантирует, что буфер будет нуль-терминирован.

```cpp
const UChar * System::String::begin() const
```


### ReturnValue

Указатель на фактический буфер строки, нуль‑терминированный или нет.
