## String::IsNull method


Проверяет, считается ли строка null. [String](../) считается null только если она создана конструктором [String()](../string/), перемещена, скопирована или присвоена из null-строки, либо был вызван метод [reset()](../reset/).

```cpp
bool System::String::IsNull() const
```


### ReturnValue

true, если строка считается null, иначе false.
