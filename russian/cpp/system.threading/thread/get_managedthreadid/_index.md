## Thread::get_ManagedThreadId method


Получает идентификатор потока. Может быть получен из ОС, но если идентификатор потока ОС превышает пределы int, идентификаторы потоков могут пересекаться.

```cpp
int System::Threading::Thread::get_ManagedThreadId() const
```


### ReturnValue

Псевдо‑уникальный идентификатор потока.
