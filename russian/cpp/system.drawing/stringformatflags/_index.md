## StringFormatFlags enum


Указывает информацию о отображении и размещении для строк текста.

```cpp
enum class StringFormatFlags
```

### Значения

| Имя | Значение | Описание |
| --- | --- | --- |
| DirectionRightToLeft | 1 | Направление текста справа налево. |
| DirectionVertical | 2 | [Text](../../system.drawing.text/) вертикально выровнен. |
| FitBlackBox | 4 | Части символов могут выступать за пределы блока размещения. |
| DisplayFormatControl | 32 | Управляющие символы отображаются. |
| NoFontFallback | 1024 | [Font](../font/) резервный шрифт отключен. |
| MeasureTrailingSpaces | 2048 | Вставляет завершающий пробел в конец каждой строки. |
| NoWrap | 4096 | [Text](../../system.drawing.text/) перенос отключен. |
| LineLimit | 8192 | Полные строки размещаются в прямоугольнике форматирования. |
| NoClip | 16384 | Разрешено отображать выступающие части глифов. |
