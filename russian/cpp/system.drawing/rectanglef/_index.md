## RectangleF class


Представляет прямоугольную область изображения, определяемую координатами X и Y верхнего левого угла в одинарной точности с плавающей запятой, а также его шириной и высотой. Этот тип следует выделять в стеке и передавать в функции по значению или по ссылке. Никогда не используйте класс [System::SmartPtr](../../system/smartptr/) для управления объектами этого типа.

```cpp
class RectangleF
```
