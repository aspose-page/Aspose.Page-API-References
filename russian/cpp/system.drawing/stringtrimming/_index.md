## StringTrimming enum


Указывает, как символы должны обрезаться из строки, которая не помещается в форму размещения.

```cpp
enum class StringTrimming
```

### Значения

| Имя | Значение | Описание |
| --- | --- | --- |
| None | 0 | Без обрезки. |
| Символ | 1 | Обрезать до ближайшего символа. |
| Слово | 2 | Обрезать до ближайшего слова. |
| EllipsisCharacter | 3 | Обрезать до ближайшего символа и вставить многоточие в конец строки. |
| EllipsisWord | 4 | Обрезать до ближайшего слова и вставить многоточие в конец строки. |
| EllipsisPath | 5 | Центр удаляется из обрезанных строк и заменяется многоточием. Сохраняйте как можно большую часть последнего сегмента строки, разделённого слешами. |
