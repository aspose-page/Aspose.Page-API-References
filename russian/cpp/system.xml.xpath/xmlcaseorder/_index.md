## XmlCaseOrder enum


Указывает порядок сортировки для заглавных и строчных букв.

```cpp
enum class XmlCaseOrder
```

### Значения

| Имя | Значение | Описание |
| --- | --- | --- |
| None | 0 | Игнорировать регистр. |
| UpperFirst | 1 | Буквы в верхнем регистре сортируются перед буквами в нижнем регистре. |
| LowerFirst | 2 | Буквы в нижнем регистре сортируются перед буквами в верхнем регистре. |
