## JobDeviceLanguage class


Описывает языки устройств, поддерживаемые для передачи данных от драйвера к физическому устройству. Это часто называется "Page Description Language". Этот ключевое слово определяет, какой язык описания страницы поддерживается драйвером и физическим устройством.
