## JobPrimaryCoverFront class


Описывает переднюю (начальную) обложку. Весь задание будет иметь один основной лист. Обложка должна печататься с использованием [PageMediaSize](../pagemediasize/) и [PageMediaType](../pagemediatype/) для первой страницы задания. Обложка должна быть интегрирована в параметры обработки (например, [JobDuplexAllDocumentsContiguously](../jobduplexalldocumentscontiguously/), [JobNUpAllDocumentsContiguously](../jobnupalldocumentscontiguously/)) в соответствии с указанным параметром [Option](../option/). [https://docs.microsoft.com/en-us/windows/win32/printdocs/jobprimarycoverfront](https://docs.microsoft.com/en-us/windows/win32/printdocs/jobprimarycoverfront).

class JobPrimaryCoverFront : public Aspose::Page::XPS::XpsMetadata::Feature,
