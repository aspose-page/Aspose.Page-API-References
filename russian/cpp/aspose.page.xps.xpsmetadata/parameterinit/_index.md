## ParameterInit class


Класс, реализующий общую инициализацию параметров [PrintTicket](../printticket/). Базовый класс для всех инициализаторов параметров, определённых схемой. Определяет значение для экземпляра элемента **ParameterDef**. Элемент **[ParameterInit](./)** является целью ссылки, сделанной элементом [ParameterRef](../parameterref/). [https://docs.microsoft.com/en-us/windows/win32/printdocs/parameterinit](https://docs.microsoft.com/en-us/windows/win32/printdocs/parameterinit).
