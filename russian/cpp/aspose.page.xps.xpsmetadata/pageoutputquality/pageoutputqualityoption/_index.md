## Поля

| Поле | Описание |
| --- | --- |
| static [Automatic](./automatic/) | Указывает назначение автоматического вывода (позволяет клиенту автоматически выбирать наилучшие настройки). |
| static [Draft](./draft/) | Указывает назначение чернового вывода (сбалансировано для текста и графики чернового качества). |
| static [Fax](./fax/) | Указывает назначение факсового вывода (сбалансировано для стандартного качества факса). |
| static [High](./high/) | Указывает назначение вывода высокого качества (сбалансировано для текста и графики высокого качества). |
| static [Normal](./normal/) | Указывает назначение обычного вывода (сбалансировано для текста и графики хорошего качества). |
| static [Photographic](./photographic/) | Указывает назначение фотоснимков (изображения должны иметь наивысшее качество). |
| static [Text](./text/) | Указывает назначение вывода только текста (графика может выводиться плохо или вовсе не выводиться). |
