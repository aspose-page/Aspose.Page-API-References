## CoverBackOption class


Описывает параметры функции [DocumentCoverBack](../).

```cpp
class CoverBackOption : public Aspose::Page::XPS::XpsMetadata::Option
```

## Поля

| Поле | Описание |
| --- | --- |
| static [BlankCover](./blankcover/) | Указывает, что должна быть напечатана пустая обложка. |
| static [NoCover](./nocover/) | Указывает, что обложка выводиться не будет. |
| static [PrintBack](./printback/) | Указывает, что обложка, указанная в \"CoverBackSource\", должна быть напечатана на обратной стороне листа обложки. Если элемент [DocumentCoverBackSource](../../documentcoverbacksource/)[ParameterInit](../../parameterinit/) не указан, эту [Option](../../option/) следует игнорировать. |
| static [PrintBoth](./printboth/) | Указывает, что обложка, указанная в \"CoverBackSource\", может быть напечатана на любой из сторон листа обложки. Если элемент [DocumentCoverBackSource](../../documentcoverbacksource/)[ParameterInit](../../parameterinit/) не указан, эту [Option](../../option/) следует игнорировать. |
| static [PrintFront](./printfront/) | Указывает, что обложка, указанная в \"CoverBackSource\", должна быть напечатана на передней стороне листа обложки. Если элемент [DocumentCoverBackSource](../../documentcoverbacksource/)[ParameterInit](../../parameterinit/) не указан, эту [Option](../../option/) следует игнорировать. |
