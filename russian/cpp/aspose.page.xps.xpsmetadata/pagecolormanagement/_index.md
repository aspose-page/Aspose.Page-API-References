## PageColorManagement class


Настраивает управление цветом для текущей страницы. Это считается автоматическим в SHIM - DM_ICMMethod Add [System](../../system/). Описывает, какой компонент должен выполнять управление цветом (т.е. драйвер). [https://docs.microsoft.com/en-us/windows/win32/printdocs/pagecolormanagement](https://docs.microsoft.com/en-us/windows/win32/printdocs/pagecolormanagement).
