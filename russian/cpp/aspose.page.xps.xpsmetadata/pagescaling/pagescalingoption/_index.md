class PageScalingOption : public Aspose::Page::XPS::XpsMetadata::Option,

## Поля

| Поле | Описание |
| --- | --- |
| static [Custom](./custom/) | Указывает, что должна быть применена пользовательская масштабировка к Application Media Size. |
| static [CustomSquare](./customsquare/) | Указывает, что должна быть применена пользовательская квадратная масштабировка к Application Media Size. |
| static [FitApplicationBleedSizeToPageImageableSize](./fitapplicationbleedsizetopageimageablesize/) | Указывает, что размер обрезки приложения должен быть масштабирован до [PageImageableSize](../../pageimageablesize/), сохраняя соотношение сторон. |
| static [FitApplicationContentSizeToPageImageableSize](./fitapplicationcontentsizetopageimageablesize/) | Указывает, что размер содержимого приложения должен быть масштабирован до [PageImageableSize](../../pageimageablesize/), сохраняя соотношение сторон. |
| static [FitApplicationMediaSizeToPageImageableSize](./fitapplicationmediasizetopageimageablesize/) | Указывает, что размер носителя приложения должен быть масштабирован до [PageImageableSize](../../pageimageablesize/), сохраняя соотношение сторон. |
| static [FitApplicationMediaSizeToPageMediaSize](./fitapplicationmediasizetopagemediasize/) | Указывает, что размер носителя приложения должен быть масштабирован до [PageMediaSize](../../pagemediasize/), сохраняя соотношение сторон. |
| static [None](./none/) | Указывает, что масштабирование не должно применяться. |
