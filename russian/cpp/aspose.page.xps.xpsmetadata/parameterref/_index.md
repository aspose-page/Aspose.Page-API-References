## ParameterRef class


Класс, реализующий общую ссылку на параметр [PrintTicket](../printticket/). Элемент **[ParameterRef](./)** определяет ссылку на элемент [ParameterInit](../parameterinit/). Элемент [ScoredProperty](../scoredproperty/), содержащий элемент [ParameterRef](./), не имеет явно заданного элемента [Value](../value/). Вместо этого элемент [ScoredProperty](../scoredproperty/) получает своё значение из элемента [ParameterInit](../parameterinit/), на который ссылается **[ParameterRef](./)** элемент. [https://docs.microsoft.com/en-us/windows/win32/printdocs/parameterref](https://docs.microsoft.com/en-us/windows/win32/printdocs/parameterref).

```cpp
class ParameterRef : public Aspose::Page::XPS::XpsMetadata::PrintTicketElement,
                     public Aspose::Page::XPS::XpsMetadata::IPrintTicketItem
```
