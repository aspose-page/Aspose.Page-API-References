**com.aspose.eps** — корневой пакет для всех классов, работающих с файлами PS/EPS.


## Классы

| Класс | Описание |
| --- | --- |
| [FontConstants](../com.aspose.eps/fontconstants) | Этот класс определяет набор констант для сохранения шрифтов. |
| [HatchPaintLibrary](../com.aspose.eps/hatchpaintlibrary) | Преобразует штриховые кисти GDI+ в изображения, подходящие для записи в XPS и PDF. |
| [LoadOptions](../com.aspose.eps/loadoptions) | Базовый класс параметров загрузки документа. |
| [PageConstants](../com.aspose.eps/pageconstants) | Этот класс определяет набор констант, описывающих страницу. |
| [PsConverterException](../com.aspose.eps/psconverterexception) | Этот класс содержит информацию об ошибке, возникающей при преобразовании PS‑файла в PDF‑документ. |
| [PsConverterLicenseException](../com.aspose.eps/psconverterlicenseexception) | Этот класс содержит информацию об ошибке, связанной с лицензированием продукта Aspose.Page. |
| [PsDocument](../com.aspose.eps/psdocument) | Этот класс инкапсулирует документы PS/EPS. |
| [PsDocumentException](../com.aspose.eps/psdocumentexception) | Этот класс содержит информацию об ошибке, которая возникает при создании, завершении и сохранении файла PS. |
| [PsLoadOptions](../com.aspose.eps/psloadoptions) | Параметры загрузки PS‑документа. |
