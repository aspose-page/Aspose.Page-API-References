Пакет **com.aspose.xps.rendering** предоставляет базовые классы для рендеринга XPS‑документа в другие форматы.


## Классы

| Класс | Описание |
| --- | --- |
| [BmpSaveOptions](../com.aspose.xps.rendering/bmpsaveoptions) | Класс параметров сохранения XPS-as-BMP. |
| [ImageSaveOptions](../com.aspose.xps.rendering/imagesaveoptions) | Базовый класс параметров сохранения XPS-as-image. |
| [JpegSaveOptions](../com.aspose.xps.rendering/jpegsaveoptions) | Класс параметров сохранения XPS-as-JPEG. |
| [PdfEncryptionDetails](../com.aspose.xps.rendering/pdfencryptiondetails) | Содержит детали шифрования PDF. |
| [PdfSaveOptions](../com.aspose.xps.rendering/pdfsaveoptions) | Класс параметров сохранения XPS-as-PDF. |
| [PngSaveOptions](../com.aspose.xps.rendering/pngsaveoptions) | Класс параметров сохранения XPS-as-PNG. |
| [TiffSaveOptions](../com.aspose.xps.rendering/tiffsaveoptions) | Класс параметров сохранения XPS-as-TIFF. |

## Интерфейсы

| Интерфейс | Описание |
| --- | --- |
| [IEventBasedModificationOptions](../com.aspose.xps.rendering/ieventbasedmodificationoptions) | Определяет параметры, относящиеся к обработке модификаций, основанных на событиях, во время сохранения документа. |
| [IPipelineOptions](../com.aspose.xps.rendering/ipipelineoptions) | Определяет параметры конвертации, связанные с конфигурацией конвейера. |
| [IXpsTextConversionOptions](../com.aspose.xps.rendering/ixpstextconversionoptions) | Определяет параметры конвертации XPS в другие форматы. |

| Перечисление | Описание |

| [InterpolationMode](../com.aspose.xps.rendering/interpolationmode) | Указывает алгоритм, используемый при масштабировании или вращении изображений. |
