Описывает полный список поддерживаемых лотков для устройства. Позволяет указывать лоток вывода для каждого документа отдельно. Ключевые слова  JobOutputBin ,  DocumentOutputBin  и  PageOutputBin  являются взаимно исключающимися; только одно должно быть указано в документе PrintTicket или Print Capabilities. https://docs.microsoft.com/en-us/windows/win32/printdocs/documentoutputbin
