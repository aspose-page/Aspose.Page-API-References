public final class DocumentStaple extends Staple implements IJobPrintTicketItem, IDocumentPrintTicketItem

Описывает характеристики скрепления вывода. Каждый документ скрепляется отдельно. Ключевые слова JobStapleAllDocuments и DocumentStaple являются взаимно исключающими. Обработка ограничений между этими ключевыми словами зависит от драйвера. https://docs.microsoft.com/en-us/windows/win32/printdocs/documentstaple
