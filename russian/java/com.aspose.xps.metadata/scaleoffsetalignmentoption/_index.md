public static final class PageScaling.ScaleOffsetAlignmentOption extends Option

Описывает параметры функций  ScaleOffsetAlignment . Указывает выравнивание масштабированного содержимого.
## Поля

| Поле | Описание |
| --- | --- |
| [BottomCenter](#BottomCenter) | Указывает, что масштабирование должно быть выровнено по центру нижнего края носителя. |
| [BottomLeft](#BottomLeft) | Указывает, что масштабирование должно быть выровнено по нижнему левому углу носителя. |
| [BottomRight](#BottomRight) | Указывает, что масштабирование должно быть выровнено по нижнему правому углу носителя. |
| [Center](#Center) | Указывает, что масштабирование должно быть центрировано на носителе. |
| [LeftCenter](#LeftCenter) | Указывает, что масштабирование должно быть выровнено по центру левой стороны носителя. |
| [RightCenter](#RightCenter) | Указывает, что масштабирование должно быть выровнено по центру правой стороны носителя. |
| [TopCenter](#TopCenter) | Указывает, что масштабирование должно быть выровнено по центру верхней стороны носителя. |
| [TopLeft](#TopLeft) | Указывает, что масштабирование должно быть выровнено по верхнему левому углу носителя. |
| [TopRight](#TopRight) | Указывает, что масштабирование должно быть выровнено по верхнему правому углу носителя. |
