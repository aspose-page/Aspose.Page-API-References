Класс, реализующий инициализатор параметров печатного билета. Класс, который реализует общий инициализатор параметров PrintTicket. Базовый класс для всех инициализаторов параметров, определённых схемой. Определяет значение для экземпляра элемента  ParameterDef  . Элемент  ParameterInit  является целью ссылки, сделанной элементом  ParameterRef . https://docs.microsoft.com/en-us/windows/win32/printdocs/parameterinit
