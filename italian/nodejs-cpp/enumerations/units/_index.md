## Units enumeration

Specifica il tipo di dimensione.

```js
enum Units
```

### Valori

| Nome | Valore | Descrizione |
| ---        | ---   | --- |
| Points | `0` | Punti (1/72 di pollice). |
| Inches | `1` | Pollici. |
| Millimeters | `2` | Millimetri. |
| Centimeters | `3` | Centimetri. |
| Percents | `4` | Percentuali della dimensione esistente. |
