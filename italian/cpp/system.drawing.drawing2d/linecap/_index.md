## LineCap enum


Indica gli stili di cappuccio disponibili con cui un oggetto [Pen](../../system.drawing/pen/) può terminare una linea.

```cpp
enum class LineCap
```

### Valori

| Nome | Valore | Descrizione |
| --- | --- | --- |
| Piatto | 0 | Cap lineare piatto. |
| Quadrato | 1 | Cap lineare quadrato. |
| Arrotonda | 2 | Cap lineare arrotondato. |
| Triangolo | 3 | Cappuccio di linea triangolare. |
| NoAnchor | 16 | Nessun ancoraggio. |
| SquareAnchor | 17 | Cap lineare di ancoraggio quadrato. |
| RoundAnchor | 18 | Cap lineare di ancoraggio rotondo. |
| DiamondAnchor | 19 | Cap lineare di ancoraggio a diamante. |
| ArrowAnchor | 20 | Cap lineare di ancoraggio a freccia. |
| AnchorMask | 240 | Una maschera usata per verificare se un cap lineare è un cap di ancoraggio. |
| Custom | 255 | Cap personalizzato. |
