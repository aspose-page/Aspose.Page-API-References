## PixelOffsetMode enum


Specifica come i pixel vengono spostati durante il rendering.

```cpp
enum class PixelOffsetMode
```

### Valori

| Nome | Valore | Descrizione |
| --- | --- | --- |
| Non valido | n/a | Valore non valido. |
| Default | n/a | La modalità predefinita. |
| HighSpeed | n/a | Modalità a bassa qualità (alta prestazione). |
| HighQuality | n/a | Modalità ad alta qualità (prestazioni inferiori). |
| None | n/a | Nessuno spostamento dei pixel. |
