## EncryptionPolicy enum


Enumera le politiche di crittografia.

```cpp
enum class EncryptionPolicy
```

### Valori

| Nome | Valore | Descrizione |
| --- | --- | --- |
| RequireEncryption | 0 | Richiedi la crittografia e non consentire mai un cifrario 'Null'. |
| AllowNoEncryption | 1 | Preferisci utilizzare la crittografia completa, ma è possibile usare un cifrario 'Null' se il server è d'accordo. |
| NoEncryption | 2 | Consenti l'assenza di crittografia e richiedi che venga usato un cifrario 'Null' se l'altro endpoint può gestire un cifrario 'Null'. |
