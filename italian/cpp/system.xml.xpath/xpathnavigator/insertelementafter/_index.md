## XPathNavigator::InsertElementAfter method


Crea un nuovo elemento fratello dopo il nodo corrente utilizzando il prefisso di spazio dei nomi, il nome locale e l'URI dello spazio dei nomi specificati, con il valore specificato.

```cpp
virtual void System::Xml::XPath::XPathNavigator::InsertElementAfter(String prefix, String localName, String namespaceURI, String value)
```


| Parametro | Tipo | Descrizione |
| --- | --- | --- |
| prefisso | String | Il prefisso dello spazio dei nomi del nuovo elemento figlio (se presente). |
| localName | String | Il nome locale del nuovo elemento figlio (se presente). |
| namespaceURI | String | L'URI dello spazio dei nomi del nuovo elemento figlio (se presente). [String::Empty](../../../system/string/empty/) e **nullptr** sono equivalenti. |
| value | String | Il valore del nuovo elemento figlio. Se [String::Empty](../../../system/string/empty/) o **nullptr** vengono passati, viene creato un elemento vuoto. |
