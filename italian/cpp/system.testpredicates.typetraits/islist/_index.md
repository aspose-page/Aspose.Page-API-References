## IsList typedef


Verifica se il tipo è una specializzazione di [System::Collections::Generic::List](../../system.collections.generic/list/). In tal caso, il membro value è impostato su true, altrimenti è impostato su false.

```cpp
using System::TestPredicates::TypeTraits::IsList =  std::is_same<T, System::Collections::Generic::List<typename T::ValueType>>
```
