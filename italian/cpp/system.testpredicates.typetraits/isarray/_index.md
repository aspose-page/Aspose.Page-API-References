## IsArray typedef


Verifica se il tipo è una specializzazione di [System::Array](../../system/array/). In tal caso, il membro value è impostato su true, altrimenti è impostato su false.

```cpp
using System::TestPredicates::TypeTraits::IsArray =  std::is_same<T, System::Array<typename T::ValueType>>
```
