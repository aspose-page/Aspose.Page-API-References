## XmlConvert::VerifyXmlChars method


Restituisce la stringa fornita se tutti i caratteri e i caratteri di coppia surrogate nella stringa passata sono caratteri XML validi, altrimenti viene sollevata un'[XmlException](../../xmlexception/) con informazioni sul primo carattere non valido riscontrato.

```cpp
static String System::Xml::XmlConvert::VerifyXmlChars(const String &content)
```


| Parametro | Tipo | Descrizione |
| --- | --- | --- |
| content | const String\& | [String](../../../system/string/) che contiene i caratteri da verificare. |

### ReturnValue

La stringa fornita se tutti i caratteri e i caratteri di coppia surrogate nella stringa passata sono caratteri XML validi, altrimenti viene sollevata un'[XmlException](../../xmlexception/) con informazioni sul primo carattere non valido riscontrato.
