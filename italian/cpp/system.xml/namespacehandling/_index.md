## NamespaceHandling enum


Specifica se rimuovere le dichiarazioni di namespace duplicate nel [XmlWriter](../xmlwriter/).

```cpp
enum class NamespaceHandling
```

### Valori

| Nome | Valore | Descrizione |
| --- | --- | --- |
| Default | 0 | Specifica che le dichiarazioni di namespace duplicate non saranno rimosse. |
| OmitDuplicates | 1 | Specifica che le dichiarazioni di namespace duplicate saranno rimosse. Per rimuovere il namespace duplicato, il prefisso e il namespace devono corrispondere. |
