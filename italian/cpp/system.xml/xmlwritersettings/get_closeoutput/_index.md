## XmlWriterSettings::get_CloseOutput method


Restituisce un valore che indica se il [XmlWriter](../../xmlwriter/) deve anche chiudere lo stream sottostante o il TextWriter quando viene chiamato il metodo [XmlWriter::Close](../../xmlwriter/close/).

```cpp
bool System::Xml::XmlWriterSettings::get_CloseOutput()
```


### ReturnValue

**true** to also close the underlying stream or TextWriter; otherwise, **false**. The default is **false**.
