## XmlWriterSettings::get_NewLineChars method


Restituisce la stringa di caratteri da utilizzare per le interruzioni di riga.

```cpp
String System::Xml::XmlWriterSettings::get_NewLineChars()
```


### ReturnValue

La stringa di caratteri da utilizzare per le interruzioni di riga. Può essere impostata a qualsiasi valore di stringa. Tuttavia, per garantire XML valido, dovresti specificare solo caratteri di spazio bianco validi, come spazi, tabulazioni, ritorni a capo o avanzamenti di riga. Il valore predefinito è **\r\n** (ritorno a capo, nuova linea).
