## XmlWriterSettings::set_NamespaceHandling method


Imposta un valore che indica se il [XmlWriter](../../xmlwriter/) deve rimuovere le dichiarazioni di namespace duplicate durante la scrittura del contenuto XML. Il comportamento predefinito è che lo scrittore emetta tutte le dichiarazioni di namespace presenti nel risolutore di namespace dello scrittore.

```cpp
void System::Xml::XmlWriterSettings::set_NamespaceHandling(System::Xml::NamespaceHandling value)
```


| Parametro | Tipo | Descrizione |
| --- | --- | --- |
| value | System::Xml::NamespaceHandling | Il valore da impostare. |
