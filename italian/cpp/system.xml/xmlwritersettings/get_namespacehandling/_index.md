## XmlWriterSettings::get_NamespaceHandling method


Restituisce un valore che indica se il [XmlWriter](../../xmlwriter/) deve rimuovere le dichiarazioni di namespace duplicate durante la scrittura del contenuto XML. Il comportamento predefinito è che lo scrittore emetta tutte le dichiarazioni di namespace presenti nel risolutore di namespace dello scrittore.

```cpp
System::Xml::NamespaceHandling System::Xml::XmlWriterSettings::get_NamespaceHandling()
```


### ReturnValue

L'enumerazione [NamespaceHandling](../../namespacehandling/) utilizzata per specificare se rimuovere le dichiarazioni di namespace duplicate nel [XmlWriter](../../xmlwriter/).
