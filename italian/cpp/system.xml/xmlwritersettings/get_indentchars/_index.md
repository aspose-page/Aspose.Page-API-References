## XmlWriterSettings::get_IndentChars method


Restituisce la stringa di caratteri da utilizzare per l'indentazione. Questa impostazione è usata quando il valore [XmlWriterSettings::set_Indent](../set_indent/) è impostato su **true**.

```cpp
String System::Xml::XmlWriterSettings::get_IndentChars()
```


### ReturnValue

La stringa di caratteri da utilizzare per l'indentazione. Può essere impostata a qualsiasi valore di stringa. Tuttavia, per garantire XML valido, dovresti specificare solo caratteri di spazio bianco validi, come spazi, tabulazioni, ritorni a capo o interruzioni di riga. Il valore predefinito è due spazi.
