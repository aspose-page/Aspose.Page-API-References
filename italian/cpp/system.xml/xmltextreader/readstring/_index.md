## XmlTextReader::ReadString method


Legge il contenuto di un elemento o di un nodo di testo come stringa.

```cpp
String System::Xml::XmlTextReader::ReadString() override
```


### ReturnValue

Il contenuto dell'elemento o del nodo di testo. Può essere una stringa vuota se il lettore è posizionato su qualcosa di diverso da un elemento o nodo di testo, o se non c'è più contenuto testuale da restituire nel contesto corrente. **Note:** Il nodo di testo può essere sia un elemento sia un nodo di testo attributo.
