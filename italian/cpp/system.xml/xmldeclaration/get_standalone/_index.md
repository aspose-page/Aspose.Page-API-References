## XmlDeclaration::get_Standalone method


Restituisce il valore dell'attributo standalone.

```cpp
String System::Xml::XmlDeclaration::get_Standalone()
```


### ReturnValue

I valori validi sono **yes** se tutte le dichiarazioni di entità richieste dal documento XML sono contenute nel documento o **no** se è necessaria una definizione esterna del tipo di documento (DTD). Se l'attributo standalone non è presente nella dichiarazione XML, questo metodo restituisce [String::Empty](../../../system/string/empty/).
