## Formatting enum


Specifica le opzioni di formattazione per il [XmlTextWriter](../xmltextwriter/).

```cpp
enum class Formatting
```

### Valori

| Nome | Valore | Descrizione |
| --- | --- | --- |
| None | 0 | Non viene applicata alcuna formattazione speciale. Questo è il valore predefinito. |
| Indented | 1 | Fa sì che gli elementi figlio vengano indentati in base alle impostazioni [XmlTextWriter::set_Indentation](../xmltextwriter/set_indentation/) e [XmlTextWriter::set_IndentChar](../xmltextwriter/set_indentchar/). |
