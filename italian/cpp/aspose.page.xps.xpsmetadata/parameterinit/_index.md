## ParameterInit class


La classe che implementa un inizializzatore di parametri [PrintTicket](../printticket/) comune. La classe base per tutti gli inizializzatori di parametri definiti dallo schema. Definisce un valore per un'istanza di un elemento **ParameterDef**. Un elemento **[ParameterInit](./)** è il bersaglio del riferimento effettuato da un elemento [ParameterRef](../parameterref/). [https://docs.microsoft.com/en-us/windows/win32/printdocs/parameterinit](https://docs.microsoft.com/en-us/windows/win32/printdocs/parameterinit).
