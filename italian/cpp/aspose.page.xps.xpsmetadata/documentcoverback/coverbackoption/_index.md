## CoverBackOption class


Descrive le opzioni della funzionalità [DocumentCoverBack](../).

```cpp
class CoverBackOption : public Aspose::Page::XPS::XpsMetadata::Option
```

## Campi

| Campo | Descrizione |
| --- | --- |
| static [BlankCover](./blankcover/) | Specifica che deve essere stampata una copertina vuota. |
| static [NoCover](./nocover/) | Specifica che non verrà prodotta alcuna copertina. |
| static [PrintBack](./printback/) | Specifica che la copertina indicata da "CoverBackSource" deve essere stampata sul lato posteriore del foglio di copertina. Se un elemento [DocumentCoverBackSource](../../documentcoverbacksource/)[ParameterInit](../../parameterinit/) non è specificato, questa [Option](../../option/) deve essere ignorata. |
| static [PrintBoth](./printboth/) | Specifica che la copertina indicata da "CoverBackSource" può essere stampata su entrambi i lati del foglio di copertina. Se un elemento [DocumentCoverBackSource](../../documentcoverbacksource/)[ParameterInit](../../parameterinit/) non è specificato, questa [Option](../../option/) deve essere ignorata. |
| static [PrintFront](./printfront/) | Specifica che la copertina indicata da "CoverBackSource" deve essere stampata sul lato anteriore del foglio di copertina. Se un elemento [DocumentCoverBackSource](../../documentcoverbacksource/)[ParameterInit](../../parameterinit/) non è specificato, questa [Option](../../option/) deve essere ignorata. |
