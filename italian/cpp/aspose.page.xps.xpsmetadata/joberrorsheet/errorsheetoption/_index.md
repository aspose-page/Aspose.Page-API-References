## Campi

| Campo | Descrizione |
| --- | --- |
| static [Custom](./custom/) | Specifica che deve essere emessa una scheda di errore personalizzata. Se un elemento [JobErrorSheetSource](../../joberrorsheetsource/)[ParameterInit](../../parameterinit/) non è specificato, questa [Option](../../option/) deve essere ignorata. |
