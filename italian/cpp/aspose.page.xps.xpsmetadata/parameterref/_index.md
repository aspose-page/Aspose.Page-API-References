## ParameterRef class


La classe che implementa un riferimento di parametro comune per [PrintTicket](../printticket/). Un elemento **[ParameterRef](./)** definisce un riferimento a un elemento [ParameterInit](../parameterinit/). Un elemento [ScoredProperty](../scoredproperty/) che contiene un elemento [ParameterRef](./) non ha un elemento [Value](../value/) impostato esplicitamente. Invece, l'elemento [ScoredProperty](../scoredproperty/) riceve il suo valore dall'elemento [ParameterInit](../parameterinit/) a cui fa riferimento un elemento **[ParameterRef](./)**. [https://docs.microsoft.com/en-us/windows/win32/printdocs/parameterref](https://docs.microsoft.com/en-us/windows/win32/printdocs/parameterref).

```cpp
class ParameterRef : public Aspose::Page::XPS::XpsMetadata::PrintTicketElement,
                     public Aspose::Page::XPS::XpsMetadata::IPrintTicketItem
```
