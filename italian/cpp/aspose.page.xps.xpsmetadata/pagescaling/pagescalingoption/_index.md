## PageScalingOption class


Descrive le opzioni delle funzionalità di [PageScaling](../).

```cpp
class PageScalingOption : public Aspose::Page::XPS::XpsMetadata::Option,
                          public Aspose::Page::XPS::XpsMetadata::PageScaling::IPageScalingItem
```

## Campi

| Campo | Descrizione |
| --- | --- |
| static [Custom](./custom/) | Specifica che una scala personalizzata dovrebbe essere applicata alla Dimensione Media dell'Applicazione. |
| static [CustomSquare](./customsquare/) | Specifica che una scala quadrata personalizzata dovrebbe essere applicata alla Dimensione Media dell'Applicazione. |
| static [FitApplicationBleedSizeToPageImageableSize](./fitapplicationbleedsizetopageimageablesize/) | Specifica che la dimensione di sbordo dell'applicazione dovrebbe essere scalata a [PageImageableSize](../../pageimageablesize/) preservando il rapporto d'aspetto. |
| static [FitApplicationContentSizeToPageImageableSize](./fitapplicationcontentsizetopageimageablesize/) | Specifica che la dimensione del contenuto dell'applicazione dovrebbe essere scalata a [PageImageableSize](../../pageimageablesize/) preservando il rapporto d'aspetto. |
| static [FitApplicationMediaSizeToPageImageableSize](./fitapplicationmediasizetopageimageablesize/) | Specifica che la dimensione media dell'applicazione dovrebbe essere scalata a [PageImageableSize](../../pageimageablesize/) preservando il rapporto d'aspetto. |
| static [FitApplicationMediaSizeToPageMediaSize](./fitapplicationmediasizetopagemediasize/) | Specifica che la dimensione media dell'applicazione dovrebbe essere scalata a [PageMediaSize](../../pagemediasize/) preservando il rapporto d'aspetto. |
| static [None](./none/) | Specifica che non deve essere applicata alcuna scalatura. |
