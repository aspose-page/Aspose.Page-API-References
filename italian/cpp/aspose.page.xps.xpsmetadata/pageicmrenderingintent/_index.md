## PageICMRenderingIntent class


Descrive l'intento di rendering così come definito dalla Specifica ICC v2. Questo valore dovrebbe essere ignorato se un'immagine o un elemento grafico ha un profilo incorporato che specifica l'intento **Rendering**. [https://docs.microsoft.com/en-us/windows/win32/printdocs/pageicmrenderingintent](https://docs.microsoft.com/en-us/windows/win32/printdocs/pageicmrenderingintent).
