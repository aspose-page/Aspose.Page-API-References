## CoverBackOption class


Descrive le opzioni della funzionalità [JobPrimaryCoverBack](../).

```cpp
class CoverBackOption : public Aspose::Page::XPS::XpsMetadata::Option
```

## Campi

| Campo | Descrizione |
| --- | --- |
| static [BlankCover](./blankcover/) | Specifica che deve essere stampata una copertina vuota. |
| static [NoCover](./nocover/) | Specifica che non verrà prodotta alcuna copertina. |
| static [PrintBack](./printback/) | Specifica che la copertina indicata da "CoverBackSource" deve essere stampata sul retro del foglio di copertina. Se un elemento [JobPrimaryCoverBackSource](../../jobprimarycoverbacksource/)[ParameterInit](../../parameterinit/) non è specificato, questa [Option](../../option/) deve essere ignorata. |
| static [PrintBoth](./printboth/) | Specifica che la copertina indicata da "CoverBackSource" può essere stampata su entrambi i lati del foglio di copertina. Se un elemento [JobPrimaryCoverBackSource](../../jobprimarycoverbacksource/)[ParameterInit](../../parameterinit/) non è specificato, questa [Option](../../option/) dovrebbe essere ignorata. |
| static [PrintFront](./printfront/) | Specifica che la copertina indicata da "CoverBackSource" deve essere stampata sul lato frontale del foglio di copertina. Se un elemento [JobPrimaryCoverBackSource](../../jobprimarycoverbacksource/)[ParameterInit](../../parameterinit/) non è specificato, questa [Option](../../option/) dovrebbe essere ignorata. |
