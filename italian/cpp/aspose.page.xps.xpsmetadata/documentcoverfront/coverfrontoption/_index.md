## CoverFrontOption class


Descrive le opzioni della funzionalità [DocumentCoverFront](../).

```cpp
class CoverFrontOption : public Aspose::Page::XPS::XpsMetadata::Option
```

## Campi

| Campo | Descrizione |
| --- | --- |
| static [BlankCover](./blankcover/) | Specifica che deve essere stampata una copertina vuota. |
| static [NoCover](./nocover/) | Specifica che non verrà prodotta alcuna copertina. |
| static [PrintBack](./printback/) | Specifica che la copertina indicata da "CoverFrontSource" deve essere stampata sul lato posteriore del foglio di copertina. Se non è specificato un elemento [DocumentCoverFrontSource](../../documentcoverfrontsource/)[ParameterInit](../../parameterinit/), questa [Option](../../option/) deve essere ignorata. |
| static [PrintBoth](./printboth/) | Specifica che la copertina indicata da "CoverFrontSource" può essere stampata su entrambi i lati del foglio di copertina. Se non è specificato un elemento [DocumentCoverFrontSource](../../documentcoverfrontsource/)[ParameterInit](../../parameterinit/), questa [Option](../../option/) deve essere ignorata. |
| static [PrintFront](./printfront/) | Specifica che la copertina indicata da "CoverFrontSource" deve essere stampata sul lato anteriore del foglio di copertina. Se non è specificato un elemento [DocumentCoverFrontSource](../../documentcoverfrontsource/)[ParameterInit](../../parameterinit/), questa [Option](../../option/) deve essere ignorata. |
