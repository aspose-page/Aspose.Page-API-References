## PageOutputQualityOption class


Definisce le opzioni della funzionalità [PageOutputQuality](../).

```cpp
class PageOutputQualityOption : public Aspose::Page::XPS::XpsMetadata::Option
```

## Campi

| Campo | Descrizione |
| --- | --- |
| static [Automatic](./automatic/) | Specifica l'intento per l'output automatico (consente al client di scegliere automaticamente le impostazioni migliori). |
| static [Draft](./draft/) | Specifica l'intento per l'output bozza (bilanciato per testo e grafica di qualità bozza). |
| static [Fax](./fax/) | Specifica l'intento per l'output fax (bilanciato per la qualità standard del fax). |
| static [High](./high/) | Specifica l'intento per l'output ad alta qualità (bilanciato per testo e grafica di alta qualità). |
| static [Normal](./normal/) | Specifica l'intento per l'output normale (bilanciato per testo e grafica di buona qualità). |
| static [Photographic](./photographic/) | Specifica l'intento per l'output fotografico (le immagini dovrebbero avere la massima qualità). |
| static [Text](./text/) | Specifica l'intento per l'output solo testo (la grafica potrebbe essere resa in modo scadente o non essere resa affatto). |
