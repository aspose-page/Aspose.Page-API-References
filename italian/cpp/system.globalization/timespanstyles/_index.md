## TimeSpanStyles enum


Specifica le opzioni di formattazione che personalizzano l'analisi delle stringhe per i metodi che convertono una rappresentazione testuale di un intervallo di tempo in un oggetto Timespan.

```cpp
enum class TimeSpanStyles : int32_t
```

### Valori

| Nome | Valore | Descrizione |
| --- | --- | --- |
| None | 0 | L'intervallo di tempo rappresentato dalla stringa inptut viene interpretato come intervallo di tempo negativo solo se è presente un segno meno. |
| AssumeNegative | 1 | L'intervallo di tempo rappresentato dalla stringa inptut viene interpretato come intervallo di tempo negativo sempre. |
