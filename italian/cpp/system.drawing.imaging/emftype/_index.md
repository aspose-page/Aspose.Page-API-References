## EmfType enum


Specifica i tipi di record che vengono inseriti in un file EMF.

```cpp
enum class EmfType
```

### Valori

| Nome | Valore | Descrizione |
| --- | --- | --- |
| EmfOnly | n/a | [Windows](../../system.windows/) metafile avanzato. Contiene comandi GDI. I metafile di questo tipo sono indicati come file EMF. |
| EmfPlusOnly | n/a | [Windows](../../system.windows/) metafile avanzato plus. Contiene comandi GDI+. I metafile di questo tipo sono indicati come file EMF+. |
| EmfPlusDual | n/a | Dual [Windows](../../system.windows/) metafile avanzato. Contiene comandi GDI e GDI+ equivalenti. I metafile di questo tipo sono indicati come file EMF+. |
