### Valori

| Nome | Valore | Descrizione |
| --- | --- | --- |
| In | 1 | Il destinatario dell'intestazione SOAP è il servizio XML [Web](../../system.web/). |
| InOut | 3 | I destinatari dell'intestazione SOAP sono sia il servizio XML [Web](../../system.web/) sia il client del servizio XML [Web](../../system.web/). |
| Out | 2 | Il destinatario dell'intestazione SOAP è il client del servizio XML [Web](../../system.web/). |
| Fault | 4 | L'intestazione SOAP viene inviata al client del servizio XML [Web](../../system.web/) quando un'eccezione è generata dal metodo del servizio XML [Web](../../system.web/). |
