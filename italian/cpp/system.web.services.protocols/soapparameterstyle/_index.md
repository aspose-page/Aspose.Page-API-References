## SoapParameterStyle enum


Enumera i formati dei parametri in un messaggio SOAP.

```cpp
enum class SoapParameterStyle
```

### Valori

| Nome | Valore | Descrizione |
| --- | --- | --- |
| Default | 0 | Se '[SoapDocumentServiceAttribute](../soapdocumentserviceattribute/)' non è applicato alla classe, allora il valore predefinito è 'Wrapped'. |
| Bare | 1 | I parametri sono posizionati in elementi XML che seguono l'elemento 'Body'. |
| Wrapped | 2 | I parametri sono incapsulati all'interno di un singolo elemento XML che segue l'elemento 'Body'. |
