## SortedSetPtr class


Puntatore per mantenere riferimenti a [SortedSet](../sortedset/). Questo tipo è un puntatore per gestire la cancellazione di altri oggetti. Dovrebbe essere allocato sullo stack e passato alle funzioni sia per valore sia per riferimento costante.

template<typename T>class SortedSetPtr : public System::SmartPtr<SortedSet<T>>
