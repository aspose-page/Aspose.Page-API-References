## HashSetPtr class


Puntatore per mantenere riferimenti a [HashSet](../hashset/). Questo tipo è un puntatore per gestire la cancellazione di altri oggetti. Dovrebbe essere allocato sullo stack e passato alle funzioni sia per valore sia per riferimento const.

```cpp
template<typename T>class HashSetPtr : public System::SmartPtr<HashSet<T>>
```
