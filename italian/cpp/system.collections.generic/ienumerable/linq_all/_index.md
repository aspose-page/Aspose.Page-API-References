## IEnumerable::LINQ_All method


Determina se tutti gli elementi di una sequenza soddisfano una condizione.

```cpp
bool System::Collections::Generic::IEnumerable<T>::LINQ_All(std::function<bool(T)> predicate)
```


| Parametro | Tipo | Descrizione |
| --- | --- | --- |
| predicato | std::function\<bool(T)> | Una funzione per verificare ogni elemento rispetto a una condizione. |

### ReturnValue

true se ogni elemento della sequenza di origine supera il test nel predicato specificato, o se la sequenza è vuota; altrimenti, false.
