

| Parametro | Descrizione |

| TKey | Tipo di chiave. |

| Parametro | Tipo | Descrizione |
| --- | --- | --- |
| sinistra | const KeyValuePair\<TKey, TValue\>\& | Operando LHS. |
| destra | const KeyValuePair\<TKey, TValue\>\& | Operando RHS. |
