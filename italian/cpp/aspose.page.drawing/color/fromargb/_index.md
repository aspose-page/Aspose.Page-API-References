## Color::FromArgb(int32_t, Aspose::Page::Drawing::Color) method


Crea una struttura [T:Aspose::Page::Drawing::Color](../) dalla struttura [T:Aspose::Page::Drawing::Color](../) specificata, ma con il nuovo valore alpha specificato. Sebbene questo metodo consenta di passare un valore a 32 bit per il valore alpha, il valore è limitato a 8 bit.

```cpp
static Aspose::Page::Drawing::Color Aspose::Page::Drawing::Color::FromArgb(int32_t alpha, Aspose::Page::Drawing::Color baseColor)
```


| Parametro | Tipo | Descrizione |
| --- | --- | --- |
| alpha | int32_t | Il valore alfa per il nuovo [T:Aspose::Page::Drawing::Color](../). I valori validi sono da 0 a 255. |
| baseColor | Aspose::Page::Drawing::Color | Il [T:Aspose::Page::Drawing::Color](../) da cui creare il nuovo [T:Aspose::Page::Drawing::Color](../). |

### ReturnValue

Il [T:Aspose::Page::Drawing::Color](../) che questo metodo crea.
