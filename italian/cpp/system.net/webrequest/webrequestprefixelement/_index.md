## WebRequestPrefixElement class


Rappresenta un prefisso HTTP o HTTPS. Gli oggetti di questa classe devono essere allocati solo utilizzando la funzione [System::MakeObject()](../../../system/makeobject/). Non creare mai un'istanza di questo tipo sullo stack o usando l'operatore new, poiché ciò provocherà errori di runtime e/o fault di asserzione.
