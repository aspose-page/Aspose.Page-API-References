## Timer::set_Enabled method


Avvia o arresta il timer. L'avvio del timer non riavvia il conteggio del tempo se il timer è già in esecuzione.

```cpp
void System::Timers::Timer::set_Enabled(bool value)
```


| Parametro | Tipo | Descrizione |
| --- | --- | --- |
| value | bool | true per avviare il timer, false per fermarlo. |
