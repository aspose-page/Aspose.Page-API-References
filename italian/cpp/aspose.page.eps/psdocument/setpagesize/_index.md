## PsDocument::SetPageSize method


Imposta la dimensione della pagina. Per creare pagine con dimensioni diverse in un unico documento, usa il metodo [SetPageDevice](../setpagedevice/) subito dopo questo metodo.


| height | float | L'altezza della pagina nel file PostScript risultante. |
