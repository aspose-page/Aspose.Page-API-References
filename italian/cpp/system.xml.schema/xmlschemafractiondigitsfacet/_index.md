## XmlSchemaFractionDigitsFacet class


Specifica una restrizione sul numero di cifre che possono essere inserite per il valore frazionario di un elemento simpleType. Il valore di fractionDigits deve essere un intero positivo. Rappresenta la faccetta **fractionDigits** del World Wide [Web](../../system.web/) Consortium (W3C).

class XmlSchemaFractionDigitsFacet : public System::Xml::Schema::XmlSchemaNumericFacet
