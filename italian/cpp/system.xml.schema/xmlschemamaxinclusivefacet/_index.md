## XmlSchemaMaxInclusiveFacet class


Rappresenta l'elemento **maxInclusive** dallo XML [Schema](../) come specificato dal World Wide [Web](../../system.web/) Consortium (W3C). Questa classe può essere usata per specificare una restrizione sul valore massimo di un elemento **simpleType**. Il valore dell'elemento deve essere minore o uguale al valore dell'elemento **maxInclusive**.

```cpp
class XmlSchemaMaxInclusiveFacet : public System::Xml::Schema::XmlSchemaFacet
```
