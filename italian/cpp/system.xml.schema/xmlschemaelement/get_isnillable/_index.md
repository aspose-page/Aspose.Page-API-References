## XmlSchemaElement::get_IsNillable method


Restituisce informazioni che indicano se **xsi:nil** può comparire nei dati di istanza. Indica se un valore nil esplicito può essere assegnato all'elemento.

```cpp
bool System::Xml::Schema::XmlSchemaElement::get_IsNillable()
```


### ReturnValue

Se nillable è **true**, questo consente a un'istanza dell'elemento di avere l'attributo **nil** impostato su **true**. L'attributo **nil** è definito come parte dello spazio dei nomi XML [Schema](../../) per le istanze. Il valore predefinito è **false**. Opzionale.
