## XmlSchemaLengthFacet class


Rappresenta la faccetta **length** dello XML [Schema](../) come specificato dal World Wide [Web](../../system.web/) Consortium (W3C). Questa classe può essere utilizzata per specificare una restrizione sulla lunghezza di un elemento **simpleType** sul tipo di dato.

```cpp
class XmlSchemaLengthFacet : public System::Xml::Schema::XmlSchemaNumericFacet
```
