## XmlSchemaContentProcessing enum


Fornisce informazioni sulla modalità di validazione delle sostituzioni degli elementi **any** e **anyAttribute**.

```cpp
enum class XmlSchemaContentProcessing
```

### Valori

| Nome | Valore | Descrizione |
| --- | --- | --- |
| None | 0 | Gli elementi del documento non sono convalidati. |
| Ignora | 1 | Gli elementi del documento devono consistere in XML ben formato e non sono convalidati dallo schema. |
| Lax | 2 | Se lo schema associato viene trovato, gli elementi del documento saranno convalidati. Altrimenti non verranno generati errori. |
| Strict | 3 | Il processore di schema deve trovare uno schema associato allo spazio dei nomi indicato per convalidare gli elementi del documento. |
