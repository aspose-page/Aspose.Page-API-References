## XmlSchemaDerivationMethod enum


Fornisce diversi metodi per impedire la derivazione.

```cpp
enum class XmlSchemaDerivationMethod
```

### Valori

| Nome | Valore | Descrizione |
| --- | --- | --- |
| Vuoto | 0 | Sovrascrivi il metodo di derivazione predefinito per consentire qualsiasi derivazione. |
| Substitution | 1 | Si riferisce alle derivazioni per **Substitution**. |
| Extension | 2 | Si riferisce alle derivazioni per **Extension**. |
| Restriction | 4 | Si riferisce alle derivazioni per **Restriction**. |
| Elenco | 8 | Si riferisce alle derivazioni per **List**. |
| Unione | 16 | Si riferisce alle derivazioni per **Union**. |
| All | 255 | **#all**. Si riferisce a tutti i metodi di derivazione. |
| None | 256 | Accetta il metodo di derivazione predefinito. |
