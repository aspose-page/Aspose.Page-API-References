## XmlSchemaComplexType::get_BlockResolved method


Restituisce il valore dopo che il tipo è stato compilato nel set di informazioni post-validazione dello schema (infoset). Questo valore indica come il tipo è applicato quando **xsi:type** è usato nel documento di istanza.

```cpp
XmlSchemaDerivationMethod System::Xml::Schema::XmlSchemaComplexType::get_BlockResolved()
```


### ReturnValue

Il valore dell'infoset post-validazione dello schema. Il valore predefinito è **BlockDefault** sull'elemento **schema**.
