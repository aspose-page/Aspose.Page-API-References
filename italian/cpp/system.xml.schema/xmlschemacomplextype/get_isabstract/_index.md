## XmlSchemaComplexType::get_IsAbstract method


Restituisce le informazioni che determinano se l'elemento **complexType** può essere usato nel documento di istanza.

```cpp
bool System::Xml::Schema::XmlSchemaComplexType::get_IsAbstract()
```


### ReturnValue

Se **true**, un elemento non può utilizzare direttamente questo elemento **complexType** e deve utilizzare un tipo complesso derivato da questo elemento **complexType**. Il valore predefinito è **false**. Opzionale.
