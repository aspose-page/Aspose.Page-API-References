## TraceLevel enum


Specifica quali messaggi inviare in output per le classi [System.Diagnostics.Debug](../debug/), [System.Diagnostics.Trace](../trace/) e System.Diagnostics.TraceSwitch.

```cpp
enum class TraceLevel
```

### Valori

| Nome | Valore | Descrizione |
| --- | --- | --- |
| Off | 0 | Non produrre messaggi di tracciamento e debug. |
| Errore | 1 | Produci messaggi di gestione degli errori. |
| Warning | 2 | Produci messaggi di avviso e di gestione degli errori. |
| Info | 3 | Genera messaggi informativi, avvisi e messaggi di gestione degli errori. |
| Dettagliato | 4 | Genera tutti i messaggi di debug e di tracciamento. |
