## PerformanceCounter class


Classe fittizia per consentire l'uso di PerformanceCounter nel codice tradotto per la compilazione. Gli oggetti di questa classe devono essere allocati solo usando la funzione [System::MakeObject()](../../system/makeobject/). Non creare mai un'istanza di questo tipo sullo stack o usando l'operatore new, poiché ciò provocherà errori di runtime e/o fault di asserzione. Avvolgere sempre questa classe in un puntatore [System::SmartPtr](../../system/smartptr/) e usare tale puntatore per passarlo alle funzioni come argomento.

```cpp
class PerformanceCounter : public System::Object
```
