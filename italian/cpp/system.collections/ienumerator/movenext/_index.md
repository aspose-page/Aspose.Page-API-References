## IEnumerator::MoveNext method


Sposta l'enumeratore al prossimo elemento. Se non è stato referenziato alcun elemento prima, imposta il riferimento al primo elemento disponibile. Se è stato raggiunto la fine del contenitore, non fa nulla.

```cpp
virtual bool System::Collections::IEnumerator::MoveNext()=0
```


### ReturnValue

True se qualche elemento è disponibile dopo la chiamata, false altrimenti.
