## Classi

| Classe | Descrizione |
| --- | --- |
| [HttpContext](./httpcontext/) | Classe fittizia che rende possibile collegare il codice tradotto con i riferimenti [HttpContext](./httpcontext/), ma senza eseguirlo. Non contiene membri implementati correttamente. Gli oggetti di questa classe devono essere allocati solo utilizzando la funzione [System::MakeObject()](../system/makeobject/). Non creare mai un'istanza di questo tipo sullo stack o usando l'operatore new, poiché ciò provocherà errori di runtime e/o fault di asserzione. Avvolgi sempre questa classe in un puntatore [System::SmartPtr](../system/smartptr/) e usa questo puntatore per passarlo alle funzioni come argomento. |
| [HttpResponse](./httpresponse/) | Classe fittizia che rende possibile collegare il codice tradotto con i riferimenti [HttpResponse](./httpresponse/), ma senza eseguirlo. Non contiene membri implementati correttamente. Gli oggetti di questa classe devono essere allocati solo utilizzando la funzione [System::MakeObject()](../system/makeobject/). Non creare mai un'istanza di questo tipo sullo stack o usando l'operatore new, poiché ciò provocherà errori di runtime e/o fault di asserzione. Avvolgi sempre questa classe in un puntatore [System::SmartPtr](../system/smartptr/) e usa questo puntatore per passarlo alle funzioni come argomento. |
| [HttpUtility](./httputility/) | Classe di servizio che codifica e decodifica le parti di URL da e verso frammenti di escape esadecimali. |
