## Classi

| Classe | Descrizione |
| --- | --- |
| [HostInformation](./hostinformation/) | Rappresenta le informazioni su un host. Gli oggetti di questa classe devono essere allocati solo utilizzando la funzione [System::MakeObject()](../system/makeobject/). Non creare mai un'istanza di questo tipo sullo stack o usando l'operatore new, poiché provocherà errori di runtime e/o fallimenti di asserzione. Avvolgere sempre questa classe in un puntatore [System::SmartPtr](../system/smartptr/) e utilizzare tale puntatore per passarlo alle funzioni come argomento. |
| [IPGlobalProperties](./ipglobalproperties/) | Rappresenta un'informazione sulla connessione di rete del computer locale. Non creare mai un'istanza di questo tipo sullo stack o usando l'operatore new, poiché ciò provocherà errori di runtime e/o fault di asserzione. Avvolgi sempre questa classe in un puntatore [System::SmartPtr](../system/smartptr/) e utilizza questo puntatore per passarla alle funzioni come argomento. |
