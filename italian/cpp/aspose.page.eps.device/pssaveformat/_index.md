## PsSaveFormat enum


Questa enumerazione contiene le opzioni disponibili per il formato di salvataggio. Può essere PS o [EPS](../../aspose.page.eps/). [EPS](../../aspose.page.eps/) è utilizzato solo per documenti a una pagina, mentre il file PS può contenere un numero qualsiasi di pagine.

```cpp
enum class PsSaveFormat
```

### Valori

| Nome | Valore | Descrizione |
| --- | --- | --- |
| PS | 0 | Questa opzione indica che il documento risultante deve essere un file PostScript (PS) o Encapsulated PostScript ([EPS](../../aspose.page.eps/)). |
| EPS | 1 | Questa opzione indica che il documento risultante deve essere un file Encapsulated PostScript ([EPS](../../aspose.page.eps/)). È utilizzato solo per documenti a una pagina;. |
