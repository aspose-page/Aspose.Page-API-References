## SystemColors class


Classe che fornisce un insieme di oggetti [Color](../color/) precreati che rappresentano i colori degli elementi di visualizzazione di [Windows](../../system.windows/). Questo è un tipo statico senza servizi di istanza. Non dovresti mai creare istanze di esso in alcun modo.

```cpp
class SystemColors
```
