## CharacterRange class


Rappresenta un intervallo di posizioni di caratteri in una stringa. Questo tipo dovrebbe essere allocato sullo stack e passato alle funzioni per valore o per riferimento. Non utilizzare la classe [System::SmartPtr](../../system/smartptr/) per gestire gli oggetti di questo tipo.

```cpp
class CharacterRange
```
