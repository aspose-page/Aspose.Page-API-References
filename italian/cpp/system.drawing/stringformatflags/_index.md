## StringFormatFlags enum


Specifica le informazioni di visualizzazione e layout per le stringhe di testo.

```cpp
enum class StringFormatFlags
```

### Valori

| Nome | Valore | Descrizione |
| --- | --- | --- |
| DirectionRightToLeft | 1 | Direzione del testo da destra a sinistra. |
| DirectionVertical | 2 | [Text](../../system.drawing.text/) è allineato verticalmente. |
| FitBlackBox | 4 | Le parti dei caratteri possono sporgere dalla casella di layout. |
| DisplayFormatControl | 32 | I caratteri di controllo sono visualizzati. |
| NoFontFallback | 1024 | Il fallback di [Font](../font/) è disabilitato. |
| MeasureTrailingSpaces | 2048 | Inserisce lo spazio finale alla fine di ogni riga. |
| NoWrap | 4096 | Il wrapping di [Text](../../system.drawing.text/) è disabilitato. |
| LineLimit | 8192 | Le linee intere sono disposte nel rettangolo di formattazione. |
