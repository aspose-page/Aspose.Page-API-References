## XpsGradientStop class


Classe che incapsula le funzionalità dell'elemento GradientStop. Questo elemento è utilizzato sia dagli elementi LinearGradientBrush sia da RadialGradientBrush per definire la posizione e l'intervallo della progressione di colore per il rendering di una sfumatura.

```cpp
class XpsGradientStop : public Aspose::Page::XPS::XpsModel::XpsObject
```

| Metodo | Descrizione |

| [get_Color](./get_color/)() | Il colore del punto di gradiente. |
| [get_Offset](./get_offset/)() const | Restituisce/imposta l'offset del gradiente. L'offset indica un punto lungo la progressione del gradiente in cui è specificato un colore. I colori tra gli offset del gradiente nella progressione sono interpolati. |
