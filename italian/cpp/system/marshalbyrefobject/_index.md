## MarshalByRefObject class


Fornisce l'accesso agli oggetti attraverso i confini del dominio dell'applicazione in applicazioni abilitate al remoting. Gli oggetti di questa classe devono essere allocati solo utilizzando la funzione [System::MakeObject()](../makeobject/). Non creare mai un'istanza di questo tipo sullo stack o usando l'operatore new, poiché ciò provocherà errori di runtime e/o fallimenti di asserzione. Avvolgere sempre questa classe in un puntatore [System::SmartPtr](../smartptr/) e utilizzare questo puntatore per passarlo alle funzioni come argomento.

```cpp
class MarshalByRefObject : public virtual System::Object
```
