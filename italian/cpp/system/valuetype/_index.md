## ValueType class


Classe base per i tipi valore con ereditarietà [Object](../object/) troncata per motivi di prestazioni. Questo tipo dovrebbe essere allocato sullo stack e passato alle funzioni per valore o per riferimento. Non utilizzare mai la classe [System::SmartPtr](../smartptr/) per gestire oggetti di questo tipo.

```cpp
class ValueType
```
