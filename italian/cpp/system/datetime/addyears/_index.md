## DateTime::AddYears method


Restituisce una nuova istanza della classe [DateTime](../) che rappresenta il valore di data e ora uguale a quello rappresentato dall'oggetto corrente con il componente anno aumentato del numero specificato.

```cpp
DateTime System::DateTime::AddYears(int value) const
```


| Parametro | Tipo | Descrizione |
| --- | --- | --- |
| value | int | Il numero di anni da aggiungere; il valore può essere positivo o negativo |

### ReturnValue

Una nuova istanza della classe [DateTime](../) che rappresenta il valore di data e ora uguale a quello rappresentato dall'oggetto corrente con il componente anno aumentato di **value**
