## DateTime::CompareTo method


Confronta due valori di data e ora rappresentati dall'oggetto corrente e dall'istanza specificata della classe [DateTime](../) e restituisce il valore che indica le posizioni relative dei valori sulla linea temporale.

```cpp
int System::DateTime::CompareTo(DateTime value) const
```


| Parametro | Tipo | Descrizione |
| --- | --- | --- |
| value | DateTime | Un'istanza della classe [DateTime](../) con cui confrontare l'oggetto corrente |

### ReturnValue

Un valore inferiore a 0 se l'oggetto corrente rappresenta un valore precedente a quello rappresentato da **value**; 0 se i valori rappresentati da entrambi gli oggetti sono uguali; un valore superiore a 0 se il valore rappresentato dall'oggetto corrente è successivo a quello rappresentato da **value**
