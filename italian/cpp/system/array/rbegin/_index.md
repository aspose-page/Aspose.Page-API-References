## Array::rbegin() const method


Restituisce un iteratore inverso al primo elemento del contenitore invertito. Corrisponde all'ultimo elemento del contenitore non invertito. Se il contenitore è vuoto, l'iteratore restituito è uguale a [rend()](../rend/).

```cpp
const_reverse_iterator System::Array<T>::rbegin() const noexcept
```


### ReturnValue

Un iteratore che punta all'ultimo elemento del contenitore qualificato come const.
