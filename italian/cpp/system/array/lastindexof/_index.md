## Array::LastIndexOf(const ArrayPtr\<ArrayType\>\&, const ValueType\&, int, int) method


Determina l'indice dell'ultima occorrenza dell'elemento specificato in un intervallo di elementi dell'array specificato dall'indice di inizio e dal numero di elementi nell'intervallo.

```cpp
template<typename ArrayType,typename ValueType> static int System::Array<T>::LastIndexOf(const ArrayPtr<ArrayType> &arr, const ValueType &value, int startIndex, int count)
```


| Parametro | Descrizione |
| --- | --- |
| ArrayType | Tipo degli elementi nell'array di destinazione |
| ValueType | tipo dell'elemento da cercare nell'array |

| Parametro | Tipo | Descrizione |
| --- | --- | --- |
| arr | const ArrayPtr\<ArrayType\>\& | [Array](../) in cui cercare l'elemento specificato |
| value | const ValueType\& | Indice dell'elemento di cui determinare l'indice |
| startIndex | int | Indice al quale inizia la ricerca |
| count | int | Numero di elementi dell'intervallo in cui cercare |

### ReturnValue

Indice dell'ultima occorrenza dell'elemento specificato se l'elemento è trovato, altrimenti -1
