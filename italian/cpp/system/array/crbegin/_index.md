## Array::crbegin method


Restituisce un iteratore inverso al primo elemento del contenitore invertito. Corrisponde all'ultimo elemento del contenitore non invertito. Se il contenitore è vuoto, l'iteratore restituito è uguale a [crend()](../crend/).

```cpp
const_reverse_iterator System::Array<T>::crbegin() const noexcept
```


### ReturnValue

Un iteratore che punta all'ultimo elemento con qualificatore const del contenitore.
