## Array::cend method


Restituisce un iteratore all'elemento successivo all'ultimo elemento del contenitore. Questo elemento funge da segnaposto; tentare di accedervi provoca un comportamento indefinito.

```cpp
const_iterator System::Array<T>::cend() const noexcept
```


### ReturnValue

Un iteratore che punta all'elemento teorico const-qualificato posto dopo l'ultimo elemento del contenitore.
