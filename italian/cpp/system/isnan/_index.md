## System::IsNaN method


Determina se il valore specificato è un valore Not-A-Number.

```cpp
template<typename T> bool System::IsNaN(const T &value)
```


| Parametro | Descrizione |
| --- | --- |
| T | Il tipo del valore controllato dalla funzione |

| Parametro | Tipo | Descrizione |
| --- | --- | --- |
| value | const T\& | Il valore da controllare |

### ReturnValue

Vero se **value** è un valore NaN, altrimenti - false
