## String::IsNull method


Verifica se la stringa è considerata nulla. [String](../) è nulla solo se è costruita tramite il costruttore [String()](../string/), spostata, copiata o assegnata da una stringa nulla o se è stato chiamato il metodo [reset()](../reset/).

```cpp
bool System::String::IsNull() const
```


### ReturnValue

true se la stringa è considerata nulla, false altrimenti.
