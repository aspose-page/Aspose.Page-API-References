## DBNull class


Rappresenta un valore non esistente. Gli oggetti di questa classe devono essere allocati solo usando la funzione [System::MakeObject()](../makeobject/) . Non creare mai un'istanza di questo tipo sullo stack o usando l'operatore new, poiché ciò provocherà errori di runtime e/o fallimenti di asserzione. Avvolgere sempre questa classe in un puntatore [System::SmartPtr](../smartptr/) e usare tale puntatore per passarla alle funzioni come argomento.

```cpp
class DBNull : public System::Object
```
