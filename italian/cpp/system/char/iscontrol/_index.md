## Char::IsControl(char_t) method


Determina se il carattere specificato è classificato come un carattere di controllo Unicode.

```cpp
static bool System::Char::IsControl(char_t c)
```


| Parametro | Tipo | Descrizione |
| --- | --- | --- |
| c | char_t | Il carattere da testare |

### ReturnValue

True se il carattere specificato è un carattere di controllo Unicode, altrimenti - false
