## GC class


Rappresenta una Garbage Collection emulata che agisce più come un stub che in pratica non fa nulla. Questo è un tipo statico senza servizi di istanza. Non dovresti mai creare istanze di esso in alcun modo.

```cpp
class GC : public System::Object
```

| static [GetTotalMemory](./gettotalmemory/)(bool) | Restituisce il numero di byte di memoria privata attualmente allocati dal processo corrente. |
