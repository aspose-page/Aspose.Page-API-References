## WeakReference class


Rappresenta un riferimento debole, che fa riferimento a un oggetto consentendone comunque la cancellazione.

```cpp
template<typename...>class WeakReference
```


| Parametro | Descrizione |
| --- | --- |
| Argomenti | Argomenti template di un riferimento debole che dichiarano il tipo di un oggetto referenziato; il numero di argomenti può essere zero o uno. |
