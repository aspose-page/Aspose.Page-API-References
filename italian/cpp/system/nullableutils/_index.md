## NullableUtils class


Rappresenta la classe statica C# [System.Nullable](../nullable/) (senza argomenti di tipo). Impossibile utilizzare il nome originale a causa dell'impossibilità di sovraccaricare i template di classe in C++. Supporta un tipo valore che può essere assegnato a null. Questa classe non può essere ereditata.
