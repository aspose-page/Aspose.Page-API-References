## SmartPtr::get_shared_count method


Ottiene il numero di puntatori condivisi esistenti per l'oggetto di riferimento, incluso quello corrente. Verifica che il puntatore corrente sia in modalità condivisa.

```cpp
int System::SmartPtr<T>::get_shared_count() const
```


### ReturnValue

Numero di smart pointer condivisi esistenti per l'oggetto di riferimento, se presenti. Se il puntatore è nullo, restituisce 0.
