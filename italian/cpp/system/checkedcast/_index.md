## System::CheckedCast method


Determina se il valore specificato rientra nell'intervallo di valori del tipo **TTo** e, in tal caso, lo converte al tipo **TTo**.

```cpp
template<typename TTo,typename TFrom> TTo System::CheckedCast(TFrom value)
```


| Parametro | Descrizione |
| --- | --- |
| TTo | Il tipo a cui il valore specificato deve essere convertito |
| TFrom | Il tipo del valore specificato |

| Parametro | Tipo | Descrizione |
| --- | --- | --- |
| value | TFrom | Il valore da convertire |

### ReturnValue
