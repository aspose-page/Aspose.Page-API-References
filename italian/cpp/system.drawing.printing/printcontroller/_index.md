## PrintController class


Controlla il modo in cui un documento viene stampato, quando si stampa da un'applicazione [Windows](../../system.windows/) Forms. Gli oggetti di questa classe devono essere allocati solo usando la funzione [System::MakeObject()](../../system/makeobject/) function. Never create instance of this type on stack or using operator new, as it will result in runtime errors and/or assertion faults. Always wrap this class into [System::SmartPtr](../../system/smartptr/) pointer and use this pointer to pass it to functions as argument.

```cpp
class PrintController : public System::Object
```
