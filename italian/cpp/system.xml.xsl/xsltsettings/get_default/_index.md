## XsltSettings::get_Default method


Restituisce un oggetto [XsltSettings](../) con impostazioni predefinite. Il supporto per la funzione XSLT **document()** e per i blocchi di script incorporati è disabilitato.

```cpp
static SharedPtr<XsltSettings> System::Xml::Xsl::XsltSettings::get_Default()
```


### ReturnValue

Un oggetto [XsltSettings](../) con le opzioni [XsltSettings::set_EnableDocumentFunction](../set_enabledocumentfunction/) e [XsltSettings::set_EnableScript](../set_enablescript/) impostate su **false**.
