## Classi

| Classe | Descrizione |
| --- | --- |
| [DataGrid](./datagrid/) | Classe fittizia che rende possibile collegare il codice tradotto con riferimenti a [DataGrid](./datagrid/), ma senza eseguirlo. Non contiene membri implementati correttamente. Gli oggetti di questa classe devono essere allocati solo utilizzando la funzione [System::MakeObject()](../system/makeobject/). Non creare mai un'istanza di questo tipo sullo stack o usando l'operatore new, poiché provocherà errori di runtime e/o fallimenti di asserzione. Avvolgere sempre questa classe in un puntatore [System::SmartPtr](../system/smartptr/) e utilizzare tale puntatore per passarlo alle funzioni come argomento. |
