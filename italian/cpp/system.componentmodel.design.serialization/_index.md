## Classi

| Classe | Descrizione |
| --- | --- |
| [InstanceDescriptor](./instancedescriptor/) | Classe fittizia per il codice che utilizza [InstanceDescriptor](./instancedescriptor/) per compilare. Gli oggetti di questa classe devono essere allocati solo utilizzando la funzione [System::MakeObject()](../system/makeobject/). Non creare mai un'istanza di questo tipo sullo stack o usando l'operatore new, poiché ciò provocherà errori di runtime e/o fault di asserzione. Avvolgi sempre questa classe in un puntatore [System::SmartPtr](../system/smartptr/) e utilizza questo puntatore per passarlo alle funzioni come argomento. |
