## Classi

| Classe | Descrizione |
| --- | --- |
| [SecurityPermission](./securitypermission/) | Classe che descrive il permesso di sicurezza. Gli oggetti di questa classe devono essere allocati solo tramite la funzione [System::MakeObject()](../system/makeobject/). Non creare mai un'istanza di questo tipo sullo stack o usando l'operatore new, poiché ciò provocherà errori di runtime e/o fallimenti di asserzione. Avvolgi sempre questa classe in un puntatore [System::SmartPtr](../system/smartptr/) e usa questo puntatore per passarlo alle funzioni come argomento. |
