## RIPEMD160 class


Calcola l'hash [RIPEMD160](./) per i dati di input. In C++ [RIPEMD160](./) non è una classe astratta. Gli oggetti di questa classe devono essere allocati solo usando la funzione [System::MakeObject()](../../system/makeobject/). Non creare mai un'istanza di questo tipo sullo stack o usando l'operatore new, poiché ciò provocherà errori di runtime e/o fault di asserzione. Avvolgi sempre questa classe in un puntatore [System::SmartPtr](../../system/smartptr/) e usa questo puntatore per passarla alle funzioni come argomento.

```cpp
class RIPEMD160 : public System::Security::Cryptography::HashAlgorithm
```
