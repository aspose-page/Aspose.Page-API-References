## SHA1 class


Calcola l'hash [SHA1](./) per i dati di input. In C++ [SHA1](./) non è una classe astratta. Gli oggetti di questa classe dovrebbero essere allocati solo usando la funzione [System::MakeObject()](../../system/makeobject/). Non creare mai un'istanza di questo tipo sullo stack o usando l'operatore new, poiché ciò provocherà errori di runtime e/o fallimenti di asserzione. Avvolgi sempre questa classe in un puntatore [System::SmartPtr](../../system/smartptr/) e usa questo puntatore per passarla alle funzioni come argomento.

```cpp
class SHA1 : public System::Security::Cryptography::HashAlgorithm
```
