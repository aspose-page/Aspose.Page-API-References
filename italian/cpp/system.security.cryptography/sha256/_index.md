## SHA256 class


Calcola l'hash [SHA256](./) per i dati di input. In C++ [SHA256](./) non è una classe astratta. Gli oggetti di questa classe devono essere allocati solo tramite la funzione [System::MakeObject()](../../system/makeobject/). Non creare mai un'istanza di questo tipo sullo stack o usando l'operatore new, poiché ciò provocherà errori di runtime e/o fallimenti di asserzione. Avvolgi sempre questa classe in un puntatore [System::SmartPtr](../../system/smartptr/) e utilizza questo puntatore per passarlo alle funzioni come argomento.

```cpp
class SHA256 : public System::Security::Cryptography::HashAlgorithm
```
