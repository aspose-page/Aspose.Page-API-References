## HMAC class


Tutte le implementazioni del Codice di [Autenticazione](../../system.security.authentication/) dei Messaggi basato su Hash ([HMAC](./)) devono ereditare questa classe astratta. Gli oggetti di questa classe devono essere allocati solo usando la funzione [System::MakeObject()](../../system/makeobject/). Non creare mai un'istanza di questo tipo sullo stack o usando l'operatore new, poiché ciò provocherà errori di runtime e/o fault di asserzione. Avvolgere sempre questa classe in un puntatore [System::SmartPtr](../../system/smartptr/) e utilizzare questo puntatore per passarlo alle funzioni come argomento.

```cpp
class HMAC : public System::Security::Cryptography::HashAlgorithm
```
