## SHA384 class


Calcola l'hash [SHA384](./) per i dati di input. In C++ [SHA384](./) non è una classe astratta. Gli oggetti di questa classe devono essere allocati solo usando la funzione [System::MakeObject()](../../system/makeobject/). Non creare mai un'istanza di questo tipo sullo stack o usando l'operatore new, poiché provocherà errori di runtime e/o fault di asserzione. Avvolgi sempre questa classe in un puntatore [System::SmartPtr](../../system/smartptr/) e usa questo puntatore per passarla alle funzioni come argomento.

```cpp
class SHA384 : public System::Security::Cryptography::HashAlgorithm
```
