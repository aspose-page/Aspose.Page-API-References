## X509KeyStorageFlags enum


Definisce come memorizzare la chiave.

```cpp
enum class X509KeyStorageFlags : int32_t
```

### Valori

| Nome | Valore | Descrizione |
| --- | --- | --- |
| DefaultKeySet | 0 | Usa il set di chiavi predefinito. |
| UserKeySet | 1 | Usa l'archivio associato all'utente invece di quello locale alla macchina. |
| MachineKeySet | 2 | Usa l'archivio della macchina locale invece di quello dell'utente. |
| Esportabile | 4 | Segna le chiavi importate come esportabili. |
| UserProtected | 8 | Notifica all'utente che la chiave è in uso. |
| PersistKeySet | 16 | La chiave viene mantenuta quando si importa il certificato. |
