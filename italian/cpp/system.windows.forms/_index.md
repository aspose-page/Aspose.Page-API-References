## Classi

| Classe | Descrizione |
| --- | --- |
| [Control](./control/) | Classe fittizia per rendere compilabile il codice tradotto che utilizza la classe [Control](./control/) . I metodi non sono implementati. Gli oggetti di questa classe dovrebbero essere allocati solo utilizzando la funzione [System::MakeObject()](../system/makeobject/) . Non creare mai un'istanza di questo tipo sullo stack o usando l'operatore new, poiché ciò provocherà errori di runtime e/o fault di asserzione. Avvolgi sempre questa classe in un puntatore [System::SmartPtr](../system/smartptr/) e usa questo puntatore per passarlo alle funzioni come argomento. |
| [Form](./form/) | Classe fittizia per rendere compilabile il codice tradotto che utilizza la classe [Form](./form/) . I metodi non sono implementati. Gli oggetti di questa classe dovrebbero essere allocati solo utilizzando la funzione [System::MakeObject()](../system/makeobject/) . Non creare mai un'istanza di questo tipo sullo stack o usando l'operatore new, poiché ciò provocherà errori di runtime e/o fault di asserzione. Avvolgi sempre questa classe in un puntatore [System::SmartPtr](../system/smartptr/) e usa questo puntatore per passarlo alle funzioni come argomento. |
| [IButtonControl](./ibuttoncontrol/) | Classe fittizia per rendere compilabile il codice tradotto che utilizza l'interfaccia [IButtonControl](./ibuttoncontrol/) . I metodi non sono implementati. Gli oggetti di questa classe dovrebbero essere allocati solo utilizzando la funzione [System::MakeObject()](../system/makeobject/) . Non creare mai un'istanza di questo tipo sullo stack o usando l'operatore new, poiché ciò provocherà errori di runtime e/o fault di asserzione. Avvolgi sempre questa classe in un puntatore [System::SmartPtr](../system/smartptr/) e usa questo puntatore per passarlo alle funzioni come argomento. |
| [Screen](./screen/) | Fornisce informazioni sui display. Solo per [Windows](../system.windows/) . Questo è un tipo singleton con la gestione della memoria effettuata tramite funzione/i di accesso. Non dovresti mai creare istanze direttamente. |
