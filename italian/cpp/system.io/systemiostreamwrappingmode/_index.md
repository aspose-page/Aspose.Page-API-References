## SystemIOStreamWrappingMode enum


Specifica la modalità delle operazioni di I/O che i wrapper eseguiranno su stream simili a [System::IO::Stream](../stream/).

```cpp
enum class SystemIOStreamWrappingMode
```

### Valori

| Nome | Valore | Descrizione |
| --- | --- | --- |
| Binary | 0 | Una modalità che consente alle operazioni di input di codificare i byte dello stream in dati di tipo char_type e di decodificare i dati di tipo char_type in byte dello stream per le operazioni di output. |
| Conversion | 1 | Una modalità che consente alle operazioni di input di convertire i byte dello stream dal tipo uint8_t al tipo char_type e viceversa per le operazioni di output. |
