## FileShare enum


Specifica che tipo di accesso possono avere altri oggetti [FileStream](../filestream/) a un file aperto.

```cpp
enum class FileShare
```

### Valori

| Nome | Valore | Descrizione |
| --- | --- | --- |
| None | 0 | Nessun accesso. |
| Read | 1 | Accesso in sola lettura. |
| Write | 2 | Accesso in sola scrittura. |
| ReadWrite | 3 | Accesso in lettura e scrittura. |
| Elimina | 4 | Il file può essere eliminato. |
| Ereditabile | 16 | Rende l'handle del file ereditabile dai processi figlio. |
