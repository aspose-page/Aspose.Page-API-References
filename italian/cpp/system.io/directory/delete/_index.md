## Directory::Delete method


Rimuove il file o la directory specificati. Non genera eccezioni.

```cpp
static void System::IO::Directory::Delete(const String &path, bool recursive=false)
```


| Parametro | Tipo | Descrizione |
| --- | --- | --- |
| percorso | const String\& | Il percorso della directory o del file da rimuovere |
| ricorsivo | bool | Se **path** specifica una directory non vuota, allora **recursive** indica se tutto il contenuto della directory deve essere rimosso ricorsivamente; se la directory specificata da **path** non è vuota e **recursive** è 'false', l'operazione fallisce |
