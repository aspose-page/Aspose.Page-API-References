## IsTemplateBaseOf typedef


Rappresenta il corrispondente di std::is_base_of<Base, Derived> che determina l'ereditarietà della classe modello Base non istanziata dalla classe modello Derived istanziata. Fallirà in caso di ereditarietà multipla o ereditarietà non pubblica da Base.

```cpp
using System::Details::IsTemplateBaseOf =  typename IsTemplateBaseOfImpl<Base, Derived>::type
```
