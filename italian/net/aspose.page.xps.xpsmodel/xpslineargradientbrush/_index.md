## XpsLinearGradientBrush class

Classe che incapsula le caratteristiche dell'elemento di proprietà LinearGradientBrush. Questo elemento viene utilizzato per specificare un pennello sfumato lineare lungo un vettore.

```csharp
public sealed class XpsLinearGradientBrush : XpsGradientBrush
```

| Nome | Descrizione |
| --- | --- |
| [ColorInterpolationMode](../../aspose.page.xps.xpsmodel/xpsgradientbrush/colorinterpolationmode/) { get; set; } | Restituisce/imposta il valore specificando la funzione gamma per l'interpolazione del colore. La regolazione gamma non deve essere applicata al componente alfa, se specificato. |
| [EndPoint](../../aspose.page.xps.xpsmodel/xpslineargradientbrush/endpoint/) { get; set; } | Restituisce/imposta il punto finale del gradiente lineare. |
