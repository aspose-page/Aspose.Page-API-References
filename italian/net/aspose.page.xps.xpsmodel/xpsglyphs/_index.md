## XpsGlyphs class

Classe che incapsula le caratteristiche dell'elemento Glyphs. Questo elemento rappresenta una sequenza di testo con formattazione uniforme da un singolo carattere. Fornisce le informazioni necessarie per un rendering accurato e supporta la ricerca e le funzionalità di selezione nella visualizzazione dei consumatori.

```csharp
public sealed class XpsGlyphs : XpsContentElement
```
