## XpsTransformableBrush.Transform property

Restituisce/imposta la trasformazione della matrice applicata allo spazio delle coordinate del pennello. La proprietà Transform è concatenata con l'attuale trasformazione di rendering effettiva per produrre una trasformazione di rendering effettiva locale al pennello. Il viewport per brush viene trasformato utilizzando la trasformazione di rendering effettiva locale.

```csharp
public XpsMatrix Transform { get; set; }
```
