## ImageDevice.IsDirectRGB property

Indica se il dispositivo utilizza la modalità RGB diretta, ovvero RGB.

```csharp
public override bool IsDirectRGB { get; }
```

### Valore di ritorno

Vero se in modalità RGB diretta e falso in caso contrario, ovvero BGR.
