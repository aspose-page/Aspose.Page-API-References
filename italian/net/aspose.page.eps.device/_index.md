Il **Aspose.Page.EPS.Device** namespace fornisce classi di possibili dispositivi e opzioni di salvataggio che possono essere utilizzate durante la conversione di PS/EPS in altri formati.

## Classi

| Classe | Descrizione |
| --- | --- |
| [ImageDevice](./imagedevice/) | Questa classe incapsula il rendering del documento in immagine. |
| [ImageSaveOptions](./imagesaveoptions/) | Questa classe contiene le opzioni necessarie per gestire il processo di salvataggio delle immagini. |
| [PdfDevice](./pdfdevice/) | Questa classe incapsula il rendering del documento in PDF. |
| [PdfSaveOptions](./pdfsaveoptions/) | Questa classe contiene flussi di input e output e altre opzioni necessarie per gestire il processo di conversione. |
| [PsSaveOptions](./pssaveoptions/) | Questa classe contiene le opzioni necessarie per gestire il processo di conversione del documento in un file PostScript (PS) o Encapsulated PostScript (EPS). |
