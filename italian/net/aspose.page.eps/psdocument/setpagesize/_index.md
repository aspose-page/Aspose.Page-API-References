## PsDocument.SetPageSize method

Imposta le dimensioni della pagina. Per creare pagine con dimensioni diverse in un unico documento utilizzare[`SetPageDevice`](../setpagedevice/) metodo subito dopo questo metodo.

```csharp
public void SetPageSize(float width, float height)
```

| Parametro | Tipo | Descrizione |
| --- | --- | --- |
| width | Single | La larghezza della pagina nel file PostScript risultante. |
| height | Single | L'altezza della pagina nel file PostScript risultante. |
