## GradientBrush class

Questa classe viene utilizzata per incapsulare LinearGradientBrush e PathGradientBrush con la possibilità di impostare la modalità wrap su clamp. I pennelli gradiente nativi generano sempre un'eccezione quando qualcuno tenta di impostare la proprietà WrapMode su WrapMode.Clamp.

```csharp
public class GradientBrush : Brush
```

## Costruttori

| Nome | Descrizione |
| --- | --- |
| [GradientBrush](gradientbrush/)(Brush) | Crea una nuova istanza di GradientBrush.  LinearGradientBrush o PathGradientBrush. Se diverso da ArgumentException verrà lanciato. |
