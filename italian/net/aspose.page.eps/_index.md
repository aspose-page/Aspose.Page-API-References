Il **Aspose.Page.EPS** è uno spazio dei nomi radice per tutte le classi che gestiscono file PS/EPS.

## Classi

| Classe | Descrizione |
| --- | --- |
| [FontConstants](./fontconstants/) | Questa classe definisce un insieme di costanti per il salvataggio dei font. |
| [GradientBrush](./gradientbrush/) | Questa classe viene utilizzata per incapsulare LinearGradientBrush e PathGradientBrush con la possibilità di impostare la modalità wrap su clamp. I pennelli gradiente nativi generano sempre un'eccezione quando qualcuno tenta di impostare la proprietà WrapMode su WrapMode.Clamp. |
| [LoadOptions](./loadoptions/) | Classe base per le opzioni di caricamento dei documenti. |
| [PageConstants](./pageconstants/) | Questa classe definisce un insieme di costanti che descrivono una pagina. Gli oggetti utili sono forniti per vari margini, orientamenti, ridimensionamento e dimensioni di pagina standard. |
| [PsConverterException](./psconverterexception/) | Questa classe contiene informazioni su un errore generato durante la conversione del file PS in documento PDF. |
| [PsDocument](./psdocument/) | Questa classe incapsula i documenti PS/EPS. |
| [PsDocumentException](./psdocumentexception/) | Questa classe contiene informazioni su un errore generato durante la creazione, la modifica e il salvataggio del file PostScript. |
| [PsLoadOptions](./psloadoptions/) | Opzioni di caricamento documento PS. |
