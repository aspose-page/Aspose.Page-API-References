## XpsDocument.CreatePolyQuadraticBezierSegment method

```csharp
public XpsPolyQuadraticBezierSegment CreatePolyQuadraticBezierSegment(PointF[] points, 
    bool isStroked = true)
```

| Parametro | Tipo | Descrizione |
| --- | --- | --- |
| points | PointF[] | Punti di controllo per più segmenti Bézier quadratici. |
| isStroked | Boolean | Specifica se viene disegnato il tratto per questo segmento del tracciato. |
