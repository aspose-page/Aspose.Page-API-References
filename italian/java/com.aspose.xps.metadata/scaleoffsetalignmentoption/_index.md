Descrive le opzioni delle funzionalità  ScaleOffsetAlignment . Specifica l'allineamento del contenuto scalato.
## Campi

| Campo | Descrizione |
| --- | --- |
| [BottomCenter](#BottomCenter) | Specifica che la scalatura deve essere allineata al centro del bordo inferiore del supporto. |
| [BottomLeft](#BottomLeft) | Specifica che la scala deve essere allineata nell'angolo inferiore sinistro del supporto. |
| [BottomRight](#BottomRight) | Specifica che la scala deve essere allineata nell'angolo inferiore destro del supporto. |
| [Center](#Center) | Specifica che la scala deve essere centrata sul supporto. |
| [LeftCenter](#LeftCenter) | Specifica che la scala deve essere allineata al centro del bordo sinistro del supporto. |
| [RightCenter](#RightCenter) | Specifica che la scala deve essere allineata al centro del bordo destro del supporto. |
| [TopCenter](#TopCenter) | Specifica che la scala deve essere allineata al centro del bordo superiore del supporto. |
| [TopLeft](#TopLeft) | Specifica che la scala deve essere allineata nell'angolo superiore sinistro del supporto. |
| [TopRight](#TopRight) | Specifica che la scala deve essere allineata nell'angolo superiore destro del supporto. |
