Indica un'intenzione di alto livello per il driver per la popolazione delle impostazioni di stampa fotografica. Queste impostazioni riguardano la qualità di output prevista che un utente può specificare quando stampa foto. https://docs.microsoft.com/en-us/windows/win32/printdocs/pagephotoprintingintent
