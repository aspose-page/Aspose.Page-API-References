Classe che implementa l'inizializzatore di parametri del ticket di stampa. La classe che implementa un inizializzatore di parametri PrintTicket comune. La classe base per tutti gli inizializzatori di parametri definiti dallo schema. Definisce un valore per un'istanza di un elemento  ParameterDef . Un elemento  ParameterInit  è il bersaglio del riferimento effettuato da un elemento  ParameterRef . https://docs.microsoft.com/en-us/windows/win32/printdocs/parameterinit
