Descrive le opzioni della funzionalità  JobOutputOptimization  .
## Campi

| Campo | Descrizione |
| --- | --- |
| [ArchiveFormat](#ArchiveFormat) | Specifica che l'elaborazione del lavoro sia ottimizzata per l'output di archivio. |
| [None](#None) | Specifica che l'elaborazione del lavoro non debba essere ottimizzata per uno scenario particolare. |
| [OptimizeForPortability](#OptimizeForPortability) | Specifica che l'elaborazione del lavoro sia ottimizzata per la portabilità (cross-device) dell'output. |
| [OptimizeForQuality](#OptimizeForQuality) | Specifica che l'elaborazione del lavoro sia ottimizzata per la qualità dell'output. |
| [OptimizeForSpeed](#OptimizeForSpeed) | Specifica che l'elaborazione del lavoro sia ottimizzata per la velocità dell'output. |
