La classe che incapsula una funzionalità comune di Print Schema. La classe base per tutte le funzionalità definite dallo schema. Un elemento  Feature  contiene un elenco completo degli elementi  Option  e  Property  che descrivono pienamente un attributo del dispositivo, un'impostazione di formattazione del lavoro o un'altra caratteristica pertinente. https://docs.microsoft.com/en-us/windows/win32/printdocs/feature
