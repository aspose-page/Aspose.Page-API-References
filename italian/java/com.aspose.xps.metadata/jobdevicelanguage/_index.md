Descrive i linguaggi del dispositivo supportati per l'invio di dati dal driver al dispositivo fisico. Questo è spesso chiamato "Linguaggio di Descrizione di Pagina". Questa parola chiave definisce quale linguaggio di descrizione di pagina è supportato dal driver e dal dispositivo fisico. https://docs.microsoft.com/en-us/windows/win32/printdocs/jobdevicelanguage
