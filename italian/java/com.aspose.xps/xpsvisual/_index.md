Classe che incapsula le funzionalità dell'elemento Visual. Questo elemento di proprietà contiene il markup che definisce il contenuto di una singola tessera di pennello visuale. La tessera può essere usata per riempire la regione geometrica a cui viene applicato il pennello visuale.
