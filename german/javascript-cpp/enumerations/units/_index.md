## Units enumeration

Gibt den Typ der Größe an.

```js
enum Units
```

### Werte

| Name | Wert | Beschreibung |
| ---        | ---   | --- |
| Points | `0` | Punkte (1/72 Zoll). |
| Inches | `1` | Zoll. |
| Millimeters | `2` | Millimeter. |
| Centimeters | `3` | Zentimeter. |
| Percents | `4` | Prozentsatz der vorhandenen Größe. |
