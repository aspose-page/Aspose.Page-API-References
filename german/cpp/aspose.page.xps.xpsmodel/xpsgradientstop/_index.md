## XpsGradientStop class


Klasse, die GradientStop-Elementeigenschaften kapselt. Dieses Element wird von sowohl LinearGradientBrush- als auch RadialGradientBrush-Elementen verwendet, um den Ort und den Bereich der Farbverläufe für die Darstellung eines Gradienten zu definieren.

```cpp
class XpsGradientStop : public Aspose::Page::XPS::XpsModel::XpsObject
```
