## Klassen

| [HostInformation](./hostinformation/) | Stellt Informationen über einen Host dar. Objekte dieser Klasse sollten ausschließlich mit der Funktion [System::MakeObject()](../system/makeobject/) alloziert werden. Erzeugen Sie niemals eine Instanz dieses Typs auf dem Stack oder mit dem Operator new, da dies zu Laufzeitfehlern und/oder Assertion-Fehlern führt.
