## IsTemplateBaseOf typedef


Stellt das Gegenstück zu std::is_base_of<Base, Derived> dar, das die Vererbung einer nicht instanziierten Base‑Template‑Klasse von einer instanziierten Derived‑Template‑Klasse bestimmt. Scheitert bei Mehrfachvererbung oder nicht‑öffentlicher Vererbung von Base.

```cpp
using System::Details::IsTemplateBaseOf =  typename IsTemplateBaseOfImpl<Base, Derived>::type
```
