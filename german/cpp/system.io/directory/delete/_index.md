## Directory::Delete method


Entfernt die angegebene Datei oder das Verzeichnis. Wirft keine Ausnahme.

```cpp
static void System::IO::Directory::Delete(const String &path, bool recursive=false)
```


| Parameter | Typ | Beschreibung |
| --- | --- | --- |
| Pfad | const String\& | Der Pfad zu dem zu entfernenden Verzeichnis oder der Datei |
| rekursiv | bool | Wenn **path** ein nicht leeres Verzeichnis angibt, dann legt **recursive** fest, ob der gesamte Inhalt des Verzeichnisses rekursiv entfernt werden soll; ist das durch **path** angegebene Verzeichnis nicht leer und **recursive** ist 'false', schlägt die Operation fehl |
