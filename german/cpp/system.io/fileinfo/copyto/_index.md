## FileInfo::CopyTo(const String\&) method


Kopiert die vom aktuellen Objekt repräsentierte Datei an den angegebenen Ort. Wenn die Zieldatei bereits existiert, schlägt das Kopieren fehl.


## FileInfo::CopyTo(const String\&, bool) method


Kopiert die vom aktuellen Objekt repräsentierte Datei an den angegebenen Ort. Ein Parameter gibt an, ob eine vorhandene Zieldatei überschrieben werden soll.

```cpp
FileInfoPtr System::IO::FileInfo::CopyTo(const String &destFileName, bool overwrite)
```


| Parameter | Typ | Beschreibung |
| --- | --- | --- |
| destFileName | const String\& | Der Zieldateiname |
| überschreiben | bool | True, wenn die vorhandene Zieldatei überschrieben werden soll, false, wenn das Kopieren fehlschlagen soll, falls die Zieldatei bereits existiert |
