## XmlReader::ReadOuterXml method


Wird in einer abgeleiteten Klasse überschrieben, liest es den Inhalt, einschließlich Markup, der diesen Knoten und alle seine Kinder darstellt.

```cpp
virtual String System::Xml::XmlReader::ReadOuterXml()
```


### ReturnValue

Wenn der Reader auf einem Element- oder Attributknoten positioniert ist, gibt diese Methode den gesamten XML-Inhalt, einschließlich Markup, des aktuellen Knotens und aller seiner Kinder zurück; andernfalls gibt sie eine leere Zeichenkette zurück.
