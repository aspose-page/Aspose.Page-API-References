## XmlReader::get_NamespaceURI method


Wird in einer abgeleiteten Klasse überschrieben, gibt die Namespace-URI (wie in der W3C-Namespace-Spezifikation definiert) des Knotens zurück, auf dem der Reader positioniert ist.

```cpp
virtual String System::Xml::XmlReader::get_NamespaceURI()=0
```


### ReturnValue

Die Namespace-URI des aktuellen Knotens; andernfalls ein leerer String.
