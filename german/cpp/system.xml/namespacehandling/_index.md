## NamespaceHandling enum


Gibt an, ob doppelte Namensraumdeklarationen im [XmlWriter](../xmlwriter/) entfernt werden sollen.

```cpp
enum class NamespaceHandling
```

### Werte

| Name | Wert | Beschreibung |
| --- | --- | --- |
| Default | 0 | Gibt an, dass doppelte Namensraumdeklarationen nicht entfernt werden. |
| OmitDuplicates | 1 | Gibt an, dass doppelte Namensraumdeklarationen entfernt werden. Damit ein doppelter Namensraum entfernt wird, müssen Präfix und Namensraum übereinstimmen. |
