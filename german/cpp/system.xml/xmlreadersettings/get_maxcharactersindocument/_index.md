## XmlReaderSettings::get_MaxCharactersInDocument method


Gibt einen Wert zurück, der die maximal zulässige Anzahl von Zeichen in einem XML-Dokument angibt. Ein Nullwert (0) bedeutet, dass es keine Begrenzung der Größe des XML-Dokuments gibt. Ein von Null verschiedener Wert gibt die maximale Größe in Zeichen an.

```cpp
int64_t System::Xml::XmlReaderSettings::get_MaxCharactersInDocument()
```


### ReturnValue

Die maximal zulässige Anzahl von Zeichen in einem XML-Dokument. Der Standardwert ist 0.
