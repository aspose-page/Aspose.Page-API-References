## XmlReaderSettings::set_MaxCharactersInDocument method


Legt einen Wert fest, der die maximal zulässige Anzahl von Zeichen in einem XML-Dokument angibt. Ein Nullwert (0) bedeutet, dass es keine Begrenzung der Größe des XML-Dokuments gibt. Ein von Null verschiedener Wert gibt die maximale Größe in Zeichen an.
