## XmlReaderSettings::get_NameTable method


Gibt das [XmlNameTable](../../xmlnametable/) zurück, das für atomisierte Zeichenkettenvergleiche verwendet wird.

```cpp
SharedPtr<XmlNameTable> System::Xml::XmlReaderSettings::get_NameTable()
```


### ReturnValue

Das [XmlNameTable](../../xmlnametable/), das alle atomisierten Zeichenketten speichert, die von allen mit diesem [XmlReaderSettings](../)-Objekt erstellten [XmlReader](../../xmlreader/)-Instanzen verwendet werden. Der Standardwert ist **nullptr**. Die erstellte [XmlReader](../../xmlreader/)-Instanz verwendet ein neues leeres [NameTable](../../nametable/), wenn dieser Wert **nullptr** ist.
