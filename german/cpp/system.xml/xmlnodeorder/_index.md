## XmlNodeOrder enum


Beschreibt die Dokumentreihenfolge eines Knotens im Vergleich zu einem zweiten Knoten.

```cpp
enum class XmlNodeOrder
```

### Werte

| Name | Wert | Beschreibung |
| --- | --- | --- |
| Before | 0 | Der aktuelle Knoten dieses Navigators befindet sich vor dem aktuellen Knoten des bereitgestellten Navigators. |
| Nach | 1 | Der aktuelle Knoten dieses Navigators befindet sich nach dem aktuellen Knoten des bereitgestellten Navigators. |
| Gleich | 2 | Die beiden Navigatoren sind auf demselben Knoten positioniert. |
| Unbekannt | 3 | Die Knotenpositionen können in Dokumentreihenfolge nicht relativ zueinander bestimmt werden. Dies könnte auftreten, wenn die beiden Knoten in unterschiedlichen Bäumen liegen. |
