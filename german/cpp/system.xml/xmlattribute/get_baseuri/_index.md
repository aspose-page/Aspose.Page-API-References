## XmlAttribute::get_BaseURI method


Gibt den Basis-Uniform Resource Identifier (URI) des Knotens zurück.

```cpp
String System::Xml::XmlAttribute::get_BaseURI() override
```


### ReturnValue

Der Ort, von dem der Knoten geladen wurde, oder [String::Empty](../../../system/string/empty/), wenn der Knoten keine Basis-URI hat. [Attribute](../../../system/attribute/) Knoten haben dieselbe Basis-URI wie ihr übergeordnetes Element. Wenn ein Attributknoten kein übergeordnetes Element hat, gibt get_BaseURI [String::Empty](../../../system/string/empty/) zurück.
