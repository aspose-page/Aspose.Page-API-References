## XmlElement::HasAttribute(String, String) method


Bestimmt, ob der aktuelle Knoten ein Attribut mit dem angegebenen lokalen Namen und der Namespace‑URI hat.

```cpp
virtual bool System::Xml::XmlElement::HasAttribute(String localName, String namespaceURI)
```


| Parameter | Typ | Beschreibung |
| --- | --- | --- |
| localName | String | Der lokale Name des Attributs, das gefunden werden soll. |
| namespaceURI | String | Der Namespace-URI des Attributs, das gefunden werden soll. |

### ReturnValue

**true** if the current node has the specified attribute; otherwise, **false**.
