## XmlElement::set_IsEmpty method


Setzt das Tag-Format des Elements.

```cpp
void System::Xml::XmlElement::set_IsEmpty(bool value)
```


| Parameter | Typ | Beschreibung |
| --- | --- | --- |
| Wert | bool | Der zu setzende Wert. Beim Setzen dieses Wertes, wenn er auf **true** gesetzt wird, werden die Kindknoten des Elements entfernt und das Element wird im Kurz-Tag-Format serialisiert. Wenn er auf **false** gesetzt wird, wird der Wert geändert (unabhängig davon, ob das Element Inhalt hat oder nicht); ist das Element leer, wird es im Langformat serialisiert. |
