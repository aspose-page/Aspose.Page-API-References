## XmlWriter::WriteQualifiedName method


Wenn in einer abgeleiteten Klasse überschrieben, schreibt es den namespace‑qualifizierten Namen. Diese Methode ermittelt das im Geltungsbereich des angegebenen Namespace vorhandene Präfix.

```cpp
virtual void System::Xml::XmlWriter::WriteQualifiedName(const String &localName, const String &ns)
```


| Parameter | Typ | Beschreibung |
| --- | --- | --- |
| localName | const String\& | Der lokale Name, der geschrieben werden soll. |
| ns | const String\& | Der Namespace-URI für den Namen. |
