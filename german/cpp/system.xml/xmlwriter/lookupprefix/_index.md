## XmlWriter::LookupPrefix method


Wird in einer abgeleiteten Klasse überschrieben, gibt das nächstgelegene Präfix zurück, das im aktuellen Namensraum‑Bereich für die Namespace‑URI definiert ist.

```cpp
virtual String System::Xml::XmlWriter::LookupPrefix(String ns)=0
```


| Parameter | Typ | Beschreibung |
| --- | --- | --- |
| ns | String | Der Namespace-URI, dessen Präfix Sie finden möchten. |

### ReturnValue

Das passende Präfix oder **nullptr**, wenn kein passender Namespace‑URI im aktuellen Bereich gefunden wird.
