## XmlWriterSettings::get_NewLineChars method


Gibt die Zeichenfolge zurück, die für Zeilenumbrüche verwendet wird.

```cpp
String System::Xml::XmlWriterSettings::get_NewLineChars()
```


### ReturnValue

Die Zeichenkette, die für Zeilenumbrüche verwendet wird. Diese kann auf jeden Zeichenwert gesetzt werden. Um jedoch gültiges XML zu gewährleisten, sollten Sie nur gültige Leerzeichen‑Zeichen angeben, wie Leerzeichen, Tabulatoren, Wagenrückläufe oder Zeilenumbrüche. Der Standardwert ist **\\r\\n** (Wagenrücklauf, neue Zeile).
