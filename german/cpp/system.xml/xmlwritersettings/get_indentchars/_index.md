## XmlWriterSettings::get_IndentChars method


Gibt die Zeichenfolge zurück, die beim Einrücken verwendet werden soll. Diese Einstellung wird verwendet, wenn der Wert [XmlWriterSettings::set_Indent](../set_indent/) auf **true** gesetzt ist.

```cpp
String System::Xml::XmlWriterSettings::get_IndentChars()
```


### ReturnValue

Die Zeichenfolge, die beim Einrücken verwendet werden soll. Sie kann auf jeden Zeichenfolgenwert gesetzt werden. Um jedoch gültiges XML sicherzustellen, sollten Sie nur gültige Leerzeichen‑Zeichen angeben, wie Leerzeichen, Tabulatoren, Wagenrückläufe oder Zeilenumbrüche. Der Standardwert sind zwei Leerzeichen.
