## XmlWriterSettings::get_CloseOutput method


Gibt einen Wert zurück, der angibt, ob der [XmlWriter](../../xmlwriter/) den zugrunde liegenden Stream oder TextWriter ebenfalls schließen soll, wenn die [XmlWriter::Close](../../xmlwriter/close/)‑Methode aufgerufen wird.

```cpp
bool System::Xml::XmlWriterSettings::get_CloseOutput()
```


### ReturnValue

**true** to also close the underlying stream or TextWriter; otherwise, **false**. The default is **false**.
