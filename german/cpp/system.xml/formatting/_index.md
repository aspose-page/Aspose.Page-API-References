## Formatting enum


Gibt Formatierungsoptionen für den [XmlTextWriter](../xmltextwriter/) an.


### Werte

| Name | Wert | Beschreibung |
| --- | --- | --- |

| Indented | 1 | Verursacht, dass Kind-Elemente gemäß den Einstellungen von [XmlTextWriter::set_Indentation](../xmltextwriter/set_indentation/) und [XmlTextWriter::set_IndentChar](../xmltextwriter/set_indentchar/) eingerückt werden. |
