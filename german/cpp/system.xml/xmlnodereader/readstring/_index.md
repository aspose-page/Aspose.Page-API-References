## XmlNodeReader::ReadString method


Liest den Inhalt eines Elements oder Textknotens als Zeichenkette.

```cpp
String System::Xml::XmlNodeReader::ReadString() override
```


### ReturnValue

Der Inhalt des Elements oder eines textähnlichen Knotens (dies kann CDATA, [Text](../../../system.text/)‑Knoten und dergleichen umfassen). Dies kann eine leere Zeichenkette sein, wenn der Reader auf etwas anderes als ein Element oder Textknoten positioniert ist oder wenn im aktuellen Kontext kein weiterer Textinhalt zurückzugeben ist. Hinweis: Der Textknoten kann entweder ein Element‑ oder ein Attribut‑Textknoten sein.
