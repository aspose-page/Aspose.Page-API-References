## XmlNodeChangedEventArgs::get_Action method


Gibt einen Wert zurück, der angibt, welcher Typ von Knotenänderungsereignis auftritt.

```cpp
XmlNodeChangedAction System::Xml::XmlNodeChangedEventArgs::get_Action()
```


### ReturnValue

Einer der Aufzählungswerte, der das Knotenänderungsereignis beschreibt. Der **[Action](../../../system/action/)**-Wert unterscheidet nicht, wann das Ereignis auftrat (vorher oder nachher). Sie können separate Ereignishandler erstellen, um beide Fälle zu behandeln.
