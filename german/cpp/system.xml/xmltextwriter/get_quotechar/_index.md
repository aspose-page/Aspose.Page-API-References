## XmlTextWriter::get_QuoteChar method


Gibt zurück, welches Zeichen zum Anführen von Attributwerten verwendet wird.

```cpp
char16_t System::Xml::XmlTextWriter::get_QuoteChar()
```


### ReturnValue

Das Zeichen, das zum Zitieren von Attributwerten verwendet wird. Es muss ein einfaches Anführungszeichen **'** oder ein doppeltes Anführungszeichen **\"** sein. Standardmäßig ist ein doppeltes Anführungszeichen.
