## XmlTextWriter::get_Indentation method


Gibt zurück, wie viele IndentChars für jede Ebene in der Hierarchie geschrieben werden, wenn [XmlTextWriter::set_Formatting](../set_formatting/) auf [Formatting::Indented](../../formatting/) gesetzt ist.

```cpp
int32_t System::Xml::XmlTextWriter::get_Indentation()
```


### ReturnValue

Anzahl der **IndentChars** pro Ebene. Der Standardwert ist 2.
