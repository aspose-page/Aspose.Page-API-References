## WhitespaceHandling enum


Gibt an, wie Leerzeichen behandelt werden.

```cpp
enum class WhitespaceHandling
```

### Werte

| Name | Wert | Beschreibung |
| --- | --- | --- |
| All | 0 | Gibt **Whitespace** und **SignificantWhitespace** Knoten zurück. Dies ist die Standardeinstellung. |
| Significant | 1 | Gibt nur **SignificantWhitespace**-Knoten zurück. |
| None | 2 | Gibt weder **Whitespace** noch **SignificantWhitespace**-Knoten zurück. |
