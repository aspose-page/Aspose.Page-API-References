## XmlTextReader::ReadString method


Liest den Inhalt eines Elements oder eines Textknotens als Zeichenkette.

```cpp
String System::Xml::XmlTextReader::ReadString() override
```


### ReturnValue

Der Inhalt des Elements oder Textknotens. Dies kann eine leere Zeichenkette sein, wenn der Reader auf etwas anderes als ein Element oder Textknoten positioniert ist oder wenn im aktuellen Kontext kein weiterer Textinhalt zurückzugeben ist. **Note:** Der Textknoten kann entweder ein Element- oder ein Attribut-Textknoten sein.
