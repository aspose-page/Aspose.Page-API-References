## XmlConvert::VerifyNCName method


Überprüft, ob der Name gemäß der W3C Extended Markup Language-Empfehlung ein gültiger **NCName** ist. Ein **NCName** ist ein Name, der keinen Doppelpunkt enthalten darf.

```cpp
static String System::Xml::XmlConvert::VerifyNCName(const String &name)
```


| Parameter | Typ | Beschreibung |
| --- | --- | --- |
| name | const String\& | Der zu überprüfende Name. |

### ReturnValue

Der Name, falls er ein gültiger NCName ist.
