## Klassen

| Klasse | Beschreibung |
| --- | --- |
| [Soap](./soap/) | Enthält die Zeichenkettenkonstanten von SOAP. |
| [Soap12](./soap12/) | Enthält die Zeichenkettenkonstanten der SOAP-Version 1.2. |
| [WebServiceAttribute](./webserviceattribute/) | Fügt dem XML [Web](../system.web/) Service zusätzliche Informationen hinzu. Objekte dieser Klasse sollten nur mit der Funktion [System::MakeObject()](../system/makeobject/) alloziert werden. Erstellen Sie niemals eine Instanz dieses Typs auf dem Stack oder mit dem Operator new, da dies zu Laufzeitfehlern und/oder Assertion-Fehlern führt. Wickeln Sie diese Klasse immer in einen [System::SmartPtr](../system/smartptr/) Zeiger ein und verwenden Sie diesen Zeiger, um ihn als Argument an Funktionen zu übergeben. |
| [WebServiceBindingAttribute](./webservicebindingattribute/) | Wird verwendet, um eine Bindung zu deklarieren, die eine oder mehrere Methoden des XML [Web](../system.web/) Service definiert. Objekte dieser Klasse sollten nur mit der Funktion [System::MakeObject()](../system/makeobject/) alloziert werden. Erstellen Sie niemals eine Instanz dieses Typs auf dem Stack oder mit dem Operator new, da dies zu Laufzeitfehlern und/oder Assertion-Fehlern führt. Wickeln Sie diese Klasse immer in einen [System::SmartPtr](../system/smartptr/) Zeiger ein und verwenden Sie diesen Zeiger, um ihn als Argument an Funktionen zu übergeben. |
