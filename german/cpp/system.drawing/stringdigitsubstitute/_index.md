## StringDigitSubstitute enum


Gibt an, wie Ziffern in einem String gemäß dem Gebietsschema oder der Sprache ersetzt werden.

```cpp
enum class StringDigitSubstitute
```

### Werte

| Name | Wert | Beschreibung |
| --- | --- | --- |
| Benutzer | 0 | Benutzerdefinierte Substitution. |
| None | 1 | Substitution ist deaktiviert. |
| National | 2 | Ersetze Ziffern, die der offiziellen Nationalsprache der Gebietseinstellung des Benutzers entsprechen. |
| Traditionell | 3 | Ersetze Ziffern, die dem nativen Schriftsystem oder der Sprache des Benutzers entsprechen, die möglicherweise nicht mit der offiziellen Nationalsprache der Gebietseinstellung des Benutzers übereinstimmen. |
