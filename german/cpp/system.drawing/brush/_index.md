## Brush class


Eine Basisklasse für Klassen, die Füllungen darstellen, die zum Ausfüllen der Innenbereiche grafischer Formen verwendet werden. Objekte dieser Klasse sollten ausschließlich mit der [System::MakeObject()](../../system/makeobject/) Funktion zugewiesen werden. Erstellen Sie niemals eine Instanz dieses Typs auf dem Stack oder mit dem Operator new, da dies zu Laufzeitfehlern und/oder Assertion-Fehlern führt. Wickeln Sie diese Klasse immer in einen [System::SmartPtr](../../system/smartptr/) Zeiger ein und verwenden Sie diesen Zeiger, um ihn als Argument an Funktionen zu übergeben.

```cpp
class Brush : public System::ICloneable,
              public System::IDisposable
```
