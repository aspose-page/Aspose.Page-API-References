## Point::Add method


Fügt die Breiten- und Höhenwerte des angegebenen [Size](../../size/) Objekts zu den X- und Y-Koordinatenwerten des angegebenen [Point](../) Objekts entsprechend hinzu.

```cpp
static Point System::Drawing::Point::Add(const Point &point, const Size &size)
```


| Parameter | Typ | Beschreibung |
| --- | --- | --- |
| Punkt | const Point\& | Der zu verschiebende Punkt |
| size | const Size\& | Das [Size](../../size/)-Objekt, das die Werte angibt, die zu den Koordinatenwerten des **point** hinzugefügt werden sollen. |

### ReturnValue

Ein neues [Point](../) Objekt, dessen X-Koordinatenwert gleich der Summe des X-Koordinatenwerts von **point** und des Breitenwerts von **size** ist und dessen Y-Koordinatenwert gleich der Summe des Y-Koordinatenwerts von **point** und des Höhenwerts von **size** ist.
