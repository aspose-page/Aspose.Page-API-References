## PointF::Add(const PointF\&, const Size\&) method


Fügt die Breiten- und Höhenwerte des angegebenen [Size](../../size/)-Objekts den X- und Y-Koordinatenwerten des angegebenen [PointF](../)-Objekts entsprechend hinzu.

```cpp
static PointF System::Drawing::PointF::Add(const PointF &point, const Size &size)
```


| Parameter | Typ | Beschreibung |
| --- | --- | --- |
| Punkt | const PointF\& | Der zu verschiebende Punkt |
| size | const Size\& | Das [Size](../../size/)-Objekt, das die Werte angibt, die zu den Koordinatenwerten des **point** hinzugefügt werden sollen. |

### ReturnValue

Ein neues [PointF](../)-Objekt, dessen X‑Koordinatenwert gleich der Summe des X‑Koordinatenwerts von **point** und des Breitenwerts von **size** ist und dessen Y‑Koordinatenwert gleich der Summe des Y‑Koordinatenwerts von **point** und des Höhenwerts von **size** ist.
