## StringFormatFlags enum


Gibt die Anzeige- und Layoutinformationen für die Textstrings an.

```cpp
enum class StringFormatFlags
```

### Werte

| Name | Wert | Beschreibung |
| --- | --- | --- |
| RichtungRechtsNachLinks | 1 | Rechts-nach-links-Textausrichtung. |
| DirectionVertical | 2 | Der [Text](../../system.drawing.text/) ist vertikal ausgerichtet. |
| FitSchwarzeBox | 4 | Teile von Zeichen können über die Layout-Box hinausragen. |
| AnzeigeFormatSteuerung | 32 | Steuerzeichen werden angezeigt. |
| NoFontFallback | 1024 | Der [Font](../font/) Fallback ist deaktiviert. |
| MesseNachlaufendeLeerzeichen | 2048 | Fügt das nachfolgende Leerzeichen am Ende jeder Zeile ein. |
| NoWrap | 4096 | Das Zeilenumbruch von [Text](../../system.drawing.text/) ist deaktiviert. |
| ZeilenGrenze | 8192 | Ganze Zeilen werden im Formatierungsrechteck angeordnet. |
| KeinClip | 16384 | Überhängende Teile von Glyphen dürfen angezeigt werden. |
