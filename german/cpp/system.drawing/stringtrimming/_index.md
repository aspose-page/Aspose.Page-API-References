## StringTrimming enum


Gibt an, wie Zeichen aus einem String, der nicht in die Layoutform passt, abgeschnitten werden sollen.

```cpp
enum class StringTrimming
```

### Werte

| Name | Wert | Beschreibung |
| --- | --- | --- |
| None | 0 | Kein Trimmen. |
| Zeichen | 1 | Auf das nächste Zeichen trimmen. |
| Wort | 2 | Auf das nächste Wort trimmen. |
| EllipsisCharacter | 3 | Auf das nächste Zeichen trimmen und am Ende der Zeichenkette ein Auslassungszeichen einfügen. |
| EllipsisWord | 4 | Auf das nächste Wort trimmen und am Ende der Zeichenkette ein Auslassungszeichen einfügen. |
| EllipsisPath | 5 | Der Mittelteil wird aus den beschnittenen Zeilen entfernt und durch ein Auslassungszeichen ersetzt. So viel wie möglich vom letzten durch Schrägstriche getrennten Segment der Zeile beibehalten. |
