## Color::get_IsNamedColor method


Gibt einen Wert zurück, der bestimmt, ob die Struktur [Color](../) eine benannte Farbe oder ein Mitglied der Aufzählung [KnownColor](../../knowncolor/) darstellt.

```cpp
bool System::Drawing::Color::get_IsNamedColor() const
```


### ReturnValue

Wahr, wenn die [Color](../) entweder mit der Methode FromName(String) oder der Methode [FromKnownColor(KnownColor)](../fromknowncolor/) erstellt wurde; andernfalls wird false zurückgegeben.
