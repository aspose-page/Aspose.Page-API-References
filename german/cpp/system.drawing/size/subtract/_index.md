## Size::Subtract method


Gibt ein neues [Size](../)-Objekt zurück, das das Ergebnis der Subtraktion von **size2** von **size1** ist, d. h. dessen Breitenwert das Ergebnis der Subtraktion des Breitenwerts von **size2's** vom Breitenwert von **size1's** ist und dessen Höhenwert das Ergebnis der Subtraktion des Höhenwerts von **size2's** vom Höhenwert von **size1's** ist.

```cpp
static Size System::Drawing::Size::Subtract(const Size &size1, const Size &size2)
```


| Parameter | Typ | Beschreibung |
| --- | --- | --- |
| size1 | const Size\& | Das [Size](../)-Objekt, von dem subtrahiert wird |
| size2 | const Size\& | Das [Size](../)-Objekt zum Subtrahieren |

### ReturnValue

Das [Size](../)-Objekt, das das Ergebnis der Subtraktion von **size2** von **size1** darstellt.
