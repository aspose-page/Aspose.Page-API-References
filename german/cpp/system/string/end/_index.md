## String::end method


Gibt einen Zeiger auf das Ende des tatsächlichen Zeichenkettenpuffers zurück. Allokiert niemals etwas neu. Garantiert nicht, dass der Puffer nullterminiert ist.

```cpp
const UChar * System::String::end() const
```


### ReturnValue

Zeiger auf das erste Element nach dem tatsächlichen String-Puffer, nullterminiert oder nicht.
