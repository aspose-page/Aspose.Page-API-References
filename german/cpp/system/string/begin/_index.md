## String::begin method


Gibt einen Zeiger auf den Anfang des tatsächlichen Zeichenkettenpuffers zurück. Allokiert niemals etwas neu. Garantiert nicht, dass der Puffer nullterminiert ist.

```cpp
const UChar * System::String::begin() const
```


### ReturnValue

Zeiger auf den tatsächlichen Zeichenkettenpuffer, nullterminiert oder nicht.
