## SmartPtr::IsAliasingPtr method


Prüft, ob der Zeiger auf ein anderes Objekt als das besessene (erstellt durch einen aliasing‑Konstruktor) zeigt.

```cpp
bool System::SmartPtr<T>::IsAliasingPtr() const
```


### ReturnValue

Wahr, wenn der Zeiger einen Unterschied zwischen dem referenzierten und dem besessenen Objekt hat, andernfalls falsch.
