## SmartPtr::get_shared_count method


Ermittelt die Anzahl der bestehenden Shared‑Pointer auf das referenzierte Objekt, einschließlich des aktuellen. Assertiert, dass der aktuelle Zeiger im shared mode ist.

```cpp
int System::SmartPtr<T>::get_shared_count() const
```


### ReturnValue

Anzahl der vorhandenen Shared‑Pointer auf das referenzierte Objekt, falls vorhanden. Ist der Zeiger null, wird 0 zurückgegeben.
