## SmartPtr::static_pointer_cast method


Wandelt den Zeiger mithilfe von static_cast auf das referenzierte Objekt in einen anderen Typ um.

```cpp
template<class Y> SmartPtr<Y> System::SmartPtr<T>::static_pointer_cast() const
```


| Parameter | Beschreibung |
| --- | --- |
| Y | Zieltyp des referenzierten Objekts. |

### ReturnValue

Zeiger des geänderten Typs, der stets im Shared‑Modus ist.
