## SmartPtr::Is method


Prüft, ob das referenzierte Objekt vom angegebenen Typ oder einem abgeleiteten Typ ist. Folgt den C#‑'is'-Semantiken.

```cpp
bool System::SmartPtr<T>::Is(const System::TypeInfo &target) const
```


| Parameter | Typ | Beschreibung |
| --- | --- | --- |
| target | const System::TypeInfo\& | Gibt den Zieltyp an, gegen den geprüft werden soll. |

### ReturnValue

Wahr, wenn die C# 'is'-Prüfung positiv ist, andernfalls falsch.
