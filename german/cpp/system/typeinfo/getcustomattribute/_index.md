## TypeInfo::GetCustomAttribute method


Sucht nach dem benutzerdefinierten Attribut, das den angegebenen Typ hat und auf den vom aktuellen Objekt dargestellten Typ angewendet wird.

```cpp
ObjectPtr System::TypeInfo::GetCustomAttribute(const TypeInfo &attributeType) const
```


| Parameter | Typ | Beschreibung |
| --- | --- | --- |
| attributeType | const TypeInfo\& | Die konstante Referenz auf das [TypeInfo](../)-Objekt, das den Typ des zu suchenden Attributs darstellt |

### ReturnValue

Ein Zeiger auf ein Objekt, das das gefundene Attribut darstellt, oder ein Nullzeiger, falls kein Attribut gefunden wurde, das den Suchkriterien entspricht
