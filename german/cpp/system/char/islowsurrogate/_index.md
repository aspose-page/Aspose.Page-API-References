## Char::IsLowSurrogate(char_t) method


Bestimmt, ob das angegebene Zeichen ein Low‑Surrogate ist.

```cpp
static bool System::Char::IsLowSurrogate(char_t c)
```


| Parameter | Typ | Beschreibung |
| --- | --- | --- |
| c | char_t | Das zu testende Zeichen |

### ReturnValue

Wahr, wenn das angegebene Zeichen ein Low Surrogate ist, andernfalls - falsch.
