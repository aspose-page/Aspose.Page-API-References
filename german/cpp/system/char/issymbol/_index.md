## Char::IsSymbol(char_t) method


Bestimmt, ob das angegebene Zeichen als Symbolzeichen klassifiziert ist.

```cpp
static bool System::Char::IsSymbol(char_t c)
```


| Parameter | Typ | Beschreibung |
| --- | --- | --- |
| c | char_t | Das zu testende Zeichen |

### ReturnValue

True, wenn das angegebene Zeichen ein Symbolzeichen ist, andernfalls - false
