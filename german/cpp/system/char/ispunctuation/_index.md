## Char::IsPunctuation(char_t) method


Bestimmt, ob das angegebene Zeichen als Satzzeichen klassifiziert ist.

```cpp
static bool System::Char::IsPunctuation(char_t c)
```


| Parameter | Typ | Beschreibung |
| --- | --- | --- |
| c | char_t | Das zu testende Zeichen |

### ReturnValue

Wahr, wenn das angegebene Zeichen ein Satzzeichen ist, sonst – falsch
