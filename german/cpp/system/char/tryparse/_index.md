## Char::TryParse method


Versucht, eine Zeichenkette, die aus einem einzelnen Zeichen besteht, in ein UTF-16‑Zeichen zu konvertieren. Die Funktion ist nur erfolgreich, wenn die Eingabezeichenkette nicht null ist und genau ein Zeichen lang ist.

```cpp
static bool System::Char::TryParse(const System::String &s, char_t &result)
```


| Parameter | Typ | Beschreibung |
| --- | --- | --- |
| s | const System::String\& | [String](../../string/) zum Konvertieren |
| result | char_t\& | Die Ausgabvariable, die das Ergebnis der Konvertierung enthält, wenn die Konvertierung erfolgreich ist. |

### ReturnValue

Wahr, wenn die Konvertierung erfolgreich war, andernfalls - falsch.
