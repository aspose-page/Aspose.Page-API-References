## TimeSpan::TimeSpan(int, int, int, int, int) constructor


Erstellt eine Instanz der Klasse [TimeSpan](../), die das Zeitintervall darstellt, das der Summe der angegebenen Anzahl von Stunden, Minuten, Sekunden und Millisekunden entspricht.

```cpp
System::TimeSpan::TimeSpan(int days, int hours, int minutes, int seconds, int milliseconds=0)
```


| Parameter | Typ | Beschreibung |
| --- | --- | --- |
| Tage | int | Die Anzahl der Tage im Tagesanteil des Zeitintervalls, das von der zu erstellenden Instanz dargestellt werden soll |
| Stunden | int | Die Anzahl der Stunden im Stundenanteil des Zeitintervalls, das von der zu erstellenden Instanz dargestellt werden soll |
| Minuten | int | Die Anzahl der Minuten im Minutenanteil des Zeitintervalls, das von der zu erstellenden Instanz dargestellt werden soll |
| Sekunden | int | Die Anzahl der Sekunden im Sekundenanteil des Zeitintervalls, das von der zu erstellenden Instanz dargestellt werden soll |
| Millisekunden | int | Die Anzahl der Millisekunden im Millisekundenanteil des Zeitintervalls, das von der zu erstellenden Instanz dargestellt werden soll |
