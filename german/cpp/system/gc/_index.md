## GC class


Stellt eine emulierte Garbage Collection dar, die eher wie ein Stub funktioniert und de facto nichts tut. Dies ist ein statischer Typ ohne Instanzdienste. Sie sollten niemals Instanzen davon auf irgendeine Weise erstellen.

```cpp
class GC : public System::Object
```
