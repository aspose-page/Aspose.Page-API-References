## Boolean::TryParse method


Konvertiert die angegebene Zeichenkette in einen bool‑Wert.

```cpp
static bool System::Boolean::TryParse(const String &value, bool &result)
```


| Parameter | Typ | Beschreibung |
| --- | --- | --- |
| Wert | const String\& | Der zu konvertierende String. |
| result | bool\& | Die Referenz auf eine bool-Variable, in die das Ergebnis der Konvertierung geschrieben wird; das Ergebnis ist true, wenn die angegebene Zeichenkette gleich "True" ist, und false, wenn die angegebene Zeichenkette gleich "False" ist. |

### ReturnValue

True, wenn die angegebene Zeichenkette entweder "True" oder "False" entspricht, sonst - false.
