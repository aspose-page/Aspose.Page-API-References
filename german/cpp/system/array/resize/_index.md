## Array::Resize method


Ändert die Größe des angegebenen Arrays auf den angegebenen Wert oder erstellt ein neues Array mit der angegebenen Größe.

```cpp
template<typename Type> static void System::Array<T>::Resize(ArrayPtr<Type> &arr, int new_size)
```


| Parameter | Typ | Beschreibung |
| --- | --- | --- |
| arr | ArrayPtr\<Type\>\& | [Array](../) zum Ändern der Größe. Wenn **arr** ein Nullzeiger ist, wird das neue Array erstellt |
| new_size | int | Die neue Größe des Arrays, oder die Größe des neuen Arrays, wenn **arr** null ist. |
