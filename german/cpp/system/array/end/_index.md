## Array::end() const method


Gibt einen Iterator auf das Element zurück, das dem letzten Element des const‑qualifizierten Containers folgt. Dieses Element dient als Platzhalter; ein Zugriff darauf führt zu undefiniertem Verhalten.

```cpp
const_iterator System::Array<T>::end() const noexcept
```


### ReturnValue

Ein Iterator, der auf das theoretische Element zeigt, das nach dem letzten Element des const‑qualifizierten Containers liegt.
