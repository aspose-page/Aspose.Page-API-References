## Enumerator::MoveNext method


Überprüft, ob der Index des aktuellen Elements nicht auf das letzte Element im Array zeigt und erhöht ihn, falls dies nicht der Fall ist.

```cpp
virtual bool System::Array<T>::Enumerator::MoveNext() override
```


### ReturnValue

Wahr, wenn der Index des aktuellen Elements nicht auf das letzte Element im Array zeigte, bevor er erhöht wurde, sonst – falsch.
