## StringComparison enum


Definiert den Stil des Zeichenkettenvergleichs.

```cpp
enum class StringComparison
```

### Werte

| Name | Wert | Beschreibung |
| --- | --- | --- |
| CurrentCulture | 0 | Verwendet die aktuelle Kultur. |
| CurrentCultureIgnoreCase | 1 | Verwendet die aktuelle Kultur, ignoriert jedoch die Groß-/Kleinschreibung. |
| InvariantCulture | 2 | Verwenden Sie die InvariantCulture. |
| InvariantCultureIgnoreCase | 3 | Verwenden Sie die InvariantCulture, aber ignorieren Sie die Groß-/Kleinschreibung. |
| Ordinal | 4 | Verwenden Sie ordinalen Sortierregeln. |
| OrdinalIgnoreCase | 5 | Verwenden Sie ordinalen Sortierregeln, aber ignorieren Sie die Groß-/Kleinschreibung. |
