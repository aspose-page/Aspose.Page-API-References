## DateTime::Equals(DateTime) const method



| Parameter | Typ | Beschreibung |
| --- | --- | --- |
| other | DateTime | Die Instanz der [DateTime](../)-Klasse, mit der das aktuelle Objekt verglichen werden soll. |

### ReturnValue

Wahr, wenn sowohl **other** als auch das aktuelle Objekt denselben Wert darstellen, sonst – falsch
