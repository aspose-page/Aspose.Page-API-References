## DateTime::SpecifyKind method


Erstellt ein neues [DateTime](../)-Objekt, das die gleiche Anzahl von Ticks wie das angegebene [DateTime](../)-Objekt enthält und lokale Zeit, UTC‑Zeit oder weder noch darstellt, wie durch das Argument **kind** angegeben.

```cpp
static DateTime System::DateTime::SpecifyKind(DateTime value, DateTimeKind kind)
```


| Parameter | Typ | Beschreibung |
| --- | --- | --- |
| value | DateTime | Das [DateTime](../)-Objekt, von dem die Anzahl der Ticks kopiert werden soll |
| Art | DateTimeKind | Gibt an, ob das neue Objekt lokale Zeit, UTC‑Zeit oder weder noch darstellen soll. |

### ReturnValue

Ein neues [DateTime](../)-Objekt, das die gleiche Anzahl von Ticks wie **value** und den [DateTimeKind](../../datetimekind/)-Wert, der durch **kind** angegeben ist, enthält.
