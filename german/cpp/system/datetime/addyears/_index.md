## DateTime::AddYears method


Gibt eine neue Instanz der [DateTime](../)-Klasse zurück, die den Datums‑ und Uhrzeitwert darstellt, der dem vom aktuellen Objekt dargestellten Wert entspricht, wobei die Jahreskomponente um die angegebene Zahl erhöht wird.

```cpp
DateTime System::DateTime::AddYears(int value) const
```


| Parameter | Typ | Beschreibung |
| --- | --- | --- |
| Wert | int | Die Anzahl der hinzuzufügenden Jahre; der Wert kann positiv oder negativ sein. |

### ReturnValue

Eine neue Instanz der [DateTime](../)-Klasse, die den Datums‑ und Uhrzeitwert darstellt, der dem vom aktuellen Objekt dargestellten Wert entspricht, wobei die Jahreskomponente um **value** erhöht wird.
