## DateTime::operator<=(DateTime) const method


Bestimmt, ob das aktuelle Objekt den Datums- und Uhrzeitwert darstellt, der früher ist oder dem gleichen Wert entspricht, der vom angegebenen [DateTime](../)-Objekt dargestellt wird.

```cpp
bool System::DateTime::operator<=(DateTime other) const
```


| Parameter | Typ | Beschreibung |
| --- | --- | --- |
| other | DateTime | Das [DateTime](../)-Objekt, mit dem das aktuelle Objekt verglichen wird |

### ReturnValue

Wahr, wenn der vom aktuellen Objekt dargestellte Datums- und Uhrzeitwert früher ist als oder dem von **other** dargestellten Wert, andernfalls – falsch
