## Uri::GetComponents method


Gibt die angegebenen Komponenten der vom aktuellen Objekt dargestellten URI zurück, wobei das angegebene Escaping verwendet wird.

```cpp
String System::Uri::GetComponents(UriComponents components, UriFormat format) const
```


| Parameter | Typ | Beschreibung |
| --- | --- | --- |
| components | UriComponents | Eine bitweise Kombination der Werte von [UriComponents](../../uricomponents/), die angibt, welche Teile der URI zurückgegeben werden sollen. |
| Format | UriFormat | Gibt an, wie Sonderzeichen escaped werden. |

### ReturnValue

Die angeforderten Komponenten
