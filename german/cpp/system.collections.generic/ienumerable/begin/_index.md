## IEnumerable::begin() method


Gibt einen Iterator zurück, der auf das erste Element (falls vorhanden) der Sammlung zeigt. Dieser Iterator kann nicht verwendet werden, um ein referenziertes Objekt zu ändern, weil [GetEnumerator()](../getenumerator/) ein Kopie-Objekt von T zurückgibt.

```cpp
iterator System::Collections::Generic::IEnumerable<T>::begin()
```


### ReturnValue

Ein Iterator, der auf das erste Element (falls vorhanden) der Sammlung zeigt
