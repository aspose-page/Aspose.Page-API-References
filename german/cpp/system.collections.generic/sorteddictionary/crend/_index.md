## SortedDictionary::crend method


Ruft einen Reverse-Iterator für ein nicht existierendes const‑qualifiziertes Element vor dem Beginn der Sammlung ab.

```cpp
const_reverse_iterator System::Collections::Generic::SortedDictionary<TKey, TValue>::crend() const noexcept
```

### ReturnValue

Ein Reverse-Iterator, der auf das theoretische const-qualifizierte Element zeigt, das dem ersten Element in der Map vorausgeht (letztes im Reverse).
