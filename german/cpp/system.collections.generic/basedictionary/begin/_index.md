## BaseDictionary::begin method


Gibt einen Iterator zum KVPair-Wrapper für das Schlüssel‑Wert‑Element des Containers zurück. Implementiert im C#‑Stil – der Iterator sollte das KVPair‑Objekt mit der get_Key()‑ und get_Value()‑Schnittstelle zurückgeben. Ist der Container leer, ist der zurückgegebene Iterator gleich [end()](../../ienumerable/end/).

```cpp
const_iterator System::Collections::Generic::BaseDictionary<Map>::begin() const noexcept
```


### ReturnValue

Ein Iterator, der auf das erste Element der Sammlung zeigt.
