## List::LastIndexOf(const T\&) const method


Durchsucht das angegebene Objekt und gibt den nullbasierten Index des letzten Vorkommens in der gesamten Liste zurück.

```cpp
int32_t System::Collections::Generic::List<T>::LastIndexOf(const T &item) const
```


| Parameter | Typ | Beschreibung |
| --- | --- | --- |
| item | const T\& | Das Objekt, das in der Liste gefunden werden soll |

### ReturnValue

Der nullbasierte Index des letzten Vorkommens des Elements innerhalb der gesamten [List](../), falls gefunden; andernfalls -1.
