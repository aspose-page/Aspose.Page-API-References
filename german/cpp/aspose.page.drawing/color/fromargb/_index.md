## Color::FromArgb(int32_t, Aspose::Page::Drawing::Color) method


Erstellt eine [T:Aspose::Page::Drawing::Color](../) Struktur aus der angegebenen [T:Aspose::Page::Drawing::Color](../) Struktur, jedoch mit dem neu angegebenen Alpha‑Wert. Obwohl diese Methode einen 32‑Bit‑Wert für den Alpha‑Wert zulässt, ist der Wert auf 8 Bit begrenzt.

```cpp
static Aspose::Page::Drawing::Color Aspose::Page::Drawing::Color::FromArgb(int32_t alpha, Aspose::Page::Drawing::Color baseColor)
```


| Parameter | Typ | Beschreibung |
| --- | --- | --- |
| alpha | int32_t | Der Alpha-Wert für das neue [T:Aspose::Page::Drawing::Color](../). Gültige Werte sind 0 bis 255. |
| baseColor | Aspose::Page::Drawing::Color | Das [T:Aspose::Page::Drawing::Color](../), aus dem die neue [T:Aspose::Page::Drawing::Color](../) erstellt werden soll. |

### ReturnValue

Das [T:Aspose::Page::Drawing::Color](../), das diese Methode erstellt.

```cpp
static Aspose::Page::Drawing::Color Aspose::Page::Drawing::Color::FromArgb(int32_t alpha, int32_t red, int32_t green, int32_t blue)
```


| Parameter | Typ | Beschreibung |

| alpha | int32_t | Die Alpha-Komponente. Gültige Werte sind 0 bis 255. |
| rot | int32_t | Die Rot-Komponente. Gültige Werte sind 0 bis 255. |
| grün | int32_t | Die Grün-Komponente. Gültige Werte sind 0 bis 255. |
| blau | int32_t | Die Blau-Komponente. Gültige Werte sind 0 bis 255. |
