## Encoding::GetMaxByteCount method


Ermittelt die maximale Anzahl von Bytes, die zum Kodieren einer angegebenen Anzahl von Zeichen benötigt werden.

```cpp
virtual int System::Text::Encoding::GetMaxByteCount(int char_count)=0
```


| Parameter | Typ | Beschreibung |
| --- | --- | --- |
| char_count | int | Anzahl der zu codierenden Zeichen. |

### ReturnValue

Anzahl der Bytes, die ausreichen, um jede Zeichenkette mit gegebener Zeichenlänge zu codieren.
