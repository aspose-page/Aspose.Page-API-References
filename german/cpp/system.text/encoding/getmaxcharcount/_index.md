## Encoding::GetMaxCharCount method

```cpp
virtual int System::Text::Encoding::GetMaxCharCount(int byte_count)=0
```


| Parameter | Typ | Beschreibung |
| --- | --- | --- |
| byte_count | int | Anzahl der zu dekodierenden Bytes. |

### ReturnValue

Anzahl der Bytes, die ausreichen, um jede Zeichenkette mit gegebener Zeichenlänge zu dekodieren.
