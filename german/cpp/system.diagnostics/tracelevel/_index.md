## TraceLevel enum


Gibt an, welche Nachrichten für die Klassen [System.Diagnostics.Debug](../debug/), [System.Diagnostics.Trace](../trace/) und System.Diagnostics.TraceSwitch ausgegeben werden sollen.

```cpp
enum class TraceLevel
```

### Werte

| Name | Wert | Beschreibung |
| --- | --- | --- |
| Aus | 0 | Gibt keine Trace‑ und Debug‑Nachrichten aus. |
| Error | 1 | Gibt Fehlermeldungen aus. |
| Warning | 2 | Gibt Warnungen und Fehlermeldungen aus. |
| Info | 3 | Gibt Informationsmeldungen, Warnungen und Fehlermeldungen aus. |
| Ausführlich | 4 | Gibt alle Debugging- und Tracing-Meldungen aus. |
