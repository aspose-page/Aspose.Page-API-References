## X500DistinguishedNameFlags enum


Formatierungsregeln für den Distinguished Name von X509-Zertifikaten.


### Werte

| Name | Wert | Beschreibung |
| --- | --- | --- |
| None | 0 | Keine besonderen Merkmale. |
| Umgekehrt | 1 | Name ist reserviert. |
| UseSemicolons | 16 | Semikolons verwenden. |
| DoNotUsePlusSign | 32 | Der Name verwendet kein Pluszeichen. |
| DoNotUseQuotes | 64 | Deaktiviert Anführungszeichen im Namen. |

| UseNewLines | 256 | Ermöglicht die Verwendung von Zeilenumbrüchen. |
