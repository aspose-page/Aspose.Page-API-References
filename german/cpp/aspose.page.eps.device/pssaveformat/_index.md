## PsSaveFormat enum


Diese Aufzählung enthält verfügbare Optionen für das Speicherformat. Es kann PS oder [EPS](../../aspose.page.eps/) sein. [EPS](../../aspose.page.eps/) wird nur für einseitige Dokumente verwendet, während eine PS-Datei beliebig viele Seiten enthalten kann.

```cpp
enum class PsSaveFormat
```

### Werte

| Name | Wert | Beschreibung |
| --- | --- | --- |
| PS | 0 | Diese Option gibt an, dass das resultierende Dokument ein PostScript (PS)- oder Encapsulated PostScript ([EPS](../../aspose.page.eps/))-Datei sein muss. |
| EPS | 1 | Diese Option gibt an, dass das resultierende Dokument eine Encapsulated PostScript ([EPS](../../aspose.page.eps/))-Datei sein muss. Sie wird nur für einseitige Dokumente verwendet;. |
