## EmfType enum

enum class EmfType

| EmfOnly | n/a | [Windows](../../system.windows/) Erweitertes Metafile. Enthält GDI‑Befehle. Metadateien dieses Typs werden als EMF‑Datei bezeichnet. |

| EmfPlusDual | n/a | Duales [Windows](../../system.windows/) Erweitertes Metafile. Enthält äquivalente GDI‑ und GDI+‑Befehle. Metadateien dieses Typs werden als EMF+‑Datei bezeichnet. |
