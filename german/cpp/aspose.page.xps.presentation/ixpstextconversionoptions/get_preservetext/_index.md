## IXpsTextConversionOptions::get_PreserveText method


In [XPS](../../../aspose.page.xps/) können einige Textelemente Verweise auf alternative Glyphenformen enthalten, die keinem Zeichencode in der Schriftart entsprechen. Ist dieses Flag auf true gesetzt, wird der Text solcher [XPS](../../../aspose.page.xps/) Elemente in grafische Formen konvertiert. Anschließend erscheint der Text selbst transparent darüber. Dadurch bleibt der Text dieser Elemente auswählbar. Der Nebeneffekt ist jedoch, dass die Ausgabedatei viel größer sein kann als das Original. Ist das Flag auf false gesetzt, werden die Zeichen, die als alternative Formen angezeigt werden sollen, durch andere Zeichen ersetzt, die den alternativen Glyphenformen zugeordnet werden. Daher wird der Text, obwohl weiterhin auswählbar, modifiziert und ist wahrscheinlich unlesbar.
