## RegexOptions enum


[Regex](../regex/) options.

```cpp
enum class RegexOptions
```

### Werte

| Name | Wert | Beschreibung |
| --- | --- | --- |
| None | 0 | Standardverhalten. |
| Compiled | 1 | Regex für bessere Leistung kompilieren. Standardmäßig immer durchgeführt. |
| CultureInvariant | 2 | Verwende kulturunabhängige Übereinstimmung. Ignoriert. |
| ECMAScript | 4 | Verwenden Sie die ECMAScript‑Syntax. Ignoriert. |
| ExplicitCapture | 8 | Nur explizites Erfassen. Ignoriert. |
| IgnoreCase | 16 | Groß-/Kleinschreibung beim Abgleichen ignorieren. |
| IgnorePatternWhitespace | 32 | Leerzeichen im Muster ignorieren. Nicht unterstützt. |
| Multiline | 64 | Behandle '^' und '$' als Anfang bzw. Ende einer Zeile, nicht des gesamten Strings. |
| RightToLeft | 128 | Rechts-nach-links-Abgleich. Nicht unterstützt. |
| Singleline | 256 | Lässt '.' jedes Zeichen ohne Ausnahmen matchen (normalerweise werden Zeilenumbrüche nicht gematcht). |
