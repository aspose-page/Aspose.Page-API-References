## Thread::get_ManagedThreadId method


Ermittelt die Kennung des Threads. Kann vom Betriebssystem erhalten werden, aber wenn die Betriebssystem‑Thread‑Kennung die int‑Grenzen überschreitet, können sich Thread‑IDs überschneiden.

```cpp
int System::Threading::Thread::get_ManagedThreadId() const
```


### ReturnValue

Pseudo-eindeutige Thread‑Kennung.
