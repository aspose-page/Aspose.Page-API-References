## CancellationToken::get_CanBeCanceled method


Ermittelt, ob dieses Token in den abgebrochenen Zustand wechseln kann.

```cpp
bool System::Threading::CancellationToken::get_CanBeCanceled() const
```


### ReturnValue

true, wenn dieser Token in den abgebrochenen Zustand wechseln kann; andernfalls false.
## Hinweise



Tokens, die aus [CancellationTokenSource](../../cancellationtokensource/) erstellt wurden, geben true zurück, während der None-Token immer false zurückgibt.
