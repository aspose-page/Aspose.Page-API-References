## XPathNamespaceScope enum


Definiert den Namensbereich.

```cpp
enum class XPathNamespaceScope
```

### Werte

| Name | Wert | Beschreibung |
| --- | --- | --- |
| All | 0 | Gibt alle Namespaces zurück, die im Geltungsbereich des aktuellen Knotens definiert sind. Dies schließt den **xmlns:xml** Namespace ein, der immer implizit deklariert wird. Die Reihenfolge der zurückgegebenen Namespaces ist nicht definiert. |
| ExcludeXml | 1 | Gibt alle Namespaces zurück, die im Geltungsbereich des aktuellen Knotens definiert sind, ausgenommen den **xmlns:xml** Namespace. Der **xmlns:xml** Namespace wird immer implizit deklariert. Die Reihenfolge der zurückgegebenen Namespaces ist nicht definiert. |
| Local | 2 | Gibt alle Namespaces zurück, die lokal am aktuellen Knoten definiert sind. |
