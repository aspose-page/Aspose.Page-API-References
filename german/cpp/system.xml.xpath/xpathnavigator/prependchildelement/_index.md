## XPathNavigator::PrependChildElement method


Erstellt ein neues Kind‑Element am Anfang der Liste der Kindknoten des aktuellen Knotens unter Verwendung des angegebenen Namespace‑Präfixes, des lokalen Namens, der angegebenen Namespace‑URI und des angegebenen Werts.

```cpp
virtual void System::Xml::XPath::XPathNavigator::PrependChildElement(String prefix, String localName, String namespaceURI, String value)
```


| Parameter | Typ | Beschreibung |
| --- | --- | --- |
| Präfix | String | Das Namespace‑Präfix des neuen Kindelements (falls vorhanden). |
| localName | String | Der lokale Name des neuen Kindelements (falls vorhanden). |
| namespaceURI | String | Die Namespace‑URI des neuen Kindelements (falls vorhanden). [String::Empty](../../../system/string/empty/) und **nullptr** sind äquivalent. |
| value | String | Der Wert des neuen Kindelements. Wenn [String::Empty](../../../system/string/empty/) oder **nullptr** übergeben werden, wird ein leeres Element erstellt. |
