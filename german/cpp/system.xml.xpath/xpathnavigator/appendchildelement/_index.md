## XPathNavigator::AppendChildElement method


Erstellt einen neuen untergeordneten Elementknoten am Ende der Liste der untergeordneten Knoten des aktuellen Knotens unter Verwendung des angegebenen Namespace-Präfixes, des lokalen Namens und des angegebenen Namespace‑URI mit dem angegebenen Wert.

```cpp
virtual void System::Xml::XPath::XPathNavigator::AppendChildElement(String prefix, String localName, String namespaceURI, String value)
```


| Parameter | Typ | Beschreibung |
| --- | --- | --- |
| Präfix | String | Das Namensraumpräfix des neuen Kindknotens (falls vorhanden). |
| localName | String | Der lokale Name des neuen Kindknotens (falls vorhanden). |
| namespaceURI | String | Der Namespace-URI des neuen Kindelementknotens (falls vorhanden). [String::Empty](../../../system/string/empty/) und **nullptr** sind äquivalent. |
| value | String | Der Wert des neuen Kindelementknotens. Wenn [String::Empty](../../../system/string/empty/) oder **nullptr** übergeben werden, wird ein leeres Element erstellt. |
