## XPathNavigator::MoveToId method


Wenn sie in einer abgeleiteten Klasse überschrieben wird, bewegt sie zum Knoten, der ein Attribut vom Typ **ID** hat, dessen Wert mit dem angegebenen [String](../../../system/string/) übereinstimmt.

```cpp
virtual bool System::Xml::XPath::XPathNavigator::MoveToId(String id)=0
```


| Parameter | Typ | Beschreibung |
| --- | --- | --- |
| id | String | Ein [String](../../../system/string/) der den **ID**-Wert des Knotens darstellt, zu dem Sie springen möchten. |

### ReturnValue

**true** if the [XPathNavigator](../) is successful moving; otherwise, **false**. If **false**, the position of the navigator is unchanged.
