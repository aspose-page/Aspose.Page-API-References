## XmlSortOrder enum


Gibt die Sortierreihenfolge an.

```cpp
enum class XmlSortOrder
```

### Werte

| Name | Wert | Beschreibung |
| --- | --- | --- |
| Aufsteigend | 1 | Knoten werden in aufsteigender Reihenfolge sortiert. Zum Beispiel, wenn die Zahlen 1,2,3 und 4 in aufsteigender Reihenfolge sortiert werden, erscheinen sie als 1,2,3,4. |
| Absteigend | 2 | Knoten werden in absteigender Reihenfolge sortiert. Zum Beispiel, wenn die Zahlen 1,2,3 und 4 in absteigender Reihenfolge sortiert werden, erscheinen sie als 4,3,2,1. |
