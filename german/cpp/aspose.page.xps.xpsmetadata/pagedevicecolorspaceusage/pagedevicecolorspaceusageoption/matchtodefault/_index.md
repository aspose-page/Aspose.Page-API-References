## MatchToDefault field


Wenn das Gerät feststellt, dass das durch den Parameter [PageDeviceColorSpaceProfileURI](../../../pagedevicecolorspaceprofileuri/) angegebene Profil als Gerätefarbraumprofil verwendet werden kann, werden alle Elemente, die dasselbe Profil verwenden, so behandelt, als seien sie bereits im Gerätefarbraum angegeben. Alle anderen Elemente MÜSSEN das für dieses Element angegebene Profil verwenden. Wenn das Profil nicht als Gerätefarbraumprofil verwendet werden kann, MÜSSEN Elemente, die das Profil verwenden, farblich verwaltet werden wie jedes andere Element, das das Farbprofil verwendet.
