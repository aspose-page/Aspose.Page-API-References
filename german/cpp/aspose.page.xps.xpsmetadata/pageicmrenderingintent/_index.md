## PageICMRenderingIntent class


Beschreibt die Renderabsicht, wie sie in der ICC v2 Spezifikation definiert ist. Dieser Wert sollte ignoriert werden, wenn ein Bild oder grafisches Element ein eingebettetes Profil hat, das die **Rendering**‑Absicht angibt. [https://docs.microsoft.com/en-us/windows/win32/printdocs/pageicmrenderingintent](https://docs.microsoft.com/en-us/windows/win32/printdocs/pageicmrenderingintent).
