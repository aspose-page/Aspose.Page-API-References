## PageDestinationColorProfileURI class


Gibt einen relativen URI-Verweis auf ein in einem [XPS](../../aspose.page.xps/)[Document](../../aspose.page/document/) enthaltenes ICC‑Profil an. Die Verarbeitung dieser Option hängt von der Einstellung der [PageDeviceColorSpaceUsage](../pagedevicecolorspaceusage/)‑Funktion ab. Alle Elemente, die dieses Profil verwenden, werden angenommen, dass sie bereits im entsprechenden Gerätefarbraum liegen und nicht vom Treiber oder Gerät farblich verwaltet werden. [https://docs.microsoft.com/en-us/windows/win32/printdocs/pagedestinationcolorprofileuri](https://docs.microsoft.com/en-us/windows/win32/printdocs/pagedestinationcolorprofileuri).
