## JobPrimaryCoverFront class


Beschreibt das vordere (anfangende) Deckblatt. Der gesamte Auftrag hat ein einziges Primärblatt. Das Deckblatt sollte auf der [PageMediaSize](../pagemediasize/) und [PageMediaType](../pagemediatype/) gedruckt werden, die für die erste Seite des Auftrags verwendet werden. Das Deckblatt sollte in Verarbeitungsoptionen (wie [JobDuplexAllDocumentsContiguously](../jobduplexalldocumentscontiguously/), [JobNUpAllDocumentsContiguously](../jobnupalldocumentscontiguously/)) integriert werden, wie durch die angegebene [Option](../option/) angegeben. [https://docs.microsoft.com/en-us/windows/win32/printdocs/jobprimarycoverfront](https://docs.microsoft.com/en-us/windows/win32/printdocs/jobprimarycoverfront).

class JobPrimaryCoverFront : public Aspose::Page::XPS::XpsMetadata::Feature,
