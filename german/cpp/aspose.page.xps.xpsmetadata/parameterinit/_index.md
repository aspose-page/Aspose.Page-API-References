## ParameterInit class


Die Klasse, die einen allgemeinen [PrintTicket](../printticket/) Parameterinitialisierer implementiert. Die Basisklasse für alle schema-definierten Parameterinitialisierer. Definiert einen Wert für eine Instanz eines **ParameterDef** Elements. Ein **[ParameterInit](./)** Element ist das Ziel der Referenz, die von einem [ParameterRef](../parameterref/) Element gemacht wird. [https://docs.microsoft.com/en-us/windows/win32/printdocs/parameterinit](https://docs.microsoft.com/en-us/windows/win32/printdocs/parameterinit).

```cpp
class ParameterInit : public Aspose::Page::XPS::XpsMetadata::PrintTicketElement,
                      public virtual Aspose::Page::XPS::XpsMetadata::IPrintTicketItem
```
