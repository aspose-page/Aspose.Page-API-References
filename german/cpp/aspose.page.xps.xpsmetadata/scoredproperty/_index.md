## ScoredProperty class


Die Klasse, die eine allgemeine [PrintTicket](../printticket/)**[ScoredProperty](./)** implementiert. Die Basisklasse für alle schemadefinierte ScoredProperties. Ein **[ScoredProperty](./)**‑Element deklariert eine Eigenschaft, die intrinsisch für eine [Option](../option/)‑Definition ist. Solche Eigenschaften sollten beim Bewerten, wie genau eine angeforderte [Option](../option/) einer geräteunterstützten [Option](../option/) entspricht, verglichen werden. [https://docs.microsoft.com/en-us/windows/win32/printdocs/scoredproperty](https://docs.microsoft.com/en-us/windows/win32/printdocs/scoredproperty).
