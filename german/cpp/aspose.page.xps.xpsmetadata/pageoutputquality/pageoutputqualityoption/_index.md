## Felder

| Feld | Beschreibung |

| static [High](./high/) | Gibt die Absicht für hochwertige Ausgabe an (ausgeglichen für hochwertige Texte und Grafiken). |

| static [Photographic](./photographic/) | Gibt die Absicht für fotografische Ausgabe an (Bilder sollten die höchste Qualität haben). |
| static [Text](./text/) | Gibt die Absicht für reine Textausgabe an (Grafiken können schlecht oder gar nicht ausgegeben werden). |
