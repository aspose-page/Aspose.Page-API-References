## ParameterRef class


Die Klasse, die eine allgemeine [PrintTicket](../printticket/)‑Parameterreferenz implementiert. Ein **[ParameterRef](./)**‑Element definiert eine Referenz zu einem [ParameterInit](../parameterinit/)‑Element. Ein [ScoredProperty](../scoredproperty/)‑Element, das ein [ParameterRef](./)‑Element enthält, hat kein explizit gesetztes [Value](../value/)‑Element. Stattdessen erhält das [ScoredProperty](../scoredproperty/)‑Element seinen Wert vom [ParameterInit](../parameterinit/)‑Element, auf das durch ein **[ParameterRef](./)**‑Element verwiesen wird. [https://docs.microsoft.com/en-us/windows/win32/printdocs/parameterref](https://docs.microsoft.com/en-us/windows/win32/printdocs/parameterref).
