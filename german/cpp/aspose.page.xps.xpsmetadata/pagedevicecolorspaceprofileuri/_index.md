## PageDeviceColorSpaceProfileURI class


Gibt einen relativen URI zum Paketstamm zu einem ICC-Profil an, das in einem [XPS](../../aspose.page.xps/)[Document](../../aspose.page/document/) enthalten ist. Die Verarbeitung dieser Option hängt von der Einstellung des [PageDeviceColorSpaceUsage](../pagedevicecolorspaceusage/) Features ab. Alle Elemente, die dieses Profil verwenden, werden angenommen, bereits im entsprechenden Gerätefarbraum zu sein, und werden nicht im Treiber oder Gerät farblich verwaltet. [https://docs.microsoft.com/en-us/windows/win32/printdocs/pagedevicecolorspaceprofileuri](https://docs.microsoft.com/en-us/windows/win32/printdocs/pagedevicecolorspaceprofileuri).
