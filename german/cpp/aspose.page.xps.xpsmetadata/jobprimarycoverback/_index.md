## JobPrimaryCoverBack class


Beschreibt das hintere (abschließende) Deckblatt. Jeder Auftrag hat ein separates Primärblatt. Das Deckblatt sollte auf der [PageMediaSize](../pagemediasize/) und [PageMediaType](../pagemediatype/) gedruckt werden, die für die letzte Seite des Auftrags verwendet werden. Das Deckblatt sollte in Verarbeitungsoptionen (wie [JobDuplexAllDocumentsContiguously](../jobduplexalldocumentscontiguously/), [JobNUpAllDocumentsContiguously](../jobnupalldocumentscontiguously/)) integriert werden, wie durch die angegebene [Option](../option/) angegeben. [https://docs.microsoft.com/en-us/windows/win32/printdocs/jobprimarycoverback](https://docs.microsoft.com/en-us/windows/win32/printdocs/jobprimarycoverback).
