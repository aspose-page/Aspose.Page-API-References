## Felder

| Feld | Beschreibung |
| --- | --- |
| static [Custom](./custom/) | Gibt an, dass ein benutzerdefiniertes Fehlerblatt ausgegeben werden soll. Wenn ein [JobErrorSheetSource](../../joberrorsheetsource/)[ParameterInit](../../parameterinit/)-Element nicht angegeben ist, sollte diese [Option](../../option/) ignoriert werden. |
| static [None](./none/) | Gibt an, dass kein Fehlerblatt ausgegeben werden soll. |
