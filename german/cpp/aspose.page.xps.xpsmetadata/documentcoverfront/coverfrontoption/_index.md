## CoverFrontOption class


Beschreibt die [DocumentCoverFront](../)-Feature-Optionen.

```cpp
class CoverFrontOption : public Aspose::Page::XPS::XpsMetadata::Option
```

## Felder

| Feld | Beschreibung |
| --- | --- |
| static [BlankCover](./blankcover/) | Gibt an, dass ein leeres Deckblatt gedruckt werden soll. |
| static [NoCover](./nocover/) | Gibt an, dass kein Deckblatt ausgegeben wird. |
| static [PrintBack](./printback/) | Gibt an, dass die durch \"CoverFrontSource\" angegebene Umschlagseite auf der Rückseite des Umschlagblatts gedruckt werden soll. Wenn ein [DocumentCoverFrontSource](../../documentcoverfrontsource/)[ParameterInit](../../parameterinit/)-Element nicht angegeben ist, sollte diese [Option](../../option/) ignoriert werden. |
| static [PrintBoth](./printboth/) | Gibt an, dass die durch \"CoverFrontSource\" angegebene Umschlagseite auf beiden Seiten des Umschlagblatts gedruckt werden kann. Wenn ein [DocumentCoverFrontSource](../../documentcoverfrontsource/)[ParameterInit](../../parameterinit/)-Element nicht angegeben ist, sollte diese [Option](../../option/) ignoriert werden. |
| static [PrintFront](./printfront/) | Gibt an, dass die durch \"CoverFrontSource\" angegebene Umschlagseite auf der Vorderseite des Umschlagblatts gedruckt werden soll. Wenn ein [DocumentCoverFrontSource](../../documentcoverfrontsource/)[ParameterInit](../../parameterinit/)-Element nicht angegeben ist, sollte diese [Option](../../option/) ignoriert werden. |
