## DocumentCoverBack class


Beschreibt das hintere (abschließende) Deckblatt. Jedes Dokument hat ein separates Blatt. Das Deckblatt sollte auf dem [PageMediaSize](../pagemediasize/) und [PageMediaType](../pagemediatype/) gedruckt werden, die für die letzte Seite des Dokuments verwendet werden. Das Deckblatt sollte in Verarbeitungsoptionen (wie [DocumentDuplex](../documentduplex/), [DocumentNUp](../documentnup/)) integriert werden, wie durch die angegebene [Option](../option/) angegeben. [https://docs.microsoft.com/en-us/windows/win32/printdocs/documentcoverback](https://docs.microsoft.com/en-us/windows/win32/printdocs/documentcoverback).
