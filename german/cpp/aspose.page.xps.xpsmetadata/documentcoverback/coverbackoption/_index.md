| Feld | Beschreibung |

| static [BlankCover](./blankcover/) | Gibt an, dass ein leeres Deckblatt gedruckt werden soll. |
| static [NoCover](./nocover/) | Gibt an, dass kein Deckblatt ausgegeben wird. |

| static [PrintFront](./printfront/) | Gibt an, dass die durch "CoverBackSource" bezeichnete Umschlagseite auf der Vorderseite des Deckblatts gedruckt werden soll.
