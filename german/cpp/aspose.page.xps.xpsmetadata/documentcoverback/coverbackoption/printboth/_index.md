## PrintBoth field


Gibt an, dass das durch \"CoverBackSource\" angegebene Deckblatt auf beiden Seiten des Deckblatts gedruckt werden kann. Wenn ein [DocumentCoverBackSource](../../../documentcoverbacksource/)[ParameterInit](../../../parameterinit/) Element nicht angegeben ist, sollte diese [Option](../../../option/) ignoriert werden.

```cpp
static System::SharedPtr<DocumentCoverBack::CoverBackOption> Aspose::Page::XPS::XpsMetadata::DocumentCoverBack::CoverBackOption::PrintBoth
```
