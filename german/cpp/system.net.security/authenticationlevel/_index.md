## AuthenticationLevel enum


WebRequest-spezifische Authentifizierungs-Flags.

```cpp
enum class AuthenticationLevel
```

### Werte

| Name | Wert | Beschreibung |
| --- | --- | --- |
| None | 0 | Für den Client und den Server ist keine Authentifizierung erforderlich. |
| MutualAuthRequested | 1 | Die Anforderung schlägt nicht fehl, wenn der Server nicht authentifiziert ist. |
| MutualAuthRequired | 2 | Die aktuelle Anwendung erhält 'IOException', wenn der Server nicht authentifiziert ist. |
