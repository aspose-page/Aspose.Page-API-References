## XmlSchemaComplexType::get_IsAbstract method


Gibt die Information zurück, die bestimmt, ob das **complexType**‑Element im Instanzdokument verwendet werden kann.

```cpp
bool System::Xml::Schema::XmlSchemaComplexType::get_IsAbstract()
```


### ReturnValue

Wenn **true**, kann ein Element dieses **complexType**-Element nicht direkt verwenden und muss einen komplexen Typ verwenden, der von diesem **complexType**-Element abgeleitet ist. Der Standardwert ist **false**. Optional.
