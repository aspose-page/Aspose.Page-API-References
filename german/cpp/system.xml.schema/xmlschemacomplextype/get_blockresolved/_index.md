## XmlSchemaComplexType::get_BlockResolved method


Gibt den Wert zurück, nachdem der Typ in das post‑Schema‑Validierungs‑Informationsset (Infoset) kompiliert wurde. Dieser Wert gibt an, wie der Typ durchgesetzt wird, wenn **xsi:type** im Instanzdokument verwendet wird.

```cpp
XmlSchemaDerivationMethod System::Xml::Schema::XmlSchemaComplexType::get_BlockResolved()
```


### ReturnValue

Der nach dem Schema validierte Infoset‑Wert. Der Standardwert ist **BlockDefault** auf dem **schema**‑Element.
