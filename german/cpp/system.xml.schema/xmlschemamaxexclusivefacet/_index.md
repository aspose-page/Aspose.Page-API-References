## XmlSchemaMaxExclusiveFacet class


Stellt das **maxExclusive**-Element aus XML [Schema](../) dar, wie vom World Wide [Web](../../system.web/) Consortium (W3C) spezifiziert. Diese Klasse kann verwendet werden, um eine Einschränkung des maximalen Wertes eines **simpleType**-Elements anzugeben. Der Elementwert muss kleiner sein als der Wert des **maxExclusive**-Elements.

```cpp
class XmlSchemaMaxExclusiveFacet : public System::Xml::Schema::XmlSchemaFacet
```
