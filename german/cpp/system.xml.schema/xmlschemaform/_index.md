## XmlSchemaForm enum


Gibt an, ob Attribute oder Elemente mit einem Namensraumpräfix qualifiziert werden müssen.

```cpp
enum class XmlSchemaForm
```

### Werte

| Name | Wert | Beschreibung |
| --- | --- | --- |
| None | 0 | Element- und Attributform ist im Schema nicht angegeben. |
| Qualifiziert | 1 | Elemente und Attribute müssen mit einem Namespace-Präfix qualifiziert werden. |
| Nicht qualifiziert | 2 | Elemente und Attribute müssen nicht mit einem Namensraumpräfix qualifiziert werden. |
