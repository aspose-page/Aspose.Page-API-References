## XmlSchemaMinInclusiveFacet class


Stellt das **minInclusive**-Element aus dem XML [Schema](../) dar, wie vom World Wide [Web](../../system.web/) Consortium (W3C) festgelegt. Diese Klasse kann verwendet werden, um eine Einschränkung des Mindestwerts eines simpleType-Elements anzugeben. Der Elementwert muss größer oder gleich dem Wert des **minInclusive**-Elements sein.

```cpp
class XmlSchemaMinInclusiveFacet : public System::Xml::Schema::XmlSchemaFacet
```
