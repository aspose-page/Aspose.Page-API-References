## XmlSchemaMinLengthFacet class


Stellt das **minLength**‑Element aus dem XML [Schema](../) dar, wie vom World Wide [Web](../../system.web/) Consortium (W3C) festgelegt. Diese Klasse kann verwendet werden, um eine Einschränkung der Mindestlänge des Datenwerts eines **simpleType**‑Elements anzugeben. Die Länge muss größer sein als der Wert des **minLength**‑Elements.

```cpp
class XmlSchemaMinLengthFacet : public System::Xml::Schema::XmlSchemaNumericFacet
```
