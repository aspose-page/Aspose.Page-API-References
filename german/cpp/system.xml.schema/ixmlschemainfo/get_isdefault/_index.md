## IXmlSchemaInfo::get_IsDefault method


Gibt einen Wert zurück, der angibt, ob dieser validierte XML-Knoten als Ergebnis einer während der XML [Schema](../../) Definition Language (XSD)-Schemavalidierung angewendeten Vorgabe gesetzt wurde.


### ReturnValue

**true** if this validated XML node was set as the result of a default being applied during schema validation; otherwise, **false**.
