## GraphicsPath::AddPie(const Rectangle\&, float, float) method


Fügt die angegebene Kontur der Kuchenform zum Pfad hinzu, der vom aktuellen Objekt dargestellt wird.

```cpp
void System::Drawing::Drawing2D::GraphicsPath::AddPie(const Rectangle &rect, float startAngle, float sweepAngle)
```


| Parameter | Typ | Beschreibung |
| --- | --- | --- |
| rect | const Rectangle\& | Das Rechteck, das die Ellipse begrenzt, aus der der Kuchen gezeichnet wird. |
| startAngle | float | Gibt den Startwinkel des Kuchens in Grad an, gemessen im Uhrzeigersinn von der X-Achse. |
| sweepAngle | float | Gibt den Winkel zwischen dem Startwinkel und dem Endwinkel des Kuchens an. |
