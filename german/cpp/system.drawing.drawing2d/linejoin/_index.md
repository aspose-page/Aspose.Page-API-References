## LineJoin enum


Gibt an, wie aufeinanderfolgende Linien oder Kurven verbunden werden.

```cpp
enum class LineJoin
```

### Werte

| Name | Wert | Beschreibung |
| --- | --- | --- |
| Miter | 0 | Gehrte Verbindung, die je nach Länge der Gehrung im Vergleich zur Gehrungsgrenze eine spitze oder abgeschnittene Ecke erzeugt. |
| Bevel | 1 | Abgeschrägte Verbindung, die eine diagonale Ecke erzeugt. |
| Runden | 2 | Runde Verbindung, die einen glatten kreisförmigen Bogen zwischen den Linien erzeugt. |
| MiterClipped | 3 | Gehrte Verbindung, die je nach Länge der Gehrung im Vergleich zur Gehrungsgrenze eine spitze oder abgeschrägte Ecke erzeugt. |
