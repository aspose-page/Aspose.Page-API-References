## SmoothingMode enum


Gibt an, welcher Glättungstyp (Antialiasing) auf Linien, Kurven und Kanten von gefüllten Bereichen angewendet wird.

```cpp
enum class SmoothingMode
```

### Werte

| Name | Wert | Beschreibung |
| --- | --- | --- |
| Ungültig | n/a | Ungültiger Wert. |
| Default | n/a | Standardmodus, kein Antialiasing. |
| Hochgeschwindigkeit | n/a | Kein Antialiasing, hohe Geschwindigkeit. |
| HoheQualität | n/a | Antialiasing-Rendering, niedrige Geschwindigkeit. |
| None | n/a | Kein Antialiasing. |
| AntiAlias | n/a | Antialiasing-Rendering. |
