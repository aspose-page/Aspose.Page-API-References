## SaveOptions.SupressErrors property

Gibt an, ob Fehler unterdrückt werden müssen oder nicht. Wenn wahr, werden unterdrückte Fehler hinzugefügt[`Exceptions`](../exceptions/) list. Wenn falsch, wird der erste Fehler das Programm beenden.

public virtual bool SupressErrors { get; set; }
