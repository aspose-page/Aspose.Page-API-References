Die **Aspose.Page** ist ein Root-Namespace für alle Klassen der Aspose.Page-Bibliothek, die entweder direkt darin enthalten sind **Gerät** oder indirekt über mehrere Unternamensräume.

## Klassen

| Klasse | Beschreibung |
| --- | --- |
| [BuildVersionInfo](./buildversioninfo/) | Diese Klasse stellt Informationen über den aktuellen Produktaufbau bereit. |
| [Device](./device/) | Diese Klasse kapselt das Rendern des Dokuments auf einem abstrakten Gerät. Das Rendern des Dokuments erfolgt Seite für Seite. |
| [Document](./document/) | Eine Oberklasse für alle gekapselten Dokumente. |
| [ExternalFontCache](./externalfontcache/) | Verwenden Sie diese Klasse, um eine Schriftkapselung in einer Form zu erhalten, die von akzeptiert wird[`Device`](../aspose.page/device/) . |
| [License](./license/) | Stellt Methoden zur Lizenzierung der Komponente bereit. |
| [Margins](./margins/) | Diese Klasse kapselt den oberen, linken, unteren und rechten Rand. |
| [Metered](./metered/) | Bietet Methoden zum Festlegen von gemessenen Schlüsseln. |
| [SaveOptions](./saveoptions/) | Diese Klasse enthält Optionen, die für die Verwaltung des Konvertierungsprozesses erforderlich sind. |
| [UserProperties](./userproperties/) | Spezielle Eigenschaftsklasse, mit der typisierte Eigenschaften festgelegt und zurückgegeben werden können. Es ermöglicht auch, dass die Verbindung von zwei Standardeigenschaftsobjekten gesucht wird, wenn dieses Eigenschaftsobjekt die Eigenschaft nicht enthält. |
## Schnittstellen

| Schnittstelle | Beschreibung |
| --- | --- |
| [IGlyph](./iglyph/) | Diese Schnittstelle ermöglicht den Zugriff auf die Hauptparameter von Glyphen. |
| [IMultiPageDevice](./imultipagedevice/) | Diese Schnittstelle enthält Methoden zum Bearbeiten von mehrseitigen Geräten. |
| [IStreamable](./istreamable/) | Diese Schnittstelle wird für Dervices verwendet, die Dokumente in einen Stream rendern. |
| [ITrFont](./itrfont/) | Diese Schnittstelle ermöglicht den Zugriff auf die Hauptparameter der Schriftart. |
