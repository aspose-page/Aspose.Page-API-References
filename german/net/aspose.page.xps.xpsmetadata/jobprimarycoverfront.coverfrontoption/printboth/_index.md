## JobPrimaryCoverFront.CoverFrontOption.PrintBoth field

Gibt an, dass das durch „CoverFrontSource“ angegebene Deckblatt auf beiden Seiten des Deckblatts gedruckt werden kann. Wenn ein[`JobPrimaryCoverFrontSource`](../../jobprimarycoverfrontsource/)[`ParameterInit`](../../parameterinit/) Element ist nicht angegeben, diese Option sollte ignoriert werden.

```csharp
public static CoverFrontOption PrintBoth;
```
