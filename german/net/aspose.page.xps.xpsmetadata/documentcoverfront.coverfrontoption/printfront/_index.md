## DocumentCoverFront.CoverFrontOption.PrintFront field

Gibt an, dass das durch „CoverFrontSource“ angegebene Deckblatt auf die Vorderseite des Deckblatts gedruckt werden soll. Wenn ein[`DocumentCoverFrontSource`](../../documentcoverfrontsource/)[`ParameterInit`](../../parameterinit/) Element ist nicht angegeben, diese Option sollte ignoriert werden.

```csharp
public static CoverFrontOption PrintFront;
```
