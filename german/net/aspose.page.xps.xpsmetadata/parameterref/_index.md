## ParameterRef class

Die Klasse, die eine gemeinsame PrintTicket-Parameterreferenz implementiert. A element definiert einen Verweis auf a[`ParameterInit`](../parameterinit/)Element. A[`ScoredProperty`](../scoredproperty/) Element, das ein ParameterRef-Element enthält, hat kein explizit festgelegtes Element[`Value`](../value/) Element. Stattdessen die[`ScoredProperty`](../scoredproperty/) element erhält seinen Wert von der[`ParameterInit`](../parameterinit/) Element, auf das von a verwiesen wird element. https://docs.microsoft.com/en-us/windows/win32/printdocs/parameterref
