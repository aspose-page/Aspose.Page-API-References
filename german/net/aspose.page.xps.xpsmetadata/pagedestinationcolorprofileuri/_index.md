## PageDestinationColorProfileURI class

Gibt einen relativen URI-Verweis auf ein ICC-Profil an, das in einem XPS-Dokument enthalten ist. Die Verarbeitung dieser Option hängt von der Einstellung der PageDeviceColorSpaceUsage-Funktion ab. Es wird davon ausgegangen, dass alle Elemente, die dieses Profil verwenden, sich bereits im entsprechenden Gerätefarbraum befinden, und werden im Treiber oder Gerät nicht farbverwaltet. https://docs.microsoft.com/en-us/windows/win32/printdocs/pagedestinationcolorprofileuri
