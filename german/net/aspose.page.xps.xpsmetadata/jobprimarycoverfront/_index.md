## JobPrimaryCoverFront class

Beschreibt das vordere (Anfangs-)Deckblatt. Der gesamte Auftrag hat ein einziges Primärblatt. Das Deckblatt sollte auf dem gedruckt werden[`PageMediaSize`](../pagemediasize/) Und[`PageMediaType`](../pagemediatype/) wird für die erste Seite des Jobs verwendet. Das Deckblatt sollte in Verarbeitungsoptionen integriert werden (wie z[`JobDuplexAllDocumentsContiguously`](../jobduplexalldocumentscontiguously/) ,[`JobNUpAllDocumentsContiguously`](../jobnupalldocumentscontiguously/) ) wie durch die angegebene Option angegeben. https://docs.microsoft.com/en-us/windows/win32/printdocs/jobprimarycoverfront
