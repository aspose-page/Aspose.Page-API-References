## PageDeviceColorSpaceProfileURI class

Gibt einen relativen URI zum Paketstamm eines in einem XPS-Dokument enthaltenen ICC-Profils an. Die Verarbeitung dieser Option hängt von der Einstellung der PageDeviceColorSpaceUsage-Funktion ab. Es wird davon ausgegangen, dass alle Elemente, die dieses Profil verwenden, bereits die entsprechende Gerätefarbe haben space, und wird im Treiber oder Gerät nicht farbverwaltet. https://docs.microsoft.com/en-us/windows/win32/printdocs/pagedevicecolorspaceprofileuri
