## DocumentCoverFront class

Beschreibt das vordere (Anfangs-)Deckblatt. Jedes Dokument hat ein separates Blatt. Das Deckblatt sollte aufgedruckt werden[`PageMediaSize`](../pagemediasize/) Und[`PageMediaType`](../pagemediatype/) wird für die erste Seite des Dokuments verwendet. Das Deckblatt sollte in Verarbeitungsoptionen integriert werden (wie z[`DocumentDuplex`](../documentduplex/) ,[`DocumentNUp`](../documentnup/) ) wie durch die angegebene Option angegeben. https://docs.microsoft.com/en-us/windows/win32/printdocs/documentcoverfront
