## JobPrimaryCoverBack.CoverBackOption.PrintBoth field

Gibt an, dass das durch „CoverBackSource“ angegebene Deckblatt auf beiden Seiten des Deckblatts gedruckt werden kann. Wenn ein[`JobPrimaryCoverBackSource`](../../jobprimarycoverbacksource/)[`ParameterInit`](../../parameterinit/) Element ist nicht angegeben, diese Option sollte ignoriert werden.

```csharp
public static CoverBackOption PrintBoth;
```
