## PageDeviceColorSpaceUsage.PageDeviceColorSpaceUsageOption.MatchToDefault field

Wenn das Gerät feststellt, dass das von der angegebene Profil[`PageDeviceColorSpaceProfileURI`](../../pagedevicecolorspaceprofileuri/) parameter kann als Gerätefarbraumprofil verwendet werden, alle Elemente, die dasselbe Profil verwenden, werden so behandelt, als seien sie bereits im Gerätefarbraum spezifiziert . Alle anderen Elemente MÜSSEN das für dieses Element angegebene Profil verwenden. Wenn das Profil nicht als Gerätefarbraumprofil verwendet werden kann, MÜSSEN Elemente, die das Profil verwenden, wie alle anderen Elemente, die das Farbprofil verwenden, farblich verwaltet werden.

```csharp
public static PageDeviceColorSpaceUsageOption MatchToDefault;
```
