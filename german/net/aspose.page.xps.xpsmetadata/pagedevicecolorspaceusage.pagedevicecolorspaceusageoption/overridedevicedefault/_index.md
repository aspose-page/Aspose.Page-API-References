## PageDeviceColorSpaceUsage.PageDeviceColorSpaceUsageOption.OverrideDeviceDefault field

Wenn das durch den Parameter PageDeviceColorSpaceProfileURI angegebene Profil eine Anzahl von Kanälen hat, die der Anzahl der Primärfarben des Geräts entspricht, SOLLTE es anstelle des geräteinternen Farbmanagements für alle Elemente verwendet werden. Elemente, die dieses Profil verwenden, werden als in Gerätefarbe angenommen Leerzeichen und wird nicht weiter farblich verwaltet.

```csharp
public static PageDeviceColorSpaceUsageOption OverrideDeviceDefault;
```
