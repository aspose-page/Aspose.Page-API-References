Die **Aspose.Page.EPS** ist ein Root-Namespace für alle Klassen, die sich mit PS/EPS-Dateien befassen.

## Klassen

| Klasse | Beschreibung |
| --- | --- |
| [FontConstants](./fontconstants/) | Diese Klasse definiert eine Reihe von Konstanten zum Speichern von Schriftarten. |
| [GradientBrush](./gradientbrush/) | Diese Klasse wird zum Einkapseln von LinearGradientBrush und PathGradientBrush mit der Möglichkeit verwendet, den Wrap-Modus auf „Clamp“ zu setzen. Native Farbverlaufspinsel lösen immer eine Ausnahme aus, wenn jemand versucht, die WrapMode-Eigenschaft auf „WrapMode.Clamp“ zu setzen. |
| [LoadOptions](./loadoptions/) | Basisklasse für Optionen zum Laden von Dokumenten. |
| [PageConstants](./pageconstants/) | Diese Klasse definiert eine Reihe von Konstanten, die eine Seite beschreiben. Hilfsobjekte werden für verschiedene Ränder, Ausrichtungen, Neuskalierung und Standardseitengrößen bereitgestellt. |
| [PsConverterException](./psconverterexception/) | Diese Klasse enthält Informationen über einen Fehler, der ausgelöst wird, während eine PS-Datei in ein PDF-Dokument konvertiert wird. |
| [PsDocument](./psdocument/) | Diese Klasse kapselt PS/EPS-Dokumente. |
| [PsDocumentException](./psdocumentexception/) | Diese Klasse enthält Informationen über einen Fehler, der ausgegeben wird, während eine PostScript-Datei erstellt, bearbeitet und gespeichert wird. |
| [PsLoadOptions](./psloadoptions/) | Optionen zum Laden von PS-Dokumenten. |
