## GradientBrush class

Diese Klasse wird zum Einkapseln von LinearGradientBrush und PathGradientBrush mit der Möglichkeit verwendet, den Wrap-Modus auf „Clamp“ zu setzen. Native Farbverlaufspinsel lösen immer eine Ausnahme aus, wenn jemand versucht, die WrapMode-Eigenschaft auf „WrapMode.Clamp“ zu setzen.

```csharp
public class GradientBrush : Brush
```
