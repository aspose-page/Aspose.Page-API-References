## PsDocument.DrawTransparentImage method

Transformiertes transparentes Bild zeichnen. Wenn das Bild keinen Alphakanal hat, wird es undurchsichtig gezeichnet image

```csharp
public void DrawTransparentImage(Bitmap image, Matrix transform, int transparencyThreshold)
```

| Parameter | Typ | Beschreibung |
| --- | --- | --- |
| image | Bitmap | Das zu zeichnende Bild. |
| transform | Matrix | Die Matrix zum Transformieren des Bildes. |
| transparencyThreshold | Int32 | Eine Schwelle, die definiert, ab welchem Transparenzwert ein Pixel als vollständig transparent interpretiert wird. Alle Werte unter diesem Schwellenwert werden als vollständig undurchsichtig interpretiert. |
