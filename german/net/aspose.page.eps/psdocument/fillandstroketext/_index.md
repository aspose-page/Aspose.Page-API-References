## FillAndStrokeText(string, Font, float, float, Brush, Pen) {#fillandstroketext_3}

Fügt eine Textzeichenfolge hinzu, indem das Innere von Glyphen gefüllt und Glyphenkonturen gezeichnet werden.

```csharp
public void FillAndStrokeText(string text, Font font, float x, float y, Brush fillPaint, Pen stroke)
```

| Parameter | Typ | Beschreibung |
| --- | --- | --- |
| text | String | Der hinzuzufügende Text. |
| font | Font | Systemschriftart, die zum Zeichnen von Text verwendet wird. |
| x | Single | X-Koordinate für den Textursprung. |
| y | Single | Y-Koordinate für den Textursprung. |
| fillPaint | Brush | Die Füllung, die zum Malen von Glyphen im Inneren verwendet wird. |
| stroke | Pen | Der Strich, der zum Zeichnen von Glyphenkonturen verwendet wird. |
