## FillAndStrokeText(string, Font, float, float, Brush, Brush, Pen) {#fillandstroketext_1}

```csharp
public void FillAndStrokeText(string text, Font font, float x, float y, Brush fillPaint, 
    Brush strokePaint, Pen stroke)
```

| Parameter | Typ | Beschreibung |
| --- | --- | --- |
| text | String | Der hinzuzufügende Text. |
| font | Font | Systemschriftart, die zum Zeichnen von Text verwendet wird. |
| x | Single | X-Koordinate für den Textursprung. |
| y | Single | Y-Koordinate für den Textursprung. |
| fillPaint | Brush | Die Füllung, die zum Malen von Glyphen im Inneren verwendet wird. |
| strokePaint | Brush | DerBrush Wird zum Malen von Glyphenkonturen verwendet. Kann eine beliebige Unterklasse von seinBrush Klasse in der .NET-Plattform. |
