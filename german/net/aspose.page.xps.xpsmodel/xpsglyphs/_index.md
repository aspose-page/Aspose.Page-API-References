## XpsGlyphs class

Klasse, die Glyphs-Elementfunktionen einkapselt. Dieses Element stellt eine Folge von einheitlich formatiertem Text aus einer einzelnen Schriftart dar. Es stellt Informationen bereit, die für eine genaue Wiedergabe erforderlich sind, und unterstützt Such- und Auswahlfunktionen beim Anzeigen von Verbrauchern.

```csharp
public sealed class XpsGlyphs : XpsContentElement
```
