## XpsGlyphs.BidiLevel property

Gibt den Wert zurück/legt ihn fest, der die bidirektionale Verschachtelungsebene des Unicode-Algorithmus angibt. Gerade Werte implizieren ein Layout von links nach rechts, ungerade Werte implizieren ein Layout von rechts nach links. Ein Layout von rechts nach links platziert den Laufursprung auf der rechten Seite der ersten Glyphe, mit positiven Vorschubbreiten (die Vorschübe nach links darstellen), wodurch nachfolgende -Glyphen links von der vorherigen Glyphe platziert werden.

```csharp
public int BidiLevel { get; set; }
```
