## PdfSaveOptions.OutlineTreeExpansionLevel property

Gibt an, bis zu welcher Ebene die Dokumentgliederung erweitert werden soll, wenn die PDF-Datei in einem Viewer geöffnet wird. 1 - nur die Gliederungselemente der ersten Ebene werden angezeigt, 2 - nur die Gliederungselemente der ersten und zweiten Ebene werden angezeigt, und usw. Standard ist 1.

```csharp
public int OutlineTreeExpansionLevel { get; set; }
```
