## PdfSaveOptions.OutlineTreeHeight property

Gibt die Höhe der zu speichernden Dokumentgliederungsstruktur an. 0 - die Gliederungsstruktur wird nicht konvertiert, 1 - nur die Gliederungselemente der ersten Ebene werden konvertiert, usw. Standard ist 10.

public int OutlineTreeHeight { get; set; }
