Die **Aspose.Page.Xps.Präsentation.Pdf** Namespace bietet Klassen zum Rendern von XPS -Dokumenten in PDF-Dokumente.

## Klassen

| Klasse | Beschreibung |
| --- | --- |
| [PdfDevice](./pdfdevice/) | Klasse, die Bildkompositionsgerät einkapselt. |
| [PdfEncryptionDetails](./pdfencryptiondetails/) | Enthält Details für eine PDF-Verschlüsselung. |
| [PdfSaveOptions](./pdfsaveoptions/) | Klasse für XPS-als-PDF-Speicheroptionen. |
## Aufzählung

| Aufzählung | Beschreibung |
| --- | --- |
| [PdfEncryptionAlgorithm](./pdfencryptionalgorithm/) | Verschlüsselungsmodus enum. Beschreiben Sie anhand von Algorithmus und Schlüssellänge. Dieses Enum wird erweitert, um die Funktionalität weiter erhöhen zu können. Dieses Enum implementiert das "Base-to-Core"-Muster. |
| [PdfImageCompression](./pdfimagecompression/) | Gibt den Komprimierungstyp an, der auf Bilder in der PDF-Datei angewendet wird. |
| [PdfTextCompression](./pdftextcompression/) | Gibt einen Komprimierungstyp an, der auf alle Inhalte in der PDF-Datei mit Ausnahme von Bildern angewendet wird. |
