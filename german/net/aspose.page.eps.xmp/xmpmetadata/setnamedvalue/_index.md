## XmpMetadata.SetNamedValue method

Setzt einen benannten Wert in eine Struktur. Der vorherige benannte Wert wird, falls bereits vorhanden, durch einen neuen ersetzt.

```csharp
public void SetNamedValue(string structureKey, string valueKey, XmpValue value)
```

| Parameter | Typ | Beschreibung |
| --- | --- | --- |
| structureKey | String | Schlüssel der im Wörterbuch zu durchsuchenden Struktur. |
| valueKey | String | Name des Wertes, der in die Struktur gesetzt werden soll. |
| value | XmpValue | Wert, der in die Struktur gesetzt werden soll. |
