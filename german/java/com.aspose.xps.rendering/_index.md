Das **com.aspose.xps.rendering**‑Paket stellt Basisklassen für das Rendern von XPS‑Dokumenten in andere Formate bereit.


## Klassen

| Klasse | Beschreibung |
| --- | --- |
| [BmpSaveOptions](../com.aspose.xps.rendering/bmpsaveoptions) | Klasse für XPS-als-BMP-Speicheroptionen. |
| [ImageSaveOptions](../com.aspose.xps.rendering/imagesaveoptions) | Grundlegende Klasse für XPS-als-Bild-Speicheroptionen. |
| [JpegSaveOptions](../com.aspose.xps.rendering/jpegsaveoptions) | Klasse für XPS-als-JPEG-Speicheroptionen. |
| [PdfEncryptionDetails](../com.aspose.xps.rendering/pdfencryptiondetails) | Enthält Details für eine PDF-Verschlüsselung. |
| [PdfSaveOptions](../com.aspose.xps.rendering/pdfsaveoptions) | Klasse für XPS-als-PDF-Speicheroptionen. |
| [PngSaveOptions](../com.aspose.xps.rendering/pngsaveoptions) | Klasse für XPS-als-PNG-Speicheroptionen. |
| [TiffSaveOptions](../com.aspose.xps.rendering/tiffsaveoptions) | Klasse für XPS-als-TIFF-Speicheroptionen. |

## Schnittstellen

| Schnittstelle | Beschreibung |
| --- | --- |
| [IEventBasedModificationOptions](../com.aspose.xps.rendering/ieventbasedmodificationoptions) | Definiert Optionen, die für die Handhabung ereignisbasierter Änderungen beim Dokumentenspeichern relevant sind. |
| [IPipelineOptions](../com.aspose.xps.rendering/ipipelineoptions) | Definiert Konvertierungsoptionen im Zusammenhang mit der Pipeline-Konfiguration. |
| [IXpsTextConversionOptions](../com.aspose.xps.rendering/ixpstextconversionoptions) | Definiert Optionen für die Konvertierung von XPS in andere Formate. |

## Aufzählungen

| Aufzählung | Beschreibung |
| --- | --- |
| [InterpolationMode](../com.aspose.xps.rendering/interpolationmode) | Gibt den Algorithmus an, der verwendet wird, wenn Bilder skaliert oder rotiert werden. |
| [PdfEncryptionAlgorithm](../com.aspose.xps.rendering/pdfencryptionalgorithm) | Aufzählung der Verschlüsselungsmodi. |
| [PdfImageCompression](../com.aspose.xps.rendering/pdfimagecompression) | Gibt den Kompressionstyp an, der auf Bilder in der PDF-Datei angewendet wird. |
| [PdfTextCompression](../com.aspose.xps.rendering/pdftextcompression) | Gibt einen Kompressionstyp an, der auf alle Inhalte in der PDF-Datei außer Bildern angewendet wird. |
| [SmoothingMode](../com.aspose.xps.rendering/smoothingmode) | Gibt an, ob Glättung (Antialiasing) auf Linien und Kurven sowie die Kanten gefüllter Bereiche angewendet wird. |
| [TextRenderingHint](../com.aspose.xps.rendering/textrenderinghint) | Gibt die Qualität der Textdarstellung an. |
