Beschreibt die Duplex-Eigenschaften der Ausgabe. Das Duplex-Feature ermöglicht das Drucken auf beiden Seiten des Mediums. Jedes Dokument wird separat duplexiert. DocumentDuplex und JobDuplexAllDocumentsContiguously schließen sich gegenseitig aus. Es liegt am Treiber, die Einschränkungsbehandlung zwischen diesen Schlüsselwörtern zu bestimmen. https://docs.microsoft.com/en-us/windows/win32/printdocs/documentduplex
