Klasse, die den Print‑Ticket‑Parameter‑Initialisierer implementiert. Die Klasse, die einen allgemeinen PrintTicket‑Parameter‑Initialisierer implementiert. Die Basisklasse für alle schemadefinierte Parameter‑Initialisierer. Definiert einen Wert für eine Instanz eines ParameterDef‑Elements. Ein ParameterInit‑Element ist das Ziel der von einem ParameterRef‑Element gemachten Referenz. https://docs.microsoft.com/en-us/windows/win32/printdocs/parameterinit
