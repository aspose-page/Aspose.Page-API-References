```
public static final class PageScaling.ScaleOffsetAlignmentOption extends Option
```

Beschreibt die  ScaleOffsetAlignment  Funktionsoptionen. Gibt die Ausrichtung des skalierten Inhalts an.
## Felder

| Feld | Beschreibung |
| --- | --- |
| [BottomCenter](#BottomCenter) | Gibt an, dass die Skalierung mittig am unteren Rand des Mediums ausgerichtet sein soll. |
| [BottomLeft](#BottomLeft) | Gibt an, dass die Skalierung an der unteren linken Ecke des Mediums ausgerichtet sein soll. |
| [BottomRight](#BottomRight) | Gibt an, dass die Skalierung an der unteren rechten Ecke des Mediums ausgerichtet sein soll. |
| [Center](#Center) | Gibt an, dass die Skalierung auf dem Medium zentriert sein soll. |
| [LeftCenter](#LeftCenter) | Gibt an, dass die Skalierung in der Mitte der linken Kante des Mediums ausgerichtet sein soll. |
| [RightCenter](#RightCenter) | Gibt an, dass die Skalierung in der Mitte der rechten Kante des Mediums ausgerichtet sein soll. |
| [TopCenter](#TopCenter) | Gibt an, dass die Skalierung in der Mitte der oberen Kante des Mediums ausgerichtet sein soll. |
| [TopLeft](#TopLeft) | Gibt an, dass die Skalierung an der oberen linken Ecke des Mediums ausgerichtet sein soll. |
| [TopRight](#TopRight) | Gibt an, dass die Skalierung an der oberen rechten Ecke des Mediums ausgerichtet sein soll. |
