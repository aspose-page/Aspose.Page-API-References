Beschreibt die von Gerät unterstützten Sprachen zum Senden von Daten vom Treiber zum physischen Gerät. Dies wird oft \"Page Description Language\" genannt. Dieses Schlüsselwort definiert, welche Page Description Language vom Treiber und physischen Gerät unterstützt wird. https://docs.microsoft.com/en-us/windows/win32/printdocs/jobdevicelanguage
