Beschreibt die Ausgabe des Fehlerblatts. Der gesamte Auftrag wird ein einzelnes Fehlerblatt haben. Das Fehlerblatt sollte auf der Standard-  PageMediaSize  und mit dem Standard-  PageMediaType  ausgegeben werden. Das Fehlerblatt sollte vom Rest des Auftrags isoliert sein. Das bedeutet, dass keine Abschluss‑ oder Verarbeitungsoptionen (wie  JobDuplex ,  JobStaple , oder  JobBinding ) das Fehlerblatt einschließen dürfen. Das Fehlerblatt sollte als letztes Blatt des Auftrags erscheinen. https://docs.microsoft.com/en-us/windows/win32/printdocs/joberrorsheet
