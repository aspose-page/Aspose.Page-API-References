java.lang.Object, java.awt.geom.Dimension2D
```
public abstract class Size2D extends Dimension2D
```

Die `Size2D`-Klasse beschreibt die Abmessungen (w x h).

Diese Klasse ist lediglich die abstrakte Oberklasse für alle Objekte, die 2D-Abmessungen speichern. Die tatsächliche Speicherrepräsentation der Abmessungen wird der Unterklasse überlassen.
