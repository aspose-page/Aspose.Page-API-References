Das **com.aspose.eps** ist ein Root‑Paket für alle Klassen, die mit PS/EPS‑Dateien arbeiten.


## Klassen

| Klasse | Beschreibung |
| --- | --- |
| [FontConstants](../com.aspose.eps/fontconstants) | Diese Klasse definiert eine Menge von Konstanten zum Speichern von Schriftarten. |
| [HatchPaintLibrary](../com.aspose.eps/hatchpaintlibrary) | Konvertiert GDI+‑Schraffur‑Pinsel in Bilder, die zum Schreiben in XPS und PDF geeignet sind. |
| [LoadOptions](../com.aspose.eps/loadoptions) | Basisklasse für Dokument‑Ladeoptionen. |
| [PageConstants](../com.aspose.eps/pageconstants) | Diese Klasse definiert eine Menge von Konstanten, die eine Seite beschreiben. |
| [PsConverterException](../com.aspose.eps/psconverterexception) | Diese Klasse enthält Informationen über einen Fehler, der auftritt, während eine PS‑Datei in ein PDF‑Dokument konvertiert wird. |
| [PsConverterLicenseException](../com.aspose.eps/psconverterlicenseexception) | Diese Klasse enthält Informationen über einen Fehler im Zusammenhang mit der Lizenzierung des Aspose.Page‑Produkts. |
| [PsDocument](../com.aspose.eps/psdocument) | Diese Klasse kapselt PS/EPS-Dokumente. |
| [PsDocumentException](../com.aspose.eps/psdocumentexception) | Diese Klasse enthält Informationen über einen Fehler, der ausgelöst wird, während die PS-Datei erstellt, fertiggestellt und gespeichert wird. |
| [PsLoadOptions](../com.aspose.eps/psloadoptions) | PS-Dokument-Ladeoptionen. |
