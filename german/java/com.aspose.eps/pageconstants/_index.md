Diese Klasse definiert eine Menge von Konstanten, die eine Seite beschreiben. Komfortobjekte werden für verschiedene Ränder, Ausrichtungen, Neuskalierung und Standardseitengrößen bereitgestellt.
## Felder

| Feld | Beschreibung |
| --- | --- |
| [BACKGROUND](#BACKGROUND) | Hintergrundschlüssel |
| [BACKGROUND_COLOR](#BACKGROUND-COLOR) | Hintergrundfarbenschlüssel |
| [FIT_TO_PAGE](#FIT-TO-PAGE) | Schlüssel zum Anpassen des Inhalts an die Seite |
| [MARGINS_LARGE](#MARGINS-LARGE) | \"Large\" Seitenrandwert |
