Das **Aspose.Page** ist ein Root‑Paket für alle Klassen der Aspose.Page‑Bibliothek, die entweder direkt darin wie **Device** oder indirekt über mehrere Unterpakete enthalten sind.


## Klassen

| Klasse | Beschreibung |
| --- | --- |
| [AsposePageException](../com.aspose.page/asposepageexception) | Stellt Fehler dar, die während der Ausführung einer Aspose.Page-Anwendung auftreten. |
| [DimensionF](../com.aspose.page/dimensionf) | Die `Dimension`-Klasse kapselt die Breite und Höhe einer Komponente (in einfacher Genauigkeit) in einem einzigen Objekt. |
| [Document](../com.aspose.page/document) | Eine Superklasse für alle gekapselten Dokumente. |
| [ExternalFontCache](../com.aspose.page/externalfontcache) | Verwenden Sie diese Klasse, um die Schriftkapselung in einer Form zu erhalten, die von Device akzeptiert wird. |
| [License](../com.aspose.page/license) | Stellt Methoden zur Lizenzierung der Komponente bereit. |
| [Metered](../com.aspose.page/metered) | Stellt Methoden zum Festlegen des gemessenen Schlüssels bereit. |
| [SaveOptions](../com.aspose.page/saveoptions) | Diese Klasse enthält Optionen, die für die Verwaltung des Konvertierungsprozesses erforderlich sind. |
| [TransformedStroke](../com.aspose.page/transformedstroke) | Ein Strich, der eine Transformation enthält. |
| [UserProperties](../com.aspose.page/userproperties) | Spezielle Eigenschaftsklasse, die das Setzen und Zurückgeben typisierter Eigenschaften ermöglicht. |
