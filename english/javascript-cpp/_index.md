> Aspose.Page for JavaScript via C++ allows developers manipulate PS,EPS,XPS files directly in the Web.
>
> Such operations are very time consuming, so we recommend using Web Worker. 


## Convert functions

| Function | Description |
| -------- | ----------- |
| [AsposePSSaveAsImage](./convert/pssaveasimage/) | Convert a Postscript file to image. |
| [AsposePSSaveAsPdf](./convert/pssaveaspdf/) | Convert a Postscript file to PDF. |
| [AsposeXPSSaveAsImage](./convert/xpssaveasimage/) | Convert a XPS file to image. |
| [AsposeXPSSaveAsPdf](./convert/xpssaveaspdf/) | Convert a XPS file to PDF. |


## Merge functions

| Function | Description |
| -------- | ----------- |
| [AsposePSMergeToPdf](./merge/psmergetopdf/) | Merge a Postscript files to PDF. |
| [AsposeXPSMergeToPdf](./merge/xpsmergetopdf/) | Merge a XPS files to PDF. |
| [AsposeXPSMergeToXps](./merge/xpsmergetopdf/) | Merge a several XPS files to one XPS. |

| Function | Description |

| [AsposeXMPAddArrayItem](./xmp/xmpaddarrayitem/) | Merge a Postscript files to PDF. |
| [AsposeXMPAddNamedValue](./xmp/xmpaddnamedvalue/) | Merge a Postscript files to PDF. |
| [AsposeXMPAddNamespace](./xmp/xmpaddnamespace/) | Merge a Postscript files to PDF. |
| [AsposeXMPAddSimpleProperties](./xmp/xmpaddsimpleproperties/) | Merge a Postscript files to PDF. |
