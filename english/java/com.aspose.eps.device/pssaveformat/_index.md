public enum PsSaveFormat extends Enum<PsSaveFormat>

This enumeration contains available options of saving format. It can be PS or EPS. EPS is used for only 1-paged documents while PS file can contain any number of pages.
## Fields

| Field | Description |
| --- | --- |
| [EPS](#EPS) | This option indicates that resulting document must be Encapsulated PostScript (EPS) file. |
| [PS](#PS) | This option indicates that resulting document must be PostScript (PS) or Encapsulated PostScript (EPS) file. |
