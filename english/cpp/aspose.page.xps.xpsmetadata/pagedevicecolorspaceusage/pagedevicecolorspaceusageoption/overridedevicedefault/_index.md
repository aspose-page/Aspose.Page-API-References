## OverrideDeviceDefault field


If the profile specified by the [PageDeviceColorSpaceProfileURI](../../../pagedevicecolorspaceprofileuri/) parameter has a number of channels matching the number of primaries of the device, it SHOULD be used instead of the devices internal color management for all elements. Elements using this profile are assumed to be in device color space and will not be color managed further.
