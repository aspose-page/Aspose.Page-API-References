## PageDeviceColorSpaceProfileURI class


Specifies a relative URI to the package root to an ICC profile contained in an [XPS](../../aspose.page.xps/)[Document](../../aspose.page/document/). The processing of this option depends of the setting of the [PageDeviceColorSpaceUsage](../pagedevicecolorspaceusage/) feature. All elements using that profile are assumed to be already in the appropriate device color space, and will not be color managed in the driver or device. [https://docs.microsoft.com/en-us/windows/win32/printdocs/pagedevicecolorspaceprofileuri](https://docs.microsoft.com/en-us/windows/win32/printdocs/pagedevicecolorspaceprofileuri).
