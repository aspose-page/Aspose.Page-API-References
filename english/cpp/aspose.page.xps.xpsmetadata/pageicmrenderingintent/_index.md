## PageICMRenderingIntent class


Describes the rendering intent as defined by the ICC v2 Specification. This value should be ignored if an image or graphical element has an embedded profile that specifies the **Rendering** intent. [https://docs.microsoft.com/en-us/windows/win32/printdocs/pageicmrenderingintent](https://docs.microsoft.com/en-us/windows/win32/printdocs/pageicmrenderingintent).
