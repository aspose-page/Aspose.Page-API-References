## Font.Equals method

Indicates whether the specified object is a [`Font`](../) and has the same FontFamily, GdiVerticalFont, GdiCharSet, [`Style`](../style/), [`Size`](../size/), and Unit property values as this [`Font`](../).

```csharp
public override bool Equals(object obj)
```

| Parameter | Type | Description |
| --- | --- | --- |
| obj | Object | The object to test. |

### Return Value

true if the *obj* parameter is a [`Font`](../) and has the same FontFamily, GdiVerticalFont, GdiCharSet, [`Style`](../style/), [`Size`](../size/), and Unit property values as this [`Font`](../); otherwise, false.
