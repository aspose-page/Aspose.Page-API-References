## JobOutputBin class

Describes the installed output bin in a device or the full list of supported bins for a device. The `JobOutputBin`, [`DocumentOutputBin`](../documentoutputbin/) and [`PageOutputBin`](../pageoutputbin/) keywords are mutually exclusive only one should be specified in a PrintTicket or Print Capabilities document. https://docs.microsoft.com/en-us/windows/win32/printdocs/joboutputbin
