## DocumentPageRanges class

Describes the output range of the document in pages. The parameter value must conform to the following structure: - PageRangeText: "PageRange" or "PageRange,PageRange" - PageRange: "PageNumber" or "PageNumber-PageNumber" - PageNumber: 1 to N, where N is the number of pages in the document.If PageNumber &gt; N, then PageNumber = N. Whitespace in the string should be ignored. https://docs.microsoft.com/en-us/windows/win32/printdocs/documentpageranges
