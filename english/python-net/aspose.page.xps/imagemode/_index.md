## ImageMode enumeration

Lists the options of fitting image within a box.

## Members
| Member name | Description |
| :- | :- |
| `FIT_TO_WIDTH` | The image fits to width. |
| `FIT_TO_HEIGHT` | The image fits to height. |
| `FIT_TO_BOX` | The image fits to box. |
