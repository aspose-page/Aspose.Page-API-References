## JobOutputBin class

Describes the installed output bin in a device or the full list of supported bins for a device.<br/>            The [JobOutputBin](/page/python-net/aspose.page.xps.xpsmetadata/joboutputbin/), [DocumentOutputBin](/page/python-net/aspose.page.xps.xpsmetadata/documentoutputbin/) and [PageOutputBin](/page/python-net/aspose.page.xps.xpsmetadata/pageoutputbin/) keywords<br/>            are mutually exclusive only one should be specified in a PrintTicket or Print Capabilities document.<br/>            https://docs.microsoft.com/en-us/windows/win32/printdocs/joboutputbin
