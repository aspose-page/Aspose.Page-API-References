## PageWatermark class

Describes the watermark setting of the output and the watermark characteristics. Watermarks apply<br/>            to the logical page, not the physical page. For example, if [DocumentDuplex](/page/python-net/aspose.page.xps.xpsmetadata/documentduplex/) is enabled,<br/>            a watermark will appear on each
