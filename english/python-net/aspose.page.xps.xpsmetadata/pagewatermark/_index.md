## PageWatermark class

Describes the watermark setting of the output and the watermark characteristics. Watermarks apply
to the logical page, not the physical page. For example, if [`DocumentDuplex`](/page/python-net/aspose.page.xps.xpsmetadata/documentduplex) is enabled,
a watermark will appear on each `NUp` page on each sheet. If [`DocumentDuplex`](/page/python-net/aspose.page.xps.xpsmetadata/documentduplex),
`PagesPerSheet`=2, then each sheet will have 2 watermarks.
https://docs.microsoft.com/en-us/windows/win32/printdocs/pagewatermark
