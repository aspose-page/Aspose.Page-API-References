## DocumentOutputBin class

Describes the full list of supported bins for the device. Allows specification of output
bin on a per document basis. The [`JobOutputBin`](/page/python-net/aspose.page.xps.xpsmetadata/joboutputbin), [`DocumentOutputBin`](/page/python-net/aspose.page.xps.xpsmetadata/documentoutputbin) and
[`PageOutputBin`](/page/python-net/aspose.page.xps.xpsmetadata/pageoutputbin) keywords are mutually exclusive only one should be specified in
a PrintTicket or Print Capabilities document.
https://docs.microsoft.com/en-us/windows/win32/printdocs/documentoutputbin
