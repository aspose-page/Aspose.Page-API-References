## DocumentCollate class

Describes the collating characteristics of the output. All pages in each individual document are collated.<br/>            DocumentCollate and JobCollateAlldocuments are mutually exclusive. The behavior and implementation of whether<br/>            both or only one of these keywords is implemented is left to the driver.<br/>            https://docs.microsoft.com/en-us/windows/win32/printdocs/documentcollate
