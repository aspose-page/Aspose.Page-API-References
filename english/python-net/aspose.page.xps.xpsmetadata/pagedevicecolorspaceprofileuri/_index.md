## PageDeviceColorSpaceProfileURI class

Specifies a relative URI to the package root to an ICC profile contained in an XPS Document.<br/>            The processing of this option depends of the setting of the PageDeviceColorSpaceUsage feature.<br/>            All elements using that profile are assumed to be already in the appropriate device color space,<br/>            and will not be color managed in the driver or device.<br/>            https://docs.microsoft.com/en-us/windows/win32/printdocs/pagedevicecolorspaceprofileuri
