## PageInputBin class

Describes the installed input bin in a device or the full list of supported bins for a device.
Allows specification of input bin on a per page basis. The [`JobInputBin`](/page/python-net/aspose.page.xps.xpsmetadata/jobinputbin), [`DocumentInputBin`](/page/python-net/aspose.page.xps.xpsmetadata/documentinputbin) and
[`PageInputBin`](/page/python-net/aspose.page.xps.xpsmetadata/pageinputbin) keywords are mutually exclusive. Both should not be specified simultaneously
in a PrintTicket or Print Capabilities document.
https://docs.microsoft.com/en-us/windows/win32/printdocs/pageinputbin
