## ParameterInit class

The class that implements a common PrintTicket parameter initializer.
The base class for all schema-defined parameter initializers.
Defines a value for an instance of a `ParameterDef` element.
A `ParameterInit` element is the target of the reference made by a [`ParameterRef`](/page/python-net/aspose.page.xps.xpsmetadata/parameterref) element.
https://docs.microsoft.com/en-us/windows/win32/printdocs/parameterinit
