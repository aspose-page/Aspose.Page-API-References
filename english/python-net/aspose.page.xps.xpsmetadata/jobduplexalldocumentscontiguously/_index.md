## JobDuplexAllDocumentsContiguously class

Describes the duplex characteristics of the output. The duplex feature allows for printing on<br/>            both sides of the media. All Documents in the job are duplexed together contiguously.<br/>            [JobDuplexAllDocumentsContiguously](/page/python-net/aspose.page.xps.xpsmetadata/jobduplexalldocumentscontiguously/) and [DocumentDuplex](/page/python-net/aspose.page.xps.xpsmetadata/documentduplex/) are mutually exclusive.<br/>            It is up to the driver to determine constraint handling between these keywords.<br/>            https://docs.microsoft.com/en-us/windows/win32/printdocs/jobduplexalldocumentscontiguously
