## Value class

The base class that incapsulates a [`Property`](/page/python-net/aspose.page.xps.xpsmetadata/property) or a [`ScoredProperty`](/page/python-net/aspose.page.xps.xpsmetadata/scoredproperty)
value in a PrintTicket document.
A Value element associates a literal with a type.
