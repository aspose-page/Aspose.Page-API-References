## ScoredProperty class

The class that implements a common PrintTicket `ScoredProperty`.
The base class for all schema-defined scored properties.
A `ScoredProperty` element declares a property that is intrinsic to an
[`Option`](/page/python-net/aspose.page.xps.xpsmetadata/option) definition. Such properties should be compared when evaluating
how closely a requested [`Option`](/page/python-net/aspose.page.xps.xpsmetadata/option) matches a device-supported [`Option`](/page/python-net/aspose.page.xps.xpsmetadata/option).
https://docs.microsoft.com/en-us/windows/win32/printdocs/scoredproperty
