## Property class

The class that implements a common PrintTicket `Property`.
The base class for all schema-defined properties.
A `Property` element declares a device, job formatting, or other relevant property
whose name is given by its name attribute. A [`Value`](/page/python-net/aspose.page.xps.xpsmetadata/value) element is used to assign
a value to the `Property`.
A `Property` can be complex, possibly containing multiple subproperties.
Subproperties are also represented by `Property` elements.
https://docs.microsoft.com/en-us/windows/win32/printdocs/property
