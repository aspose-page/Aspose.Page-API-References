## JobPrimaryBannerSheet class

Describes the banner sheet to be output for the job. The banner sheet should be output on the default<br/>            [PageMediaSize](/page/python-net/aspose.page.xps.xpsmetadata/pagemediasize/) and using the default [PageMediaType](/page/python-net/aspose.page.xps.xpsmetadata/pagemediatype/). The banner sheet should<br/>            be isolated from the remainder of the job. This means that any finishing or processing options (such as<br/>            [JobDuplexAllDocumentsContiguously](/page/python-net/aspose.page.xps.xpsmetadata/jobduplexalldocumentscontiguously/), [JobStapleAllDocuments](/page/python-net/aspose.page.xps.xpsmetadata/jobstaplealldocuments/), or [JobBindAllDocuments](/page/python-net/aspose.page.xps.xpsmetadata/jobbindalldocuments/))<br/>            should not include the banner sheet. The banner sheet should occur as the first sheet of the job.
