## PageDeviceColorSpaceUsage class

In conjunction with the [`PageDeviceColorSpaceProfileURI`](/page/python-net/aspose.page.xps.xpsmetadata/pagedevicecolorspaceprofileuri) parameter, this parameter defines
the rendering behavior for elements presented in a device color space.
https://docs.microsoft.com/en-us/windows/win32/printdocs/pagedevicecolorspaceusage
