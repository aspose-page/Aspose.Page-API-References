## XpsTransformableBrush class

Class incapsulating common features of transformable brushes elements (all except SolidColorBrush).

**Inheritance:** `XpsTransformableBrush` → [`XpsBrush`](/page/python-net/aspose.page.xps.xpsmodel/xpsbrush) → [`XpsObject`](/page/python-net/aspose.page.xps.xpsmodel/xpsobject)
