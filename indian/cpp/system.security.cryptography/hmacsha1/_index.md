## HMACSHA1 class


हैश-आधारित संदेश [Authentication](../../system.security.authentication/) कोड जो [SHA1](../sha1/) हैश फ़ंक्शन का उपयोग करता है। लागू नहीं किया गया है। इस क्लास की वस्तुओं को केवल [System::MakeObject()](../../system/makeobject/) फ़ंक्शन का उपयोग करके आवंटित किया जाना चाहिए। इस प्रकार का उदाहरण स्टैक पर या operator new का उपयोग करके कभी न बनाएँ, क्योंकि इससे रनटाइम त्रुटियाँ और/या असर्शन त्रुटियाँ उत्पन्न होंगी। हमेशा इस क्लास को [System::SmartPtr](../../system/smartptr/) पॉइंटर में लपेटें और इस पॉइंटर का उपयोग करके इसे फ़ंक्शनों में तर्क के रूप में पास करें।

```cpp
class HMACSHA1 : public System::Security::Cryptography::HashAlgorithm
```
