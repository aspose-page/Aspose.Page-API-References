## TraceLevel enum


[System.Diagnostics.Debug](../debug/), [System.Diagnostics.Trace](../trace/) और System.Diagnostics.TraceSwitch क्लासों के लिए कौन से संदेश आउटपुट करने हैं, यह निर्दिष्ट करता है।

```cpp
enum class TraceLevel
```

### मान

| नाम | मान | विवरण |
| --- | --- | --- |
| Off | 0 | कोई ट्रेसिंग और डिबगिंग संदेश आउटपुट नहीं करता। |
| Error | 1 | त्रुटि-हैंडलिंग संदेश आउटपुट करता है। |
| Warning | 2 | चेतावनियों और त्रुटि-हैंडलिंग संदेशों को आउटपुट करता है। |
| Info | 3 | सूचनात्मक संदेश, चेतावनियाँ, और त्रुटि-प्रबंधन संदेश आउटपुट करें। |
| विस्तृत | 4 | सभी डिबगिंग और ट्रेसिंग संदेश आउटपुट करें। |
