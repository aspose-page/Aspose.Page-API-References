## XmlConvert::ToString(DateTimeOffset, const String\&) method

```cpp

```


| पैरामीटर | प्रकार | विवरण |
| --- | --- | --- |
| value | DateTimeOffset | परिवर्तित किए जाने वाले [DateTimeOffset](../../../system/datetimeoffset/)। |
