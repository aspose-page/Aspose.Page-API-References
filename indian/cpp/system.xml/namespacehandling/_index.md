## NamespaceHandling enum


डुप्लिकेट नेमस्पेस डिक्लेरेशन्स को [XmlWriter](../xmlwriter/) में हटाना है या नहीं, यह निर्दिष्ट करता है।

```cpp
enum class NamespaceHandling
```

### मान

| नाम | मान | विवरण |
| --- | --- | --- |
| Default | 0 | निर्दिष्ट करता है कि डुप्लिकेट नेमस्पेस डिक्लेरेशन्स को नहीं हटाया जाएगा। |
| OmitDuplicates | 1 | निर्दिष्ट करता है कि डुप्लिकेट नेमस्पेस डिक्लेरेशन्स को हटाया जाएगा। डुप्लिकेट नेमस्पेस को हटाने के लिए, प्रीफ़िक्स और नेमस्पेस मेल खाने चाहिए। |
