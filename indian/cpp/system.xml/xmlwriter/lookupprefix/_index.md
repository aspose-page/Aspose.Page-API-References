## XmlWriter::LookupPrefix method

```cpp
virtual String System::Xml::XmlWriter::LookupPrefix(String ns)=0
```


| पैरामीटर | प्रकार | विवरण |
| --- | --- | --- |
| ns | String | जिस namespace URI का प्रीफ़िक्स आप खोजना चाहते हैं। |

### ReturnValue

यदि वर्तमान स्कोप में मिलते-जुलते नेमस्पेस URI नहीं मिलता है तो मिलते-जुलते प्रीफ़िक्स या **nullptr**।
