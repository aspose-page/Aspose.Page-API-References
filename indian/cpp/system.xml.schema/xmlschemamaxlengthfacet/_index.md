## XmlSchemaMaxLengthFacet class


XML [Schema](../) से **maxLength** तत्व का प्रतिनिधित्व करता है जैसा कि World Wide [Web](../../system.web/) Consortium (W3C) द्वारा निर्दिष्ट है। इस क्लास का उपयोग **simpleType** तत्व के डेटा मान की अधिकतम लंबाई पर प्रतिबंध निर्दिष्ट करने के लिए किया जा सकता है। लंबाई **maxLength** तत्व के मान से कम होनी चाहिए।

```cpp
class XmlSchemaMaxLengthFacet : public System::Xml::Schema::XmlSchemaNumericFacet
```
