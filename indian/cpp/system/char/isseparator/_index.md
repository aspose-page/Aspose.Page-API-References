```cpp
static bool System::Char::IsSeparator(char_t c)
```


| पैरामीटर | प्रकार | विवरण |
| --- | --- | --- |

### ReturnValue

True यदि निर्दिष्ट अक्षर एक विभाजक अक्षर है, अन्यथा - false

## Char::IsSeparator(const char_t *, int) method

```cpp
static bool System::Char::IsSeparator(const char_t *str, int idx)
```


| पैरामीटर | प्रकार | विवरण |
| --- | --- | --- |
| str | const char_t * | अक्षर बफ़र की शुरुआत की ओर संकेत करने वाला पॉइंटर |
| idx | int | परीक्षण के अक्षर के निर्दिष्ट बफ़र में शून्य-आधारित सूचकांक |

### ReturnValue

True यदि निर्दिष्ट इंडेक्स पर अक्षर एक विभाजक अक्षर है, अन्यथा - false
