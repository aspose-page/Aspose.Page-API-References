## Array::rbegin() const method


रिवर्स इटरेटर को रिवर्स्ड कंटेनर के पहले तत्व के लिए लौटाता है। यह नॉन-रिवर्स्ड कंटेनर के अंतिम तत्व के अनुरूप है। यदि कंटेनर खाली है, तो लौटाया गया इटरेटर [rend()](../rend/) के बराबर होता है।

```cpp
const_reverse_iterator System::Array<T>::rbegin() const noexcept
```


### ReturnValue

एक इटरेटर जो const-योग्य कंटेनर के अंतिम तत्व की ओर इंगित करता है।
