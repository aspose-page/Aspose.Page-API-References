## DateTime::Compare method


निर्दिष्ट [DateTime](../) क्लास के इंस्टेंस द्वारा दर्शाए गए दो मानों की तुलना करता है और समयरेखा पर मानों की सापेक्ष स्थितियों को दर्शाने वाला मान लौटाता है।

```cpp
static constexpr int System::DateTime::Compare(DateTime t1, DateTime t2)
```


| पैरामीटर | प्रकार | विवरण |
| --- | --- | --- |
| t1 | DateTime | पहला तुलनात्मक मान |
| t2 | DateTime | दूसरा तुलनात्मक मान |

### ReturnValue

एक मान जो 0 से कम होता है यदि **t1** **t2** से पहले है; 0 यदि **t1** **t2** के समान है; एक मान जो 0 से बड़ा होता है यदि **t1** **t2** के बाद है।
