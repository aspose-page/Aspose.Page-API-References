## TransitionTime::CreateTransitionTime method


एक [TransitionTime](../) क्लास का उदाहरण बनाता है जो निर्दिष्ट पैरामीटरों के साथ वर्णित समय परिवर्तन को दर्शाता है।

```cpp
static TransitionTime System::TimeZoneInfo::TransitionTime::CreateTransitionTime(DateTime time_of_day, int month, int week, int day, DayOfWeek day_of_week, bool is_fixed_date_rule)
```


| पैरामीटर | प्रकार | विवरण |
| --- | --- | --- |
| time_of_day | DateTime | समय परिवर्तन के होने का विशिष्ट समय। |
| महीना | int | समय परिवर्तन के होने का वर्ष का महीना। |
| week | int | समय परिवर्तन के होने का महीने का सप्ताह। |
| दिन | int | समय परिवर्तन होने वाला दिन। |
| day_of_week | DayOfWeek | समय परिवर्तन के होने का सप्ताह का दिन। |
| is_fixed_date_rule | bool | मान जो दर्शाता है कि समय परिवर्तन एक निश्चित तिथि और समय पर होता है या एक परिवर्तनीय तिथि और समय पर। |
