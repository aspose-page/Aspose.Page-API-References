## System::MakeSharedPtr(const X *) method


रॉ पॉइंटर को स्मार्ट पॉइंटर में बदलता है। कॉन्स्ट पॉइंटर के लिए ओवरलोड। उदाहरण के लिए, C# मेथड्स में 'this' वेरिएबल का उपयोग करने पर, जो कॉन्स्ट के रूप में अनुवादित होते हैं, यह उपयोगी है।

```cpp
template<class X> SmartPtr<X> System::MakeSharedPtr(const X *p)
```


| पैरामीटर | विवरण |
| --- | --- |
| X | पॉइंटी प्रकार। |

| पैरामीटर | प्रकार | विवरण |
| --- | --- | --- |
| p | const X * | ऑब्जेक्ट का रॉ पॉइंटर। |

### ReturnValue

ऑब्जेक्ट का साझा स्मार्ट पॉइंटर।
