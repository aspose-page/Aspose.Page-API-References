## GC class


एक अनुकरणीय गैरेज कलेक्शन का प्रतिनिधित्व करता है जो अधिकतर एक स्टब की तरह कार्य करता है और प्रभावी रूप से कुछ नहीं करता। यह एक स्थैतिक प्रकार है जिसमें कोई इंस्टेंस सेवाएँ नहीं हैं। आपको इसे किसी भी माध्यम से इंस्टेंस नहीं बनाना चाहिए।

```cpp
class GC : public System::Object
```
