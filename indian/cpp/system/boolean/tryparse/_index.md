## Boolean::TryParse method


निर्दिष्ट स्ट्रिंग को बूल प्रकार के मान में परिवर्तित करता है।

```cpp
static bool System::Boolean::TryParse(const String &value, bool &result)
```


| पैरामीटर | प्रकार | विवरण |
| --- | --- | --- |
| मान | const String\& | परिवर्तित करने के लिए स्ट्रिंग। |
| परिणाम | bool\& | परिणाम को रखने वाले bool वेरिएबल का रेफ़रेंस; परिणाम true है यदि निर्दिष्ट स्ट्रिंग \"True\" के बराबर है और false है यदि स्ट्रिंग \"False\" के बराबर है। |

### ReturnValue

यदि निर्दिष्ट स्ट्रिंग \"True\" या \"False\" में से किसी एक के बराबर है तो true, अन्यथा false।
