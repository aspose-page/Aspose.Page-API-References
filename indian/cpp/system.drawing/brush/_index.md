## Brush class


ग्राफ़िकल आकारों के अंदरूनी भाग को भरने के लिए उपयोग किए जाने वाले फ़िलर को दर्शाने वाली क्लासों के लिए एक बेस क्लास है। इस क्लास की वस्तुओं को केवल [System::MakeObject()](../../system/makeobject/) फ़ंक्शन का उपयोग करके आवंटित किया जाना चाहिए। इस प्रकार का इंस्टेंस स्टैक पर या operator new का उपयोग करके कभी न बनाएँ, क्योंकि इससे रनटाइम त्रुटियाँ और/या असर्शन फॉल्ट्स हो सकते हैं। हमेशा इस क्लास को [System::SmartPtr](../../system/smartptr/) पॉइंटर में लपेटें और इस पॉइंटर का उपयोग करके इसे फ़ंक्शनों को आर्ग्यूमेंट के रूप में पास करें।

```cpp
class Brush : public System::ICloneable,
              public System::IDisposable
```
