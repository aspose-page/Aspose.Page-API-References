## PointF::Subtract(const PointF\&, const Size\&) method


निर्दिष्ट [Size](../../size/) ऑब्जेक्ट की चौड़ाई और ऊँचाई मानों को क्रमशः निर्दिष्ट [PointF](../) ऑब्जेक्ट के X और Y निर्देशांक मानों से घटाता है।

```cpp
static PointF System::Drawing::PointF::Subtract(const PointF &point, const Size &size)
```


| पैरामीटर | प्रकार | विवरण |
| --- | --- | --- |
| बिंदु | const PointF\& | अनुवाद करने के लिए बिंदु |
| size | const Size\& | वह [Size](../../size/) ऑब्जेक्ट जो **point** के निर्देशांक मानों से घटाने के लिए मान निर्दिष्ट करता है |

### ReturnValue

एक नया [PointF](../) ऑब्जेक्ट जिसका X निर्देशांक मान **size** की चौड़ाई मान को **point** के X निर्देशांक मान से घटाने के परिणाम के बराबर होता है और Y निर्देशांक मान **size** की ऊँचाई मान को **point** के Y निर्देशांक मान से घटाने के परिणाम के बराबर होता है।
