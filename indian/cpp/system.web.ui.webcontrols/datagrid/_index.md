## DataGrid class


डमी क्लास जो अनुवादित कोड को [DataGrid](./) संदर्भों से लिंक करना संभव बनाता है, लेकिन इसे निष्पादित नहीं करता। इसमें कोई सही तरीके से लागू किए गए सदस्य नहीं हैं। इस क्लास की वस्तुओं को केवल [System::MakeObject()](../../system/makeobject/) फ़ंक्शन का उपयोग करके आवंटित किया जाना चाहिए। इस प्रकार को स्टैक पर या operator new का उपयोग करके कभी भी इंस्टेंस न बनाएँ, क्योंकि इससे रनटाइम त्रुटियाँ और/या असर्शन त्रुटियाँ उत्पन्न होंगी। हमेशा इस क्लास को [System::SmartPtr](../../system/smartptr/) पॉइंटर में लपेटें और इस पॉइंटर का उपयोग करके इसे फ़ंक्शनों को तर्क के रूप में पास करें।

```cpp
class DataGrid : public System::Object
```
