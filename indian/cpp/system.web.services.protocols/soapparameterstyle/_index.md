## SoapParameterStyle enum

```cpp
enum class SoapParameterStyle
```

### मान

| नाम | मान | विवरण |
| --- | --- | --- |
| Default | 0 | यदि '[SoapDocumentServiceAttribute](../soapdocumentserviceattribute/)' क्लास पर लागू नहीं किया गया है, तो डिफ़ॉल्ट मान 'Wrapped' है। |

| Wrapped | 2 | पैरामीटर 'Body' तत्व के बाद आने वाले एकल XML तत्व के भीतर संलग्न होते हैं। |
