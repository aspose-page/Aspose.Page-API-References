## WrapMode enum


निर्दिष्ट करता है कि जब टेक्सचर या ग्रेडिएंट भरे जाने वाले क्षेत्र से छोटा हो तो उसे कैसे टाइल किया जाता है।

```cpp
enum class WrapMode
```

### मान

| नाम | मान | विवरण |
| --- | --- | --- |
| Tile | 0 | ग्रेडिएंट या टेक्सचर को टाइल करता है। |
| TileFlipX | 1 | टेक्सचर या ग्रेडिएंट को क्षैतिज रूप से उलटता है और फिर उसे टाइल करता है। |
| TileFlipY | 2 | टेक्सचर या ग्रेडिएंट को लंबवत रूप से उलटता है और फिर टेक्सचर या ग्रेडिएंट को टाइल करता है। |
| TileFlipXY | 3 | टेक्सचर या ग्रेडिएंट को क्षैतिज और लंबवत दोनों दिशा में उलटता है और फिर टेक्सचर या ग्रेडिएंट को टाइल करता है। |
| Clamp | 4 | टेक्सचर या ग्रेडिएंट को ऑब्जेक्ट की सीमा तक क्लैम्प करता है। |
