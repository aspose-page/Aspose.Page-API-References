## PageDeviceColorSpaceProfileURI class


एक ICC प्रोफ़ाइल के लिए पैकेज रूट से सापेक्ष URI निर्दिष्ट करता है जो एक [XPS](../../aspose.page.xps/)[Document](../../aspose.page/document/) में सम्मिलित है। इस विकल्प की प्रोसेसिंग [PageDeviceColorSpaceUsage](../pagedevicecolorspaceusage/) फीचर की सेटिंग पर निर्भर करती है। इस प्रोफ़ाइल का उपयोग करने वाले सभी तत्वों को मान लिया जाता है कि वे पहले से ही उपयुक्त डिवाइस कलर स्पेस में हैं, और ड्राइवर या डिवाइस में रंग प्रबंधन नहीं किया जाएगा। [https://docs.microsoft.com/en-us/windows/win32/printdocs/pagedevicecolorspaceprofileuri](https://docs.microsoft.com/en-us/windows/win32/printdocs/pagedevicecolorspaceprofileuri).

```cpp
class PageDeviceColorSpaceProfileURI : public Aspose::Page::XPS::XpsMetadata::StringParameterInit,

                                       public Aspose::Page::XPS::XpsMetadata::IDocumentPrintTicketItem,

```
