## ParameterRef class


एक सामान्य [PrintTicket](../printticket/) पैरामीटर रेफ़रेंस को लागू करने वाली क्लास। एक **[ParameterRef](./)** तत्व एक [ParameterInit](../parameterinit/) तत्व के लिए रेफ़रेंस परिभाषित करता है। एक [ScoredProperty](../scoredproperty/) तत्व जिसमें एक [ParameterRef](./) तत्व होता है, उसमें स्पष्ट रूप से सेट किया गया [Value](../value/) तत्व नहीं होता। इसके बजाय, [ScoredProperty](../scoredproperty/) तत्व अपना मान उस [ParameterInit](../parameterinit/) तत्व से प्राप्त करता है जो **[ParameterRef](./)** तत्व द्वारा संदर्भित है। [https://docs.microsoft.com/en-us/windows/win32/printdocs/parameterref](https://docs.microsoft.com/en-us/windows/win32/printdocs/parameterref).

```cpp
class ParameterRef : public Aspose::Page::XPS::XpsMetadata::PrintTicketElement,
                     public Aspose::Page::XPS::XpsMetadata::IPrintTicketItem
```
