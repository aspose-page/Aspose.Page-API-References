## XmlAttributeOverrides class


जब [XmlSerializer](../xmlserializer/) का उपयोग किसी ऑब्जेक्ट को सीरियलाइज़ या डीसीरियलाइज़ करने के लिए किया जाता है, तो यह एट्रिब्यूट्स को ओवरराइड करने की अनुमति देता है। इस क्लास की ऑब्जेक्ट्स को केवल [System::MakeObject()](../../system/makeobject/) फ़ंक्शन का उपयोग करके ही आवंटित किया जाना चाहिए। इस प्रकार का इंस्टेंस स्टैक पर या operator new का उपयोग करके कभी न बनाएँ, क्योंकि इससे रनटाइम त्रुटियाँ और/या असर्शन फॉल्ट्स हो सकते हैं। हमेशा इस क्लास को [System::SmartPtr](../../system/smartptr/) पॉइंटर में लपेटें और इस पॉइंटर का उपयोग फ़ंक्शनों में आर्ग्यूमेंट के रूप में पास करने के लिए करें।

```cpp
class XmlAttributeOverrides : public System::Object
```
