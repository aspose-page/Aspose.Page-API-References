## Units enumeration

आकार के प्रकार को निर्दिष्ट करता है।

```js
enum Units
```

### मान

| नाम | मान | विवरण |
| ---        | ---   | --- |
| Points | `0` | पॉइंट्स (इंच का 1/72)। |
| Inches | `1` | इंच। |
| Millimeters | `2` | मिलीमीटर। |
| Centimeters | `3` | सेंटीमीटर। |
| Percents | `4` | मौजूदा आकार के प्रतिशत। |
