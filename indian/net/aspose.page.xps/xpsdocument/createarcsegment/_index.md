## XpsDocument.CreateArcSegment method

एक नया अण्डाकार चाप खंड बनाता है।

```csharp
public XpsArcSegment CreateArcSegment(PointF point, SizeF size, float rotationAngle, 
    bool isLargeArc, XpsSweepDirection sweepDirection, bool isStroked = true)
```

| पैरामीटर | प्रकार | विवरण |
| --- | --- | --- |
| point | PointF | अण्डाकार चाप का समापन बिंदु। |
| size | SizeF | अण्डाकार चाप की x और y त्रिज्या एक x, y जोड़ी के रूप में। |
| rotationAngle | Single | इंगित करता है कि दीर्घवृत्त वर्तमान समन्वय प्रणाली के सापेक्ष कैसे घूमता है। |
| isLargeArc | Boolean | यह निर्धारित करता है कि चाप 180 या उससे अधिक के स्वीप के साथ खींचा गया है या नहीं। |
| sweepDirection | XpsSweepDirection | जिस दिशा में चाप खींचा जाता है। |
| isStroked | Boolean | निर्दिष्ट करता है कि पथ के इस खंड के लिए स्ट्रोक खींचा गया है या नहीं। |
