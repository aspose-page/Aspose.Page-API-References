## SaveOptions.JpegQualityLevel property

गुणवत्ता श्रेणी एक छवि के लिए संपीड़न के स्तर को निर्दिष्ट करती है। उपलब्ध मान 0 से 100 हैं। निर्दिष्ट संख्या जितनी कम होगी, संपीड़न उतना ही अधिक होगा और इसलिए छवि की गुणवत्ता कम होगी। 0 मान सबसे कम गुणवत्ता वाली छवि देता है, जबकि 100 मान उच्चतम गुणवत्ता देता है.

public int JpegQualityLevel { get; set; }
