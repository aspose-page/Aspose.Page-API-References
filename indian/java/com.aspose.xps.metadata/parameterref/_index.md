एक सामान्य PrintTicket पैरामीटर रेफ़रेंस को लागू करने वाली क्लास। एक ParameterRef तत्व ParameterInit तत्व के लिए एक रेफ़रेंस परिभाषित करता है। एक ScoredProperty तत्व जिसमें ParameterRef तत्व होता है, उसमें स्पष्ट रूप से सेट किया गया Value तत्व नहीं होता। इसके बजाय, ScoredProperty तत्व अपना मान ParameterRef तत्व द्वारा संदर्भित ParameterInit तत्व से प्राप्त करता है। https://docs.microsoft.com/en-us/windows/win32/printdocs/parameterref
