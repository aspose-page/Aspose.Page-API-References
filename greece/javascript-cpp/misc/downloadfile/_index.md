_Δημιουργήστε έναν σύνδεσμο σε HTML για λήψη του αρχείου._

```js
function DownloadFile(
    filename,
    mime 
)
```

**Parameters**:

* **fileName** file name
* **mime** string "mime/type"

**Doesn't work in Web Worker.**
