Το **com.aspose.eps** είναι ένα βασικό πακέτο για όλες τις κλάσεις που ασχολούνται με αρχεία PS/EPS.


## Κλάσεις

| Κλάση | Περιγραφή |
| --- | --- |
| [FontConstants](../com.aspose.eps/fontconstants) | Αυτή η κλάση ορίζει ένα σύνολο σταθερών για την αποθήκευση γραμματοσειρών. |
| [HatchPaintLibrary](../com.aspose.eps/hatchpaintlibrary) | Μετατρέπει τα πινέλα GDI+ hatch σε εικόνες κατάλληλες για εγγραφή σε XPS και PDF. |
| [LoadOptions](../com.aspose.eps/loadoptions) | Βασική κλάση για επιλογές φόρτωσης εγγράφου. |
| [PageConstants](../com.aspose.eps/pageconstants) | Αυτή η κλάση ορίζει ένα σύνολο σταθερών που περιγράφουν μια σελίδα. |
| [PsConverterException](../com.aspose.eps/psconverterexception) | Αυτή η κλάση περιέχει πληροφορίες σχετικά με ένα σφάλμα που προκύπτει όταν το αρχείο PS μετατρέπεται σε έγγραφο PDF. |
| [PsConverterLicenseException](../com.aspose.eps/psconverterlicenseexception) | Αυτή η κλάση περιέχει πληροφορίες σχετικά με ένα σφάλμα που σχετίζεται με την άδεια του προϊόντος Aspose.Page. |
| [PsDocument](../com.aspose.eps/psdocument) | Αυτή η κλάση περιλαμβάνει έγγραφα PS/EPS. |
| [PsDocumentException](../com.aspose.eps/psdocumentexception) | Αυτή η κλάση περιέχει πληροφορίες σχετικά με ένα σφάλμα που προκύπτει κατά τη δημιουργία, ολοκλήρωση και αποθήκευση του αρχείου PS. |
| [PsLoadOptions](../com.aspose.eps/psloadoptions) | Επιλογές φόρτωσης εγγράφου PS. |
