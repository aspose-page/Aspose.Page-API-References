Το **Aspose.Page** είναι ένα βασικό πακέτο για όλες τις κλάσεις της βιβλιοθήκης Aspose.Page, οι οποίες είτε βρίσκονται απευθείας σε αυτό όπως η **Device**, είτε έμμεσα μέσω πολλών υποπακέτων.

| [DimensionF](../com.aspose.page/dimensionf) | Η κλάση `Dimension` περιλαμβάνει το πλάτος και το ύψος ενός στοιχείου (σε μονή ακρίβεια) σε ένα ενιαίο αντικείμενο. |
| [Document](../com.aspose.page/document) | Μια υπερκλάση για όλα τα ενσωματωμένα έγγραφα. |
| [ExternalFontCache](../com.aspose.page/externalfontcache) | Χρησιμοποιήστε αυτήν την κλάση για να αποκτήσετε ενσωμάτωση γραμματοσειράς σε μορφή που γίνεται αποδεκτή από το Device. |
| [License](../com.aspose.page/license) | Παρέχει μεθόδους για την άδεια του στοιχείου. |
| [Metered](../com.aspose.page/metered) | Παρέχει μεθόδους για τον ορισμό κλειδιού μέτρησης. |

| [TransformedStroke](../com.aspose.page/transformedstroke) | Μία γραμμή που περιλαμβάνει μετασχηματισμό. |
| [UserProperties](../com.aspose.page/userproperties) | Ειδική κλάση ιδιοτήτων που επιτρέπει τον ορισμό και την επιστροφή τυποποιημένων ιδιοτήτων. |
