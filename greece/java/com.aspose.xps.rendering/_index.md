Το πακέτο **com.aspose.xps.rendering** παρέχει βασικές κλάσεις για την απόδοση του εγγράφου XPS σε άλλες μορφές.


## Κλάσεις

| Κλάση | Περιγραφή |
| --- | --- |
| [BmpSaveOptions](../com.aspose.xps.rendering/bmpsaveoptions) | Κλάση για επιλογές αποθήκευσης XPS-as-BMP. |
| [ImageSaveOptions](../com.aspose.xps.rendering/imagesaveoptions) | Βασική κλάση για επιλογές αποθήκευσης XPS-as-image. |
| [JpegSaveOptions](../com.aspose.xps.rendering/jpegsaveoptions) | Κλάση για επιλογές αποθήκευσης XPS-as-JPEG. |
| [PdfEncryptionDetails](../com.aspose.xps.rendering/pdfencryptiondetails) | Περιέχει λεπτομέρειες για κρυπτογράφηση pdf. |
| [PdfSaveOptions](../com.aspose.xps.rendering/pdfsaveoptions) | Κλάση για επιλογές αποθήκευσης XPS-as-PDF. |
| [PngSaveOptions](../com.aspose.xps.rendering/pngsaveoptions) | Κλάση για επιλογές αποθήκευσης XPS-as-PNG. |
| [TiffSaveOptions](../com.aspose.xps.rendering/tiffsaveoptions) | Κλάση για επιλογές αποθήκευσης XPS-as-TIFF. |

## Διεπαφές

| Διεπαφή | Περιγραφή |
| --- | --- |
| [IEventBasedModificationOptions](../com.aspose.xps.rendering/ieventbasedmodificationoptions) | Ορίζει επιλογές σχετικές με τη διαχείριση τροποποιήσεων βάσει συμβάντων κατά την αποθήκευση του εγγράφου. |
| [IPipelineOptions](../com.aspose.xps.rendering/ipipelineoptions) | Ορίζει επιλογές μετατροπής σχετικές με τη διαμόρφωση της γραμμής επεξεργασίας. |
| [IXpsTextConversionOptions](../com.aspose.xps.rendering/ixpstextconversionoptions) | Ορίζει επιλογές για τη μετατροπή του XPS σε άλλες μορφές. |

## Απαριθμήσεις
