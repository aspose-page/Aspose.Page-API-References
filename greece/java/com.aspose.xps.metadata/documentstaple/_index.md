Περιγράφει τα χαρακτηριστικά δέσμευσης του εξόδου. Κάθε έγγραφο δένεται ξεχωριστά. Οι λέξεις-κλειδιά  JobStapleAllDocuments  και  DocumentStaple  είναι αμοιβαία αποκλειστικές. Εξαρτάται από τον οδηγό να καθορίσει τη διαχείριση περιορισμών μεταξύ αυτών των λέξεων-κλειδιά. https://docs.microsoft.com/en-us/windows/win32/printdocs/documentstaple
