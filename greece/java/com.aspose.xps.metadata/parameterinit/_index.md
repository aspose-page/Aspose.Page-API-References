Κλάση που υλοποιεί τον αρχικοποιητή παραμέτρων του δελτίου εκτύπωσης. Η κλάση που υλοποιεί έναν κοινό αρχικοποιητή παραμέτρων PrintTicket. Η βασική κλάση για όλους τους αρχικοποιητές παραμέτρων που ορίζονται από το σχήμα. Ορίζει μια τιμή για μια παρουσία ενός  ParameterDef  στοιχείου. Ένα  ParameterInit  στοιχείο είναι ο στόχος της αναφοράς που γίνεται από ένα  ParameterRef  στοιχείο. https://docs.microsoft.com/en-us/windows/win32/printdocs/parameterinit
