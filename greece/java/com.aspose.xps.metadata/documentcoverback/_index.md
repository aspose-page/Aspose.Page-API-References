Περιγράφει το πίσω (τελικό) φύλλο εξώφυλλου. Κάθε έγγραφο θα έχει ξεχωριστό φύλλο. Το φύλλο εξώφυλλου πρέπει να εκτυπώνεται στο  PageMediaSize  και  PageMediaType  που χρησιμοποιούνται για τη τελευταία σελίδα του εγγράφου. Το φύλλο εξώφυλλου πρέπει να ενσωματώνεται στις επιλογές επεξεργασίας (όπως  DocumentDuplex ,  DocumentNUp ) όπως υποδεικνύεται από την Option που καθορίζεται. https://docs.microsoft.com/en-us/windows/win32/printdocs/documentcoverback
