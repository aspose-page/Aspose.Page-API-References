| [DataType](../com.aspose.page.plugins/datatype) | Αντιπροσωπεύει τους πιθανούς τύπους δεδομένων για την επεξεργασία του πρόσθετου. |
| [FileDataSource](../com.aspose.page.plugins/filedatasource) | Αντιπροσωπεύει την πηγή δεδομένων αρχείου για λειτουργίες φόρτωσης και αποθήκευσης ενός πρόσθετου. |
| [FileResult](../com.aspose.page.plugins/fileresult) | Αντιπροσωπεύει το αποτέλεσμα της λειτουργίας με τη μορφή διαδρομής string προς το αρχείο. |
| [PluginExceptionMessages](../com.aspose.page.plugins/pluginexceptionmessages) | Αυτή η κλάση αντιπροσωπεύει μια αποθήκη για μηνύματα που χρησιμοποιούνται σε εξαίρεσεις που προβάλλονται από πρόσθετα. |

| [StreamDataSource](../com.aspose.page.plugins/streamdatasource) | Αντιπροσωπεύει την πηγή δεδομένων ροής για λειτουργίες φόρτωσης και αποθήκευσης ενός πρόσθετου. |
| [StreamResult](../com.aspose.page.plugins/streamresult) | Αντιπροσωπεύει το αποτέλεσμα της λειτουργίας με τη μορφή Stream. |
| [StringResult](../com.aspose.page.plugins/stringresult) | Αντιπροσωπεύει το αποτέλεσμα της λειτουργίας με τη μορφή string. |
