## Aspose.Page για .NET

{{% alert color="primary" %}}

![Aspose.Page για το λογότυπο προϊόντος .NET](../home_1.png)


Το Aspose.Page είναι ένα API .NET και έχει δημιουργηθεί για να επιτρέπει στους προγραμματιστές να εργάζονται με έγγραφα XPS και EPS/PS. Χρησιμοποιώντας το API μπορείτε να δημιουργήσετε, να επεξεργαστείτε και να αποθηκεύσετε υπάρχοντα καθώς και νέα έγγραφα XPS. Επιπλέον, μπορείτε να μετατρέψετε έγγραφα XPS και EPS/PS σε PDF και Εικόνες. Το Aspose.Page για .NET υλοποιείται με χρήση C# και μπορεί να χρησιμοποιηθεί με οποιαδήποτε γλώσσα .NET, όπως C#, VB.NET και J# κ.λπ. Μπορεί να ενσωματωθεί με οποιοδήποτε είδος εφαρμογής είτε είναι ASP.NET Web Application είτε Windows Εφαρμογή.

{{% /alert %}}

## Aspose.Page για C++

{{% alert color="primary" %}}

![Aspose.Page για το λογότυπο προϊόντος C++](../home_3.png)


Το Aspose.Page για C++ είναι ένα αξιόπιστο και αυτόνομο API χειρισμού και μετατροπής αρχείων XPS και PostScript. Σας επιτρέπει να δημιουργείτε, να επεξεργάζεστε και να μετατρέπετε αρχεία XPS χωρίς να χρειάζεστε πρόσθετα εργαλεία. Το API επιτρέπει λειτουργίες όπως χειρισμό σελίδων και στοιχείων εγγράφου, δημιουργία διανυσματικών γραφικών, ομαδοποίηση σχημάτων και καθορισμό χρωμάτων σε διαφορετικούς χρωματικούς χώρους. Επιπλέον, επιτρέπει στους προγραμματιστές να μετατρέπουν αρχεία PS και EPS σε έγγραφα PDF, εικόνες ράστερ όπως BMP, EMF, GIF, JPEG, PNG, TIFF και άλλα.
