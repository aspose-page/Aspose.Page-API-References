## AssemblyConstants class


Ορίζει τις σταθερές που συμμετέχουν στον έλεγχο άδειας για το στοιχείο. Αυτές ορίζονταν προηγουμένως απευθείας ως χαρακτηριστικά συναρμολόγησης, αλλά τις μετακίνησα σε ξεχωριστή κλάση επειδή στο .NET Compact Framework δεν μπορώ να προσπελάσω τα χαρακτηριστικά συναρμολόγησης. Τώρα ο κώδικας αδειοδότησης όταν μεταγλωττίζεται για το .NET Compact Framework χρησιμοποιεί αυτές τις σταθερές αντί για τα χαρακτηριστικά συναρμολόγησης.

```cpp
class AssemblyConstants : public System::Object
```
