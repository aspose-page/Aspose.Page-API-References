## Metered::SetMeteredKey method


Ορίζει το μετρημένο δημόσιο και ιδιωτικό κλειδί. Εάν αγοράσετε άδεια μετρημένης χρήσης, όταν ξεκινάτε την εφαρμογή, αυτό το API πρέπει να κληθεί, κανονικά, αυτό είναι αρκετό. Ωστόσο, εάν αποτυγχάνει συνεχώς η μεταφόρτωση των δεδομένων κατανάλωσης και υπερβαίνει τις 24 ώρες, η άδεια θα οριστεί σε κατάσταση αξιολόγησης· για να αποφύγετε αυτήν την περίπτωση, θα πρέπει να ελέγχετε τακτικά την κατάσταση της άδειας· εάν είναι σε κατάσταση αξιολόγησης, καλέστε ξανά αυτό το API.
