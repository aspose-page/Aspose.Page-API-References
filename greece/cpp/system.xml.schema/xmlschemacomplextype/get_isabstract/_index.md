## XmlSchemaComplexType::get_IsAbstract method


Επιστρέφει τις πληροφορίες που καθορίζουν αν το στοιχείο **complexType** μπορεί να χρησιμοποιηθεί στο έγγραφο παρουσίασης.

```cpp
bool System::Xml::Schema::XmlSchemaComplexType::get_IsAbstract()
```


### ReturnValue

Αν **true**, ένα στοιχείο δεν μπορεί να χρησιμοποιήσει άμεσα αυτό το στοιχείο **complexType** και πρέπει να χρησιμοποιήσει έναν σύνθετο τύπο που προέρχεται από αυτό το στοιχείο **complexType**. Η προεπιλογή είναι **false**. Προαιρετικό.
