## XmlSchemaComplexType::get_BlockResolved method


Επιστρέφει την τιμή μετά τη μεταγλώττιση του τύπου στο σύνολο πληροφοριών μετά-επαλήθευσης σχήματος (infoset). Αυτή η τιμή υποδεικνύει πώς επιβάλλεται ο τύπος όταν χρησιμοποιείται το **xsi:type** στο έγγραφο παρουσίασης.

```cpp
XmlSchemaDerivationMethod System::Xml::Schema::XmlSchemaComplexType::get_BlockResolved()
```


### ReturnValue

Η τιμή του infoset μετά την επικύρωση του σχήματος. Η προεπιλογή είναι η τιμή **BlockDefault** στο στοιχείο **schema**.
