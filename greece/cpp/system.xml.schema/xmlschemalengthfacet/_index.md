## XmlSchemaLengthFacet class


Αναπαριστά το **length** χαρακτηριστικό από το XML [Schema](../) όπως καθορίζεται από το World Wide [Web](../../system.web/) Consortium (W3C). Αυτή η κλάση μπορεί να χρησιμοποιηθεί για να καθορίσει περιορισμό στο μήκος ενός **simpleType** στοιχείου στον τύπο δεδομένων.

```cpp
class XmlSchemaLengthFacet : public System::Xml::Schema::XmlSchemaNumericFacet
```
