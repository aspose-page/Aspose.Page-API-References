## XmlSchemaTotalDigitsFacet class


Αντιπροσωπεύει το χαρακτηριστικό **totalDigits** από το XML [Schema](../) όπως καθορίζεται από το Παγκόσμιου Ιστού [Web](../../system.web/) Consortium (W3C). Αυτή η κλάση μπορεί να χρησιμοποιηθεί για να ορίσει περιορισμό στον αριθμό των ψηφίων που μπορούν να εισαχθούν για την τιμή ενός στοιχείου **simpleType**. Η τιμή **totalDigits** πρέπει να είναι θετικός ακέραιος.

```cpp
class XmlSchemaTotalDigitsFacet : public System::Xml::Schema::XmlSchemaNumericFacet
```
