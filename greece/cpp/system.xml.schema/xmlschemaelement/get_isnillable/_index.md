## XmlSchemaElement::get_IsNillable method


Επιστρέφει πληροφορίες που υποδεικνύουν εάν το **xsi:nil** μπορεί να εμφανιστεί στα δεδομένα στιγμιοτύπου. Υποδεικνύει εάν μια ρητή τιμή nil μπορεί να εκχωρηθεί στο στοιχείο.

```cpp
bool System::Xml::Schema::XmlSchemaElement::get_IsNillable()
```


### ReturnValue

Αν το nillable είναι **true**, αυτό επιτρέπει σε μια παρουσία του στοιχείου να έχει το χαρακτηριστικό **nil** ορισμένο σε **true**. Το χαρακτηριστικό **nil** ορίζεται ως μέρος του χώρου ονομάτων XML [Schema](../../) για παρουσίες. Η προεπιλογή είναι **false**. Προαιρετικό.
