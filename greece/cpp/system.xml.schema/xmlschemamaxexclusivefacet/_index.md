## XmlSchemaMaxExclusiveFacet class


Αντιπροσωπεύει το στοιχείο **maxExclusive** από το XML [Schema](../) όπως καθορίζεται από το Παγκόσμιου Ιστού [Web](../../system.web/) Consortium (W3C). Αυτή η κλάση μπορεί να χρησιμοποιηθεί για να ορίσει περιορισμό στη μέγιστη τιμή ενός στοιχείου **simpleType**. Η τιμή του στοιχείου πρέπει να είναι μικρότερη από την τιμή του στοιχείου **maxExclusive**.

```cpp
class XmlSchemaMaxExclusiveFacet : public System::Xml::Schema::XmlSchemaFacet
```
