Αυτή η κλάση μπορεί να χρησιμοποιηθεί για να ορίσει περιορισμό στην ελάχιστη τιμή ενός στοιχείου **simpleType**. Η τιμή του στοιχείου πρέπει να είναι μεγαλύτερη από την τιμή του στοιχείου **minExclusive**.

