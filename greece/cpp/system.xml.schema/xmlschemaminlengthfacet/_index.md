## XmlSchemaMinLengthFacet class


Αντιπροσωπεύει το στοιχείο **minLength** από το XML [Schema](../) όπως ορίζεται από το World Wide [Web](../../system.web/) Consortium (W3C). Αυτή η κλάση μπορεί να χρησιμοποιηθεί για να καθορίσει έναν περιορισμό στο ελάχιστο μήκος της τιμής δεδομένων ενός στοιχείου **simpleType**. Το μήκος πρέπει να είναι μεγαλύτερο από την τιμή του στοιχείου **minLength**.

```cpp
class XmlSchemaMinLengthFacet : public System::Xml::Schema::XmlSchemaNumericFacet
```
