## XmlSchemaMaxLengthFacet class


Αντιπροσωπεύει το στοιχείο **maxLength** από το XML [Schema](../) όπως καθορίζεται από το World Wide [Web](../../system.web/) Consortium (W3C). Αυτή η κλάση μπορεί να χρησιμοποιηθεί για να ορίσει περιορισμό στο μέγιστο μήκος της τιμής δεδομένων ενός στοιχείου **simpleType**. Το μήκος πρέπει να είναι μικρότερο από την τιμή του στοιχείου **maxLength**.

```cpp
class XmlSchemaMaxLengthFacet : public System::Xml::Schema::XmlSchemaNumericFacet
```
