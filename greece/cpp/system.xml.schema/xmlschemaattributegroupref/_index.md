## XmlSchemaAttributeGroupRef class


Αντιπροσωπεύει το στοιχείο **attributeGroup** με το χαρακτηριστικό **ref** από το XML [Schema](../) όπως καθορίζεται από το [World Wide Web Consortium (W3C)](https://go.microsoft.com/fwlink/?LinkId=49454). Το AttributesGroupRef είναι η αναφορά για ένα attributeGroup, η ιδιότητα name περιέχει το attribute group που αναφέρεται.

```cpp
class XmlSchemaAttributeGroupRef : public System::Xml::Schema::XmlSchemaAnnotated
```
