## XmlSchemaForm enum


Δείχνει εάν τα χαρακτηριστικά ή τα στοιχεία χρειάζεται να προσδιορίζονται με πρόθεμα ονοματοχώρου.

```cpp
enum class XmlSchemaForm
```

### Τιμές

| Όνομα | Τιμή | Περιγραφή |
| --- | --- | --- |
| None | 0 | Η μορφή στοιχείου και χαρακτηριστικού δεν έχει καθοριστεί στο σχήμα. |
| Προσδιορισμένο | 1 | Τα στοιχεία και τα χαρακτηριστικά πρέπει να είναι προσδιορισμένα με πρόθεμα χώρου ονομάτων. |
| Μη προσδιορισμένο | 2 | Τα στοιχεία και τα χαρακτηριστικά δεν απαιτείται να προσδιορίζονται με πρόθεμα ονοματοχώρου. |
