## TraceLevel enum


Καθορίζει ποια μηνύματα θα εμφανίζονται για τις κλάσεις [System.Diagnostics.Debug](../debug/), [System.Diagnostics.Trace](../trace/) και System.Diagnostics.TraceSwitch.

```cpp
enum class TraceLevel
```

### Τιμές

| Όνομα | Τιμή | Περιγραφή |
| --- | --- | --- |
| Off | 0 | Δεν εμφανίζει μηνύματα εντοπισμού και αποσφαλμάτωσης. |
| Σφάλμα | 1 | Εμφανίζει μηνύματα διαχείρισης σφαλμάτων. |
| Warning | 2 | Εμφανίζει προειδοποιήσεις και μηνύματα διαχείρισης σφαλμάτων. |
| Info | 3 | Εκτυπώνει πληροφοριακά μηνύματα, προειδοποιήσεις και μηνύματα διαχείρισης σφαλμάτων. |
| Αναλυτικό | 4 | Εκτυπώνει όλα τα μηνύματα αποσφαλμάτωσης και εντοπισμού. |
