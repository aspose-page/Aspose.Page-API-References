## PsSaveFormat enum


Αυτή η απαρίθμηση περιέχει τις διαθέσιμες επιλογές μορφής αποθήκευσης. Μπορεί να είναι PS ή [EPS](../../aspose.page.eps/). Το [EPS](../../aspose.page.eps/) χρησιμοποιείται μόνο για έγγραφα με μία σελίδα, ενώ το αρχείο PS μπορεί να περιέχει οποιονδήποτε αριθμό σελίδων.

```cpp
enum class PsSaveFormat
```

### Τιμές

| Όνομα | Τιμή | Περιγραφή |
| --- | --- | --- |
| PS | 0 | Αυτή η επιλογή υποδεικνύει ότι το τελικό έγγραφο πρέπει να είναι αρχείο PostScript (PS) ή Encapsulated PostScript ([EPS](../../aspose.page.eps/)). |
| EPS | 1 | Αυτή η επιλογή υποδεικνύει ότι το τελικό έγγραφο πρέπει να είναι αρχείο Encapsulated PostScript ([EPS](../../aspose.page.eps/)). Χρησιμοποιείται μόνο για έγγραφα με μία σελίδα. |
