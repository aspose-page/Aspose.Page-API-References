### Τιμές

| Όνομα | Τιμή | Περιγραφή |
| --- | --- | --- |
| None | 0 | Προεπιλεγμένη συμπεριφορά. |
| Compiled | 1 | Συμπιέστε το regex για απόδοση. Πάντα γίνεται προεπιλογή. |
| CultureInvariant | 2 | Χρησιμοποιήστε αντιστοίχιση ανεξάρτητη από πολιτισμό. Αγνοείται. |
| ECMAScript | 4 | Χρησιμοποιήστε τη σύνταξη ECMAScript. Αγνοήθηκε. |
| ExplicitCapture | 8 | Μόνο ρητή σύλληψη. Αγνοήθηκε. |
| IgnoreCase | 16 | Αγνοήστε τη διάκριση πεζών-κεφαλαίων κατά την αντιστοίχιση. |
| IgnorePatternWhitespace | 32 | Αγνοήστε τα κενά στο μοτίβο. Δεν υποστηρίζεται. |
| Multiline | 64 | Θεωρείτε το '^' και το '$' ως αρχή και τέλος γραμμής, όχι ολόκληρης συμβολοσειράς. |
| RightToLeft | 128 | Αντιστοίχιση από δεξιά προς αριστερά. Δεν υποστηρίζεται. |
| Singleline | 256 | Κάνει το '.' να ταιριάζει με οποιονδήποτε χαρακτήρα χωρίς εξαιρέσεις (κανονικά, οι χαρακτήρες νέας γραμμής δεν ταιριάζουν). |
