## AutoSizeMode enum


Ορίζει πώς λειτουργεί η αυτόματη προσαρμογή μεγέθους.

```cpp
enum class AutoSizeMode
```

### Τιμές

| Όνομα | Τιμή | Περιγραφή |
| --- | --- | --- |
| GrowAndShrink | 0 | [Control](../control/) μεγαλώνει και μικραίνει όπως απαιτείται. Δεν μπορεί να αλλάξει μέγεθος χειροκίνητα. |
| GrowOnly | 1 | [Control](../control/) μεγαλώνει όπως απαιτείται αλλά δεν μπορεί να μικραίνει κάτω από το ελάχιστο μέγεθος. Μπορεί να αλλάξει μέγεθος χειροκίνητα. |
