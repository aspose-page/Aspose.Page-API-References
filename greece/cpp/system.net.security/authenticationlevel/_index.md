## AuthenticationLevel enum


Σημαίες πιστοποίησης ειδικές για WebRequest.

```cpp
enum class AuthenticationLevel
```

### Τιμές

| Όνομα | Τιμή | Περιγραφή |
| --- | --- | --- |
| None | 0 | Δεν απαιτείται έλεγχος ταυτότητας για τον πελάτη και τον διακομιστή. |
| MutualAuthRequested | 1 | Η αίτηση δεν αποτυγχάνει εάν ο διακομιστής δεν είναι αυθεντικοποιημένος. |
| MutualAuthRequired | 2 | Η τρέχουσα εφαρμογή θα λάβει το 'IOException' όταν ο διακομιστής δεν είναι αυθεντικοποιημένος. |
