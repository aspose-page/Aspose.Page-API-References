## EncryptionPolicy enum


Απαριθμεί τις πολιτικές κρυπτογράφησης.

```cpp
enum class EncryptionPolicy
```

### Τιμές

| Όνομα | Τιμή | Περιγραφή |
| --- | --- | --- |
| RequireEncryption | 0 | Απαιτεί κρυπτογράφηση και ποτέ δεν επιτρέπει έναν κρυπτογράφο 'Null'. |
| AllowNoEncryption | 1 | Προτιμά τη χρήση πλήρους κρυπτογράφησης, αλλά μπορεί να χρησιμοποιηθεί ένας κρυπτογράφος 'Null' εάν ο διακομιστής συμφωνήσει. |
| NoEncryption | 2 | Επιτρέπει την απουσία κρυπτογράφησης και ζητά να χρησιμοποιηθεί ένας κρυπτογράφος 'Null' εάν το άλλο άκρο μπορεί να διαχειριστεί έναν κρυπτογράφο 'Null'. |
