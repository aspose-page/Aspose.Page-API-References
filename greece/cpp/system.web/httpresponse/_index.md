## HttpResponse class


Ψεύτικη κλάση που καθιστά δυνατή τη σύνδεση του μεταφρασμένου κώδικα με αναφορές [HttpResponse](./), αλλά δεν την εκτελεί. Δεν περιέχει σωστά υλοποιημένα μέλη. Τα αντικείμενα αυτής της κλάσης πρέπει να δημιουργούνται μόνο μέσω της συνάρτησης [System::MakeObject()](../../system/makeobject/). Ποτέ μην δημιουργείτε αντίγραφο αυτού του τύπου στο stack ή χρησιμοποιώντας τον τελεστή new, καθώς θα οδηγήσει σε σφάλματα χρόνου εκτέλεσης και/ή σφάλματα assert. Πάντα τυλίξτε αυτήν την κλάση σε δείκτη [System::SmartPtr](../../system/smartptr/) και χρησιμοποιήστε αυτόν τον δείκτη για να τη περάσετε σε συναρτήσεις ως όρισμα.

```cpp
class HttpResponse : public System::Object
```
