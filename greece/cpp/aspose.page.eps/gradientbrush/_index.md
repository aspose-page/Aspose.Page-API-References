## GradientBrush class


Αυτή η κλάση χρησιμοποιείται για την ενσωμάτωση των LinearGradientBrush και PathGradientBrush με δυνατότητα ορισμού της λειτουργίας περιτύλιξης σε clamp. Οι εγγενείς πινέζες διαβάθμισης πάντα προκαλούν εξαίρεση όταν κάποιος προσπαθεί να ορίσει την ιδιότητα WrapMode σε WrapMode.Clamp.

```cpp
class GradientBrush : public System::Drawing::Brush
```
