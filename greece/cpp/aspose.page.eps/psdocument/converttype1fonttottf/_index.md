## PsDocument::ConvertType1FontToTTF method


Μετατρέπει τη γραμματοσειρά Type 1 σε **TrueType**. Το όνομα της μετατρεπόμενης γραμματοσειράς TTF θα είναι το ίδιο με τη γραμματοσειρά Type 1 εισόδου με επέκταση ".ttf". Το αρχείο TTF θα αποθηκευτεί στον καθορισμένο φάκελο εξόδου.

```cpp
void Aspose::Page::EPS::PsDocument::ConvertType1FontToTTF(System::String type1FontFilePath, System::String outputDir)
```


| Parameter | Type | Περιγραφή |
| --- | --- | --- |
| type1FontFilePath | System::String | Η διαδρομή αρχείου γραμματοσειράς Type 1. |
| outputDir | System::String | Κατάλογος εξόδου όπου θα αποθηκευτεί η προκύπτουσα γραμματοσειρά **TrueType**. |
