## HMACSHA512 class


Κώδικας ελέγχου ακεραιότητας μηνύματος [Authentication](../../system.security.authentication/) που χρησιμοποιεί τη συνάρτηση κατακερματισμού [SHA512](../sha512/). Μερικώς υλοποιημένο. Τα αντικείμενα αυτής της κλάσης πρέπει να δημιουργούνται μόνο με τη συνάρτηση [System::MakeObject()](../../system/makeobject/). Ποτέ μην δημιουργείτε ένα αντίτυπο αυτού του τύπου στη στοίβα ή χρησιμοποιώντας τον τελεστή new, καθώς θα προκαλέσει σφάλματα χρόνου εκτέλεσης και/ή σφάλματα ελέγχου. Πάντα τυλίξτε αυτήν την κλάση σε δείκτη [System::SmartPtr](../../system/smartptr/) και χρησιμοποιήστε αυτόν τον δείκτη για να τη περάσετε σε συναρτήσεις ως όρισμα.

```cpp
class HMACSHA512 : public System::Security::Cryptography::HashAlgorithm
```
