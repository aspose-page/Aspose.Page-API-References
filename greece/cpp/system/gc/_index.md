## GC class


Αντιπροσωπεύει μια προσομοιωμένη Συλλογή Σκυβωδών που λειτουργεί περισσότερο ως υποδοχέας και ουσιαστικά δεν κάνει τίποτα. Αυτός είναι ένας στατικός τύπος χωρίς υπηρεσίες στιγμιοτύπων. Δεν πρέπει ποτέ να δημιουργείτε στιγμιότυπα του με κανένα τρόπο.

```cpp
class GC : public System::Object
```
