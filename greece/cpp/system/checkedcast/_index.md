## System::CheckedCast method

```cpp
template<typename TTo,typename TFrom> TTo System::CheckedCast(TFrom value)
```


| Parameter | Περιγραφή |

| TTo | Ο τύπος στον οποίο πρέπει να μετατραπεί η καθορισμένη τιμή |
| TFrom | Ο τύπος της καθορισμένης τιμής |

| Parameter | Type | Περιγραφή |
| --- | --- | --- |
| τιμή | TFrom | Η τιμή προς μετατροπή |
