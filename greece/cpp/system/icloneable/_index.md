## ICloneable class


Ορίζει μια μέθοδο που επιτρέπει την κλωνοποίηση αντικειμένων - τη δημιουργία αντιγράφου ενός αντικειμένου. Τα αντικείμενα αυτής της κλάσης πρέπει να δημιουργούνται μόνο μέσω της συνάρτησης [System::MakeObject()](../makeobject/). Ποτέ μην δημιουργείτε μια υπόδειξη αυτού του τύπου στη στοίβα ή χρησιμοποιώντας τον τελεστή new, καθώς αυτό θα οδηγήσει σε σφάλματα χρόνου εκτέλεσης και/ή σφάλματα ελέγχου. Πάντα να τυλίγετε αυτήν την κλάση σε δείκτη [System::SmartPtr](../smartptr/) και να χρησιμοποιείτε αυτόν τον δείκτη για να τη μεταβιβάζετε σε συναρτήσεις ως όρισμα.

```cpp
class ICloneable : public virtual System::Object
```
