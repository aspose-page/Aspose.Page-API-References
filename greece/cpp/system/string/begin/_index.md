## String::begin method


Επιστρέφει δείκτη στην αρχή του πραγματικού buffer της συμβολοσειράς. Ποτέ δεν επανεκχωρεί τίποτα. Δεν εγγυάται ότι το buffer θα είναι τερματισμένο με null.

```cpp
const UChar * System::String::begin() const
```


### ReturnValue

Δείκτης στο πραγματικό buffer της συμβολοσειράς, είτε είναι null-terminated είτε όχι.
