## ValueType class


Βασική κλάση για τύπους τιμής με κληρονομικότητα [Object](../object/) που περικόπτεται για λόγους απόδοσης. Αυτός ο τύπος πρέπει να κατανεμηθεί στη στοίβα και να περάσει σε συναρτήσεις με τιμή ή με αναφορά. Ποτέ μην χρησιμοποιείτε την [System::SmartPtr](../smartptr/) κλάση για τη διαχείριση αντικειμένων αυτού του τύπου.

```cpp
class ValueType
```
