## TypeInfo::GetCustomAttribute method


Αναζητά το προσαρμοσμένο χαρακτηριστικό που έχει τον καθορισμένο τύπο και εφαρμόζεται στον τύπο που αντιπροσωπεύεται από το τρέχον αντικείμενο.

```cpp
ObjectPtr System::TypeInfo::GetCustomAttribute(const TypeInfo &attributeType) const
```


| Parameter | Type | Περιγραφή |
| --- | --- | --- |
| attributeType | const TypeInfo\& | Η σταθερή αναφορά στο αντικείμενο [TypeInfo](../) που αντιπροσωπεύει τον τύπο του χαρακτηριστικού προς αναζήτηση |

### ReturnValue

Ένας δείκτης σε αντικείμενο που αντιπροσωπεύει το ευρεθέν χαρακτηριστικό, ή null-pointer εάν δεν βρέθηκε κανένα χαρακτηριστικό που να ταιριάζει με τα κριτήρια αναζήτησης
