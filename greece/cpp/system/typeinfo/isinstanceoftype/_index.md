

| Parameter | Type | Περιγραφή |
| --- | --- | --- |
| obj | const SharedPtr\<Object\>\& | Το αντικείμενο για σύγκριση με τον τρέχοντα τύπο |

### ReturnValue

true εάν ο τρέχων τύπος βρίσκεται στην ιεραρχία κληρονομικότητας του αντικειμένου που αντιπροσωπεύεται από το obj
