## TypeInfo::IsDefined method


ΔΕΝ ΕΦΑΡΜΟΣΤΕΙ. Δείχνει εάν ένα ή περισσότερα χαρακτηριστικά του καθορισμένου τύπου ή των παραγώγων του τύπου εφαρμόζονται σε αυτό το μέλος.

```cpp
bool System::TypeInfo::IsDefined(const TypeInfo &attributeType, bool inherit) const
```


| Parameter | Type | Περιγραφή |
| --- | --- | --- |
| attributeType | const TypeInfo\& | Ο τύπος του προσαρμοσμένου χαρακτηριστικού για αναζήτηση. Η αναζήτηση περιλαμβάνει παράγωγους τύπους. |
| inherit | bool | true για να αναζητήσετε την αλυσίδα κληρονομικότητας αυτού του μέλους ώστε να βρείτε τα χαρακτηριστικά· διαφορετικά, false. Αυτή η παράμετρος αγνοείται για ιδιότητες και συμβάντα. |

### ReturnValue

true εάν ένα ή περισσότερα στιγμιότυπα του attributeType ή οποιοσδήποτε από τους παράγωγους τύπους του έχουν εφαρμοστεί σε αυτό το μέλος· διαφορετικά, false.
