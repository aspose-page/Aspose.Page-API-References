## TransitionTime::CreateTransitionTime method


Δημιουργεί μια παρουσία της κλάσης [TransitionTime](../) που αντιπροσωπεύει μια αλλαγή ώρας που περιγράφεται με τις καθορισμένες παραμέτρους.

```cpp
static TransitionTime System::TimeZoneInfo::TransitionTime::CreateTransitionTime(DateTime time_of_day, int month, int week, int day, DayOfWeek day_of_week, bool is_fixed_date_rule)
```


| Parameter | Type | Περιγραφή |
| --- | --- | --- |
| time_of_day | DateTime | Η συγκεκριμένη ώρα στην οποία συμβαίνει η αλλαγή ώρας. |
| μήνας | int | Ο μήνας του έτους στον οποίο συμβαίνει η αλλαγή ώρας. |
| εβδομάδα | int | Η εβδομάδα του μήνα στην οποία συμβαίνει η αλλαγή ώρας. |
| ημέρα | int | Η ημέρα κατά την οποία πραγματοποιείται η αλλαγή ώρας. |
| day_of_week | DayOfWeek | Η ημέρα της εβδομάδας στην οποία συμβαίνει η αλλαγή ώρας. |
| is_fixed_date_rule | bool | Τιμή που υποδεικνύει εάν η αλλαγή ώρας πραγματοποιείται σε σταθερή ημερομηνία και ώρα ή σε κυμαινόμενη ημερομηνία και ώρα. |

### ReturnValue

Μια παρουσία της κλάσης [TransitionTime](../) που αντιπροσωπεύει την περιγραφόμενη αλλαγή ώρας.
