## Array::rbegin() const method


Επιστρέφει έναν αντίστροφο επαναλήπτη στο πρώτο στοιχείο του αντιστροφικού container. Αντιστοιχεί στο τελευταίο στοιχείο του μη-αντιστροφικού container. Εάν το container είναι κενό, ο επιστραφέν επαναλήπτης είναι ίσος με [rend()](../rend/).

```cpp
const_reverse_iterator System::Array<T>::rbegin() const noexcept
```


### ReturnValue
