## Array::cbegin method


Επιστρέφει έναν iterator προς το πρώτο στοιχείο με const-προσδιορισμό του container. Εάν το container είναι κενό, ο επιστραφέν iterator θα είναι ίσος με [cend()](../cend/).

```cpp
const_iterator System::Array<T>::cbegin() const noexcept
```


### ReturnValue

Ένας iterator που δείχνει στο πρώτο στοιχείο με const-προσδιορισμό του container.
