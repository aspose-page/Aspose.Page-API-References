## Array::crend method


Επιστρέφει έναν reverse iterator προς το στοιχείο που ακολουθεί το τελευταίο στοιχείο του reversed container. Αντιστοιχεί στο στοιχείο που προηγείται του πρώτου στοιχείου του non‑reversed container. Αυτό το στοιχείο λειτουργεί ως σύμβολο κράτησης· η προσπάθεια πρόσβασης σε αυτό οδηγεί σε ακαθόριστη συμπεριφορά.

```cpp
const_reverse_iterator System::Array<T>::crend() const noexcept
```


### ReturnValue

Ένας επαναλήπτης που δείχνει στο θεωρητικό στοιχείο με const-προσδιορισμό που προηγείται του πρώτου στοιχείου του δοχείου.
