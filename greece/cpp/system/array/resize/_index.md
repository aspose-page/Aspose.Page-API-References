```cpp
template<typename Type> static void System::Array<T>::Resize(ArrayPtr<Type> &arr, int new_size)
```


| Parameter | Type | Περιγραφή |
| --- | --- | --- |
| arr | ArrayPtr\<Type\>\& | [Array](../) για αλλαγή μεγέθους. Εάν το **arr** είναι δείκτης null, θα δημιουργηθεί ο νέος πίνακας |
| new_size | int | Το νέο μέγεθος του πίνακα, ή το μέγεθος του νέου πίνακα εάν το **arr** είναι null |
