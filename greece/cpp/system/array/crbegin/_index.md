## Array::crbegin method


Επιστρέφει έναν αντίστροφο iterator προς το πρώτο στοιχείο του αντιστραμμένου container. Αντιστοιχεί στο τελευταίο στοιχείο του μη αντιστραμμένου container. Εάν το container είναι κενό, ο επιστραφέν iterator είναι ίσος με [crend()](../crend/).

```cpp
const_reverse_iterator System::Array<T>::crbegin() const noexcept
```


### ReturnValue

Ένας iterator που δείχνει στο τελευταίο στοιχείο με const-προσδιορισμό του container.
