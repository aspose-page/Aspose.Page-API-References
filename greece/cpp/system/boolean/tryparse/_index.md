| Parameter | Type | Περιγραφή |
| --- | --- | --- |
| τιμή | const String\& | Η συμβολοσειρά προς μετατροπή. |
| αποτέλεσμα | bool\& | Η αναφορά σε μια μεταβλητή bool όπου το αποτέλεσμα της μετατροπής τοποθετείται· το αποτέλεσμα είναι true εάν η συγκεκριμένη συμβολοσειρά είναι ίση με \"True\" και false εάν είναι ίση με \"False\". |
