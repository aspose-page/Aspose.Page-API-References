## DateTime::Compare method


Συγκρίνει δύο τιμές που αντιπροσωπεύονται από τις καθορισμένες περιπτώσεις της [DateTime](../) κλάσης και επιστρέφει την τιμή που υποδεικνύει τη σχετική θέση των τιμών στην χρονογραμμή.

```cpp
static constexpr int System::DateTime::Compare(DateTime t1, DateTime t2)
```


| Parameter | Type | Περιγραφή |
| --- | --- | --- |
| t1 | DateTime | Ο πρώτος συγκριτέος |
| t2 | DateTime | Ο δεύτερος συγκριτέος |

### ReturnValue

Μια τιμή που είναι μικρότερη του 0 αν **t1** είναι νωρίτερη από **t2**· 0 αν **t1** είναι ίδια με **t2**· μια τιμή που είναι μεγαλύτερη του 0 αν **t1** είναι μεταγενέστερη από **t2**
