## DateTime::SpecifyKind method


Δημιουργεί ένα νέο αντικείμενο [DateTime](../) που αντιπροσωπεύει τον ίδιο αριθμό ticks όπως το καθορισμένο αντικείμενο [DateTime](../) και αντιπροσωπεύει τοπική ώρα, ώρα UTC ή καμία, όπως καθορίζεται από το όρισμα **kind**.

```cpp
static DateTime System::DateTime::SpecifyKind(DateTime value, DateTimeKind kind)
```


| Parameter | Type | Περιγραφή |
| --- | --- | --- |
| value | DateTime | Το αντικείμενο [DateTime](../) από το οποίο θα αντιγραφεί ο αριθμός των ticks |
| είδος | DateTimeKind | Καθορίζει εάν το νέο αντικείμενο πρέπει να αντιπροσωπεύει τοπική ώρα, ώρα UTC ή καμία. |

### ReturnValue

Ένα νέο αντικείμενο [DateTime](../) που αντιπροσωπεύει τον ίδιο αριθμό ticks όπως **value** και τιμή [DateTimeKind](../../datetimekind/) που καθορίζεται από **kind**.
