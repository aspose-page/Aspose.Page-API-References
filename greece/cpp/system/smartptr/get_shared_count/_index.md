## SmartPtr::get_shared_count method


Λαμβάνει τον αριθμό των κοινών δεικτών που υπάρχουν προς το αναφερόμενο αντικείμενο, συμπεριλαμβανομένου του τρέχοντος. Επιβεβαιώνει ότι ο τρέχων δείκτης βρίσκεται σε κοινή λειτουργία.

```cpp
int System::SmartPtr<T>::get_shared_count() const
```


### ReturnValue

Αριθμός των κοινόχρηστων δεικτών που υπάρχουν προς το αναφερόμενο αντικείμενο, εάν υπάρχουν. Εάν ο δείκτης είναι null, επιστρέφει 0.
