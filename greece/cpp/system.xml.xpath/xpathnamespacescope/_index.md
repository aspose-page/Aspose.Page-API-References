## XPathNamespaceScope enum


Ορίζει το πεδίο του χώρου ονομάτων.

```cpp
enum class XPathNamespaceScope
```

### Τιμές

| Όνομα | Τιμή | Περιγραφή |
| --- | --- | --- |
| All | 0 | Επιστρέφει όλους τους χώρους ονομάτων που ορίζονται στο πεδίο του τρέχοντος κόμβου. Αυτό περιλαμβάνει το χώρο ονομάτων **xmlns:xml** που πάντα δηλώνεται έμμεσα. Η σειρά των επιστρεφόμενων χώρων ονομάτων δεν ορίζεται. |
| ExcludeXml | 1 | Επιστρέφει όλους τους χώρους ονομάτων που ορίζονται στο πεδίο του τρέχοντος κόμβου, εξαιρώντας το χώρο ονομάτων **xmlns:xml**. Ο χώρος ονομάτων **xmlns:xml** δηλώνεται πάντα έμμεσα. Η σειρά των επιστρεφόμενων χώρων ονομάτων δεν ορίζεται. |
| Local | 2 | Επιστρέφει όλους τους χώρους ονομάτων που ορίζονται τοπικά στον τρέχοντα κόμβο. |
