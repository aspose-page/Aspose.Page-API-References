## DocumentInputBin class


Περιγράφει το εγκατεστημένο κουτί εισόδου σε μια συσκευή ή τη πλήρη λίστα των υποστηριζόμενων κουτιών για μια συσκευή. Οι λέξεις-κλειδιά [JobInputBin](../jobinputbin/), [DocumentInputBin](./), και [PageInputBin](../pageinputbin/) είναι αμοιβαία αποκλειστικές. Δεν πρέπει να καθορίζονται ταυτόχρονα σε ένα [PrintTicket](../printticket/) ή έγγραφο Print Capabilities.
