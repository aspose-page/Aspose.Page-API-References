## ScoredProperty class


Η κλάση που υλοποιεί μια κοινή [PrintTicket](../printticket/)**[ScoredProperty](./)**. Η βασική κλάση για όλες τις ιδιότητες που ορίζονται στο σχήμα. Ένα **[ScoredProperty](./)** στοιχείο δηλώνει μια ιδιότητα που είναι εγγενής σε έναν ορισμό [Option](../option/). Τέτοιες ιδιότητες πρέπει να συγκρίνονται κατά την αξιολόγηση του πόσο στενά μια ζητούμενη [Option](../option/) ταιριάζει με μια υποστηριζόμενη από τη συσκευή [Option](../option/). [https://docs.microsoft.com/en-us/windows/win32/printdocs/scoredproperty](https://docs.microsoft.com/en-us/windows/win32/printdocs/scoredproperty).
