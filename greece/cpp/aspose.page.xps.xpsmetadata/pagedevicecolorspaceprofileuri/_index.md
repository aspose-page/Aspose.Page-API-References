## PageDeviceColorSpaceProfileURI class


Καθορίζει ένα σχετικό URI προς τη ρίζα του πακέτου για ένα προφίλ ICC που περιέχεται σε ένα [XPS](../../aspose.page.xps/)[Document](../../aspose.page/document/). Η επεξεργασία αυτής της επιλογής εξαρτάται από τη ρύθμιση της δυνατότητας [PageDeviceColorSpaceUsage](../pagedevicecolorspaceusage/). Υποτίθεται ότι όλα τα στοιχεία που χρησιμοποιούν αυτό το προφίλ βρίσκονται ήδη στο κατάλληλο χρωματικό χώρο της συσκευής και δεν θα υποστούν διαχείριση χρώματος από τον οδηγό ή τη συσκευή. [https://docs.microsoft.com/en-us/windows/win32/printdocs/pagedevicecolorspaceprofileuri](https://docs.microsoft.com/en-us/windows/win32/printdocs/pagedevicecolorspaceprofileuri).

class PageDeviceColorSpaceProfileURI : public Aspose::Page::XPS::XpsMetadata::StringParameterInit,

                                       public Aspose::Page::XPS::XpsMetadata::IDocumentPrintTicketItem,
