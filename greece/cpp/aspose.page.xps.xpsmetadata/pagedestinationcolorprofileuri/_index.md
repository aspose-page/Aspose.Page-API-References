## PageDestinationColorProfileURI class


Καθορίζει μια σχετική αναφορά URI σε ένα προφίλ ICC που περιέχεται σε ένα [XPS](../../aspose.page.xps/)[Document](../../aspose.page/document/). Η επεξεργασία αυτής της επιλογής εξαρτάται από τη ρύθμιση της δυνατότητας [PageDeviceColorSpaceUsage](../pagedevicecolorspaceusage/). Υποτίθεται ότι όλα τα στοιχεία που χρησιμοποιούν αυτό το προφίλ βρίσκονται ήδη στο κατάλληλο χρωματικό χώρο της συσκευής και δεν θα υποστούν διαχείριση χρώματος στον οδηγό ή στη συσκευή. [https://docs.microsoft.com/en-us/windows/win32/printdocs/pagedestinationcolorprofileuri](https://docs.microsoft.com/en-us/windows/win32/printdocs/pagedestinationcolorprofileuri).

class PageDestinationColorProfileURI : public Aspose::Page::XPS::XpsMetadata::StringParameterInit,

                                       public Aspose::Page::XPS::XpsMetadata::IDocumentPrintTicketItem,
