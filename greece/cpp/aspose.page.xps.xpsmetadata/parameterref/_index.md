## ParameterRef class


Η κλάση που υλοποιεί μια κοινή αναφορά παραμέτρου [PrintTicket](../printticket/). Ένα **[ParameterRef](./)** στοιχείο ορίζει μια αναφορά σε ένα στοιχείο [ParameterInit](../parameterinit/). Ένα [ScoredProperty](../scoredproperty/) στοιχείο που περιέχει ένα [ParameterRef](./) στοιχείο δεν έχει ρητά ορισμένο [Value](../value/) στοιχείο. Αντίθετα, το [ScoredProperty](../scoredproperty/) στοιχείο λαμβάνει την τιμή του από το [ParameterInit](../parameterinit/) στοιχείο στο οποίο αναφέρεται ένα **[ParameterRef](./)** στοιχείο. [https://docs.microsoft.com/en-us/windows/win32/printdocs/parameterref](https://docs.microsoft.com/en-us/windows/win32/printdocs/parameterref).

```cpp
class ParameterRef : public Aspose::Page::XPS::XpsMetadata::PrintTicketElement,
                     public Aspose::Page::XPS::XpsMetadata::IPrintTicketItem
```
