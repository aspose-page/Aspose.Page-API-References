## PrintBoth field


Καθορίζει ότι το εξώφυλλο που υποδεικνύεται από "CoverBackSource" μπορεί να εκτυπωθεί σε οποιαδήποτε από τις δύο πλευρές του φύλλου εξωφύλλου. Εάν δεν καθοριστεί ένα στοιχείο [DocumentCoverBackSource](../../../documentcoverbacksource/)[ParameterInit](../../../parameterinit/), αυτή η [Option](../../../option/) πρέπει να αγνοηθεί.

```cpp
static System::SharedPtr<DocumentCoverBack::CoverBackOption> Aspose::Page::XPS::XpsMetadata::DocumentCoverBack::CoverBackOption::PrintBoth
```
