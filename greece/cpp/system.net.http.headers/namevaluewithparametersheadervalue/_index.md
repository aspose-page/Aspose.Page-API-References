## NameValueWithParametersHeaderValue class


Αντιπροσωπεύει ένα ζεύγος κλειδί/τιμή με παραμέτρους για χρήση σε κεφαλίδες. Αντικείμενα αυτής της κλάσης πρέπει να δημιουργούνται μόνο με τη λειτουργία [System::MakeObject()](../../system/makeobject/) function. Πάντα τυλίξτε αυτήν την κλάση σε δείκτη [System::SmartPtr](../../system/smartptr/) pointer και χρησιμοποιήστε αυτόν τον δείκτη για να τη μεταβιβάζετε σε συναρτήσεις ως όρισμα.
