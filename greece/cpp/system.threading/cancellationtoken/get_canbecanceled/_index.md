## CancellationToken::get_CanBeCanceled method


Καθορίζει εάν αυτό το token μπορεί να βρίσκεται στην κατάσταση ακυρωμένου.

```cpp
bool System::Threading::CancellationToken::get_CanBeCanceled() const
```


### ReturnValue

true εάν αυτό το token μπορεί να βρίσκεται σε κατάσταση ακύρωσης· διαφορετικά, false.
## Παρατηρήσεις



Tokens που δημιουργούνται από το [CancellationTokenSource](../../cancellationtokensource/) θα επιστρέψουν true, ενώ το token None θα επιστρέφει πάντα false.
