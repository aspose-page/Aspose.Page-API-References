## File::Copy method


Αντιγράφει το καθορισμένο αρχείο στην καθορισμένη τοποθεσία. Εάν το αρχείο προορισμού υπάρχει ήδη, μια παράμετρος καθορίζει αν πρέπει να αντικατασταθεί.

```cpp
static void System::IO::File::Copy(const String &sourceFileName, const String &destFileName, bool overwrite=false)
```


| Parameter | Type | Περιγραφή |
| --- | --- | --- |
| sourceFileName | const String\& | Μια διαδρομή του αρχείου προς αντιγραφή |
| destFileName | const String\& | Μια διαδρομή της νέας τοποθεσίας του αρχείου προς αντιγραφή |
| αντικατάσταση | bool | Αληθές εάν το υπάρχον αρχείο προορισμού πρέπει να αντικατασταθεί, ψευδές εάν η αντιγραφή πρέπει να αποτύχει εάν το αρχείο προορισμού υπάρχει ήδη |
