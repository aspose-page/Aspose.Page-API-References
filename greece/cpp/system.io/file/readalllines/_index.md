## File::ReadAllLines method


Διαβάζει το περιεχόμενο του καθορισμένου αρχείου κειμένου γραμμή προς γραμμή σε έναν πίνακα συμβολοσειρών χρησιμοποιώντας την καθορισμένη κωδικοποίηση χαρακτήρων.

```cpp
static ArrayPtr<String> System::IO::File::ReadAllLines(const String &path, const EncodingPtr &encoding=Text::Encoding::get_UTF8())
```


| Parameter | Type | Περιγραφή |
| --- | --- | --- |
| path | const String\& | Η διαδρομή του αρχείου που θα διαβαστεί |
| encoding | const EncodingPtr\& | Η κωδικοποίηση χαρακτήρων που θα χρησιμοποιηθεί |

### ReturnValue

Ένας πίνακας συμβολοσειρών, όπου κάθε στοιχείο αντιπροσωπεύει μια μεμονωμένη γραμμή από το καθορισμένο αρχείο
