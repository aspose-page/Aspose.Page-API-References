## STDIOStreamWrappingMode enum


Καθορίζει τη λειτουργία των λειτουργιών I/O που θα εκτελούν τα περιτυλίγματα σε ροές τύπου std::iostreams.

```cpp
enum class STDIOStreamWrappingMode
```

### Τιμές

| Όνομα | Τιμή | Περιγραφή |
| --- | --- | --- |
| Binary | 0 | Μια λειτουργία που επιτρέπει στις λειτουργίες εισόδου να αποκωδικοποιούν δεδομένα ροής τύπου char_type σε byte και να κωδικοποιούν byte σε δεδομένα τύπου char_type για λειτουργίες εξόδου. |
| Conversion | 1 | Μια λειτουργία που επιτρέπει στις λειτουργίες εισόδου να μετατρέπουν δεδομένα ροής από τύπο char_type σε τύπο uint8_t και αντίστροφα για λειτουργίες εξόδου. |
