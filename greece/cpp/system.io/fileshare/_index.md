## FileShare enum


Καθορίζει τι είδους πρόσβαση μπορούν να έχουν άλλα αντικείμενα [FileStream](../filestream/) σε ένα αρχείο που ανοίγεται.

```cpp
enum class FileShare
```

### Τιμές

| Όνομα | Τιμή | Περιγραφή |
| --- | --- | --- |
| None | 0 | Καμία πρόσβαση. |
| Read | 1 | Πρόσβαση μόνο για ανάγνωση. |
| Write | 2 | Πρόσβαση μόνο για εγγραφή. |
| ReadWrite | 3 | Πρόσβαση ανάγνωσης και εγγραφής. |
| Διαγραφή | 4 | Το αρχείο μπορεί να διαγραφεί. |
| Κληρονομήσιμο | 16 | Κάνει το χειριστήριο αρχείου κληρονομήσιμο από διεργασίες-παιδιά. |
