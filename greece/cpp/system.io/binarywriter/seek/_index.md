## BinaryWriter::Seek method


Ορίζει τη θέση της ροής που αντιπροσωπεύεται από το τρέχον αντικείμενο.

```cpp
int64_t System::IO::BinaryWriter::Seek(int offset, System::IO::SeekOrigin origin=System::IO::SeekOrigin::Begin)
```


| Parameter | Type | Περιγραφή |
| --- | --- | --- |
| offset | int | Η μετατόπιση σε byte σε σχέση με μια θέση που καθορίζεται από το **origin** |
| origin | System::IO::SeekOrigin | Καθορίζει τη θέση από την οποία και την κατεύθυνση προς την οποία υπολογίζεται η μετατόπιση. |

### ReturnValue

Η νέα θέση της ροής.
