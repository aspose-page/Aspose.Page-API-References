## NamespaceHandling enum


Καθορίζει εάν θα αφαιρεθούν οι διπλές δηλώσεις χώρου ονομάτων στον [XmlWriter](../xmlwriter/).

```cpp
enum class NamespaceHandling
```

### Τιμές

| Όνομα | Τιμή | Περιγραφή |
| --- | --- | --- |
| Default | 0 | Καθορίζει ότι οι διπλές δηλώσεις χώρου ονομάτων δεν θα αφαιρεθούν. |
| OmitDuplicates | 1 | Καθορίζει ότι οι διπλές δηλώσεις χώρου ονομάτων θα αφαιρεθούν. Για να αφαιρεθεί η διπλή δήλωση χώρου ονομάτων, το πρόθεμα και ο χώρος ονομάτων πρέπει να ταιριάζουν. |
