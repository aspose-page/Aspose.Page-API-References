## XmlElement::set_IsEmpty method


Ορίζει τη μορφή ετικέτας του στοιχείου.

```cpp
void System::Xml::XmlElement::set_IsEmpty(bool value)
```


| Parameter | Type | Περιγραφή |
| --- | --- | --- |
| τιμή | bool | Η τιμή προς ορισμό. Κατά τον ορισμό αυτής της τιμής, εάν οριστεί σε **true**, τα παιδιά του στοιχείου αφαιρούνται και το στοιχείο σειριοποιείται σε μορφή σύντομης ετικέτας. Εάν οριστεί σε **false**, η τιμή αλλάζει (ανεξάρτητα από το αν το στοιχείο έχει περιεχόμενο ή όχι); εάν το στοιχείο είναι κενό, σειριοποιείται σε μορφή μακράς ετικέτας. |
