## XmlWriter::get_XmlSpace method


Όταν παρακαμφθεί σε μια κληρονομημένη κλάση, λαμβάνει ένα [XmlSpace](../../xmlspace/) που αντιπροσωπεύει την τρέχουσα εμβέλεια **xml:space**.

```cpp
virtual System::Xml::XmlSpace System::Xml::XmlWriter::get_XmlSpace()
```


### ReturnValue

Ένα [XmlSpace](../../xmlspace/) που αντιπροσωπεύει την τρέχουσα εμβέλεια **xml:space**.

|||
|-|-|
| Τιμή | Σημασία |
| None | Αυτή είναι η προεπιλογή εάν δεν υπάρχει εμβέλεια xml:space. |
| Default | Η τρέχουσα εμβέλεια είναι xml:space=\"default\". |
| Preserve | Η τρέχουσα εμβέλεια είναι xml:space=\"preserve\". |
