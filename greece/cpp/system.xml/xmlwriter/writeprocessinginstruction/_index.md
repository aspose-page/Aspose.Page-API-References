## XmlWriter::WriteProcessingInstruction method


Όταν αντικαθίσταται σε μια παράγωγη κλάση, γράφει μια οδηγία επεξεργασίας με κενό μεταξύ του ονόματος και του κειμένου ως εξής: **<?name text?>**.

```cpp
virtual void System::Xml::XmlWriter::WriteProcessingInstruction(String name, String text)=0
```


| Parameter | Type | Περιγραφή |
| --- | --- | --- |
| όνομα | String | Το όνομα της οδηγίας επεξεργασίας. |
| text | String | Το κείμενο που πρέπει να συμπεριληφθεί στην οδηγία επεξεργασίας. |
