## XmlWriter::LookupPrefix method


Όταν παρακαμφθεί σε μια παράγωγη κλάση, επιστρέφει το πλησιέστερο πρόθεμα που ορίζεται στο τρέχον πεδίο ονοματοχώρου για το URI του ονοματοχώρου.

```cpp
virtual String System::Xml::XmlWriter::LookupPrefix(String ns)=0
```


| Parameter | Type | Περιγραφή |
| --- | --- | --- |
| ns | String | Το URI του ονοματοχώρου του οποίου το πρόθεμα θέλετε να βρείτε. |

### ReturnValue

Το αντίστοιχο πρόθεμα ή **nullptr** εάν δεν βρεθεί αντίστοιχο URI ονοματοχώρου στην τρέχουσα εμβέλεια.
