## XmlDeclaration::get_Standalone method


Επιστρέφει την τιμή του χαρακτηριστικού standalone.

```cpp
String System::Xml::XmlDeclaration::get_Standalone()
```


### ReturnValue

Οι έγκυρες τιμές είναι **yes** εάν όλες οι δηλώσεις οντοτήτων που απαιτούνται από το έγγραφο XML περιλαμβάνονται στο έγγραφο ή **no** εάν απαιτείται εξωτερικός ορισμός τύπου εγγράφου (DTD). Εάν το χαρακτηριστικό standalone δεν υπάρχει στη δήλωση XML, αυτή η μέθοδος επιστρέφει [String::Empty](../../../system/string/empty/).
