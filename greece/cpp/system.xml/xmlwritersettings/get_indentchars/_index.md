## XmlWriterSettings::get_IndentChars method


Επιστρέφει τη συμβολοσειρά χαρακτήρων που χρησιμοποιείται για εσοχή. Αυτή η ρύθμιση χρησιμοποιείται όταν η τιμή [XmlWriterSettings::set_Indent](../set_indent/) ορίζεται σε **true**.

```cpp
String System::Xml::XmlWriterSettings::get_IndentChars()
```


### ReturnValue

Η συμβολοσειρά χαρακτήρων που χρησιμοποιείται για εσοχή. Μπορεί να οριστεί σε οποιαδήποτε τιμή συμβολοσειράς. Ωστόσο, για να εξασφαλίσετε έγκυρο XML, θα πρέπει να καθορίζετε μόνο έγκυρους χαρακτήρες λευκού διαστήματος, όπως χαρακτήρες διαστήματος, καρτέλες, επιστροφές καρτέλας ή αλλαγές γραμμής. Η προεπιλογή είναι δύο διαστήματα.
