## XmlWriterSettings::get_NewLineChars method


Επιστρέφει τη συμβολοσειρά χαρακτήρων που θα χρησιμοποιηθεί για αλλαγές γραμμής.

```cpp
String System::Xml::XmlWriterSettings::get_NewLineChars()
```


### ReturnValue

Η συμβολοσειρά χαρακτήρων που χρησιμοποιείται για αλλαγές γραμμής. Μπορεί να οριστεί σε οποιαδήποτε τιμή συμβολοσειράς. Ωστόσο, για να εξασφαλίσετε έγκυρο XML, θα πρέπει να καθορίζετε μόνο έγκυρους χαρακτήρες λευκού διαστήματος, όπως χαρακτήρες διαστήματος, καρτέλες, επιστροφές καρέρ και αλλαγές γραμμής. Η προεπιλογή είναι **\\r\\n** (επιστροφή καρέρ, νέα γραμμή).
