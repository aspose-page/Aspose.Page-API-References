## XmlWriterSettings::get_CheckCharacters method


Επιστρέφει μια τιμή που υποδεικνύει εάν ο XML writer πρέπει να ελέγχει ώστε να διασφαλιστεί ότι όλοι οι χαρακτήρες στο έγγραφο συμμορφώνονται με την ενότητα \"2.2 Characters\" της σύστασης W3C [XML 1.0 Recommendation](https://www.w3.org/TR/REC-xml/#charsets).

```cpp
bool System::Xml::XmlWriterSettings::get_CheckCharacters()
```


### ReturnValue

**true** to do character checking; otherwise, **false**. The default is **true**.
