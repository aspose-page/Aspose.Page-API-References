## XmlAttribute::get_BaseURI method


Επιστρέφει το βασικό Uniform Resource Identifier (URI) του κόμβου.

```cpp
String System::Xml::XmlAttribute::get_BaseURI() override
```


### ReturnValue

Η θέση από την οποία φορτώθηκε ο κόμβος ή [String::Empty](../../../system/string/empty/) εάν ο κόμβος δεν έχει βασικό URI. Οι κόμβοι [Attribute](../../../system/attribute/) έχουν το ίδιο βασικό URI με το στοιχείο ιδιοκτήτη τους. Εάν ένας κόμβος χαρακτηριστικού δεν έχει στοιχείο ιδιοκτήτη, το get_BaseURI επιστρέφει [String::Empty](../../../system/string/empty/).
