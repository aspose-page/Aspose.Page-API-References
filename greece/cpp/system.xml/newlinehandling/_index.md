## NewLineHandling enum


Καθορίζει πώς να διαχειριστείτε τις αλλαγές γραμμής.

```cpp
enum class NewLineHandling
```

### Τιμές

| Όνομα | Τιμή | Περιγραφή |
| --- | --- | --- |
| Replace | 0 | Οι χαρακτήρες νέας γραμμής αντικαθίστανται ώστε να ταιριάζουν με τον χαρακτήρα που καθορίζεται στην τιμή του [XmlWriterSettings::set_NewLineChars](../xmlwritersettings/set_newlinechars/). |
| Entitize | 1 | Οι χαρακτήρες νέας γραμμής κωδικοποιούνται. Αυτή η ρύθμιση διατηρεί όλους τους χαρακτήρες όταν η έξοδος διαβάζεται από έναν κανονικοποιητικό [XmlReader](../xmlreader/). |
| None | 2 | Οι χαρακτήρες νέας γραμμής παραμένουν αμετάβλητοι. Η έξοδος είναι η ίδια με την είσοδο. |
