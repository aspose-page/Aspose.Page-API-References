## XmlReaderSettings::get_ValidationFlags method


Επιστρέφει μια τιμή που υποδεικνύει τις ρυθμίσεις επικύρωσης σχήματος. Αυτή η ρύθμιση εφαρμόζεται σε αντικείμενα [XmlReader](../../xmlreader/) που επικυρώνουν σχήματα (η τιμή [XmlReaderSettings::get_ValidationType](../get_validationtype/) είναι [ValidationType::Schema](../../validationtype/)).

```cpp
Schema::XmlSchemaValidationFlags System::Xml::XmlReaderSettings::get_ValidationFlags()
```


### ReturnValue

Ένας δυαδικός συνδυασμός τιμών απαρίθμησης που καθορίζουν επιλογές επικύρωσης. Τα XmlSchemaValidationFlags::ProcessIdentityConstraints και XmlSchemaValidationFlags::AllowXmlAttributes είναι ενεργοποιημένα από προεπιλογή. Τα XmlSchemaValidationFlags::ProcessInlineSchema, XmlSchemaValidationFlags::ProcessSchemaLocation και XmlSchemaValidationFlags::ReportValidationWarnings είναι απενεργοποιημένα από προεπιλογή.
