## XmlReader::ReadInnerXml method


Όταν υπερκαλύπτεται σε μια παράγωγη κλάση, διαβάζει όλο το περιεχόμενο, συμπεριλαμβανομένου του σήματος, ως συμβολοσειρά.

```cpp
virtual String System::Xml::XmlReader::ReadInnerXml()
```


### ReturnValue

Όλο το περιεχόμενο XML, συμπεριλαμβανομένης της σήμανσης, στον τρέχοντα κόμβο. Εάν ο τρέχων κόμβος δεν έχει παιδιά, επιστρέφεται μια κενή συμβολοσειρά. Εάν ο τρέχων κόμβος δεν είναι ούτε στοιχείο ούτε χαρακτηριστικό, επιστρέφεται μια κενή συμβολοσειρά.
