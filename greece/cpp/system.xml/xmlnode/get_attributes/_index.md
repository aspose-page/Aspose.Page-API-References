## XmlNode::get_Attributes method


Επιστρέφει ένα [XmlAttributeCollection](../../xmlattributecollection/) που περιέχει τα χαρακτηριστικά αυτού του κόμβου.

```cpp
virtual SharedPtr<XmlAttributeCollection> System::Xml::XmlNode::get_Attributes() final
```


### ReturnValue

Ένα [XmlAttributeCollection](../../xmlattributecollection/) που περιέχει τα χαρακτηριστικά του κόμβου. Εάν ο κόμβος είναι τύπου [XmlNodeType::Element](../../xmlnodetype/), επιστρέφονται τα χαρακτηριστικά του κόμβου. Διαφορετικά, αυτή η μέθοδος επιστρέφει **nullptr**.
