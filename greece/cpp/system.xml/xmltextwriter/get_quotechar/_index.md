## XmlTextWriter::get_QuoteChar method


Επιστρέφει ποιον χαρακτήρα να χρησιμοποιήσει για την παράθεση τιμών χαρακτηριστικών.

```cpp
char16_t System::Xml::XmlTextWriter::get_QuoteChar()
```


### ReturnValue

Ο χαρακτήρας που χρησιμοποιείται για την παράθεση τιμών χαρακτηριστικών. Πρέπει να είναι μονό απόστροφο **'** ή διπλό απόστροφο **\"**. Η προεπιλογή είναι το διπλό απόστροφο.
