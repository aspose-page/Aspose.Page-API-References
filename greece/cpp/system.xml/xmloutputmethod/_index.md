## XmlOutputMethod enum


Καθορίζει τη μέθοδο που χρησιμοποιείται για τη σειριοποίηση της εξόδου του [XmlWriter](../xmlwriter/).

```cpp
enum class XmlOutputMethod
```

### Τιμές

| Όνομα | Τιμή | Περιγραφή |
| --- | --- | --- |
| Xml | 0 | Σειριοποιήστε σύμφωνα με τους κανόνες XML 1.0. |
| Html | 1 | Σειριοποιήστε σύμφωνα με τους κανόνες HTML που καθορίζονται από το XSLT. |
| Text | 2 | Σειριοποιήστε μόνο μπλοκ κειμένου. |
| AutoDetect | 3 | Χρησιμοποιήστε τους κανόνες XSLT για να επιλέξετε μεταξύ των μεθόδων εξόδου [XmlOutputMethod::Xml](./) και [XmlOutputMethod::Html](./) κατά την εκτέλεση. |
