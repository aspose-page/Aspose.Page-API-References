## XmlNodeOrder enum


Περιγράφει τη σειρά εγγράφου ενός κόμβου σε σύγκριση με έναν δεύτερο κόμβο.

```cpp
enum class XmlNodeOrder
```

### Τιμές

| Όνομα | Τιμή | Περιγραφή |
| --- | --- | --- |
| Before | 0 | Ο τρέχων κόμβος αυτού του περιηγητή είναι πριν τον τρέχοντα κόμβο του παρεχόμενου περιηγητή. |
| Μετά | 1 | Ο τρέχων κόμβος αυτού του περιηγητή είναι μετά τον τρέχοντα κόμβο του παρεχόμενου περιηγητή. |
| Ίδιο | 2 | Οι δύο περιηγητές είναι τοποθετημένοι στον ίδιο κόμβο. |
| Άγνωστο | 3 | Οι θέσεις των κόμβων δεν μπορούν να προσδιοριστούν με τη σειρά του εγγράφου, σε σχέση μεταξύ τους. Αυτό μπορεί να συμβεί εάν οι δύο κόμβοι βρίσκονται σε διαφορετικά δέντρα. |
