## XmlNodeReader::ReadString method


Διαβάζει τα περιεχόμενα ενός στοιχείου ή κόμβου κειμένου ως συμβολοσειρά.

```cpp
String System::Xml::XmlNodeReader::ReadString() override
```


### ReturnValue

Τα περιεχόμενα του στοιχείου ή του κόμβου τύπου κειμένου (Μπορεί να περιλαμβάνει CDATA, [Text](../../../system.text/) κόμβους, κ.λπ.). Αυτό μπορεί να είναι μια κενή συμβολοσειρά εάν ο αναγνώστης είναι τοποθετημένος σε κάτι διαφορετικό από ένα στοιχείο ή κόμβο κειμένου, ή εάν δεν υπάρχει άλλο περιεχόμενο κειμένου για επιστροφή στο τρέχον πλαίσιο. Σημείωση: Ο κόμβος κειμένου μπορεί να είναι είτε στοιχείο είτε κόμβος κειμένου χαρακτηριστικού.
