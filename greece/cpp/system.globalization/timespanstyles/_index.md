## TimeSpanStyles enum


Καθορίζει τις επιλογές μορφοποίησης που προσαρμόζουν την ανάλυση συμβολοσειράς για μεθόδους που μετατρέπουν μια συμβολοσειρά αναπαράστασης χρονικού διαστήματος σε αντικείμενο Timespan.

```cpp
enum class TimeSpanStyles : int32_t
```

### Τιμές

| Όνομα | Τιμή | Περιγραφή |
| --- | --- | --- |
| None | 0 | Το χρονικό διάστημα που αντιπροσωπεύεται από τη συμβολοσειρά inptut ερμηνεύεται ως αρνητικό μόνο εάν υπάρχει αρνητικό πρόσημο. |
