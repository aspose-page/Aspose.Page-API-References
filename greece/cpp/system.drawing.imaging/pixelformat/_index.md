## PixelFormat enum


Καθορίζει τη μορφή δεδομένων χρώματος ενός pixel.

```cpp
enum class PixelFormat
```

### Τιμές

| Όνομα | Τιμή | Περιγραφή |
| --- | --- | --- |
| Indexed | 65536 | Καθορίζει ότι τα δεδομένα pixel περιέχουν τιμές χρωμάτων με ευρετήριο, που σημαίνει ότι είναι δείκτης σε χρώματα στον σύστημα χρωματικού πίνακα. |
| Gdi | 131072 | Καθορίζει ότι τα δεδομένα pixel περιέχουν χρώματα GDI. |
| Alpha | 262144 | Καθορίζει ότι τα δεδομένα pixel περιέχουν τιμές άλφα που δεν είναι προ-πολλαπλασιασμένες. |
| PAlpha | 524288 | Καθορίζει ότι τα δεδομένα pixel περιέχουν προ-πολλαπλασιασμένες τιμές άλφα. |
| Εκτεταμένο | 1048576 | Δεσμευμένο. |
| Κανονικό | 2097152 | Καθορίζει τη μορφή pixel των 32 bits ανά pixel με βάθος χρώματος 24-bit και κανάλι άλφα 8-bit. |
| Απροσδιόριστο | 0 | Καθορίζει ότι η μορφή pixel είναι απροσδιόριστη. |
| DontCare | 0 | Η μορφή pixel δεν έχει καθοριστεί. |
| Format1bppIndexed | n/a | Καθορίζει ότι η μορφή pixel είναι χρωματική με ευρετήριο 1 bit ανά pixel. |
| Format4bppIndexed | n/a | Καθορίζει ότι η μορφή pixel είναι χρωματική με ευρετήριο 4 bits ανά pixel. |
| Format8bppIndexed | n/a | Καθορίζει ότι η μορφή pixel είναι χρωματική με ευρετήριο 8 bits ανά pixel. |
| Format16bppGrayScale | n/a | Καθορίζει ότι η μορφή pixel είναι 16 bits ανά pixel. Οι πληροφορίες χρώματος καθορίζουν 65536 αποχρώσεις του γκρι. |
| Format16bppRgb555 | n/a | Καθορίζει ότι η μορφή pixel είναι 16 bits ανά pixel με 5 bits για κάθε ένα από τα συστατικά κόκκινο, πράσινο και μπλε και το υπόλοιπο bit δεν χρησιμοποιείται. |
| Format16bppRgb565 | n/a | Καθορίζει ότι η μορφή pixel είναι 16 bits ανά pixel με 5 bits για το κόκκινο, 6 bits για το πράσινο και 5 bits για το μπλε. |
| Format16bppArgb1555 | n/a | Καθορίζει ότι η μορφή pixel είναι 16 bits ανά pixel με 5 bits για κάθε ένα από τα συστατικά κόκκινο, πράσινο και μπλε και 1 bit για το άλφα. |
| Format24bppRgb | n/a | Καθορίζει ότι η μορφή pixel είναι 24 bits ανά pixel με 8 bits για κάθε ένα από τα συστατικά κόκκινο, πράσινο και μπλε. |
| Format32bppRgb | n/a | Καθορίζει ότι η μορφή εικονοστοιχείου είναι 32 bit ανά εικονοστοιχείο με 8 bit για κάθε ένα από τα κόκκινα, πράσινα και μπλε συστατικά και τα υπόλοιπα 8 bit δεν χρησιμοποιούνται. |
| Format32bppArgb | n/a | Καθορίζει ότι η μορφή εικονοστοιχείου είναι 32 bit ανά εικονοστοιχείο με 8 bit για κάθε ένα από τα κόκκινα, πράσινα και μπλε συστατικά και 8 bit για το άλφα. |
| Format32bppPArgb | n/a | Καθορίζει ότι η μορφή εικονοστοιχείου είναι 32 bit ανά εικονοστοιχείο με 8 bit για κάθε ένα από τα κόκκινα, πράσινα και μπλε συστατικά και 8 bit για το άλφα. Τα κόκκινα, πράσινα και μπλε συστατικά προ-πολλαπλασιάζονται σύμφωνα με την τιμή του συστατικού άλφα. |
| Format48bppRgb | n/a | Καθορίζει ότι η μορφή εικονοστοιχείου είναι 48 bit ανά εικονοστοιχείο με 16 bit για κάθε ένα από τα κόκκινα, πράσινα και μπλε συστατικά. |
| Format64bppArgb | n/a | Καθορίζει ότι η μορφή εικονοστοιχείου είναι 64 bit ανά εικονοστοιχείο με 16 bit για κάθε ένα από τα κόκκινα, πράσινα και μπλε συστατικά και 16 bit για το άλφα. |
| Format64bppPArgb | n/a | Καθορίζει ότι η μορφή εικονοστοιχείου είναι 64 bit ανά εικονοστοιχείο με 16 bit για κάθε ένα από τα κόκκινα, πράσινα και μπλε συστατικά και 16 bit για το άλφα. Τα κόκκινα, πράσινα και μπλε συστατικά προ-πολλαπλασιάζονται σύμφωνα με την τιμή του συστατικού άλφα. |
| Format32bppCMYK | n/a | Καθορίζει ότι η μορφή εικονοστοιχείου είναι 32 bit ανά εικονοστοιχείο με 8 bit για κάθε ένα από τα κυανό, ματζέντα, κίτρινο και μαύρο συστατικά. |
| Max | 16 | Η μέγιστη τιμή αυτού του enum. |
