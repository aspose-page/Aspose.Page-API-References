## ImageFlags enum


Αντιπροσωπεύει τα χαρακτηριστικά των δεδομένων εικονοστοιχείου που αντιπροσωπεύονται από ένα αντικείμενο [Image](../../system.drawing/image/).

```cpp
enum class ImageFlags
```

| Όνομα | Τιμή | Περιγραφή |
| --- | --- | --- |
| None | 0 |  |
| Κλιμακώσιμο | 1 | Κλιμακώσιμο. |
| HasAlpha | 2 | Περιέχει πληροφορίες alpha. |
| HasTranslucent | 4 | Υπάρχουν τιμές alpha μεγαλύτερες από 0 και μικρότερες από 255. |
| PartiallyScalable | 8 |  |
| ColorSpaceRgb | 16 | Τα δεδομένα pixel αναπαρίστανται στο χρωματικό χώρο RGB. |

| ColorSpaceGray | 64 | Τα δεδομένα pixel είναι σε αποχρώσεις του γκρι. |
| ColorSpaceYcbcr | 128 | Τα δεδομένα pixel αναπαρίστανται στο χρωματικό χώρο YCBCR. |
| ColorSpaceYcck | 256 | Τα δεδομένα pixel αναπαρίστανται στο χρωματικό χώρο YCCK. |
| HasRealDpi | 4096 | Οι πληροφορίες DPI αποθηκεύονται στην εικόνα. |
