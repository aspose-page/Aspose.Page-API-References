## SystemFonts class


Παρέχει ένα σύνολο προ-δημιουργημένων αντικειμένων [Font](../font/) που αντιπροσωπεύουν γραμματοσειρές που χρησιμοποιούνται για την εμφάνιση κειμένου σε στοιχεία εμφάνισης των [Windows](../../system.windows/). Αυτός είναι ένας στατικός τύπος χωρίς υπηρεσίες στιγμιοτύπων. Δεν πρέπει ποτέ να δημιουργείτε στιγμιότυπα αυτού με κανέναν τρόπο.

```cpp
class SystemFonts : public System::Object
```
