## Point::Add method


Προσθέτει τις τιμές πλάτους και ύψους του καθορισμένου αντικειμένου [Size](../../size/) στις τιμές των συντεταγμένων X και Y του καθορισμένου αντικειμένου [Point](../) αντίστοιχα.

```cpp
static Point System::Drawing::Point::Add(const Point &point, const Size &size)
```


| Parameter | Type | Περιγραφή |
| --- | --- | --- |
| σημείο | const Point\& | Το σημείο προς μετατόπιση |
| size | const Size\& | Το αντικείμενο [Size](../../size/) που καθορίζει τις τιμές προς προσθήκη στις τιμές των συντεταγμένων του **point** |

### ReturnValue

Ένα νέο αντικείμενο [Point](../) του οποίου η τιμή της συντεταγμένης X είναι ίση με το άθροισμα της τιμής X του **point** και της τιμής πλάτους του **size**, και η τιμή της συντεταγμένης Y είναι ίση με το άθροισμα της τιμής Y του **point** και της τιμής ύψους του **size**
