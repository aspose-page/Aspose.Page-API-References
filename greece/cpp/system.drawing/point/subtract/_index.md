## Point::Subtract method


Αφαιρεί τις τιμές του πλάτους και του ύψους του καθορισμένου αντικειμένου [Size](../../size/) από τις τιμές των συντεταγμένων X και Y του καθορισμένου αντικειμένου [Point](../) αντίστοιχα.

```cpp
static Point System::Drawing::Point::Subtract(const Point &point, const Size &size)
```


| Parameter | Type | Περιγραφή |
| --- | --- | --- |
| σημείο | const Point\& | Το σημείο προς μετατόπιση |
| size | const Size\& | Το αντικείμενο [Size](../../size/) που καθορίζει τις τιμές προς αφαίρεση από τις τιμές των συντεταγμένων του **point** |

### ReturnValue

Ένα νέο αντικείμενο [Point](../) του οποίου η τιμή της συντεταγμένης X είναι ίση με το αποτέλεσμα της αφαίρεσης της τιμής του πλάτους του **size** από την τιμή της συντεταγμένης X του **point** και η τιμή της συντεταγμένης Y είναι ίση με το αποτέλεσμα της αφαίρεσης της τιμής του ύψους του **size** από την τιμή της συντεταγμένης Y του **point**
