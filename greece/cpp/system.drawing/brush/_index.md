## Brush class


Μια βασική κλάση για κλάσεις που αντιπροσωπεύουν γεμιστικά που χρησιμοποιούνται για τη γέμιση των εσωτερικών περιοχών γραφικών σχημάτων. Τα αντικείμενα αυτής της κλάσης πρέπει να κατανεμηθούν μόνο με τη λειτουργία [System::MakeObject()](../../system/makeobject/). Ποτέ μην δημιουργείτε παρουσία αυτού του τύπου στη στοίβα ή χρησιμοποιώντας τον τελεστή new, καθώς αυτό θα οδηγήσει σε σφάλματα χρόνου εκτέλεσης και/ή σφάλματα ελέγχου. Πάντα τυλίξτε αυτήν την κλάση σε δείκτη [System::SmartPtr](../../system/smartptr/) και χρησιμοποιήστε αυτόν τον δείκτη για να τη μεταβιβάζετε στις συναρτήσεις ως όρισμα.

```cpp
class Brush : public System::ICloneable,
              public System::IDisposable
```
