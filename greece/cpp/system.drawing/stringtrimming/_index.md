## StringTrimming enum


Καθορίζει πώς πρέπει να περικόπτονται χαρακτήρες από μια συμβολοσειρά που δεν χωράει στο σχήμα διάταξης.

```cpp
enum class StringTrimming
```

### Τιμές

| Όνομα | Τιμή | Περιγραφή |
| --- | --- | --- |
| None | 0 | Χωρίς περικοπή. |
| Character | 1 | Περικοπή στον πλησιέστερο χαρακτήρα. |
| Word | 2 | Περικοπή στην πλησιέστερη λέξη. |
| EllipsisCharacter | 3 | Περικοπή στον πλησιέστερο χαρακτήρα και εισαγωγή αποσιωπητικών στο τέλος της συμβολοσειράς. |
| EllipsisWord | 4 | Περικοπή στην πλησιέστερη λέξη και εισαγωγή αποσιωπητικών στο τέλος της συμβολοσειράς. |
| EllipsisPath | 5 | Το κέντρο αφαιρείται από τις περικομμένες γραμμές και αντικαθίσταται από αποσιωπητικά. Διατηρήστε όσο το δυνατόν περισσότερο το τελευταίο τμήμα της γραμμής που διαχωρίζεται με κάθετο. |
