## PointF::Subtract(const PointF\&, const Size\&) method


Αφαιρεί τις τιμές πλάτους και ύψους του καθορισμένου αντικειμένου [Size](../../size/) από τις τιμές των συντεταγμένων X και Y του καθορισμένου αντικειμένου [PointF](../) αντίστοιχα.

```cpp
static PointF System::Drawing::PointF::Subtract(const PointF &point, const Size &size)
```


| Parameter | Type | Περιγραφή |
| --- | --- | --- |
| σημείο | const PointF\& | Το σημείο προς μετατόπιση |
| size | const Size\& | Το αντικείμενο [Size](../../size/) που καθορίζει τις τιμές προς αφαίρεση από τις τιμές των συντεταγμένων του **point** |

### ReturnValue

Ένα νέο αντικείμενο [PointF](../) του οποίου η τιμή της συντεταγμένης X είναι ίση με το αποτέλεσμα της αφαίρεσης της τιμής πλάτους του **size** από την τιμή της συντεταγμένης X του **point** και η τιμή της συντεταγμένης Y είναι ίση με το αποτέλεσμα της αφαίρεσης της τιμής ύψους του **size** από την τιμή της συντεταγμένης Y του **point**
