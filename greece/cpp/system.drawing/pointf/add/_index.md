## PointF::Add(const PointF\&, const Size\&) method


Προσθέτει τις τιμές πλάτους και ύψους του καθορισμένου αντικειμένου [Size](../../size/) στις τιμές των συντεταγμένων X και Y του καθορισμένου αντικειμένου [PointF](../) αντίστοιχα.

```cpp
static PointF System::Drawing::PointF::Add(const PointF &point, const Size &size)
```


| Parameter | Type | Περιγραφή |
| --- | --- | --- |
| σημείο | const PointF\& | Το σημείο προς μετατόπιση |
| size | const Size\& | Το αντικείμενο [Size](../../size/) που καθορίζει τις τιμές προς προσθήκη στις τιμές των συντεταγμένων του **point** |

### ReturnValue

Ένα νέο αντικείμενο [PointF](../) του οποίου η τιμή της συντεταγμένης X είναι ίση με το άθροισμα της τιμής της συντεταγμένης X του **point** και της τιμής πλάτους του **size**, και η τιμή της συντεταγμένης Y είναι ίση με το άθροισμα της τιμής της συντεταγμένης Y του **point** και της τιμής ύψους του **size**.
