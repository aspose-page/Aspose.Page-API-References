## XpsDocument.CreateArcSegment method

Δημιουργεί ένα νέο τμήμα ελλειπτικού τόξου.

```csharp
public XpsArcSegment CreateArcSegment(PointF point, SizeF size, float rotationAngle, 
    bool isLargeArc, XpsSweepDirection sweepDirection, bool isStroked = true)
```

| Παράμετρος | Τύπος | Περιγραφή |
| --- | --- | --- |
| point | PointF | Το τελικό σημείο του ελλειπτικού τόξου. |
| size | SizeF | Η ακτίνα x και y του ελλειπτικού τόξου ως ζεύγος x,y. |
| rotationAngle | Single | Υποδεικνύει πώς περιστρέφεται η έλλειψη σε σχέση με το τρέχον σύστημα συντεταγμένων. |
| isLargeArc | Boolean | Καθορίζει εάν το τόξο σχεδιάζεται με σάρωση 180 ή μεγαλύτερη. |
| sweepDirection | XpsSweepDirection | Η κατεύθυνση προς την οποία σχεδιάζεται το τόξο. |
| isStroked | Boolean | Καθορίζει εάν σχεδιάζεται η διαδρομή για αυτό το τμήμα της διαδρομής. |
