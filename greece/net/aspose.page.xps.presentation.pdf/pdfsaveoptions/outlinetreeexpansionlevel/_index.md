## PdfSaveOptions.OutlineTreeExpansionLevel property

Καθορίζει μέχρι ποιο επίπεδο θα πρέπει να επεκτείνεται το περίγραμμα του εγγράφου όταν ανοίγει το αρχείο PDF σε πρόγραμμα προβολής. 1 - εμφανίζονται μόνο τα στοιχεία περίγραμμα πρώτου επιπέδου, 2 - εμφανίζονται μόνο τα στοιχεία περιγράμματος πρώτου και δεύτερου επιπέδου, και ούτω καθεξής. Η προεπιλογή είναι 1.

```csharp
public int OutlineTreeExpansionLevel { get; set; }
```
