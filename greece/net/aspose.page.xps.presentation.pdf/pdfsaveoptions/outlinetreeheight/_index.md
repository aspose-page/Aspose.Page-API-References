## PdfSaveOptions.OutlineTreeHeight property

Καθορίζει το ύψος του δέντρου περιγράμματος του εγγράφου προς αποθήκευση. 0 - το δέντρο περιγράμματος δεν θα μετατραπεί, 1 - θα μετατραπούν μόνο τα στοιχεία του πρώτου επιπέδου, και ούτω καθεξής. Η προεπιλογή είναι 10.

```csharp
public int OutlineTreeHeight { get; set; }
```
