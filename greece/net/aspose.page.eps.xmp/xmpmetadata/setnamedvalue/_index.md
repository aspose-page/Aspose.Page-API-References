## XmpMetadata.SetNamedValue method

Ορίζει την ονομασμένη τιμή σε μια δομή. Η προηγούμενη ονομασμένη τιμή, εάν υπάρχει ήδη, θα αντικατασταθεί με νέα.

```csharp
public void SetNamedValue(string structureKey, string valueKey, XmpValue value)
```

| Παράμετρος | Τύπος | Περιγραφή |
| --- | --- | --- |
| structureKey | String | Κλειδί της δομής για αναζήτηση στο λεξικό. |
| valueKey | String | Όνομα της τιμής που θα οριστεί στη δομή. |
| value | XmpValue | Αξία που πρέπει να τεθεί στη δομή. |
