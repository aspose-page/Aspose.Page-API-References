## FillAndStrokeText(string, Font, float, float, Brush, Pen) {#fillandstroketext_3}

Προσθέτει μια συμβολοσειρά κειμένου συμπληρώνοντας το εσωτερικό των γλυφών και σχεδιάζοντας περιγράμματα γλυφών.

```csharp
public void FillAndStrokeText(string text, Font font, float x, float y, Brush fillPaint, Pen stroke)
```

| Παράμετρος | Τύπος | Περιγραφή |
| --- | --- | --- |
| text | String | Το κείμενο για προσθήκη. |
| font | Font | Γραμματοσειρά συστήματος που θα χρησιμοποιηθεί για τη σχεδίαση κειμένου. |
| x | Single | X συντεταγμένη για την προέλευση κειμένου. |
| y | Single | Y συντεταγμένες για την προέλευση κειμένου. |
| fillPaint | Brush | Το γέμισμα που χρησιμοποιείται για τη ζωγραφική εσωτερικών γλυφών. |
| stroke | Pen | Το εγκεφαλικό επεισόδιο που χρησιμοποιείται για τη χάραξη περιγραμμάτων γλυφών. |
