## PsDocument.DrawTransparentImage method

Σχεδιάστε μετασχηματισμένη διαφανή εικόνα. Εάν η εικόνα δεν έχει κανάλι Alpha, θα σχεδιαστεί ως αδιαφανές image

```csharp
public void DrawTransparentImage(Bitmap image, Matrix transform, int transparencyThreshold)
```

| Παράμετρος | Τύπος | Περιγραφή |
| --- | --- | --- |
| image | Bitmap | Η εικόνα για να σχεδιάσετε. |
| transform | Matrix | Η μήτρα για τη μετατροπή της εικόνας. |
| transparencyThreshold | Int32 | Ένα όριο που καθορίζει από ποια τιμή του εικονοστοιχείου διαφάνειας θα ερμηνευτεί ως πλήρως διαφανές. Όλες οι τιμές κάτω από αυτό το όριο θα ερμηνευθούν ως πλήρως αδιαφανείς. |
