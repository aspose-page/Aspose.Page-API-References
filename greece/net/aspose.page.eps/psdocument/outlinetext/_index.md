## OutlineText(string, float[], Font, float, float) {#outlinetext}

```csharp
public void OutlineText(string text, float[] advances, Font font, float x, float y)
```

| Παράμετρος | Τύπος | Περιγραφή |

| text | String | Το κείμενο για προσθήκη. |
| advances | Single[] | Μια σειρά από γλυφές πλάτους. Το μήκος του πρέπει να είναι σύμφωνο με τον αριθμό των γλυφών στη χορδή. |

| y | Single | Y συντεταγμένες για την προέλευση κειμένου. |
