Το **Aspose.Page.EPS** είναι ένας χώρος ονομάτων ρίζας για όλες τις κλάσεις που ασχολούνται με αρχεία PS/EPS.

## Τάξεις

| Τάξη | Περιγραφή |
| --- | --- |
| [FontConstants](./fontconstants/) | Αυτή η κλάση ορίζει ένα σύνολο σταθερών για την αποθήκευση γραμματοσειρών. |
| [GradientBrush](./gradientbrush/) | Αυτή η κλάση χρησιμοποιείται για την ενθυλάκωση του LinearGradientBrush και του PathGradientBrush με δυνατότητα ρύθμισης της λειτουργίας αναδίπλωσης σε clamp. Οι εγγενείς βούρτσες διαβάθμισης κάνουν πάντα μια εξαίρεση όταν κάποιος προσπαθεί να ορίσει την ιδιότητα WrapMode σε WrapMode.Clamp. |
| [LoadOptions](./loadoptions/) | Βασική κλάση για επιλογές φόρτωσης εγγράφων. |
| [PageConstants](./pageconstants/) | Αυτή η κλάση ορίζει ένα σύνολο σταθερών που περιγράφουν μια σελίδα. Παρέχονται αντικείμενα ευκολίας για διάφορα περιθώρια, προσανατολισμούς, αλλαγή κλίμακας και τυπικά μεγέθη σελίδας. |
| [PsConverterException](./psconverterexception/) | Αυτή η κλάση περιέχει πληροφορίες σχετικά με ένα σφάλμα που εμφανίζεται ενώ το αρχείο PS μετατρέπεται σε έγγραφο PDF. |
| [PsDocument](./psdocument/) | Αυτή η τάξη ενσωματώνει έγγραφα PS/EPS. |
| [PsDocumentException](./psdocumentexception/) | Αυτή η κλάση περιέχει πληροφορίες σχετικά με ένα σφάλμα που εμφανίζεται κατά τη δημιουργία, επεξεργασία και αποθήκευση του αρχείου PostScript. |
| [PsLoadOptions](./psloadoptions/) | Επιλογές φόρτωσης εγγράφου PS. |
