## GradientBrush class

Αυτή η κλάση χρησιμοποιείται για την ενθυλάκωση του LinearGradientBrush και του PathGradientBrush με δυνατότητα ρύθμισης της λειτουργίας αναδίπλωσης σε clamp. Οι εγγενείς βούρτσες διαβάθμισης κάνουν πάντα μια εξαίρεση όταν κάποιος προσπαθεί να ορίσει την ιδιότητα WrapMode σε WrapMode.Clamp.

```csharp
public class GradientBrush : Brush
```
