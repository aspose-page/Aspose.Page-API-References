## SaveOptions.AdditionalFontsFolders property

Καθορίζει πρόσθετους φακέλους στους οποίους ο μετατροπέας θα πρέπει να βρει γραμματοσειρές για το έγγραφο εισόδου. Ο προεπιλεγμένος φάκελος είναι φάκελος τυπικών γραμματοσειρών όπου το OS βρίσκει γραμματοσειρές για εσωτερικές ανάγκες.

```csharp
public string[] AdditionalFontsFolders { get; set; }
```
