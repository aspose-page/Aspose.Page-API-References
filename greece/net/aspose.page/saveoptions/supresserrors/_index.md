## SaveOptions.SupressErrors property

Καθορίζει εάν τα σφάλματα πρέπει να καταστραφούν ή όχι. Εάν προστεθούν αληθινά κατασταλμένα σφάλματα[`Exceptions`](../exceptions/) list. Εάν είναι false, το πρώτο σφάλμα θα τερματίσει το πρόγραμμα.

```csharp
public virtual bool SupressErrors { get; set; }
```
