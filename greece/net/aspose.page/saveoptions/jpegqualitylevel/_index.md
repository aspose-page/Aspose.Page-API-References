## SaveOptions.JpegQualityLevel property

Η κατηγορία Ποιότητα καθορίζει το επίπεδο συμπίεσης για μια εικόνα. Οι διαθέσιμες τιμές είναι από 0 έως 100. Όσο μικρότερος είναι ο καθορισμένος αριθμός, τόσο μεγαλύτερη είναι η συμπίεση και επομένως χαμηλότερη η ποιότητα της εικόνας. Η τιμή 0 έχει ως αποτέλεσμα εικόνα χαμηλότερης ποιότητας, ενώ η τιμή 100 έχει ως αποτέλεσμα την υψηλότερη.

```csharp
public int JpegQualityLevel { get; set; }
```
