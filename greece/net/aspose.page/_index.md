Το **Aspose.Page**είναι ένας ριζικός χώρος ονομάτων για όλες τις κλάσεις της βιβλιοθήκης Aspose.Page που είτε βρίσκονται απευθείας σε αυτήν like **Συσκευή** ή έμμεσα μέσω πολλών υποχώρων ονομάτων.

## Τάξεις

| Τάξη | Περιγραφή |
| --- | --- |
| [BuildVersionInfo](./buildversioninfo/) | Αυτή η κλάση παρέχει πληροφορίες σχετικά με την τρέχουσα κατασκευή προϊόντων. |
| [Device](./device/) | Αυτή η κλάση ενσωματώνει την απόδοση του εγγράφου σε μια αφηρημένη συσκευή. Η απόδοση του εγγράφου πραγματοποιείται σελίδα προς σελίδα. |
| [Document](./document/) | Μια υπερκλάση για όλα τα ενθυλακωμένα έγγραφα. |
| [ExternalFontCache](./externalfontcache/) | Χρησιμοποιήστε αυτήν την κλάση για να αποκτήσετε ενθυλάκωση γραμματοσειράς σε μορφή που είναι αποδεκτή από[`Device`](../aspose.page/device/) . |
| [License](./license/) | Παρέχει μεθόδους για την άδεια χρήσης του στοιχείου. |
| [Margins](./margins/) | Αυτή η κλάση περικλείει πάνω, αριστερά, κάτω και δεξιά περιθώρια. |
| [Metered](./metered/) | Παρέχει μεθόδους για να ορίσετε μετρημένο κλειδί. |
| [SaveOptions](./saveoptions/) | Αυτή η κλάση περιέχει επιλογές που είναι απαραίτητες για τη διαχείριση της διαδικασίας μετατροπής. |
| [UserProperties](./userproperties/) | Ειδική κλάση ιδιοτήτων που επιτρέπει τον ορισμό πληκτρολογημένων ιδιοτήτων και την επιστροφή . Επιτρέπει επίσης την αναζήτηση της σύνδεσης δύο προεπιλεγμένων ιδιοτήτων objects εάν αυτό το αντικείμενο ιδιότητας δεν περιέχει την ιδιότητα. |
## Διεπαφές

| Διεπαφή | Περιγραφή |
| --- | --- |
| [IGlyph](./iglyph/) | Αυτή η διεπαφή παρέχει πρόσβαση στις κύριες παραμέτρους των γλυφών. |
| [IMultiPageDevice](./imultipagedevice/) | Αυτή η διεπαφή περιέχει μεθόδους χειρισμού πολυσέλιδης συσκευής. |
| [IStreamable](./istreamable/) | Αυτή η διεπαφή χρησιμοποιείται για συσκευές που αποδίδουν το έγγραφο σε μια ροή. |
| [ITrFont](./itrfont/) | Αυτή η διεπαφή παρέχει πρόσβαση στις κύριες παραμέτρους της γραμματοσειράς. |
