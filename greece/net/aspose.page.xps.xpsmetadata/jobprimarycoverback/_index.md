## JobPrimaryCoverBack class

Περιγράφει το πίσω (τελικό) φύλλο εξωφύλλου. Κάθε εργασία θα έχει ένα ξεχωριστό κύριο φύλλο. Το εξώφυλλο πρέπει να τυπωθεί στο[`PageMediaSize`](../pagemediasize/) και[`PageMediaType`](../pagemediatype/) Το χρησιμοποιείται για την τελική σελίδα της εργασίας. Το εξώφυλλο θα πρέπει να ενσωματωθεί στο processing options (όπως π.χ[`JobDuplexAllDocumentsContiguously`](../jobduplexalldocumentscontiguously/) ,[`JobNUpAllDocumentsContiguously`](../jobnupalldocumentscontiguously/) ) όπως υποδεικνύεται από την καθορισμένη επιλογή. https://docs.microsoft.com/en-us/windows/win32/printdocs/jobprimarycoverback

```csharp
public sealed class JobPrimaryCoverBack : Feature, IJobPrintTicketItem
```
