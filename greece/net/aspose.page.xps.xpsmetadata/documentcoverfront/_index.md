## DocumentCoverFront class

Περιγράφει το μπροστινό (αρχικό) φύλλο εξωφύλλου. Κάθε έγγραφο θα έχει ξεχωριστό φύλλο. Το εξώφυλλο θα πρέπει να τυπωθεί στο[`PageMediaSize`](../pagemediasize/) και[`PageMediaType`](../pagemediatype/) χρησιμοποιείται για την πρώτη σελίδα του εγγράφου. Το εξώφυλλο θα πρέπει να ενσωματωθεί στο processing options (όπως π.χ[`DocumentDuplex`](../documentduplex/) ,[`DocumentNUp`](../documentnup/) ) όπως υποδεικνύεται από την καθορισμένη επιλογή. https://docs.microsoft.com/en-us/windows/win32/printdocs/documentcoverfront

```csharp
public sealed class DocumentCoverFront : Feature, IDocumentPrintTicketItem, IJobPrintTicketItem
```
