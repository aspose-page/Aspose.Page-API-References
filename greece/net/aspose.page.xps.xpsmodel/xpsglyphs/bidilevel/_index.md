## XpsGlyphs.BidiLevel property

Επιστρέφει/ορίζει την τιμή που καθορίζει το επίπεδο ένθεσης αμφίδρομης κατεύθυνσης του αλγόριθμου Unicode. Οι ζυγές τιμές υποδηλώνουν διάταξη από αριστερά προς τα δεξιά, οι περιττές τιμές υποδηλώνουν διάταξη από δεξιά προς τα αριστερά. Η διάταξη από δεξιά προς αριστερά τοποθετεί την αρχή εκτέλεσης στη δεξιά πλευρά του πρώτου γλυφού, με θετικά πλάτη προώθησης (που αντιπροσωπεύουν προόδους προς τα αριστερά) τοποθετώντας επόμενα γλυφές στα αριστερά της προηγούμενης γλυφής.

```csharp
public int BidiLevel { get; set; }
```
