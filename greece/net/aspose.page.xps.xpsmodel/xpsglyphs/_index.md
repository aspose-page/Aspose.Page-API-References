## XpsGlyphs class

Χαρακτηριστικά στοιχείου Glyphs incapsulating Class. Αυτό το στοιχείο αντιπροσωπεύει μια σειρά ομοιόμορφου μορφοποιημένου κειμένου από μία μόνο γραμματοσειρά. Παρέχει πληροφορίες που είναι απαραίτητες για ακριβή απόδοση και υποστηρίζει λειτουργίες search και επιλογής στην προβολή των καταναλωτών.

public sealed class XpsGlyphs : XpsContentElement
