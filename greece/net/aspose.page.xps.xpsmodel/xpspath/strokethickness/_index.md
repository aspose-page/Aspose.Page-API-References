## XpsPath.StrokeThickness property

Επιστρέφει/ορίζει το πάχος μιας διαδρομής, σε μονάδες του ενεργού χώρου συντεταγμένων (περιλαμβάνει τον μετασχηματισμό απόδοσης της διαδρομής). Η διαδρομή σχεδιάζεται πάνω από το όριο της γεωμετρίας specified από την ιδιότητα Δεδομένα του στοιχείου Διαδρομή. Το μισό του StrokeThickness extends εκτός της γεωμετρίας που καθορίζεται από την ιδιότητα Data και το άλλο μισό εκτείνεται εντός της γεωμετρίας.

```csharp
public float StrokeThickness { get; set; }
```
