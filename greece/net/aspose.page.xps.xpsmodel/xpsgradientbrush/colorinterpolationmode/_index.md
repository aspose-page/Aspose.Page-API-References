## XpsGradientBrush.ColorInterpolationMode property

Επιστρέφει/ορίζει την τιμή καθορίζοντας τη συνάρτηση γάμμα για παρεμβολή χρώματος. Το gamma adjustment δεν θα πρέπει να εφαρμόζεται στο στοιχείο άλφα, εάν καθορίζεται.

```csharp
public XpsColorInterpolationMode ColorInterpolationMode { get; set; }
```
