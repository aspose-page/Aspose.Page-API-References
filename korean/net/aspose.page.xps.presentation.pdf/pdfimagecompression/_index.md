## PdfImageCompression enumeration

PDF 파일의 이미지에 적용되는 압축 유형을 지정합니다.

```csharp
public enum PdfImageCompression
```

### 가치

| 이름 | 값 | 설명 |
| --- | --- | --- |
| Auto | `0` | 각 이미지에 가장 적합한 압축을 자동으로 선택합니다. |
| None | `1` | 원시 이미지 바이트를 저장하여 PDF 파일 크기가 커집니다. |
| Rle | `2` | 실행 길이 압축. |
| Flate | `3` | 플랫 압축. |
| LzwBaselinePredictor | `4` | 프로세스 속도를 높이기 위해 예측자 선택이 PNG 경로 예측자로 제한됩니다. 실제로 는 놀라울 정도로 잘 수행됩니다. 보다 낫다LzwOptimizedPredictor . |
| LzwOptimizedPredictor | `5` | 예측자 선택이 더 복잡하고 이미지 크기가 더 작아야 하지만 더 많은 시간이 걸립니다. |
| Jpeg | `6` | JPEG 압축. 투명도를 지원하지 않습니다. |
