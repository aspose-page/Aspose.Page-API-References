## OpenPage(string) {#openpage_1}

페이지 렌더링 전에 필요한 장치 준비를 합니다.

```csharp
public bool OpenPage(string title)
```

| 모수 | 유형 | 설명 |
| --- | --- | --- |
| title | String | 페이지 제목입니다. |

### 반환 값

페이지가 요청된 범위에 있으면 True, 그렇지 않으면 false. 지정된 페이지 번호 배열을 렌더링할 수 있는 장치에서 사용됩니다.

## OpenPage(float, float) {#openpage}

각 페이지 렌더링 전에 필요한 장치 준비를 합니다.

```csharp
public bool OpenPage(float width, float height)
```

| 모수 | 유형 | 설명 |
| --- | --- | --- |
| width | Single | 페이지의 너비입니다. |
| height | Single | 페이지의 높이입니다. |

### 반환 값

열린 페이지에 선택한 페이지 번호 범위에 속하는 번호가 있으면 true를 반환하고 그렇지 않으면 false를 반환합니다.
