**Aspose.Page**다음과 같은 Aspose.Page 라이브러리의 모든 클래스에 대한 루트 네임스페이스입니다. **장치** 또는 여러 하위 네임스페이스를 통해 간접적으로.

## 클래스

| 수업 | 설명 |
| --- | --- |
| [BuildVersionInfo](./buildversioninfo/) | 이 클래스는 현재 제품 빌드에 대한 정보를 제공합니다. |
| [Device](./device/) | 이 클래스는 문서 렌더링을 추상 장치로 캡슐화합니다. 문서 렌더링은 페이지별로 수행됩니다. |
| [Document](./document/) | 모든 캡슐화된 문서에 대한 수퍼클래스입니다. |
| [ExternalFontCache](./externalfontcache/) | 이 클래스를 사용하여 허용되는 형식의 글꼴 캡슐화를 얻습니다.[`Device`](../aspose.page/device/) . |
| [License](./license/) | 구성 요소에 라이선스를 부여하는 방법을 제공합니다. |
| [Margins](./margins/) | 이 클래스는 위쪽, 왼쪽, 아래쪽 및 오른쪽 여백을 캡슐화합니다. |
| [Metered](./metered/) | 측정 키를 설정하는 방법을 제공합니다. |
| [SaveOptions](./saveoptions/) | 이 클래스에는 변환 프로세스 관리에 필요한 옵션이 포함되어 있습니다. |
| [UserProperties](./userproperties/) | 유형이 지정된 속성을 설정하고 반환할 수 있는 특수 속성 클래스입니다. 또한 이 속성 개체에 속성이 포함되어 있지 않은 경우 두 개의 기본 속성 objects 의 연결을 검색할 수 있습니다. |
