```csharp
    PointF center, PointF gradientOrigin, float radiusX, float radiusY)
```

| 모수 | 유형 | 설명 |
| --- | --- | --- |
| gradientStops | List`1 | 그래디언트 중지 목록입니다. |
| center | PointF | 방사형 그래디언트의 중심점(즉, 타원의 중심)입니다. |
| gradientOrigin | PointF | 방사형 그래디언트의 원점입니다. |
| radiusX | Single | 방사형 그래디언트를 정의하는 타원의 x 차원 반지름입니다. |
| radiusY | Single | 방사형 그래디언트를 정의하는 타원의 y 차원 반지름입니다. |

### 반환 값
