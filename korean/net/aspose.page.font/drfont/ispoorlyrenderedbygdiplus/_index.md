## DrFont.IsPoorlyRenderedByGdiPlus method

"Microsoft Sans Serif" 글꼴에 대해 True를 반환합니다. 이것은 GDI+에 의해 제대로 렌더링되지 않습니다. Test286 및 Gemini-6959. 를 참조하십시오.

```csharp
public static bool IsPoorlyRenderedByGdiPlus(string fontName)
```

| 모수 | 유형 | 설명 |
| --- | --- | --- |
| fontName | String | 글꼴의 이름입니다. |

### 반환 값

`진실` if [GDI 플러스에 의해 제대로 렌더링되지 않음] [지정된 글꼴 이름]; 그렇지 않으면,`거짓` .
