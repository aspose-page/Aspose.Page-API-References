## PsDocument.SetPageSize method

페이지 크기를 설정합니다. 한 문서에서 크기가 다른 페이지를 만들려면[`SetPageDevice`](../setpagedevice/) 이 메서드 바로 뒤에 있는 메서드.

```csharp
public void SetPageSize(float width, float height)
```

| 모수 | 유형 | 설명 |
| --- | --- | --- |
| width | Single | 결과 PostScript 파일의 페이지 너비입니다. |
| height | Single | 결과 PostScript 파일의 페이지 높이입니다. |
