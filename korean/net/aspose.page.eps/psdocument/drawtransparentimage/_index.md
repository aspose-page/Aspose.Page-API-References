## PsDocument.DrawTransparentImage method

변형된 투명 이미지를 그립니다. 이미지에 알파 채널이 없으면 불투명한 image 로 그려집니다.

```csharp
public void DrawTransparentImage(Bitmap image, Matrix transform, int transparencyThreshold)
```

| 모수 | 유형 | 설명 |
| --- | --- | --- |
| image | Bitmap | 그릴 이미지입니다. |
| transform | Matrix | 이미지를 변환하는 행렬입니다. |
| transparencyThreshold | Int32 | 투명도 픽셀이 완전히 투명한 것으로 해석되는 값을 정의하는 임계값입니다. 이 임계값 미만의 모든 값은 완전히 불투명한 것으로 해석됩니다. |
