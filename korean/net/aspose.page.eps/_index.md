**Aspose.Page.EPS** PS/EPS 파일을 다루는 모든 클래스의 루트 네임스페이스입니다.

## 클래스

| 수업 | 설명 |
| --- | --- |
| [FontConstants](./fontconstants/) | 이 클래스는 글꼴 저장을 위한 상수 집합을 정의합니다. |
| [GradientBrush](./gradientbrush/) | 이 클래스는 래핑 모드를 클램프로 설정할 가능성이 있는 LinearGradientBrush 및 PathGradientBrush를 캡슐화하는 데 사용됩니다. |
| [LoadOptions](./loadoptions/) | 문서 로딩 옵션의 기본 클래스. |
| [PageConstants](./pageconstants/) | 이 클래스는 페이지를 설명하는 일련의 상수를 정의합니다. 다양한 여백, 방향, 크기 조정 및 표준 페이지 크기에 대한 편의 개체가 제공됩니다. |
| [PsConverterException](./psconverterexception/) | 이 클래스에는 PS 파일을 PDF 문서로 변환하는 동안 발생하는 is 오류에 대한 정보가 포함되어 있습니다. |
| [PsDocument](./psdocument/) | 이 클래스는 PS/EPS 문서를 캡슐화합니다. |
| [PsDocumentException](./psdocumentexception/) | 이 클래스에는 PostScript 파일을 생성, 편집 및 저장하는 동안 발생한 오류에 대한 정보가 포함되어 있습니다. |
| [PsLoadOptions](./psloadoptions/) | PS 문서 로딩 옵션. |
