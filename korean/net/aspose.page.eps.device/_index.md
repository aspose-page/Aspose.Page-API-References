**Aspose.Page.EPS.Device** 네임스페이스는 PS/EPS를 다른 형식으로 변환하는 동안 사용할 수 있는 가능한 장치 및 저장 옵션의 클래스를 제공합니다.

## 클래스

| 수업 | 설명 |
| --- | --- |
| [ImageDevice](./imagedevice/) | 이 클래스는 문서 렌더링을 이미지로 캡슐화합니다. |
| [ImageSaveOptions](./imagesaveoptions/) | 이 클래스에는 이미지 저장 프로세스를 관리하는 데 필요한 옵션이 포함되어 있습니다. |
| [PdfDevice](./pdfdevice/) | 이 클래스는 문서를 PDF로 렌더링하는 것을 캡슐화합니다. |
| [PdfSaveOptions](./pdfsaveoptions/) | 이 클래스에는 입력 및 출력 스트림과 변환 프로세스 관리에 필요한 기타 옵션이 포함되어 있습니다. |
| [PsSaveOptions](./pssaveoptions/) | 이 클래스에는 문서를 PostScript(PS) 또는 Encapsulated PostScript(EPS) 파일로 변환하는 프로세스를 관리하는 데 필요한 옵션이 포함되어 있습니다. |
