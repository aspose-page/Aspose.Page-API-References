## XmlDeclaration::get_Standalone method


standalone 속성의 값을 반환합니다.

```cpp
String System::Xml::XmlDeclaration::get_Standalone()
```


### ReturnValue

유효한 값은 XML 문서에 필요한 모든 엔터티 선언이 문서 내에 포함된 경우 **yes**, 외부 문서 유형 정의(DTD)가 필요한 경우 **no**입니다. XML 선언에 standalone 속성이 없으면 이 메서드는 [String::Empty](../../../system/string/empty/)을 반환합니다.
