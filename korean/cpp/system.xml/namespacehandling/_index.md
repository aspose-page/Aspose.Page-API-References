## NamespaceHandling enum


[XmlWriter](../xmlwriter/)에서 중복 네임스페이스 선언을 제거할지 여부를 지정합니다.

```cpp
enum class NamespaceHandling
```

### 값

| 이름 | 값 | 설명 |
| --- | --- | --- |
| Default | 0 | 중복 네임스페이스 선언이 제거되지 않음을 지정합니다. |
| OmitDuplicates | 1 | 중복 네임스페이스 선언이 제거되도록 지정합니다. 중복 네임스페이스가 제거되려면 접두사와 네임스페이스가 일치해야 합니다. |
