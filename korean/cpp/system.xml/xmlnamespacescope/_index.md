enum class XmlNamespaceScope

### 값

| 이름 | 값 | 설명 |
| --- | --- | --- |
| All | 0 | 현재 노드 범위에 정의된 모든 네임스페이스입니다. 여기에는 항상 암시적으로 선언되는 xmlns:xml 네임스페이스가 포함됩니다. 반환되는 네임스페이스의 순서는 정의되지 않습니다. |

| Local | 2 | 현재 노드에서 로컬로 정의된 모든 네임스페이스입니다. |
