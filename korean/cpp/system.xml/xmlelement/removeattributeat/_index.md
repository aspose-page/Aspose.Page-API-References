## XmlElement::RemoveAttributeAt method


요소에서 지정된 인덱스를 가진 속성 노드를 제거합니다. (제거된 속성이 기본값을 가지고 있는 경우 즉시 대체됩니다.)

```cpp
virtual SharedPtr<XmlNode> System::Xml::XmlElement::RemoveAttributeAt(int32_t i)
```


| 매개변수 | 형식 | 설명 |
| --- | --- | --- |
| i | int32_t | 제거할 노드의 인덱스입니다. 첫 번째 노드의 인덱스는 0입니다. |

### ReturnValue

제거된 속성 노드이며, 지정된 인덱스에 노드가 없으면 **nullptr**입니다.
