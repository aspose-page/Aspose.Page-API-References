## PsSaveFormat enum


이 열거형은 저장 형식에 사용할 수 있는 옵션을 포함합니다. PS 또는 [EPS](../../aspose.page.eps/) 중 하나를 선택할 수 있습니다. [EPS](../../aspose.page.eps/)는 1페이지 문서에만 사용되며, PS 파일은 페이지 수에 제한이 없습니다.

```cpp
enum class PsSaveFormat
```

### 값

| 이름 | 값 | 설명 |
| --- | --- | --- |
| PS | 0 | 이 옵션은 결과 문서가 PostScript (PS) 또는 Encapsulated PostScript ([EPS](../../aspose.page.eps/)) 파일이어야 함을 나타냅니다. |
| EPS | 1 | 이 옵션은 결과 문서가 Encapsulated PostScript ([EPS](../../aspose.page.eps/)) 파일이어야 함을 나타냅니다. 이는 1페이지 문서에만 사용됩니다. |
