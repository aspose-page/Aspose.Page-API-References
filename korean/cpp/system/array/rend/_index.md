## Array::rend() const method


역전된 컨테이너의 마지막 요소 다음에 있는 요소에 대한 역방향 반복자를 반환합니다. 이는 역전되지 않은 컨테이너의 첫 번째 요소 이전에 해당합니다. 이 요소는 자리표시자 역할을 하며, 접근을 시도하면 정의되지 않은 동작이 발생합니다.

```cpp
const_reverse_iterator System::Array<T>::rend() const noexcept
```
