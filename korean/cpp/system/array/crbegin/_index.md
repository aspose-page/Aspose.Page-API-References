## Array::crbegin method


역전된 컨테이너의 첫 번째 요소에 대한 역방향 반복자를 반환합니다. 이는 역전되지 않은 컨테이너의 마지막 요소에 해당합니다. 컨테이너가 비어 있는 경우, 반환된 반복자는 [crend()](../crend/)와 같습니다.

```cpp
const_reverse_iterator System::Array<T>::crbegin() const noexcept
```


### ReturnValue

컨테이너의 마지막 const 한정 요소를 가리키는 반복자입니다.
