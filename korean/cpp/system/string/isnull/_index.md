## String::IsNull method


문자열이 null로 간주되는지 확인합니다. [String](../)은(는) String() 생성자를 통해 생성되었거나, 이동되었거나, 복사되었거나, null 문자열로부터 할당되었거나, [reset()](../reset/) 메서드가 호출된 경우에만 null입니다.

```cpp
bool System::String::IsNull() const
```


### ReturnValue

문자열이 null로 간주되면 true, 그렇지 않으면 false.
