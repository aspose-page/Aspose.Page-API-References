## String::IsNormalized method

```cpp
bool System::String::IsNormalized(System::Text::NormalizationForm form) const
```


| 매개변수 | 형식 | 설명 |
| --- | --- | --- |
| form | System::Text::NormalizationForm | 정규화 형식, 자세한 내용은 [System::Text::NormalizationForm](../../../system.text/normalizationform/)을(를) 참조하십시오. |

### ReturnValue

문자열이 이미 정규화된 경우 true, 그렇지 않으면 false.
