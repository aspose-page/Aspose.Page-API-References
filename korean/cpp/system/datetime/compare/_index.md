## DateTime::Compare method


지정된 [DateTime](../) 클래스 인스턴스가 나타내는 두 값을 비교하고 값들의 시간 축에서 상대적 위치를 반환합니다.

```cpp
static constexpr int System::DateTime::Compare(DateTime t1, DateTime t2)
```


| 매개변수 | 형식 | 설명 |
| --- | --- | --- |
| t1 | DateTime | 첫 번째 비교값 |
| t2 | DateTime | 두 번째 비교값 |

### ReturnValue

**t1**이 **t2**보다 이전이면 0보다 작고; **t1**이 **t2**와 동일하면 0이며; **t1**이 **t2**보다 이후이면 0보다 큰 값을 반환합니다.
