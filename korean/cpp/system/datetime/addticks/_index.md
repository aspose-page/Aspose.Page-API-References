```cpp
DateTime System::DateTime::AddTicks(int64_t value) const
```


| 매개변수 | 형식 | 설명 |
| --- | --- | --- |
| value | int64_t | 추가할 100나노초 간격 수; 값은 양수 또는 음수가 될 수 있습니다. |
