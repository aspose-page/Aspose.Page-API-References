## DateTime::Add method


[DateTime](../) 클래스의 새 인스턴스를 반환하며, 이는 현재 객체가 나타내는 날짜 및 시간 값에 지정된 시간 간격을 더한 결과 날짜 및 시간 값을 나타냅니다.

```cpp
DateTime System::DateTime::Add(TimeSpan value) const
```


| 매개변수 | 형식 | 설명 |
| --- | --- | --- |
| value | TimeSpan | 추가할 시간 간격 |

### ReturnValue

현재 객체가 나타내는 날짜 및 시간 값에 **tm** 시간 간격을 더한 결과인 날짜 및 시간 값을 나타내는 새로운 [DateTime](../) 클래스 인스턴스
