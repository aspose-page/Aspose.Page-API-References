## DateTime::CompareTo method


현재 객체와 지정된 [DateTime](../) 클래스 인스턴스가 나타내는 두 날짜 및 시간 값을 비교하고, 시간 축에서 값들의 상대적 위치를 나타내는 값을 반환합니다.

```cpp
int System::DateTime::CompareTo(DateTime value) const
```


| 매개변수 | 형식 | 설명 |
| --- | --- | --- |
| value | DateTime | 현재 객체와 비교할 [DateTime](../) 클래스의 인스턴스 |

### ReturnValue

**value**가 나타내는 값보다 현재 객체가 더 이전을 나타내면 0보다 작고, 두 객체가 나타내는 값이 동일하면 0이며, 현재 객체가 **value**가 나타내는 값보다 나중을 나타내면 0보다 큰 값을 반환합니다.
