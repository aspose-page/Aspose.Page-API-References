## DateTime::AddYears method


[DateTime](../) 클래스의 새로운 인스턴스를 반환합니다. 이 인스턴스는 현재 객체가 나타내는 날짜와 시간 값에 연도 구성 요소를 지정된 수만큼 증가시킨 값을 나타냅니다.

```cpp
DateTime System::DateTime::AddYears(int value) const
```


| 매개변수 | 형식 | 설명 |
| --- | --- | --- |
| value | int | 추가할 연도 수; 값은 양수이거나 음수일 수 있습니다. |

### ReturnValue

[DateTime](../) 클래스의 새로운 인스턴스로, 현재 객체가 나타내는 날짜와 시간 값에 연도 구성 요소를 **value**만큼 증가시킨 값을 나타냅니다.
