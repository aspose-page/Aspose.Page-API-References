## Char::TryParse method


단일 문자로 구성된 문자열을 UTF-16 문자로 변환하려 시도합니다. 입력 문자열이 null이 아니고 정확히 한 문자 길이를 가질 때만 함수가 성공합니다.

```cpp
static bool System::Char::TryParse(const System::String &s, char_t &result)
```


| 매개변수 | 형식 | 설명 |
| --- | --- | --- |
| s | const System::String\& | [String](../../string/) 변환 |
| result | char_t\& | 변환이 성공하면 결과를 담게 되는 출력 변수 |

### ReturnValue

변환이 성공하면 true, 그렇지 않으면 false
