## Char::IsLowSurrogate(char_t) method


지정된 문자가 하위 서러게이트인지 여부를 결정합니다.

```cpp
static bool System::Char::IsLowSurrogate(char_t c)
```


| 매개변수 | 형식 | 설명 |
| --- | --- | --- |
| c | char_t | 테스트할 문자 |

### ReturnValue

지정된 문자가 로우 서러게이트이면 true, 그렇지 않으면 false
