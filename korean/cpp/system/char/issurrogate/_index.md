```cpp

```


| 매개변수 | 형식 | 설명 |
| --- | --- | --- |
| c | char_t | 문자 하나 |

### ReturnValue

지정된 문자가 UTF-16 서러게이트 코드 단위이면 True, 그렇지 않으면 false
