## System::CheckedCast method


지정된 값이 타입 **TTo**의 값 범위에 속하는지 확인하고, 속한다면 해당 값을 타입 **TTo**로 캐스팅합니다.

```cpp
template<typename TTo,typename TFrom> TTo System::CheckedCast(TFrom value)
```


| 매개변수 | 설명 |
| --- | --- |
| TTo | 지정된 값을 캐스팅할 대상 타입 |
| TFrom | 지정된 값의 타입 |

| 매개변수 | 형식 | 설명 |
| --- | --- | --- |
| value | TFrom | 캐스팅할 값 |

### ReturnValue

타입 **TTo**의 값으로, **value**와 동일합니다.
