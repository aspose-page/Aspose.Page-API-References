## MarshalByRefObject class


원격 호출이 가능한 애플리케이션에서 애플리케이션 도메인 경계 너머의 객체에 접근할 수 있게 합니다. 이 클래스의 객체는 [System::MakeObject()](../makeobject/) 함수를 사용하여 할당해야 합니다. 스택에 직접 생성하거나 operator new를 사용하여 인스턴스를 만들면 런타임 오류 및/또는 어설션 오류가 발생합니다. 항상 이 클래스를 [System::SmartPtr](../smartptr/) 포인터로 래핑하고, 함수 인수로 전달할 때 이 포인터를 사용하십시오.

```cpp
class MarshalByRefObject : public virtual System::Object
```
