## WeakReference class


객체를 참조하면서도 해당 객체가 삭제될 수 있도록 허용하는 약한 참조를 나타냅니다.

```cpp
template<typename...>class WeakReference
```


| 매개변수 | 설명 |
| --- | --- |
| 인수 | 약한 참조의 템플릿 인수는 참조된 객체의 유형을 선언하며, 인수 개수는 0개 또는 1개일 수 있습니다. |
