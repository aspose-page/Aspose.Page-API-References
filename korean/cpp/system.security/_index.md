## 클래스

| 클래스 | 설명 |
| --- | --- |
| [SecureString](./securestring/) | Secure string은 기밀로 유지되어야 하는 텍스트를 나타냅니다. 이 클래스는 내부 데이터를 암호화하지 않습니다. 이 클래스의 객체는 [System::MakeObject()](../system/makeobject/) 함수를 사용하여 할당해야 합니다. 스택에 직접 생성하거나 operator new를 사용하여 인스턴스를 만들면 런타임 오류 및/또는 어설션 오류가 발생합니다. 항상 이 클래스를 [System::SmartPtr](../system/smartptr/) 포인터로 감싸고, 해당 포인터를 함수 인수로 전달하십시오. |
| [SecureStringMarshal](./securestringmarshal/) | 관리되지 않는 메모리 블록을 할당하고 복사하기 위한 메서드 컬렉션입니다. |
| [SecurityElement](./securityelement/) | 보안 객체를 인코딩하기 위한 XML 객체 모델입니다. 구현되지 않았습니다. 이 클래스의 객체는 [System::MakeObject()](../system/makeobject/) 함수를 사용하여 할당해야 합니다. 스택에 직접 생성하거나 operator new를 사용하여 인스턴스를 만들면 런타임 오류 및/또는 어설션 오류가 발생합니다. 항상 이 클래스를 [System::SmartPtr](../system/smartptr/) 포인터로 감싸고, 해당 포인터를 함수 인수로 전달하십시오. |
