## HttpContext class


번역된 코드와 [HttpContext](./) 참조를 연결할 수 있게 하는 더미 클래스이지만 실행되지 않습니다. 제대로 구현된 멤버가 없습니다. 이 클래스의 객체는 [System::MakeObject()](../../system/makeobject/) 함수를 사용하여 할당해야 합니다. 스택에 직접 생성하거나 operator new를 사용해 인스턴스를 만들면 런타임 오류 및/또는 어설션 오류가 발생합니다. 항상 이 클래스를 [System::SmartPtr](../../system/smartptr/) 포인터로 감싸고, 해당 포인터를 함수 인자로 전달하는 데 사용하십시오.

```cpp
class HttpContext : public System::Object
```
