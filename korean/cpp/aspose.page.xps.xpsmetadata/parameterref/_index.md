## ParameterRef class


일반 [PrintTicket](../printticket/) 매개변수 참조를 구현하는 클래스입니다. **[ParameterRef](./)** 요소는 [ParameterInit](../parameterinit/) 요소에 대한 참조를 정의합니다. [ParameterRef](./) 요소를 포함하는 [ScoredProperty](../scoredproperty/) 요소는 명시적으로 설정된 [Value](../value/) 요소가 없습니다. 대신, [ScoredProperty](../scoredproperty/) 요소는 **[ParameterRef](./)** 요소가 참조하는 [ParameterInit](../parameterinit/) 요소로부터 값을 받습니다. [https://docs.microsoft.com/en-us/windows/win32/printdocs/parameterref](https://docs.microsoft.com/en-us/windows/win32/printdocs/parameterref).

```cpp
class ParameterRef : public Aspose::Page::XPS::XpsMetadata::PrintTicketElement,
                     public Aspose::Page::XPS::XpsMetadata::IPrintTicketItem
```
