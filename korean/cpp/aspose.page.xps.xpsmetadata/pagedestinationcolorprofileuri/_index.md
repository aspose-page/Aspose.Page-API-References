## PageDestinationColorProfileURI class


ICC 프로파일에 대한 상대 URI 참조를 [XPS](../../aspose.page.xps/)[Document](../../aspose.page/document/)에 포함된 경우 지정합니다. 이 옵션의 처리는 [PageDeviceColorSpaceUsage](../pagedevicecolorspaceusage/) 기능 설정에 따라 달라집니다. 해당 프로파일을 사용하는 모든 요소는 이미 적절한 장치 색상 공간에 있다고 가정되며, 드라이버나 장치에서 색상 관리되지 않습니다. [https://docs.microsoft.com/en-us/windows/win32/printdocs/pagedestinationcolorprofileuri](https://docs.microsoft.com/en-us/windows/win32/printdocs/pagedestinationcolorprofileuri).
