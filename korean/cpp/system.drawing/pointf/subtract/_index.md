## PointF::Subtract(const PointF\&, const Size\&) method


지정된 [Size](../../size/) 객체의 너비와 높이 값을 지정된 [PointF](../) 객체의 X와 Y 좌표 값에서 각각 뺍니다.

```cpp
static PointF System::Drawing::PointF::Subtract(const PointF &point, const Size &size)
```


| 매개변수 | 형식 | 설명 |
| --- | --- | --- |
| 점 | const PointF\& | 변환할 점 |
| size | const Size\& | 좌표 값에서 빼야 할 값을 지정하는 [Size](../../size/) 객체 **point** |

### ReturnValue

새로운 [PointF](../) 객체로, X 좌표 값은 **point**의 X 좌표 값에서 **size**의 너비 값을 뺀 결과와 같으며, Y 좌표 값은 **point**의 Y 좌표 값에서 **size**의 높이 값을 뺀 결과와 같습니다.
