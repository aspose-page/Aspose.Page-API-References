## PointF::Add(const PointF\&, const Size\&) method


지정된 [Size](../../size/) 객체의 너비와 높이 값을 지정된 [PointF](../) 객체의 X 및 Y 좌표 값에 각각 더합니다.

```cpp
static PointF System::Drawing::PointF::Add(const PointF &point, const Size &size)
```


| 매개변수 | 형식 | 설명 |
| --- | --- | --- |
| 점 | const PointF\& | 변환할 점 |
| size | const Size\& | 좌표 값에 더할 값을 지정하는 [Size](../../size/) 객체 (**point**) |

### ReturnValue

새로운 [PointF](../) 객체로, X 좌표 값은 **point**의 X 좌표 값과 **size**의 너비 값을 더한 값과 같으며, Y 좌표 값은 **point**의 Y 좌표 값과 **size**의 높이 값을 더한 값과 같습니다.
