## StringTrimming enum


레이아웃 형태에 맞지 않는 문자열에서 문자를 어떻게 잘라낼지 지정합니다.

```cpp
enum class StringTrimming
```

### 값

| 이름 | 값 | 설명 |
| --- | --- | --- |
| None | 0 | 잘라내지 않음. |
| 문자 | 1 | 가장 가까운 문자까지 잘라냅니다. |
| 단어 | 2 | 가장 가까운 단어까지 잘라냅니다. |
| EllipsisCharacter | 3 | 가장 가까운 문자까지 잘라내고 문자열 끝에 생략 부호를 삽입합니다. |
| EllipsisWord | 4 | 가장 가까운 단어까지 잘라내고 문자열 끝에 생략 부호를 삽입합니다. |
| EllipsisPath | 5 | 잘라낸 라인에서 가운데 부분을 제거하고 생략 부호로 대체합니다. 가능한 한 라인의 마지막 슬래시로 구분된 구간을 많이 유지합니다. |
