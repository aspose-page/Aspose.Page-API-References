## Size::Subtract method


새로운 [Size](../) 객체를 반환합니다. 이는 **size1**에서 **size2**를 빼는 결과이며, 즉 **size1's** 너비 값에서 **size2's** 너비 값을 뺀 결과가 너비가 되고, **size1's** 높이 값에서 **size2's** 높이 값을 뺀 결과가 높이가 됩니다.

```cpp
static Size System::Drawing::Size::Subtract(const Size &size1, const Size &size2)
```


| 매개변수 | 형식 | 설명 |
| --- | --- | --- |
| size1 | const Size\& | 빼기 대상이 되는 [Size](../) 객체 |
| size2 | const Size\& | 빼기 위한 [Size](../) 객체 |

### ReturnValue

**size1**에서 **size2**를 빼서 얻은 결과인 [Size](../) 객체
