## RIPEMD160 class


입력 데이터에 대한 [RIPEMD160](./) 해시를 계산합니다. C++에서 [RIPEMD160](./)은 추상 클래스가 아닙니다. 이 클래스의 객체는 [System::MakeObject()](../../system/makeobject/) 함수를 사용하여 할당해야 합니다. 스택에 직접 생성하거나 operator new를 사용하면 런타임 오류 및/또는 어설션 오류가 발생합니다. 항상 이 클래스를 [System::SmartPtr](../../system/smartptr/) 포인터로 감싸고, 해당 포인터를 함수 인수로 전달하십시오.
