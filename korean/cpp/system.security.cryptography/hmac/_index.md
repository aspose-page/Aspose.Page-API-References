## HMAC class


해시 기반 메시지 [Authentication](../../system.security.authentication/) 코드([HMAC](./))의 모든 구현은 이 추상 클래스를 상속해야 합니다. 이 클래스의 객체는 [System::MakeObject()](../../system/makeobject/) 함수를 사용하여 할당해야 합니다. 스택에 직접 생성하거나 operator new를 사용하면 런타임 오류 및/또는 어설션 오류가 발생합니다. 항상 이 클래스를 [System::SmartPtr](../../system/smartptr/) 포인터로 감싸고, 해당 포인터를 함수 인수로 전달하십시오.

```cpp
class HMAC : public System::Security::Cryptography::HashAlgorithm
```
