## KeyInfoClause class


Todas las implementaciones de subelementos [KeyInfo](../keyinfo/) deben heredar de esta clase abstracta. Los objetos de esta clase solo deben asignarse mediante la función [System::MakeObject()](../../system/makeobject/). Nunca cree una instancia de este tipo en la pila ni usando el operador new, ya que provocará errores de tiempo de ejecución y/o fallas de aserción. Siempre envuelva esta clase en un puntero [System::SmartPtr](../../system/smartptr/) y use ese puntero para pasarla a funciones como argumento.

```cpp
class KeyInfoClause : public System::Object
```
