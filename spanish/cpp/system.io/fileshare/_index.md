## FileShare enum


Especifica qué tipo de acceso pueden tener otros objetos [FileStream](../filestream/) a un archivo que se está abriendo.

```cpp
enum class FileShare
```

### Valores

| Nombre | Valor | Descripción |
| --- | --- | --- |
| None | 0 | Sin acceso. |
| Leer | 1 | Acceso de solo lectura. |
| Write | 2 | Acceso de solo escritura. |
| ReadWrite | 3 | Acceso de lectura y escritura. |
| Eliminar | 4 | El archivo puede ser eliminado. |
| Hereditable | 16 | Hace que el descriptor de archivo sea heredable por procesos hijos. |
