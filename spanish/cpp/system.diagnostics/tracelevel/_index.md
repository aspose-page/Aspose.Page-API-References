## TraceLevel enum


Especifica qué mensajes se deben emitir para las clases [System.Diagnostics.Debug](../debug/), [System.Diagnostics.Trace](../trace/) y System.Diagnostics.TraceSwitch.

```cpp
enum class TraceLevel
```

### Valores

| Nombre | Valor | Descripción |
| --- | --- | --- |
| Off | 0 | No emite mensajes de rastreo ni de depuración. |
| Error | 1 | Emite mensajes de manejo de errores. |
| Warning | 2 | Emite mensajes de advertencia y de manejo de errores. |
| Info | 3 | Salida de mensajes informativos, advertencias y mensajes de manejo de errores. |
| Detallado | 4 | Salida de todos los mensajes de depuración y rastreo. |
