## SHA256 class


Calcula el hash [SHA256](./) para los datos de entrada. En C++ [SHA256](./) no es una clase abstracta. Los objetos de esta clase solo deben asignarse usando la función [System::MakeObject()](../../system/makeobject/). Nunca cree una instancia de este tipo en la pila ni usando el operador new, ya que provocará errores en tiempo de ejecución y/o fallas de aserción. Siempre envuelva esta clase en un puntero [System::SmartPtr](../../system/smartptr/) y use ese puntero para pasarlo a funciones como argumento.

```cpp
class SHA256 : public System::Security::Cryptography::HashAlgorithm
```
