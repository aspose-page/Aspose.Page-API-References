## HMACSHA1 class


Código de autenticación de mensaje basado en hash [Authentication](../../system.security.authentication/) que utiliza la función hash [SHA1](../sha1/). No implementado. Los objetos de esta clase solo deben asignarse usando la función [System::MakeObject()](../../system/makeobject/). Nunca cree una instancia de este tipo en la pila ni usando el operador new, ya que provocará errores en tiempo de ejecución y/o fallas de aserción. Siempre envuelva esta clase en un puntero [System::SmartPtr](../../system/smartptr/) y use ese puntero para pasarlo a funciones como argumento.

class HMACSHA1 : public System::Security::Cryptography::HashAlgorithm
