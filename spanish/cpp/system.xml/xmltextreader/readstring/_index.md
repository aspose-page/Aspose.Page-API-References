## XmlTextReader::ReadString method


Lee el contenido de un elemento o de un nodo de texto como una cadena.

```cpp
String System::Xml::XmlTextReader::ReadString() override
```


### ReturnValue

El contenido del elemento o del nodo de texto. Puede ser una cadena vacía si el lector está posicionado en algo que no sea un elemento o nodo de texto, o si no hay más contenido de texto que devolver en el contexto actual. **Note:** El nodo de texto puede ser un elemento o un nodo de texto de atributo.
