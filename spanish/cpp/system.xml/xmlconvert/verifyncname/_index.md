## XmlConvert::VerifyNCName method


Verifica que el nombre sea un **NCName** válido según la recomendación de W3C Extended Markup Language. Un **NCName** es un nombre que no puede contener dos puntos.

```cpp
static String System::Xml::XmlConvert::VerifyNCName(const String &name)
```


| Parámetro | Tipo | Descripción |
| --- | --- | --- |
| name | const String\& | El nombre a verificar. |

### ReturnValue

El nombre, si es un NCName válido.
