## XmlConvert::VerifyXmlChars method


Devuelve la cadena proporcionada si todos los caracteres y pares sustitutos en el argumento de cadena son caracteres XML válidos; de lo contrario, se lanza una [XmlException](../../xmlexception/) con información sobre el primer carácter inválido encontrado.

```cpp
static String System::Xml::XmlConvert::VerifyXmlChars(const String &content)
```


| Parámetro | Tipo | Descripción |
| --- | --- | --- |
| content | const String\& | [String](../../../system/string/) que contiene caracteres para verificar. |

### ReturnValue

La cadena proporcionada si todos los caracteres y pares sustitutos en el argumento de cadena son caracteres XML válidos; de lo contrario, se lanza una [XmlException](../../xmlexception/) con información sobre el primer carácter inválido encontrado.
