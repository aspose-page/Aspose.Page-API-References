## XmlNodeChangedEventArgs::get_Action method


Devuelve un valor que indica qué tipo de evento de cambio de nodo está ocurriendo.

```cpp
XmlNodeChangedAction System::Xml::XmlNodeChangedEventArgs::get_Action()
```


### ReturnValue

Uno de los valores de enumeración que describen el evento de cambio de nodo. El valor **[Action](../../../system/action/)** no diferencia entre cuándo ocurrió el evento (antes o después). Puedes crear manejadores de eventos separados para manejar ambas instancias.
