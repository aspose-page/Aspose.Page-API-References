## NamespaceHandling enum


Especifica si se deben eliminar las declaraciones de espacio de nombres duplicadas en el [XmlWriter](../xmlwriter/).

```cpp
enum class NamespaceHandling
```

### Valores

| Nombre | Valor | Descripción |
| --- | --- | --- |
| Default | 0 | Especifica que las declaraciones de espacio de nombres duplicadas no se eliminarán. |
| OmitDuplicates | 1 | Especifica que las declaraciones de espacio de nombres duplicadas se eliminarán. Para que se elimine el espacio de nombres duplicado, el prefijo y el espacio de nombres deben coincidir. |
