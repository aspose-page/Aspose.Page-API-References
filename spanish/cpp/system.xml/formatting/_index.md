## Formatting enum


Especifica opciones de formato para el [XmlTextWriter](../xmltextwriter/).


### Valores

| Nombre | Valor | Descripción |
| --- | --- | --- |
| None | 0 | No se aplica formato especial. Este es el valor predeterminado. |
| Indented | 1 | Hace que los elementos hijos se sangren según la configuración de [XmlTextWriter::set_Indentation](../xmltextwriter/set_indentation/) y [XmlTextWriter::set_IndentChar](../xmltextwriter/set_indentchar/). |
