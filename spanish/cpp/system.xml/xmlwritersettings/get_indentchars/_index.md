## XmlWriterSettings::get_IndentChars method


Devuelve la cadena de caracteres a usar al sangrar. Esta configuración se utiliza cuando el valor [XmlWriterSettings::set_Indent](../set_indent/) está establecido en **true**.

```cpp
String System::Xml::XmlWriterSettings::get_IndentChars()
```


### ReturnValue

La cadena de caracteres a usar al sangrar. Puede establecerse a cualquier valor de cadena. Sin embargo, para garantizar XML válido, debe especificar solo caracteres de espacio en blanco válidos, como caracteres de espacio, tabulaciones, retornos de carro o saltos de línea. El valor predeterminado son dos espacios.
