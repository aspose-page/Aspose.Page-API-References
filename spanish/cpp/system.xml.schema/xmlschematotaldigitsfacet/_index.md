## XmlSchemaTotalDigitsFacet class


Representa la faceta **totalDigits** del XML [Schema](../) según lo especificado por el Consorcio de la World Wide [Web](../../system.web/) (W3C). Esta clase puede usarse para especificar una restricción sobre el número de dígitos que pueden ingresarse para el valor de un elemento **simpleType**. Ese valor de **totalDigits** debe ser un entero positivo.

```cpp
class XmlSchemaTotalDigitsFacet : public System::Xml::Schema::XmlSchemaNumericFacet
```
