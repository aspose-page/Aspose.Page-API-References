## XmlSchemaFractionDigitsFacet class


Especifica una restricción en el número de dígitos que pueden ingresarse para el valor fraccionario de un elemento simpleType. El valor de fractionDigits debe ser un entero positivo. Representa el World Wide [Web](../../system.web/) Consortium (W3C) **fractionDigits** faceta.

class XmlSchemaFractionDigitsFacet : public System::Xml::Schema::XmlSchemaNumericFacet
