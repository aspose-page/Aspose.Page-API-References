## XmlSchemaLengthFacet class


Representa la faceta **length** del XML [Schema](../) según lo especificado por el World Wide [Web](../../system.web/) Consortium (W3C). Esta clase puede usarse para especificar una restricción en la longitud de un elemento **simpleType** del tipo de datos.

```cpp
class XmlSchemaLengthFacet : public System::Xml::Schema::XmlSchemaNumericFacet
```
