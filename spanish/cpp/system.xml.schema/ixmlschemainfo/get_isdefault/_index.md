## IXmlSchemaInfo::get_IsDefault method


Devuelve un valor que indica si este nodo XML validado se estableció como resultado de una configuración predeterminada aplicada durante la validación del [Schema](../../) Lenguaje de Definición de Esquemas XML (XSD).

```cpp
virtual bool System::Xml::Schema::IXmlSchemaInfo::get_IsDefault()=0
```


### ReturnValue

**true** if this validated XML node was set as the result of a default being applied during schema validation; otherwise, **false**.
