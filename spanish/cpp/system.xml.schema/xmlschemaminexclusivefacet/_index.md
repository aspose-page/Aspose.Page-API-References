## XmlSchemaMinExclusiveFacet class


Representa el elemento **minExclusive** del XML [Schema](../) según lo especificado por el World Wide [Web](../../system.web/) Consortium (W3C). Esta clase puede usarse para especificar una restricción sobre el valor mínimo de un elemento **simpleType**. El valor del elemento debe ser mayor que el valor del elemento **minExclusive**.

```cpp
class XmlSchemaMinExclusiveFacet : public System::Xml::Schema::XmlSchemaFacet
```
