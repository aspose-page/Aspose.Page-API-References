## XmlSchemaUnique class

class XmlSchemaUnique : public System::Xml::Schema::XmlSchemaIdentityConstraint

## Observaciones



Los objetos de esta clase solo deben asignarse usando la función [System::MakeObject()](../../system/makeobject/).
