## XmlSchemaMinLengthFacet class


Representa el elemento **minLength** del XML [Schema](../) según lo especificado por el Consorcio de la World Wide [Web](../../system.web/). Esta clase puede usarse para especificar una restricción sobre la longitud mínima del valor de datos de un elemento **simpleType**. La longitud debe ser mayor que el valor del elemento **minLength**.

```cpp
class XmlSchemaMinLengthFacet : public System::Xml::Schema::XmlSchemaNumericFacet
```
