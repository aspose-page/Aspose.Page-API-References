## ParameterRef class


La clase que implementa una referencia de parámetro común de [PrintTicket](../printticket/). Un elemento **[ParameterRef](./)** define una referencia a un elemento [ParameterInit](../parameterinit/). Un elemento [ScoredProperty](../scoredproperty/) que contiene un elemento [ParameterRef](./) no tiene un elemento [Value](../value/) establecido explícitamente. En su lugar, el elemento [ScoredProperty](../scoredproperty/) recibe su valor del elemento [ParameterInit](../parameterinit/) referenciado por un elemento **[ParameterRef](./)**. [https://docs.microsoft.com/en-us/windows/win32/printdocs/parameterref](https://docs.microsoft.com/en-us/windows/win32/printdocs/parameterref).

```cpp
class ParameterRef : public Aspose::Page::XPS::XpsMetadata::PrintTicketElement,
                     public Aspose::Page::XPS::XpsMetadata::IPrintTicketItem
```
