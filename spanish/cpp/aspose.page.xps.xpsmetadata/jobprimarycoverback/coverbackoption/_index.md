## Campos

| Campo | Descripción |
| --- | --- |
| static [BlankCover](./blankcover/) | Especifica que se debe imprimir una hoja de cubierta en blanco. |
| static [NoCover](./nocover/) | Especifica que no se imprimirá ninguna cubierta. |
| static [PrintBack](./printback/) | Especifica que la cubierta indicada por "CoverBackSource" debe imprimirse en el reverso de la hoja de cubierta. Si no se especifica un elemento [JobPrimaryCoverBackSource](../../jobprimarycoverbacksource/)[ParameterInit](../../parameterinit/), esta [Option](../../option/) debe ser ignorada. |
| static [PrintBoth](./printboth/) | Especifica que la cubierta indicada por \"CoverBackSource\" puede imprimirse en cualquiera de los dos lados de la hoja de cubierta. Si no se especifica un elemento [JobPrimaryCoverBackSource](../../jobprimarycoverbacksource/)[ParameterInit](../../parameterinit/), esta [Opción](../../option/) debe ser ignorada. |
| static [PrintFront](./printfront/) | Especifica que la cubierta indicada por \"CoverBackSource\" debe imprimirse en el lado frontal de la hoja de cubierta. Si no se especifica un elemento [JobPrimaryCoverBackSource](../../jobprimarycoverbacksource/)[ParameterInit](../../parameterinit/), esta [Opción](../../option/) debe ser ignorada. |
