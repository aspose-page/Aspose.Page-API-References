## JobPrimaryCoverBack class


Describe la hoja de cubierta posterior (final). Cada trabajo tendrá una hoja primaria separada. La hoja de cubierta debe imprimirse con el [PageMediaSize](../pagemediasize/) y [PageMediaType](../pagemediatype/) utilizados para la página final del trabajo. La hoja de cubierta debe integrarse en las opciones de procesamiento (como [JobDuplexAllDocumentsContiguously](../jobduplexalldocumentscontiguously/), [JobNUpAllDocumentsContiguously](../jobnupalldocumentscontiguously/)) según lo indique la [Option](../option/) especificada.
