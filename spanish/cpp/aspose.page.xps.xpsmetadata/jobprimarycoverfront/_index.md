## JobPrimaryCoverFront class


Describe la hoja de cubierta frontal (inicial). Todo el trabajo tendrá una única hoja primaria. La hoja de cubierta debe imprimirse con el [PageMediaSize](../pagemediasize/) y [PageMediaType](../pagemediatype/) utilizados para la primera página del trabajo. La hoja de cubierta debe integrarse en las opciones de procesamiento (como [JobDuplexAllDocumentsContiguously](../jobduplexalldocumentscontiguously/), [JobNUpAllDocumentsContiguously](../jobnupalldocumentscontiguously/)) según lo indique la [Option](../option/) especificada. [https://docs.microsoft.com/en-us/windows/win32/printdocs/jobprimarycoverfront](https://docs.microsoft.com/en-us/windows/win32/printdocs/jobprimarycoverfront).
