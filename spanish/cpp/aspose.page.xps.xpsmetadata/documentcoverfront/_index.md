## DocumentCoverFront class


Describe la hoja de cubierta frontal (inicial). Cada documento tendrá una hoja separada. La hoja de cubierta debe imprimirse con el [PageMediaSize](../pagemediasize/) y [PageMediaType](../pagemediatype/) utilizados para la primera página del documento. La hoja de cubierta debe integrarse en las opciones de procesamiento (como [DocumentDuplex](../documentduplex/), [DocumentNUp](../documentnup/)) según lo indique la [Option](../option/) especificada. [https://docs.microsoft.com/en-us/windows/win32/printdocs/documentcoverfront](https://docs.microsoft.com/en-us/windows/win32/printdocs/documentcoverfront).
