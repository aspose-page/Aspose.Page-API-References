## PageForceFrontSide class


Obliga a que la salida aparezca en el frente de una hoja de medio. Relevante para hojas de medio con superficies diferentes en cada lado. En los casos en que esta característica interfiere con las opciones de procesamiento (como [DocumentDuplex](../documentduplex/)), **[PageForceFrontSide](./)** tiene precedencia para la página específica a la que se aplica la característica.

```cpp
class PageForceFrontSide : public Aspose::Page::XPS::XpsMetadata::Feature,
                           public Aspose::Page::XPS::XpsMetadata::IJobPrintTicketItem,
                           public Aspose::Page::XPS::XpsMetadata::IDocumentPrintTicketItem,
                           public Aspose::Page::XPS::XpsMetadata::IPagePrintTicketItem
```
