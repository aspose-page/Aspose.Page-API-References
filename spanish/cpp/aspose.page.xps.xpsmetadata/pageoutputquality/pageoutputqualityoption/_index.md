## PageOutputQualityOption class


Define las opciones de la característica [PageOutputQuality](../).

```cpp
class PageOutputQualityOption : public Aspose::Page::XPS::XpsMetadata::Option
```

## Campos

| Campo | Descripción |
| --- | --- |
| static [Automatic](./automatic/) | Especifica la intención para la salida automática (permite que el cliente elija automáticamente la mejor configuración). |
| static [Draft](./draft/) | Especifica la intención para la salida preliminar (equilibrada para texto y gráficos de calidad preliminar). |
| static [Fax](./fax/) | Especifica la intención para la salida de fax (equilibrada para la calidad estándar de fax). |
| static [High](./high/) | Especifica la intención para la salida de alta calidad (equilibrada para texto y gráficos de alta calidad). |
| static [Normal](./normal/) | Especifica la intención para la salida normal (equilibrada para texto y gráficos de buena calidad). |
| static [Photographic](./photographic/) | Especifica la intención para la salida fotográfica (las imágenes deben tener la máxima calidad). |
| static [Text](./text/) | Especifica la intención para la salida solo de texto (los gráficos pueden producirse de forma deficiente o no producirse en absoluto). |
