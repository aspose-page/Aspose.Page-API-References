## PsSaveFormat enum


Esta enumeración contiene las opciones disponibles de formato de guardado. Puede ser PS o [EPS](../../aspose.page.eps/). [EPS](../../aspose.page.eps/) se usa solo para documentos de una página, mientras que un archivo PS puede contener cualquier número de páginas.

```cpp
enum class PsSaveFormat
```

### Valores

| Nombre | Valor | Descripción |
| --- | --- | --- |
| PS | 0 | Esta opción indica que el documento resultante debe ser un archivo PostScript (PS) o Encapsulated PostScript ([EPS](../../aspose.page.eps/)). |
| EPS | 1 | Esta opción indica que el documento resultante debe ser un archivo Encapsulated PostScript ([EPS](../../aspose.page.eps/)). Se usa solo para documentos de una página;. |
