## PdfSaveOptions::set_OutlineTreeHeight method


Especifica la altura del árbol de esquema del documento a guardar. 0 - el árbol de esquema no se convertirá, 1 - solo se convertirán los elementos del esquema de primer nivel, y así sucesivamente. El valor predeterminado es 10.
