## PdfSaveOptions::set_TextCompression method


Especifica en qué nivel del esquema del documento se deben mostrar los objetos [ApsBookmark](../). 0 - no se muestra. 1 en el primer nivel y así sucesivamente. El valor predeterminado es 0.

## Observaciones


Especifica el tipo de compresión que se usará para todas las secuencias de contenido, excepto las imágenes. El valor predeterminado es [PdfTextCompression::Flate](../../pdftextcompression/).
