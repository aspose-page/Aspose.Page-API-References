## EmfType enum


Especifica los tipos de registros que se colocan en un archivo EMF.

```cpp
enum class EmfType
```

### Valores

| Nombre | Valor | Descripción |
| --- | --- | --- |
| EmfOnly | n/a | [Windows](../../system.windows/) metarchivo mejorado. Contiene comandos GDI. Los metarchivos de este tipo se denominan archivo EMF. |
| EmfPlusOnly | n/a | [Windows](../../system.windows/) metarchivo mejorado plus. Contiene comandos GDI+. Los metarchivos de este tipo se denominan archivo EMF+. |
| EmfPlusDual | n/a | Dual [Windows](../../system.windows/) metarchivo mejorado. Contiene comandos equivalentes GDI y GDI+. Los metarchivos de este tipo se denominan archivo EMF+. |
