## ColorMatrixFlag enum


Especifica los tipos de imágenes y colores que se verán afectados por la configuración de ajuste de color y escala de grises de un objeto [ImageAttributes](../imageattributes/).

```cpp
enum class ColorMatrixFlag
```

### Valores

| Nombre | Valor | Descripción |
| --- | --- | --- |
| Default | 0 | Tanto los colores como los valores de escala de grises se ajustan de color. |
| SkipGrays | 1 | Los valores de escala de grises no se ajustan de color. |
| AltGrays | 2 | Solo los valores de escala de grises se ajustan de color. |
