## RegexOptions enum


[Regex](../regex/) options.

```cpp
enum class RegexOptions
```

### Valores

| Nombre | Valor | Descripción |
| --- | --- | --- |
| None | 0 | Comportamiento predeterminado. |
| Compiled | 1 | Compila la expresión regular para rendimiento. Siempre se hace por defecto. |
| CultureInvariant | 2 | Utiliza coincidencia independiente de la cultura. Ignorado. |
| ECMAScript | 4 | Utiliza la sintaxis ECMAScript. Ignorado. |
| ExplicitCapture | 8 | Captura explícita solamente. Ignorado. |
| IgnoreCase | 16 | Ignorar mayúsculas y minúsculas al coincidir. |
| IgnorePatternWhitespace | 32 | Ignorar espacios en blanco en el patrón. No compatible. |
| Multiline | 64 | Tratar '^' y '$' como inicio y fin de línea, no de la cadena completa. |
| RightToLeft | 128 | Coincidencia de derecha a izquierda. No compatible. |
| Singleline | 256 | Hace que '.' coincida con cualquier carácter sin excepciones (normalmente, los caracteres de nueva línea no coinciden). |
