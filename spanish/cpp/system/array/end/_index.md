## Array::end() const method


Devuelve un iterador al elemento que sigue al último elemento del contenedor calificado como const. Este elemento actúa como un marcador de posición; intentar acceder a él produce un comportamiento indefinido.

```cpp
const_iterator System::Array<T>::end() const noexcept
```


### ReturnValue

Un iterador que apunta al elemento teórico colocado después del último elemento del contenedor calificado como const.
