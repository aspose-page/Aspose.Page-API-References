## IComparable::CompareTo method


Compara el objeto actual con el objeto especificado.

```cpp
virtual int System::IComparable<T>::CompareTo(T other)=0
```


| Parámetro | Tipo | Descripción |
| --- | --- | --- |
| otro | T | El objeto con el que comparar el objeto actual |

### ReturnValue

Un número negativo si el objeto actual precede al objeto especificado en el orden de clasificación; 0 si los objetos comparados son iguales; un número positivo si el objeto actual sigue al objeto especificado en el orden de clasificación
