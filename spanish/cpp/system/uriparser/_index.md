## UriParser class


Se usa para analizar un nuevo esquema URI. Los objetos de esta clase solo deben asignarse mediante la función [System::MakeObject()](../makeobject/) function. Nunca cree una instancia de este tipo en la pila ni usando el operador new, ya que provocará errores en tiempo de ejecución y/o fallas de aserción. Siempre envuelva esta clase en un puntero [System::SmartPtr](../smartptr/) y use ese puntero para pasarlo a funciones como argumento.

```cpp
class UriParser
```
