## System::Equals method


Determina la igualdad de dos valores aplicando [operator==()](../operator==/) a ellos.

```cpp
template<typename TA,typename TB> bool System::Equals(const TA &a, const TB &b)
```


| Parámetro | Descripción |
| --- | --- |
| TA | El tipo del primer comparando |
| TB | El tipo del segundo comparando |

| Parámetro | Tipo | Descripción |
| --- | --- | --- |
| a | const TA\& | El primer comparando |
| b | const TB\& | El segundo comparando |

### ReturnValue

The boolean value returned by [operator==()](../operator==/) applied to **a** and **b**
