## SmartPtr::Is method


Comprueba si el objeto apuntado es de un tipo específico o de su tipo hijo. Sigue la semántica 'is' de C#.

```cpp
bool System::SmartPtr<T>::Is(const System::TypeInfo &target) const
```


| Parámetro | Tipo | Descripción |
| --- | --- | --- |
| target | const System::TypeInfo\& | Especifica el tipo objetivo contra el cual comprobar. |

### ReturnValue

Verdadero si la verificación al estilo 'is' de C# es positiva y falso en caso contrario.
