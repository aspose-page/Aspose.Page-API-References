## TimeZone::IsDaylightSavingTime method


Determina si el valor de fecha y hora representado por el objeto [DateTime](../../datetime/) especificado se encuentra dentro del rango de horario de verano para la zona horaria representada por el objeto [TimeZone](../) actual.

```cpp
virtual bool System::TimeZone::IsDaylightSavingTime(DateTime time)=0
```


### ReturnValue

True si el valor de fecha y hora representado por el objeto [DateTime](../../datetime/) especificado se encuentra dentro del rango de horario de verano de la zona horaria representada por el objeto [TimeZone](../) actual, de lo contrario - false
