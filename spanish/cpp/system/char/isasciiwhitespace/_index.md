## Char::IsAsciiWhiteSpace method


Determina si el carácter especificado se clasifica como un carácter de espacio en blanco ASCII.

```cpp
static constexpr bool System::Char::IsAsciiWhiteSpace(char_t c)
```


| Parámetro | Tipo | Descripción |
| --- | --- | --- |
| c | char_t | El carácter a probar |

### ReturnValue

True si el carácter especificado es un carácter de espacio en blanco ASCII, de lo contrario - false
