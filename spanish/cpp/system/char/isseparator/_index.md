## Char::IsSeparator(char_t) method


Determina si el carácter especificado está clasificado como un carácter separador.

```cpp
static bool System::Char::IsSeparator(char_t c)
```


| Parámetro | Tipo | Descripción |
| --- | --- | --- |
| c | char_t | El carácter a probar |

### ReturnValue

Verdadero si el carácter especificado es un carácter separador, de lo contrario - falso
