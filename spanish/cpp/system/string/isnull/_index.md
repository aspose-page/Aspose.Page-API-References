## String::IsNull method


Verifica si la cadena se considera nula. [String](../) es nula solo si se construye mediante el constructor [String()](../string/), se mueve, copia o asigna desde una cadena nula o se llamó al método [reset()](../reset/).

```cpp
bool System::String::IsNull() const
```


### ReturnValue

true si la cadena se considera nula, false en caso contrario.
