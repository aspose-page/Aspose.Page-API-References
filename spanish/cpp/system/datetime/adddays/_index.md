## DateTime::AddDays method


Devuelve una nueva instancia de la clase [DateTime](../) que representa el valor de fecha y hora que es la suma del valor representado por el objeto actual y el número especificado de días.

```cpp
DateTime System::DateTime::AddDays(double value) const
```


| Parámetro | Tipo | Descripción |
| --- | --- | --- |
| value | double | El número de días a añadir; el valor puede ser positivo o negativo. |

### ReturnValue

Una nueva instancia de la clase [DateTime](../) que representa el valor de fecha y hora que es la suma del valor representado por el objeto actual y **value** días.
