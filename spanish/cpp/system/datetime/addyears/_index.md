## DateTime::AddYears method


Devuelve una nueva instancia de la clase [DateTime](../) que representa el valor de fecha y hora igual al representado por el objeto actual con el componente de año incrementado en el número especificado.

```cpp
DateTime System::DateTime::AddYears(int value) const
```


| Parámetro | Tipo | Descripción |
| --- | --- | --- |
| value | int | El número de años a añadir; el valor puede ser positivo o negativo |

### ReturnValue

Una nueva instancia de la clase [DateTime](../) que representa el valor de fecha y hora igual al representado por el objeto actual con el componente de año incrementado en **value**
