## DateTime::FromFileTime method

```cpp
static DateTime System::DateTime::FromFileTime(int64_t value)
```


| Parámetro | Tipo | Descripción |
| --- | --- | --- |
| value | int64_t | Tiempo de archivo a convertir |

### ReturnValue
