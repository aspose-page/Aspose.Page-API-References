## ValueType class


Clase base para tipos de valor con herencia de [Object](../object/) truncada por razones de rendimiento. Este tipo debe asignarse en la pila y pasarse a funciones por valor o por referencia. Nunca use la clase [System::SmartPtr](../smartptr/) para gestionar objetos de este tipo.
