## XPathNavigator::MoveToId method


Cuando se sobrescribe en una clase derivada, se mueve al nodo que tiene un atributo de tipo **ID** cuyo valor coincide con la [String](../../../system/string/) especificada.

```cpp
virtual bool System::Xml::XPath::XPathNavigator::MoveToId(String id)=0
```


| Parámetro | Tipo | Descripción |
| --- | --- | --- |
| id | String | Una [String](../../../system/string/) que representa el valor **ID** del nodo al que desea moverse. |

### ReturnValue

**true** if the [XPathNavigator](../) is successful moving; otherwise, **false**. If **false**, the position of the navigator is unchanged.
