## IEnumerable::LINQ_All method


Determina si todos los elementos de una secuencia cumplen una condición.

```cpp
bool System::Collections::Generic::IEnumerable<T>::LINQ_All(std::function<bool(T)> predicate)
```


| Parámetro | Tipo | Descripción |
| --- | --- | --- |
| predicado | std::function\<bool(T)> | Una función para probar cada elemento con una condición. |

### ReturnValue

true si cada elemento de la secuencia de origen pasa la prueba en el predicado especificado, o si la secuencia está vacía; de lo contrario, false.
