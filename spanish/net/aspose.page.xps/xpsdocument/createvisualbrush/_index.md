## XpsDocument.CreateVisualBrush method

Crea un nuevo pincel visual.

```csharp
public XpsVisualBrush CreateVisualBrush(XpsContentElement element, RectangleF viewbox, 
    RectangleF viewport)
```

| Parámetro | Escribe | Descripción |
| --- | --- | --- |
| element | XpsContentElement | El elemento XPS (Canvas, Path o Glyphs) para la propiedad visual del pincel visual. |
| viewbox | RectangleF | La posición y las dimensiones del contenido de origen del pincel. |
| viewport | RectangleF | La región en el espacio de coordenadas que contiene el mosaico principal brush que se aplica (posiblemente repetidamente) para llenar la región a la que se aplica el pincel. |

### Valor_devuelto

Nuevo pincel visual.
