El **Aspose.Page.EPS** es un espacio de nombres raíz para todas las clases que tratan con archivos PS/EPS.

## Clases

| Clase | Descripción |
| --- | --- |
| [FontConstants](./fontconstants/) | Esta clase define un conjunto de constantes para guardar fuentes. |
| [GradientBrush](./gradientbrush/) | Esta clase se utiliza para encapsular LinearGradientBrush y PathGradientBrush con la posibilidad de establecer el modo de ajuste en clamp. Los pinceles de degradado nativos siempre lanzan una excepción cuando alguien intenta establecer la propiedad WrapMode en WrapMode.Clamp. |
| [LoadOptions](./loadoptions/) | Clase básica para opciones de carga de documentos. |
| [PageConstants](./pageconstants/) | Esta clase define un conjunto de constantes que describen una página. Se proporcionan objetos prácticos para varios márgenes, orientaciones, cambio de escala y tamaños de página estándar. |
| [PsConverterException](./psconverterexception/) | Esta clase contiene información sobre un error que se produce mientras el archivo PS se convierte en un documento PDF. |
| [PsDocument](./psdocument/) | Esta clase encapsula documentos PS/EPS. |
| [PsDocumentException](./psdocumentexception/) | Esta clase contiene información sobre un error que se produce mientras se crea, edita y guarda el archivo PostScript. |
| [PsLoadOptions](./psloadoptions/) | Opciones de carga de documentos PS. |
