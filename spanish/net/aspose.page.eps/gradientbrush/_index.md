## GradientBrush class

Esta clase se utiliza para encapsular LinearGradientBrush y PathGradientBrush con la posibilidad de establecer el modo de ajuste en clamp. Los pinceles de degradado nativos siempre lanzan una excepción cuando alguien intenta establecer la propiedad WrapMode en WrapMode.Clamp.

```csharp
public class GradientBrush : Brush
```

## Constructores

| Nombre | Descripción |
| --- | --- |
| [GradientBrush](gradientbrush/)(Brush) | Crea una nueva instancia de GradientBrush.  LinearGradientBrush o PathGradientBrush. Si se lanzará otra excepción que no sea ArgumentException. |
