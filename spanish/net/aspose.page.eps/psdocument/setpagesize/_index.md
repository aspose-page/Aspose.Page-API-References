## PsDocument.SetPageSize method

Establece el tamaño de la página. Para crear páginas con diferentes tamaños en un documento, use[`SetPageDevice`](../setpagedevice/) método justo después de este método.

```csharp
public void SetPageSize(float width, float height)
```

| Parámetro | Escribe | Descripción |
| --- | --- | --- |
| width | Single | El ancho de la página en el archivo PostScript resultante. |
| height | Single | La altura de la página en el archivo PostScript resultante. |
