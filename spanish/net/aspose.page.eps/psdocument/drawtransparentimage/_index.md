## PsDocument.DrawTransparentImage method

Dibujar imagen transparente transformada. Si la imagen no tiene canal alfa, se dibujará como imagen opaca

```csharp
public void DrawTransparentImage(Bitmap image, Matrix transform, int transparencyThreshold)
```

| Parámetro | Escribe | Descripción |
| --- | --- | --- |
| image | Bitmap | La imagen para dibujar. |
| transform | Matrix | La matriz para transformar la imagen. |
| transparencyThreshold | Int32 | Un umbral que define a partir de qué valor de píxel de transparencia se interpretará como totalmente transparente. Todos los valores por debajo de este umbral se interpretarán como totalmente opacos. |
