## FillAndStrokeText(string, Font, float, float, Brush, Pen) {#fillandstroketext_3}

Agrega una cadena de texto llenando el interior de los glifos y dibujando los contornos de los glifos.

```csharp
public void FillAndStrokeText(string text, Font font, float x, float y, Brush fillPaint, Pen stroke)
```

| Parámetro | Escribe | Descripción |
| --- | --- | --- |
| text | String | El texto a agregar. |
| font | Font | Fuente del sistema que se utilizará para dibujar texto. |
| x | Single | Coordenada X para el origen del texto. |
| y | Single | Coordenada Y para el origen del texto. |
| fillPaint | Brush | El relleno utilizado para pintar el interior de los glifos. |
| stroke | Pen | El trazo utilizado para dibujar contornos de glifos. |
