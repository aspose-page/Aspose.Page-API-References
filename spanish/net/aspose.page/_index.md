El **Aspose.Page** es un espacio de nombres raíz para todas las clases de la biblioteca Aspose.Page que están directamente en ella como **Dispositivo** o indirectamente a través de varios subnamespaces.

## Clases

| Clase | Descripción |
| --- | --- |
| [BuildVersionInfo](./buildversioninfo/) | Esta clase proporciona información sobre la compilación actual del producto. |
| [Device](./device/) | Esta clase encapsula la representación del documento en un dispositivo abstracto. La representación del documento se realiza página por página. |
| [Document](./document/) | Una superclase para todos los documentos encapsulados. |
| [ExternalFontCache](./externalfontcache/) | Use esta clase para obtener la encapsulación de fuentes en una forma que sea aceptada por[`Device`](../aspose.page/device/) . |
| [License](./license/) | Proporciona métodos para licenciar el componente. |
| [Margins](./margins/) | Esta clase encapsula los márgenes superior, izquierdo, inferior y derecho. |
| [Metered](./metered/) | Proporciona métodos para configurar la clave medida. |
| [SaveOptions](./saveoptions/) | Esta clase contiene las opciones necesarias para administrar el proceso de conversión. |
| [UserProperties](./userproperties/) | Clase de propiedad especial que permite establecer propiedades escritas y devolver . También permite buscar la conexión de dos objetos de propiedad predeterminados si este objeto de propiedad no contiene la propiedad. |
## Interfaces

| Interfaz | Descripción |
| --- | --- |
| [IGlyph](./iglyph/) | Esta interfaz da acceso a los principales parámetros de glyphs. |
| [IMultiPageDevice](./imultipagedevice/) | Esta interfaz contiene métodos para manipular dispositivos de varias páginas. |
| [IStreamable](./istreamable/) | Esta interfaz se usa para dispositivos que procesan documentos en una secuencia. |
| [ITrFont](./itrfont/) | Esta interfaz da acceso a los principales parámetros de font. |
## Enumeración

| Enumeración | Descripción |
| --- | --- |
| [TextRenderingMode](./textrenderingmode/) | Esta enumeración contiene valores posibles para el modo de representación de texto. |
