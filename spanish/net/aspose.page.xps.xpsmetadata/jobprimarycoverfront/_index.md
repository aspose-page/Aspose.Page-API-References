## JobPrimaryCoverFront class

Describe la portada (principio) delantera. Todo el trabajo tendrá una única hoja principal. La portada debe imprimirse en la[`PageMediaSize`](../pagemediasize/) y[`PageMediaType`](../pagemediatype/) utilizado para la primera página del trabajo. La portada debe integrarse en las opciones de procesamiento (como[`JobDuplexAllDocumentsContiguously`](../jobduplexalldocumentscontiguously/) ,[`JobNUpAllDocumentsContiguously`](../jobnupalldocumentscontiguously/) ) como lo indica la opción especificada. https://docs.microsoft.com/en-us/windows/win32/printdocs/jobprimarycoverfront
