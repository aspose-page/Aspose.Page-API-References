## PageDeviceColorSpaceUsage.PageDeviceColorSpaceUsageOption.MatchToDefault field

Si el dispositivo determina que el perfil especificado por el[`PageDeviceColorSpaceProfileURI`](../../pagedevicecolorspaceprofileuri) El parámetro se puede usar como un perfil de espacio de color del dispositivo, todos los elementos que usan el mismo perfil se tratan como si ya estuvieran especificados en el espacio de color del dispositivo. Todos los demás elementos DEBEN utilizar el perfil especificado para ese elemento. Si el perfil no se puede usar como perfil de espacio de color del dispositivo, los elementos que usan el perfil DEBEN administrarse en color como cualquier otro elemento que usa el perfil de color.

public static PageDeviceColorSpaceUsageOption MatchToDefault;
