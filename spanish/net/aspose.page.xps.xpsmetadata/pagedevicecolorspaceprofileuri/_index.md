## PageDeviceColorSpaceProfileURI class

Especifica un URI relativo a la raíz del paquete para un perfil ICC contenido en un documento XPS. El procesamiento de esta opción depende de la configuración de la función PageDeviceColorSpaceUsage. Se supone que todos los elementos que usan ese perfil ya tienen el color de dispositivo apropiado space, y no se administrará el color en el controlador o dispositivo. https://docs.microsoft.com/en-us/windows/win32/printdocs/pagedevicecolorspaceprofileuri
