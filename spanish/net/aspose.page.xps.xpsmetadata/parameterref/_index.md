## ParameterRef class

La clase que implementa una referencia de parámetro PrintTicket común. A elemento define una referencia a un[`ParameterInit`](../parameterinit/)elemento. A[`ScoredProperty`](../scoredproperty/) elemento que contiene un elemento ParameterRef no tiene un valor establecido explícitamente[`Value`](../value/) elemento. En cambio, el[`ScoredProperty`](../scoredproperty/) element recibe su valor del[`ParameterInit`](../parameterinit/) elemento referenciado por un elemento. https://docs.microsoft.com/en-us/windows/win32/printdocs/parameterref
