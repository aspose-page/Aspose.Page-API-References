## PdfSaveOptions.OutlineTreeExpansionLevel property

Especifica hasta qué nivel debe expandirse el esquema del documento cuando se visualiza el archivo PDF. 1: solo se muestran los elementos del esquema del primer nivel, 2: solo se muestran los elementos del esquema del primer y segundo nivel, y así sucesivamente. El valor predeterminado es 1.

public int OutlineTreeExpansionLevel { get; set; }
