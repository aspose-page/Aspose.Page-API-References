## PdfSaveOptions.OutlineTreeHeight property

Especifica la altura del árbol de esquema del documento para guardar. 0: el árbol de esquema no se convertirá, 1: solo se convertirán los elementos de esquema de primer nivel, y así sucesivamente. El valor predeterminado es 10.

```csharp
public int OutlineTreeHeight { get; set; }
```
