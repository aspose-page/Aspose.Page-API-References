## XpsRadialGradientBrush class

Clase que encapsula las características del elemento de propiedad RadialGradientBrush. Este elemento se utiliza para especificar un pincel de degradado radial.

```csharp
public sealed class XpsRadialGradientBrush : XpsGradientBrush
```

## Propiedades

| Nombre | Descripción |
| --- | --- |
| [Center](../../aspose.page.xps.xpsmodel/xpsradialgradientbrush/center/) { get; set; } | Devuelve/establece el punto central del gradiente radial (es decir, el centro de la elipse). |
| [ColorInterpolationMode](../../aspose.page.xps.xpsmodel/xpsgradientbrush/colorinterpolationmode/) { get; set; } | Devuelve/establece el valor que especifica la función gamma para la interpolación de color. El ajuste gamma no debe aplicarse al componente alfa, si se especifica. |
| [GradientOrigin](../../aspose.page.xps.xpsmodel/xpsradialgradientbrush/gradientorigin/) { get; set; } | Devuelve/establece el punto de origen del degradado radial. |
| [GradientStops](../../aspose.page.xps.xpsmodel/xpsgradientbrush/gradientstops/) { get; set; } | Devuelve/establece la lista de paradas de gradiente que componen el gradiente. |
| [Opacity](../../aspose.page.xps.xpsmodel/xpsbrush/opacity/) { get; set; } | Devuelve/establece el valor que define la transparencia uniforme del relleno de pincel. |
| [RadiusX](../../aspose.page.xps.xpsmodel/xpsradialgradientbrush/radiusx/) { get; set; } | Devuelve/establece el radio en la dimensión x de la elipse que define el gradiente radial. |
| [RadiusY](../../aspose.page.xps.xpsmodel/xpsradialgradientbrush/radiusy/) { get; set; } | Devuelve/establece el radio en la dimensión y de la elipse que define el gradiente radial. |
| [SpreadMethod](../../aspose.page.xps.xpsmodel/xpsgradientbrush/spreadmethod/) { get; set; } | Devuelve/establece el valor que describe cómo el pincel debe llenar el área de contenido fuera del área primaria, degradado inicial. |
| [Transform](../../aspose.page.xps.xpsmodel/xpstransformablebrush/transform/) { get; set; } | Devuelve/establece la transformación matricial aplicada al espacio de coordenadas del pincel. La ventana gráfica de brush se transforma utilizando la transformación de renderización efectiva local. |
