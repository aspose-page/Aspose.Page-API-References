## XpsGradientStop class

Clase que encapsula las características del elemento GradientStop. Este elemento lo utilizan los elementos LinearGradientBrush y RadialGradientBrush para definir la ubicación y el rango de progresión de color para representar un degradado.

```csharp
public sealed class XpsGradientStop : XpsObject
```

## Propiedades

| Nombre | Descripción |
| --- | --- |
| [Color](../../aspose.page.xps.xpsmodel/xpsgradientstop/color/) { get; } | El color de parada del degradado. |
| [Offset](../../aspose.page.xps.xpsmodel/xpsgradientstop/offset/) { get; } | Devuelve/establece el desplazamiento del degradado. El desplazamiento indica un punto a lo largo de la progresión de el degradado en el que se especifica un color. Los colores entre las compensaciones de degradado en la progresión se interpolan. |
