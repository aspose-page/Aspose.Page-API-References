## XpsTransformableBrush.Transform property

Devuelve/establece la transformación matricial aplicada al espacio de coordenadas del pincel. La propiedad Transform se concatena con la transformación de renderización efectiva actual para producir una transformación de renderización efectiva local para el pincel. La ventana gráfica de brush se transforma utilizando la transformación de renderización efectiva local.

```csharp
public XpsMatrix Transform { get; set; }
```
