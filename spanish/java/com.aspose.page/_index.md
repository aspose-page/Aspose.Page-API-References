El **Aspose.Page** es un paquete raíz para todas las clases de la biblioteca Aspose.Page que están ya sea directamente en él como **Device** o indirectamente a través de varios subpaquetes.


## Clases

| Clase | Descripción |
| --- | --- |
| [AsposePageException](../com.aspose.page/asposepageexception) | Representa los errores que ocurren durante la ejecución de la aplicación Aspose.Page. |
| [DimensionF](../com.aspose.page/dimensionf) | La clase `Dimension` encapsula el ancho y la altura de un componente (en precisión simple) en un solo objeto. |
| [Document](../com.aspose.page/document) | Una superclase para todos los documentos encapsulados. |
| [ExternalFontCache](../com.aspose.page/externalfontcache) | Utilice esta clase para obtener la encapsulación de fuentes en un formato que sea aceptado por Device. |
| [License](../com.aspose.page/license) | Proporciona métodos para licenciar el componente. |
| [Metered](../com.aspose.page/metered) | Proporciona métodos para establecer la clave medida. |
| [SaveOptions](../com.aspose.page/saveoptions) | Esta clase contiene opciones necesarias para gestionar el proceso de conversión. |
| [TransformedStroke](../com.aspose.page/transformedstroke) | Un trazo que contiene transformación. |
| [UserProperties](../com.aspose.page/userproperties) | Clase de propiedad especial que permite establecer y devolver propiedades tipadas. |
