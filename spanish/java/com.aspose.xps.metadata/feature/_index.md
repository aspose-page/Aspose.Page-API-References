La clase que encapsula una característica común de Print Schema. La clase base para todas las características definidas por el esquema. Un elemento  Feature  contiene una lista completa de los elementos  Option  y  Property  que describen completamente un atributo del dispositivo, una configuración de formato de trabajo u otra característica relevante. https://docs.microsoft.com/en-us/windows/win32/printdocs/feature
