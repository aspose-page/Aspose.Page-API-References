## XpsPath.StrokeThickness property

Returnerar/ställer in tjockleken på ett streck, i enheter av det effektiva koordinatutrymmet (inkluderar banans renderingstransform). Sträcket ritas ovanpå gränsen för geometrin specificerad av Path-elementets Data-egenskap. Hälften av StrokeThickness sträcker sig utanför geometrin som specificeras av dataegenskapen och den andra halvan sträcker sig inuti geometrin.

```csharp
public float StrokeThickness { get; set; }
```
