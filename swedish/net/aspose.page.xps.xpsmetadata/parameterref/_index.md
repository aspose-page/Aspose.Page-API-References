## ParameterRef class

Klassen som implementerar en gemensam PrintTicket-parameterreferens. A element definierar en referens till en[`ParameterInit`](../parameterinit/)element. A[`ScoredProperty`](../scoredproperty/) element som innehåller ett ParameterRef-element har inte en explicit-set[`Value`](../value/) element. Istället[`ScoredProperty`](../scoredproperty/) element får sitt värde från[`ParameterInit`](../parameterinit/) element som refereras av en element. https://docs.microsoft.com/en-us/windows/win32/printdocs/parameterref
