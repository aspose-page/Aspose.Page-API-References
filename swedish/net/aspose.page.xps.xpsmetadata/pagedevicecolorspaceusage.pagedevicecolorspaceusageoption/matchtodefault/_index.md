## PageDeviceColorSpaceUsage.PageDeviceColorSpaceUsageOption.MatchToDefault field

Om enheten bestämmer att profilen som anges av[`PageDeviceColorSpaceProfileURI`](../../pagedevicecolorspaceprofileuri) parameter kan användas som en enhetsfärgrymdsprofil, alla element som använder samma profil behandlas som redan specificerade i enhetens färgrymd. Alla andra element MÅSTE använda den profil som anges för det elementet. Om profilen inte kan användas som en enhetsfärgrymdsprofil, MÅSTE element som använder profilen färghanteras som alla andra element som använder färgprofilen.
