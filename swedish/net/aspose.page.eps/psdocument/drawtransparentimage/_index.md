## PsDocument.DrawTransparentImage method

Rita transformerad transparent bild. Om bilden inte har alfakanal kommer den att ritas som ogenomskinlig image

```csharp
public void DrawTransparentImage(Bitmap image, Matrix transform, int transparencyThreshold)
```

| Parameter | Typ | Beskrivning |
| --- | --- | --- |
| image | Bitmap | Bilden att rita. |
| transform | Matrix | Matrisen för att transformera bilden. |
| transparencyThreshold | Int32 | En tröskel som definierar från vilket värde på transparenspixeln kommer att tolkas som helt transparent. Alla värden under denna tröskel kommer att tolkas som helt ogenomskinliga. |
