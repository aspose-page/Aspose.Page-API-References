## PsDocument.ClipAndNewPath method

Lägger till klipp till aktuell grafikstatus och sedan skriver "newpath"-operatorn. Det är nödvändigt att göra för att escape av sammanflödet av denna urklippsbana och några efterföljande sökvägar, såsom glyfer som beskrivs med "charpath"-operatorn.

```csharp
public void ClipAndNewPath(GraphicsPath s)
```
