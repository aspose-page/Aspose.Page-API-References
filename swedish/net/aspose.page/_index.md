Den **Aspose.Page** är ett rotnamnområde för alla klasser av Aspose.Page-biblioteket som antingen finns direkt i det **Enhet** eller indirekt genom flera undernamnsutrymmen.

## Klasser

| Klass | Beskrivning |
| --- | --- |
| [BuildVersionInfo](./buildversioninfo/) | Den här klassen ger information om aktuell produktbygge. |
| [Device](./device/) | Denna klass kapslar in rendering av dokument till abstrakt enhet. Rendering av dokumentet utförs sida för sida. |
| [Document](./document/) | En superklass för alla inkapslade dokument. |
| [ExternalFontCache](./externalfontcache/) | Använd den här klassen för att erhålla teckensnittsinkapsling i en form som accepteras av[`Device`](../aspose.page/device/) . |
| [License](./license/) | Tillhandahåller metoder för att licensiera komponenten. |
| [Margins](./margins/) | Denna klass kapslar in topp-, vänster-, botten- och högermarginaler. |
| [Metered](./metered/) | Tillhandahåller metoder för att ställa in mätnyckel. |
| [SaveOptions](./saveoptions/) | Den här klassen innehåller alternativ som är nödvändiga för att hantera konverteringsprocessen. |
| [UserProperties](./userproperties/) | Särskild egenskapsklass som gör att inskrivna egenskaper kan ställas in och returneras. Det tillåter också att kopplingen av två standardegenskapsobjekt objekt kan sökas om detta egenskapsobjekt inte innehåller egenskapen. |
## Gränssnitt

| Gränssnitt | Beskrivning |
| --- | --- |
| [IGlyph](./iglyph/) | Detta gränssnitt ger tillgång till huvudparametrar för glyfer. |
| [IMultiPageDevice](./imultipagedevice/) | Det här gränssnittet innehåller metoder för att manipulera flersidiga enheter. |
| [IStreamable](./istreamable/) | Det här gränssnittet används för enheter som renderar dokument till en ström. |
| [ITrFont](./itrfont/) | Detta gränssnitt ger tillgång till huvudparametrarna för teckensnitt. |
