Klass som implementerar utskriftsbiljett‑parameterinitierare. Klassen som implementerar en gemensam PrintTicket‑parameterinitierare. Bas‑klassen för alla schemadefinierade parameterinitierare. Definierar ett värde för en instans av ett  ParameterDef  -element. Ett  ParameterInit  -element är målet för referensen som görs av ett  ParameterRef  -element. https://docs.microsoft.com/en-us/windows/win32/printdocs/parameterinit
