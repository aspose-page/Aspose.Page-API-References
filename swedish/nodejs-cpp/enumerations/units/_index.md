## Units enumeration

Anger storlekstyp.

```js
enum Units
```

### Värden

| Namn | Värde | Beskrivning |
| ---        | ---   | --- |
| Points | `0` | Punkter (1/72 tum). |
| Inches | `1` | Tum. |
| Millimeters | `2` | Millimeter. |
| Centimeters | `3` | Centimeter. |
| Percents | `4` | Procent av befintlig storlek. |
