## SystemFonts class


Tillhandahåller en uppsättning förskapade [Font](../font/) objekt som representerar typsnitt som används för att visa text i [Windows](../../system.windows/) display‑element. Detta är en statisk typ utan instansfunktioner. Du bör aldrig skapa instanser av den på något sätt.

```cpp
class SystemFonts : public System::Object
```
