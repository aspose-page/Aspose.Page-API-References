## StringTrimming enum


Anger hur tecken ska trunkeras från en sträng som inte får plats i layoutformen.

```cpp
enum class StringTrimming
```

### Värden

| Namn | Värde | Beskrivning |
| --- | --- | --- |
| None | 0 | Ingen trimning. |
| Tecken | 1 | Trimma till närmaste tecken. |
| Ord | 2 | Trimma till närmaste ord. |
| EllipsisCharacter | 3 | Trimma till närmaste tecken och infoga ellipsis i slutet av strängen. |
| EllipsisWord | 4 | Trimma till närmaste ord och infoga ellipsis i slutet av strängen. |
| EllipsisPath | 5 | Centrala delen tas bort från trimmade rader och ersätts av en ellipsis. Behåll så mycket som möjligt av det sista snedstrecksavgränsade segmentet av raden. |
