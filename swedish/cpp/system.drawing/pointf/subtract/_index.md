## PointF::Subtract(const PointF\&, const Size\&) method


Subtraherar bredd- och höjdvärden från det specificerade [Size](../../size/)-objektet från X- och Y-koordinatvärdena i det specificerade [PointF](../)-objektet på motsvarande sätt.

```cpp
static PointF System::Drawing::PointF::Subtract(const PointF &point, const Size &size)
```


| Parameter | Type | Beskrivning |
| --- | --- | --- |
| punkt | const PointF\& | Punkten att översätta |
| size | const Size\& | Det [Size](../../size/)-objekt som specificerar värdena att subtrahera från koordinatvärdena för **point** |

### ReturnValue

Ett nytt [PointF](../)-objekt vars X-koordinatvärde är lika med resultatet av subtraktionen av breddvärdet för **size** från X-koordinatvärdet för **point**, och vars Y-koordinatvärde är lika med resultatet av subtraktionen av höjdvärdet för **size** från Y-koordinatvärdet för **point**
