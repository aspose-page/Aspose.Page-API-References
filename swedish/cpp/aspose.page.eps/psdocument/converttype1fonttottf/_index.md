## PsDocument::ConvertType1FontToTTF method


Konverterar Type 1-typsnitt till **TrueType**. Namnet på det konverterade TTF-typsnittet blir samma som inmatat Type 1-typsnitt med \".ttf\"-ändelse. TTF-filen sparas till den tilldelade utdatamappen.
