### Värden

| Namn | Värde | Beskrivning |
| --- | --- | --- |
| None | 0 | Standardbeteende. |
| Compiled | 1 | Kompilera regex för prestanda. Görs alltid som standard. |
| CultureInvariant | 2 | Använd kultur‑invariant matchning. Ignoreras. |
| ECMAScript | 4 | Använd ECMAScript-syntax. Ignorerad. |
| ExplicitCapture | 8 | Endast explicit fångst. Ignorerad. |
| IgnoreCase | 16 | Ignorera skiftläge vid matchning. |
| IgnorePatternWhitespace | 32 | Ignorera blanksteg i mönster. Stöds inte. |
| Multiline | 64 | Behandla '^' och '$' som början och slut på rad, inte hela strängen. |
| RightToLeft | 128 | Höger-till-vänster-matchning. Stöds inte. |
| Singleline | 256 | Gör så att '.' matchar alla tecken utan undantag (normalt matchas inte radbrytningstecken). |
