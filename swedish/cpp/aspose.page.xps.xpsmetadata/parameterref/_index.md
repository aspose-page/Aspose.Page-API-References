## ParameterRef class


Klassen som implementerar en gemensam [PrintTicket](../printticket/) parameterreferens. Ett **[ParameterRef](./)** element definierar en referens till ett [ParameterInit](../parameterinit/) element. Ett [ScoredProperty](../scoredproperty/) element som innehåller ett [ParameterRef](./) element har inget explicit angivet [Value](../value/) element. Istället får [ScoredProperty](../scoredproperty/) elementet sitt värde från [ParameterInit](../parameterinit/) elementet som refereras av ett **[ParameterRef](./)** element.
