## MatchToDefault field


Om enheten fastställer att profilen som anges av parametern [PageDeviceColorSpaceProfileURI](../../../pagedevicecolorspaceprofileuri/) kan användas som en enhetsfärgrymdprofil, behandlas alla element som använder samma profil som om de redan är specificerade i enhetens färgrymd. Alla andra element MÅSTE använda den profil som anges för det elementet. Om profilen inte kan användas som en enhetsfärgrymdprofil, MÅSTE element som använder profilen färghanteras på samma sätt som alla andra element som använder färgprofilen.
