## IEnumerable::begin() method


Hämtar en iterator som pekar på det första elementet (om något) i samlingen. Denna iterator kan inte användas för att ändra ett refererat objekt eftersom [GetEnumerator()](../getenumerator/) returnerar ett kopierat objekt av T.

```cpp
iterator System::Collections::Generic::IEnumerable<T>::begin()
```
