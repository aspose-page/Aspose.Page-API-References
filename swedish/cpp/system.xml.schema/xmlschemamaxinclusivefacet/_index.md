## XmlSchemaMaxInclusiveFacet class


Representerar **maxInclusive**-elementet från XML [Schema](../) enligt World Wide [Web](../../system.web/) Consortium (W3C). Denna klass kan användas för att ange en restriktion på det maximala värdet för ett **simpleType**-element. Elementvärdet måste vara mindre än eller lika med värdet för **maxInclusive**-elementet.

```cpp
class XmlSchemaMaxInclusiveFacet : public System::Xml::Schema::XmlSchemaFacet
```
