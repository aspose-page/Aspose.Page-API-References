## File::Copy method


Kopierar den angivna filen till den angivna platsen. Om målfilen redan finns anger en parameter om den ska skrivas över.

```cpp
static void System::IO::File::Copy(const String &sourceFileName, const String &destFileName, bool overwrite=false)
```


| Parameter | Type | Beskrivning |
| --- | --- | --- |
| sourceFileName | const String\& | Sökvägen till filen som ska kopieras |
| destFileName | const String\& | Sökvägen till den nya platsen för filen som ska kopieras |
| skriv över | bool | Sant om den befintliga destinationsfilen ska skrivas över, falskt om kopieringen ska misslyckas om destinationsfilen redan finns |
