## NamespaceHandling enum


Anger om duplikata namnrymdsdeklarationer ska tas bort i [XmlWriter](../xmlwriter/).

```cpp
enum class NamespaceHandling
```

### Värden

| Namn | Värde | Beskrivning |
| --- | --- | --- |
| Default | 0 | Anger att duplikata namnrymdsdeklarationer inte kommer att tas bort. |
| OmitDuplicates | 1 | Anger att duplikata namnrymdsdeklarationer ska tas bort. För att en duplikatnamnrymd ska tas bort måste prefixet och namnrymden matcha. |
