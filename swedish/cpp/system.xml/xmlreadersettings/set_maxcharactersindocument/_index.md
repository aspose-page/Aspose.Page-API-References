## XmlReaderSettings::set_MaxCharactersInDocument method


Ställer in ett värde som anger det maximalt tillåtna antalet tecken i ett XML-dokument. Ett nollvärde (0) betyder att det inte finns några begränsningar för storleken på XML-dokumentet. Ett icke‑nollvärde specificerar den maximala storleken i tecken.
