## XmlElement::HasAttribute(String, String) method


Avgör om den aktuella noden har ett attribut med det angivna lokala namnet och namnrymdens URI.

```cpp
virtual bool System::Xml::XmlElement::HasAttribute(String localName, String namespaceURI)
```


| Parameter | Type | Beskrivning |
| --- | --- | --- |
| localName | String | Det lokala namnet på attributet som ska hittas. |
| namespaceURI | String | Namnområdets URI för attributet som ska hittas. |

### ReturnValue

**true** if the current node has the specified attribute; otherwise, **false**.
