## XmlNamespaceScope enum


Definierar namnrymdens omfattning.

```cpp
enum class XmlNamespaceScope
```

### Värden

| Namn | Värde | Beskrivning |
| --- | --- | --- |
| Alla | 0 | Alla namnrymder som definieras i omfattningen för den aktuella noden. Detta inkluderar xmlns:xml-namnrymden som alltid deklareras implicit. Ordningen på de returnerade namnrymderna är inte definierad. |
| ExcludeXml | 1 | Alla namnrymder som definieras i omfattningen för den aktuella noden, exklusive xmlns:xml-namnrymden, som alltid deklareras implicit. Ordningen på de returnerade namnrymderna är inte definierad. |
| Local | 2 | Alla namnrymder som definieras lokalt på den aktuella noden. |
