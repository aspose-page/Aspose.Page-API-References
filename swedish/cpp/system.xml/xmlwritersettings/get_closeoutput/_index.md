## XmlWriterSettings::get_CloseOutput method


Returnerar ett värde som indikerar om [XmlWriter](../../xmlwriter/) även ska stänga den underliggande strömmen eller TextWriter när [XmlWriter::Close](../../xmlwriter/close/)‑metoden anropas.

```cpp
bool System::Xml::XmlWriterSettings::get_CloseOutput()
```


### ReturnValue

**true** to also close the underlying stream or TextWriter; otherwise, **false**. The default is **false**.
