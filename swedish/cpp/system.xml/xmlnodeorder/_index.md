## XmlNodeOrder enum


Beskriver dokumentordningen för en nod jämfört med en andra nod.

```cpp
enum class XmlNodeOrder
```

### Värden

| Namn | Värde | Beskrivning |
| --- | --- | --- |
| Före | 0 | Den aktuella noden för den här navigatören är före den aktuella noden för den angivna navigatören. |
| Efter | 1 | Den aktuella noden för den här navigatören är efter den aktuella noden för den angivna navigatören. |
| Samma | 2 | De två navigatörerna är placerade på samma nod. |
| Okänd | 3 | Nodpositionerna kan inte bestämmas i dokumentordning i förhållande till varandra. Detta kan inträffa om de två noderna finns i olika träd. |
