## IXpsTextConversionOptions::set_PreserveText method


I [XPS](../../../aspose.page.xps/) kan vissa textelement innehålla referenser till alternativa glyfformer som inte motsvarar någon teckenkod i teckensnittet. Om denna flagga är satt till true konverteras texten från sådana [XPS](../../../aspose.page.xps/)‑element till grafiska former. Därefter visas själva texten transparent ovanpå. Detta gör att texten i sådana element kan markeras. Men en bieffekt är att utdatafilen kan bli mycket större än originalet. Om flaggan är satt till false ersätts tecknen som skulle visas som alternativa former med andra tecken som mappas till de alternativa glyfformerna. Därför kommer texten, även om den fortfarande kan markeras, att modifieras och sannolikt bli oläslig.
