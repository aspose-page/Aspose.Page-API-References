## String::IsNull method


Kontrollerar om strängen anses vara null. [String](../) är null endast om den konstruerats via [String()](../string/)‑konstruktorn, har flyttats, kopierats eller tilldelats från en null‑sträng eller om [reset()](../reset/)‑metoden har anropats.

```cpp
bool System::String::IsNull() const
```


### ReturnValue

true om strängen anses vara null, false annars.
