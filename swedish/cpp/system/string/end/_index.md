## String::end method


Returnerar en pekare till slutet av den faktiska strängbufferten. Återallokerar aldrig något. Garanterar inte att bufferten är null-terminerad.

```cpp
const UChar * System::String::end() const
```


### ReturnValue

Pekare till det första elementet efter den faktiska strängbufferten, null-terminerad eller ej.
