## System::CheckedCast method


Avgör om det angivna värdet faller inom intervallet av värden av typen **TTo** och om så är fallet kastar det till typen **TTo**.

```cpp
template<typename TTo,typename TFrom> TTo System::CheckedCast(TFrom value)
```


| Parameter | Beskrivning |
| --- | --- |
| TTo | Typen som det angivna värdet ska kastas till |
| TFrom | Typen för det angivna värdet |

| Parameter | Type | Beskrivning |
| --- | --- | --- |
| value | TFrom | Värdet att kasta |

### ReturnValue

Värdet av typen **TTo** som motsvarar **value**
