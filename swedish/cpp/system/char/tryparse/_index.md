## Char::TryParse method


Försöker omvandla en sträng bestående av ett enda tecken till ett UTF-16‑tecken. Funktionen lyckas endast när indatasträngen inte är null och har en längd på exakt ett tecken.

```cpp
static bool System::Char::TryParse(const System::String &s, char_t &result)
```


| Parameter | Type | Beskrivning |
| --- | --- | --- |
| s | const System::String\& | [String](../../string/) att konvertera |
| resultat | char_t\& | Utdata‑variabeln som kommer att innehålla konverteringsresultatet om konverteringen lyckas |

### ReturnValue

Sant om konverteringen lyckades, annars - falskt
