## XpsGlyphs.BidiLevel property

Unicode アルゴリズムの双方向のネスティング レベルを指定する値を返す/設定します。 偶数の値は左から右のレイアウトを意味し、奇数の値は右から左のレイアウトを意味します。 右から左のレイアウトは実行原点を右側に配置します。最初のグリフの を正の送り幅 (左への送りを表す) で、後続の グリフを前のグリフの左側に配置する.

```csharp
public int BidiLevel { get; set; }
```
