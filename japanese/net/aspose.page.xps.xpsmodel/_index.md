**Aspose.Page.Xps.XpsModel**名前空間は、XPS ドキュメントの要素を記述するクラスを提供します.

## クラス

| クラス | 説明 |
| --- | --- |
| [XpsArcSegment](./xpsarcsegment/) | ArcSegment 要素の機能をカプセル化するクラス. この要素は楕円弧を記述します. |
| [XpsArray&lt;T&gt;](./xpsarray-1/) | 一般的な XPS モデル配列オブジェクト機能をカプセル化するクラス。 |
| [XpsBrush](./xpsbrush/) | すべてのブラシ要素の共通機能をカプセル化したクラス. |
| [XpsCanvas](./xpscanvas/) | Canvas 要素の機能をカプセル化するクラス。 この要素は、要素をグループ化します。たとえば、Glyphs および Path 要素 をキャンバスでグループ化して、(ハイパーリンク先として) ユニットとして識別したり、 を構成したプロパティ値を各子および先祖要素に適用したりできます。 |
| [XpsColor](./xpscolor/) | 一般的な色の機能をカプセル化する基本クラス。 |
| [XpsContentElement](./xpscontentelement/) | XPS コンテンツ要素の機能をカプセル化します: Canvas、Path、および Glyphs. |
| [XpsElement](./xpselement/) | 一般的な XPS 要素の機能をカプセル化するクラス。 |
| [XpsElementLinkTarget](./xpselementlinktarget/) | 相対名前付きアドレス ハイパーリンク ターゲットをカプセル化するクラス。 |
| [XpsExternalLinkTarget](./xpsexternallinktarget/) | 外部ハイパーリンク ターゲットをカプセル化するクラス。 |
| [XpsFileResource](./xpsfileresource/) | すべてのファイル リソースの共通機能をカプセル化したクラス。 |
| [XpsFont](./xpsfont/) | TrueType フォント リソースをカプセル化するクラス。 |
| [XpsGlyphs](./xpsglyphs/) | Glyphs 要素の機能をカプセル化するクラス。 この要素は、単一のフォントから一様にフォーマットされた一連のテキストを表します。 正確なレンダリングに必要な情報を提供し、コンシューマーを表示する際の search および選択機能をサポートします。 |
| [XpsGradientBrush](./xpsgradientbrush/) | LinerGradientBrush 要素と RadialGradientBrush 要素の共通機能をカプセル化するクラス。 |
| [XpsGradientStop](./xpsgradientstop/) | GradientStop 要素の機能をカプセル化するクラス. この要素は、LinearGradientBrush 要素と RadialGradientBrush 要素の両方で使用され、 グラデーションをレンダリングするための色の進行の位置と範囲を定義します. |
| [XpsHyperlinkElement](./xpshyperlinkelement/) | ハイパーリンクにできる XPS 要素の共通機能をカプセル化します。 |
| [XpsHyperlinkTarget](./xpshyperlinktarget/) | ハイパーリンク ターゲットの基本クラス。 |
| [XpsIccBasedColor](./xpsiccbasedcolor/) | ICC ベースの色をカプセル化します。 |
| [XpsIccProfile](./xpsiccprofile/) | ICC プロファイル リソースをカプセル化するクラス。 |
| [XpsImage](./xpsimage/) | 画像リソースをカプセル化するクラス. |
| [XpsImageBrush](./xpsimagebrush/) | ImageBrush プロパティ要素の機能をカプセル化するクラス. この要素は、領域を画像で塗りつぶすために使用されます. |
| [XpsLinearGradientBrush](./xpslineargradientbrush/) | LinearGradientBrush プロパティ要素の機能をカプセル化するクラス. この要素は、ベクトルに沿って線形グラデーション ブラシを指定するために使用されます. |
| [XpsMatrix](./xpsmatrix/) | MatrixTransform プロパティ要素の機能をカプセル化するクラス. この要素は、要素の座標 システムを操作するために使用される任意のアフィン行列変換を定義します. |
| [XpsObject](./xpsobject/) | 一般的な XPS モデル オブジェクト機能をカプセル化するクラス。 |
| [XpsPage](./xpspage/) | FixedPage 要素の機能をカプセル化するクラス. この要素にはページのコンテンツが含まれ、FixedPage パーツのルート要素です. |
| [XpsPageLinkTarget](./xpspagelinktarget/) | ページのハイパーリンク ターゲットをカプセル化するクラス。 |
| [XpsPath](./xpspath/) | パス要素の機能をカプセル化するクラス. この要素は、固定ページにベクトル グラフィックと画像を追加する唯一の手段です. ページにレンダリングされる単一のベクトル グラフィックを定義します. |
| [XpsPathFigure](./xpspathfigure/) | PathFigure 要素の機能をカプセル化するクラス. この要素は、1 つまたは複数の直線または曲線セグメントのセットで構成されます. |
| [XpsPathGeometry](./xpspathgeometry/) | PathGeometry プロパティ要素の機能をカプセル化するクラス。 この要素には、Figures 属性または子 PathFigure 要素を持つ で指定された一連のパス図が含まれます。 |
| [XpsPathPolySegment](./xpspathpolysegment/) | PolyLineSegment、PolyBézierSegment、および PolyQuadraticBézierSegment 要素の共通機能をカプセル化するクラス。 |
| [XpsPathSegment](./xpspathsegment/) | すべてのパス セグメント要素の共通機能をカプセル化するクラス。 |
| [XpsPolyBezierSegment](./xpspolybeziersegment/) | PolyBezierSegment 要素の機能をカプセル化するクラス. この要素は、3 次ベジエ曲線のセットを記述します. |
| [XpsPolyLineSegment](./xpspolylinesegment/) | PolyLineSegment 要素の機能をカプセル化するクラス. この要素は、任意の数の個々の頂点を含む多角形の描画を記述します. |
| [XpsPolyQuadraticBezierSegment](./xpspolyquadraticbeziersegment/) | PolyQuadraticBezierSegment 要素の機能をカプセル化するクラス. この要素は、指定された制御点を使用して、頂点のセットを通るパス図の の前の点からの二次ベジエ曲線のセットを記述します. |
| [XpsRadialGradientBrush](./xpsradialgradientbrush/) | RadialGradientBrush プロパティ要素の機能をカプセル化するクラス. この要素は、放射状グラデーション ブラシを指定するために使用されます. |
| [XpsRgbColor](./xpsrgbcolor/) | 任意の色空間 (sRGB または scRGB) の RGB 色をカプセル化します。 |
| [XpsSolidColorBrush](./xpssolidcolorbrush/) | SolidColorBrush プロパティ要素の機能をカプセル化するクラス. この要素は、定義された幾何学的領域を単色で塗りつぶすために使用されます. |
| [XpsTilingBrush](./xpstilingbrush/) | タイリング ブラシ要素 (VisualBrush および ImageBrush) の共通機能をカプセル化するクラス。 |
| [XpsTransformableBrush](./xpstransformablebrush/) | 変換可能なブラシ要素 (SolidColorBrush を除くすべて) の共通機能をカプセル化するクラス。 |
| [XpsVisualBrush](./xpsvisualbrush/) | VisualBrush プロパティ要素の機能をカプセル化するクラス. この要素は、領域を描画で塗りつぶすために使用されます. |
