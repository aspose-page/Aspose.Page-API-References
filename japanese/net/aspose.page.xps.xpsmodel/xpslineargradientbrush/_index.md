## XpsLinearGradientBrush class

LinearGradientBrush プロパティ要素の機能をカプセル化するクラス. この要素は、ベクトルに沿って線形グラデーション ブラシを指定するために使用されます.

```csharp
public sealed class XpsLinearGradientBrush : XpsGradientBrush
```
