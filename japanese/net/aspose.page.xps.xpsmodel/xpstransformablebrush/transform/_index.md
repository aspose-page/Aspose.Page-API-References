## XpsTransformableBrush.Transform property

ブラシの座標空間に適用されるマトリックス変換を返す/設定します。 Transform プロパティは、現在の有効なレンダリング transform と連結され、ブラシに対してローカルな有効なレンダリング変換を生成します。 brush のビューポートは、ローカルの効果的なレンダリング変換を使用して変換されます。

public XpsMatrix Transform { get; set; }
