## ImageDevice.IsDirectRGB property

デバイスがダイレクト RGB モード、つまり RGB を使用するかどうかを示します。

```csharp
public override bool IsDirectRGB { get; }
```

### 戻り値

直接 RGB モードの場合は true、それ以外の場合は false、つまり BGR です。
