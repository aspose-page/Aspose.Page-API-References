## PdfDevice.SetTransform method

現在の変換を指定します。ほとんどの出力形式はこの機能を実装していないため、 currentTransform の逆変換が計算され、 変換が乗算されて設定されます。その結果は、call によって writeTransform(Transform). に転送されます。

```csharp
public override void SetTransform(Matrix transform)
```
