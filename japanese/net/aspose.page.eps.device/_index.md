**Aspose.Page.EPS.Device**名前空間は、可能なデバイスのクラスと、PS/EPS を他の形式に変換する際に 使用できる保存オプションを提供します.

## クラス

| クラス | 説明 |
| --- | --- |
| [ImageDevice](./imagedevice/) | このクラスは、ドキュメントからイメージへのレンダリングをカプセル化します。 |
| [ImageSaveOptions](./imagesaveoptions/) | このクラスには、画像保存プロセスを管理するために必要なオプションが含まれています。 |
| [PdfDevice](./pdfdevice/) | このクラスは、ドキュメントの PDF へのレンダリングをカプセル化します。 |
| [PdfSaveOptions](./pdfsaveoptions/) | このクラスには、入力ストリームと出力ストリーム、および変換プロセスの管理に必要なその他のオプションが含まれています。 |
| [PsSaveOptions](./pssaveoptions/) | このクラスには、ドキュメントを PostScript (PS) または Encapsulated PostScript (EPS) ファイルに変換するプロセスを管理するために必要なオプションが含まれています。 |
