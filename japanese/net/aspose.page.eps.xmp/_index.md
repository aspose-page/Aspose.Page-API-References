**Aspose.Page.EPS.XMP**名前空間は、EPS ファイルの XMP メタデータを読み取り、変更するためのクラスを提供します。

## クラス

| クラス | 説明 |
| --- | --- |
| [XmpField](./xmpfield/) | XMP フィールドを表します。 |
| [XmpMetadata](./xmpmetadata/) | XMP メタデータ ストリームへのアクセスを提供します。 |
| [XmpValue](./xmpvalue/) | XMP 値を表します |
## 列挙

| 列挙 | 説明 |
| --- | --- |
| [XmpFieldType](./xmpfieldtype/) | この列挙型は、XMP フィールドのタイプを表します。 |
