## PageDeviceColorSpaceUsage.PageDeviceColorSpaceUsageOption.OverrideDeviceDefault field

PageDeviceColorSpaceProfileURI パラメータで指定されたプロファイルに、デバイスのプライマリの数 に一致するチャネル数がある場合、すべての要素のデバイス内部カラー管理の代わりに使用する必要があります. このプロファイルを使用する要素は、デバイスの色であると見なされます。スペースがあり、それ以上カラー管理されません.

```csharp
public static PageDeviceColorSpaceUsageOption OverrideDeviceDefault;
```
