## PageDeviceColorSpaceProfileURI class

XPS ドキュメントに含まれる ICC プロファイルへのパッケージ ルートへの相対 URI を指定します。 このオプションの処理は、PageDeviceColorSpaceUsage 機能の設定によって異なります。 そのプロファイルを使用するすべての要素は、既に適切なデバイス カラーであると見なされます。スペース, であり、ドライバーまたはデバイスで色が管理されません.

```csharp
public sealed class PageDeviceColorSpaceProfileURI : StringParameterInit, IDocumentPrintTicketItem, 
    IJobPrintTicketItem, IPagePrintTicketItem
```
