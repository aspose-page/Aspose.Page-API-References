## PageDestinationColorProfileURI class

XPS ドキュメントに含まれる ICC プロファイルへの相対 URI 参照を指定します。 このオプションの処理は、PageDeviceColorSpaceUsage 機能の設定によって異なります。 そのプロファイルを使用するすべての要素は、適切なデバイスの色空間に既にあると想定されます, ドライバーまたはデバイスで色が管理されません.

public sealed class PageDestinationColorProfileURI : StringParameterInit, IDocumentPrintTicketItem,
