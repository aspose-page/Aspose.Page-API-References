## GradientBrush class

このクラスは、LinearGradientBrush と PathGradientBrush をカプセル化するために使用され、ラップ モードをクランプに設定する可能性があります。

```csharp
public class GradientBrush : Brush
```

## コンストラクター

| 名前 | 説明 |
| --- | --- |
| [GradientBrush](gradientbrush/)(Brush) | GradientBrush の新しいインスタンスを作成します。  LinearGradientBrush または PathGradientBrush。 ArgumentException 以外の場合はスローされます。 |

## プロパティ

| 名前 | 説明 |
| --- | --- |
| [Bounds](../../aspose.page.eps/gradientbrush/bounds/) { get; set; } | このグラデーション ブラシの境界を返すか指定します。 |
| [NativeBrush](../../aspose.page.eps/gradientbrush/nativebrush/) { get; } | ネイティブグラデーションブラシを返します. |
| [WrapMode](../../aspose.page.eps/gradientbrush/wrapmode/) { get; set; } | このグラデーション ブラシのラップ モードを返すか指定します。これは、ネイティブ グラデーション ブラシで例外をスローする原因となる WrapMode.Clamp である可能性があります。 |
