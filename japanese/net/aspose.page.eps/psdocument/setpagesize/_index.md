## PsDocument.SetPageSize method

```csharp
public void SetPageSize(float width, float height)
```

| パラメータ | タイプ | 説明 |
| --- | --- | --- |
| width | Single | 結果の PostScript ファイルのページの幅。 |
| height | Single | 結果の PostScript ファイルのページの高さ。 |
