**Aspose.Page.EPS** PS/EPS ファイルを扱うすべてのクラスのルート名前空間です。

## クラス

| クラス | 説明 |
| --- | --- |
| [FontConstants](./fontconstants/) | このクラスは、フォントを保存するための一連の定数を定義します。 |
| [GradientBrush](./gradientbrush/) | このクラスは、LinearGradientBrush と PathGradientBrush をカプセル化するために使用され、ラップ モードをクランプに設定する可能性があります。 |
| [LoadOptions](./loadoptions/) | ドキュメント読み込みオプションの基本クラス。 |
| [PageConstants](./pageconstants/) | このクラスは、ページを記述する一連の定数を定義します。 さまざまな余白、向き、 再スケーリング、および標準のページ サイズ用の便利なオブジェクトが提供されています。 |
| [PsConverterException](./psconverterexception/) | このクラスには、PS ファイルを PDF ドキュメントに変換する際に スローされるエラーに関する情報が含まれます。 |
| [PsDocument](./psdocument/) | このクラスは、PS/EPS ドキュメントをカプセル化します。 |
| [PsDocumentException](./psdocumentexception/) | このクラスには、PostScript ファイルの作成、編集、および保存中に スローされるエラーに関する情報が含まれます。 |
| [PsLoadOptions](./psloadoptions/) | PS ドキュメント読み込みオプション. |
