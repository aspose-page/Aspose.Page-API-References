## Units enumeration

サイズの種類を指定します。

```js
enum Units
```

### 値

| 名前 | 値 | 説明 |
| ---        | ---   | --- |
| Points | `0` | ポイント (1/72 インチ)。 |
| Inches | `1` | インチ。 |
| Millimeters | `2` | ミリメートル。 |
| Centimeters | `3` | センチメートル。 |
| Percents | `4` | 既存サイズのパーセント。 |
