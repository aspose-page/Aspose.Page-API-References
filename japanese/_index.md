## Aspose.Page for .NET

{{% alert color="primary" %}}

![Aspose.Page for .NET 製品のロゴ](../home_1.png)


Aspose.Page は .NET API であり、開発者が XPS および EPS/PS ドキュメントを操作できるように構築されています。 API を使用して、既存の XPS ドキュメントと新しい XPS ドキュメントを作成、編集、保存できます。 さらに、XPS および EPS/PS ドキュメントを PDF および画像に変換できます。 Aspose.Page for .NET は C# を使用して実装され、C#、VB.NET、J# などの任意の .NET 言語で使用できます。ASP.NET Web アプリケーションまたは Windows など、あらゆる種類のアプリケーションと統合できます。 応用。

{{% /alert %}}

## Aspose.Page for Java

{{% alert color="primary" %}}

![Aspose.Page for Java 製品のロゴ](../home_2.png)


Aspose.Page は、開発者が XPS および EPS/PS ドキュメントを操作できるように構築された Java API です。 API を使用して、既存の XPS ドキュメントと新しい XPS ドキュメントを作成、編集、保存できます。 さらに、XPS および EPS/PS ドキュメントを PDF および画像に変換できます。 広範な操作機能を提供し、あらゆる種類のデスクトップ GUI アプリケーション、Web アプリケーション、およびコンソール アプリケーションと統合できます。

{{% /alert %}}

## Aspose.Page for C++

{{% alert color="primary" %}}

![Aspose.Page for C++ 製品ロゴ](../home_3.png)


Aspose.Page for C++ は、信頼性の高いスタンドアロンの XPS および PostScript ファイル操作および変換 API です。 追加のツールを必要とせずに、XPS ファイルを作成、編集、および変換できます。 API により、ドキュメント ページと要素の操作、ベクター グラフィックの作成、図形のグループ化、さまざまな色空間での色の指定などの操作が可能になります。 さらに、開発者は PS および EPS ファイルを PDF ドキュメント、BMP、EMF、GIF、JPEG、PNG、TIFF などのラスター イメージに変換できます。

{{% /alert %}}

これらは、いくつかの有用なリソースへのリンクです。
