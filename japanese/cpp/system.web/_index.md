## クラス

| クラス | 説明 |
| --- | --- |
| [HttpContext](./httpcontext/) | ダミークラスは、翻訳されたコードを [HttpContext](./httpcontext/) 参照とリンクできるようにしますが、実行はしません。適切に実装されたメンバーは含まれていません。このクラスのオブジェクトは、[System::MakeObject()](../system/makeobject/) 関数を使用してのみ割り当てるべきです。スタック上や operator new を使用してこの型のインスタンスを作成しないでください。そうしないと実行時エラーやアサーション障害が発生します。常にこのクラスを [System::SmartPtr](../system/smartptr/) ポインタでラップし、そのポインタを関数への引数として使用してください。 |
| [HttpResponse](./httpresponse/) | ダミークラスは、翻訳されたコードを [HttpResponse](./httpresponse/) 参照とリンクできるようにしますが、実行はしません。適切に実装されたメンバーは含まれていません。このクラスのオブジェクトは、[System::MakeObject()](../system/makeobject/) 関数を使用してのみ割り当てるべきです。スタック上や operator new を使用してこの型のインスタンスを作成しないでください。そうしないと実行時エラーやアサーション障害が発生します。常にこのクラスを [System::SmartPtr](../system/smartptr/) ポインタでラップし、そのポインタを関数への引数として使用してください。 |
| [HttpUtility](./httputility/) | URL 部分を 16 進エスケープフラグメントにエンコードおよびデコードするサービスクラス。 |
