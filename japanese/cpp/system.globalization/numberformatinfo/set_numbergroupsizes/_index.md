## NumberFormatInfo::set_NumberGroupSizes method

```cpp

```


| パラメーター | 型 | 説明 |
| --- | --- | --- |
| value | const ArrayPtr\<int\>\& | [Array](../../../system/array/) はグループごとの桁数を左から右へ並べたものです。各要素は 1 から 9 の間でなければならず、最後の要素は 0 にでき、これは「combine all」を意味します。最後の要素は繰り返されます。 |
