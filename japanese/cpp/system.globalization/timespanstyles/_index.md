## TimeSpanStyles enum


文字列で表された時間間隔を Timespan オブジェクトに変換するメソッドの文字列解析をカスタマイズする書式設定オプションを指定します。

```cpp
enum class TimeSpanStyles : int32_t
```

### 値

| 名前 | 値 | 説明 |
| --- | --- | --- |
| None | 0 | 入力文字列で表される時間間隔は、負の記号が存在する場合にのみ負の時間間隔として解釈されます。 |
| AssumeNegative | 1 | 入力文字列で表される時間間隔は常に負の時間間隔として解釈されます。 |
