## CultureTypes enum


カルチャーカテゴリのビットマスクエントリ。

```cpp
enum class CultureTypes : int32_t
```

### 値

| 名前 | 値 | 説明 |
| --- | --- | --- |
| NeutralCultures | 1 | 言語には特化しているが、国や地域には特化していないカルチャー。 |
| SpecificCultures | 2 | 国/地域に特化しています。 |
| InstalledWin32Cultures | 4 | OS にインストールされているカルチャー。 |
| AllCultures | n/a | 利用可能なすべてのカルチャー。 |
| UserCustomCulture | 8 | ユーザー定義のカルチャー。 |
| ReplacementCultures | 16 | 既存のカルチャーに対するユーザー定義の置換。 |
| WindowsOnlyCultures | 32 | 非推奨。 |
| FrameworkCultures | 64 | 非推奨。 |
