## クラス

| クラス | 説明 |
| --- | --- |
| [SecureString](./securestring/) | Secure string は、機密として保持すべきテキストを表します。このクラスは内部データを暗号化しません。このクラスのオブジェクトは [System::MakeObject()](../system/makeobject/) 関数を使用してのみ割り当てるべきです。スタック上や operator new を使用してこの型のインスタンスを作成しないでください。そうしないと実行時エラーやアサーション障害が発生します。このクラスは常に [System::SmartPtr](../system/smartptr/) ポインタでラップし、そのポインタを関数への引数として使用してください。 |
| [SecureStringMarshal](./securestringmarshal/) | アンマネージド メモリブロックの割り当てとコピーのためのメソッドコレクションです。 |
| [SecurityElement](./securityelement/) | セキュリティオブジェクトをエンコードするための XML オブジェクトモデルです。実装されていません。このクラスのオブジェクトは [System::MakeObject()](../system/makeobject/) 関数を使用してのみ割り当てるべきです。スタック上や operator new を使用してこの型のインスタンスを作成しないでください。そうしないと実行時エラーやアサーション障害が発生します。このクラスは常に [System::SmartPtr](../system/smartptr/) ポインタでラップし、そのポインタを関数への引数として使用してください。 |
