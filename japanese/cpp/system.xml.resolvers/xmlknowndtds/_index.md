## XmlKnownDtds enum


この [Resolvers::XmlKnownDtds](./) 列挙体は [Resolvers::XmlPreloadedResolver](../xmlpreloadedresolver/) によって使用され、[Resolvers::XmlPreloadedResolver](../xmlpreloadedresolver/) が認識する既知の DTD を定義します。

```cpp
enum class XmlKnownDtds
```

### 値

| 名前 | 値 | 説明 |
| --- | --- | --- |
| None | 0 | この [Resolvers::XmlPreloadedResolver](../xmlpreloadedresolver/) が事前定義された DTD を一切認識しないことを指定します。 |
| Xhtml10 | 1 | この [Resolvers::XmlPreloadedResolver](../xmlpreloadedresolver/) が XHTML 1.0 で定義された DTD とエンティティを認識することを指定します。 |
| Rss091 | 2 | この [Resolvers::XmlPreloadedResolver](../xmlpreloadedresolver/) が RSS 0.91 で定義された DTD とエンティティを認識することを指定します。 |
| All | 65535 | この [Resolvers::XmlPreloadedResolver](../xmlpreloadedresolver/) が現在サポートされているすべての DTD を認識することを指定します。これはデフォルトの動作です。 |
