## クラス

| クラス | 説明 |
| --- | --- |
| [Soap](./soap/) | SOAP の文字列定数を含みます。 |
| [Soap12](./soap12/) | SOAP バージョン 1.2 の文字列定数を含みます。 |
| [WebServiceAttribute](./webserviceattribute/) | XML [Web](../system.web/) サービスに追加情報を加えます。このクラスのオブジェクトは [System::MakeObject()](../system/makeobject/) 関数を使用してのみ割り当てるべきです。スタック上や operator new を使用してこの型のインスタンスを作成しないでください。そうしないと実行時エラーやアサーション障害が発生します。常にこのクラスを [System::SmartPtr](../system/smartptr/) ポインタでラップし、そのポインタを関数への引数として使用してください。 |
| [WebServiceBindingAttribute](./webservicebindingattribute/) | XML [Web](../system.web/) サービスの 1 つまたは複数のメソッドを定義するバインディングを宣言するために使用されます。このクラスのオブジェクトは [System::MakeObject()](../system/makeobject/) 関数を使用してのみ割り当てるべきです。スタック上や operator new を使用してこの型のインスタンスを作成しないでください。そうしないと実行時エラーやアサーション障害が発生します。常にこのクラスを [System::SmartPtr](../system/smartptr/) ポインタでラップし、そのポインタを関数への引数として使用してください。 |
