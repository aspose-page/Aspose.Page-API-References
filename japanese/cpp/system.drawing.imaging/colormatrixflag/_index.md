## ColorMatrixFlag enum


カラーおよびグレースケール調整設定の影響を受ける画像と色のタイプを [ImageAttributes](../imageattributes/) オブジェクトで指定します。

```cpp
enum class ColorMatrixFlag
```

### 値

| 名前 | 値 | 説明 |
| --- | --- | --- |
| Default | 0 | 色とグレースケールの値の両方がカラー調整されます。 |
| SkipGrays | 1 | グレースケールの値はカラー調整されません。 |
| AltGrays | 2 | グレースケールの値のみがカラー調整されます。 |
