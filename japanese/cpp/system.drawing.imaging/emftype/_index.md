## EmfType enum


EMF ファイルに配置されるレコードのタイプを指定します。

```cpp
enum class EmfType
```

### 値

| 名前 | 値 | 説明 |
| --- | --- | --- |
| EmfOnly | n/a | [Windows](../../system.windows/) 拡張メタファイル。GDI コマンドを含みます。このタイプのメタファイルは EMF ファイルと呼ばれます。 |
| EmfPlusOnly | n/a | [Windows](../../system.windows/) 拡張メタファイルプラス。GDI+ コマンドを含みます。このタイプのメタファイルは EMF+ ファイルと呼ばれます。 |
| EmfPlusDual | n/a | デュアル [Windows](../../system.windows/) 拡張メタファイル。等価な GDI と GDI+ コマンドを含みます。このタイプのメタファイルは EMF+ ファイルと呼ばれます。 |
