## PixelFormat enum


ピクセルのカラー データ形式を指定します。

```cpp
enum class PixelFormat
```

### 値

| 名前 | 値 | 説明 |
| --- | --- | --- |
| Indexed | 65536 | ピクセル データがカラーインデックス値を含むことを指定します。これはシステム カラーテーブル内の色へのインデックスを意味します。 |
| Gdi | 131072 | ピクセル データが GDI カラーを含むことを指定します。 |
| Alpha | 262144 | ピクセル データが事前乗算されていないアルファ値を含むことを指定します。 |
| PAlpha | 524288 | ピクセル データが事前乗算されたアルファ値を含むことを指定します。 |
| 拡張 | 1048576 | 予約済み。 |
| 標準 | 2097152 | 32ビット/ピクセル、24ビットのカラーデプス、8ビットのアルファチャンネルのピクセルフォーマットを指定します。 |
| 未定義 | 0 | ピクセルフォーマットが未定義であることを指定します。 |
| 指定なし | 0 | ピクセルフォーマットは指定されていません。 |
| Format1bppIndexed | n/a | ピクセルフォーマットが1ビット/ピクセルのインデックスカラーであることを指定します。 |
| Format4bppIndexed | n/a | ピクセルフォーマットが4ビット/ピクセルのインデックスカラーであることを指定します。 |
| Format8bppIndexed | n/a | ピクセルフォーマットが8ビット/ピクセルのインデックスカラーであることを指定します。 |
| Format16bppGrayScale | n/a | ピクセルフォーマットが16ビット/ピクセルであることを指定します。カラー情報は65536階調のグレースケールを表します。 |
| Format16bppRgb555 | n/a | ピクセルフォーマットが16ビット/ピクセルで、赤・緑・青各コンポーネントが5ビットで、残りのビットは使用されません。 |
| Format16bppRgb565 | n/a | ピクセルフォーマットが16ビット/ピクセルで、赤が5ビット、緑が6ビット、青が5ビットのコンポーネントで構成されていることを指定します。 |
| Format16bppArgb1555 | n/a | ピクセルフォーマットが16ビット/ピクセルで、赤・緑・青各コンポーネントが5ビット、アルファが1ビットであることを指定します。 |
| Format24bppRgb | n/a | ピクセルフォーマットが24ビット/ピクセルで、赤・緑・青各コンポーネントが8ビットであることを指定します。 |
| Format32bppRgb | n/a | ピクセル形式が 1 ピクセルあたり 32 ビットで、赤、緑、青の各コンポーネントが 8 ビットずつ、残りの 8 ビットは使用されないことを指定します。 |
| Format32bppArgb | n/a | ピクセル形式が 1 ピクセルあたり 32 ビットで、赤、緑、青の各コンポーネントが 8 ビットずつ、アルファが 8 ビットであることを指定します。 |
| Format32bppPArgb | n/a | ピクセル形式が 1 ピクセルあたり 32 ビットで、赤、緑、青の各コンポーネントが 8 ビットずつ、アルファが 8 ビットであることを指定します。赤、緑、青のコンポーネントはアルファ値に従って事前に乗算されます。 |
| Format48bppRgb | n/a | ピクセル形式が 1 ピクセルあたり 48 ビットで、赤、緑、青の各コンポーネントが 16 ビットであることを指定します。 |
| Format64bppArgb | n/a | ピクセル形式が 1 ピクセルあたり 64 ビットで、赤、緑、青の各コンポーネントが 16 ビットずつ、アルファが 16 ビットであることを指定します。 |
| Format64bppPArgb | n/a | ピクセル形式が 1 ピクセルあたり 64 ビットで、赤、緑、青の各コンポーネントが 16 ビットずつ、アルファが 16 ビットであることを指定します。赤、緑、青のコンポーネントはアルファ値に従って事前に乗算されます。 |
| Format32bppCMYK | n/a | ピクセル形式が 1 ピクセルあたり 32 ビットで、シアン、マゼンタ、イエロー、キー（黒）各コンポーネントが 8 ビットであることを指定します。 |
| Max | 16 | この列挙体の最大値です。 |
