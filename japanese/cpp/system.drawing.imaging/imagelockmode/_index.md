## ImageLockMode enum


ロックされている画像領域のプロパティを指定します。

```cpp
enum class ImageLockMode
```

### 値

| 名前 | 値 | 説明 |
| --- | --- | --- |
| 読み取り専用 | 1 | 領域は読み取り専用でロックされています。 |
| WriteOnly | 2 | 領域は書き込み専用でロックされています。 |
| ReadWrite | n/a | 領域は読み取りと書き込みの両方でロックされています。 |
| UserInputBuffer | 4 | 画像はユーザー入力バッファに格納されており、そのアクセスはユーザーが制御します。 |
