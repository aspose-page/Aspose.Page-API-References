## XmlSchemaMaxExclusiveFacet class


XML [Schema](../) の **maxExclusive** 要素を、World Wide [Web](../../system.web/) コンソーシアム (W3C) が定める通りに表します。このクラスは **simpleType** 要素の最大値に対する制限を指定するために使用できます。要素の値は **maxExclusive** 要素の値未満でなければなりません。
