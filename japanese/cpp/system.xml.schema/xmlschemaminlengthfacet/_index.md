## XmlSchemaMinLengthFacet class


World Wide [Web](../../system.web/) Consortium (W3C) が定義する XML [Schema](../) の **minLength** 要素を表します。このクラスは **simpleType** 要素のデータ値の最小長さに対する制限を指定するために使用できます。長さは **minLength** 要素の値より大きくなければなりません。

```cpp
class XmlSchemaMinLengthFacet : public System::Xml::Schema::XmlSchemaNumericFacet
```
