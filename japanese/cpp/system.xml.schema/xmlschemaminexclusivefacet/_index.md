## XmlSchemaMinExclusiveFacet class


XML [Schema](../) の **minExclusive** 要素を、World Wide [Web](../../system.web/) Consortium (W3C) によって指定されたものとして表します。このクラスは **simpleType** 要素の最小値に対する制限を指定するために使用できます。要素の値は **minExclusive** 要素の値より大きくなければなりません。
