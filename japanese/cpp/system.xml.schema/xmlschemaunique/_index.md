## XmlSchemaUnique class


XML [Schema](../) の **unique** 要素を、World Wide [Web](../../system.web/) Consortium (W3C) が定義する通りに表します。このクラスは要素の集合の中から一意制約を識別するために使用できます。

```cpp
class XmlSchemaUnique : public System::Xml::Schema::XmlSchemaIdentityConstraint
```

## 備考



このクラスのオブジェクトは、[System::MakeObject()](../../system/makeobject/) 関数を使用してのみ割り当てるべきです。スタック上や operator new を使用してインスタンスを作成しないでください。そうすると実行時エラーやアサーション障害が発生します。このクラスは常に [System::SmartPtr](../../system/smartptr/) ポインタでラップし、そのポインタを関数への引数として渡すようにしてください。
