## XmlSchemaLengthFacet class


XML [Schema](../) からの **length** ファセットを、World Wide [Web](../../system.web/) Consortium (W3C) が定義するものとして表します。このクラスはデータ型の **simpleType** 要素の長さに制限を指定するために使用できます。

```cpp
class XmlSchemaLengthFacet : public System::Xml::Schema::XmlSchemaNumericFacet
```
