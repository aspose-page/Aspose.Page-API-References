## XmlSchemaMinInclusiveFacet class


XML [Schema](../) の **minInclusive** 要素を、World Wide [Web](../../system.web/) Consortium (W3C) が定義するように表します。このクラスは simpleType 要素の最小値に対する制限を指定するために使用できます。要素の値は **minInclusive** 要素の値以上でなければなりません。
