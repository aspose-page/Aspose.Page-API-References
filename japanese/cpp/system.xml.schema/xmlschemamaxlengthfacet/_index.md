## XmlSchemaMaxLengthFacet class


XML [Schema](../) の **maxLength** 要素を World Wide [Web](../../system.web/) Consortium (W3C) が指定する形で表します。このクラスは **simpleType** 要素のデータ値の最大長さに対する制限を指定するために使用できます。長さは **maxLength** 要素の値未満でなければなりません。

```cpp
class XmlSchemaMaxLengthFacet : public System::Xml::Schema::XmlSchemaNumericFacet
```
