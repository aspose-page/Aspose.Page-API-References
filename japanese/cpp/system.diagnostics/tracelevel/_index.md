## TraceLevel enum


[System.Diagnostics.Debug](../debug/)、[System.Diagnostics.Trace](../trace/) および System.Diagnostics.TraceSwitch クラスに出力するメッセージを指定します。

```cpp
enum class TraceLevel
```

### 値

| 名前 | 値 | 説明 |
| --- | --- | --- |
| オフ | 0 | トレースやデバッグメッセージを出力しません。 |
| エラー | 1 | エラーハンドリングメッセージを出力します。 |
| 警告 | 2 | 警告とエラーハンドリングメッセージを出力します。 |
| Info | 3 | 情報メッセージ、警告、およびエラー処理メッセージを出力します。 |
| 詳細 | 4 | すべてのデバッグおよびトレースメッセージを出力します。 |
