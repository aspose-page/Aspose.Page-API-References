## Size::Subtract method


新しい [Size](../) オブジェクトを返します。これは **size2** から **size1** を減算した結果で、幅の値は **size1's** の幅から **size2's** の幅を減算した結果、 高さの値は **size1's** の高さから **size2's** の高さを減算した結果です。

```cpp
static Size System::Drawing::Size::Subtract(const Size &size1, const Size &size2)
```


| パラメーター | 型 | 説明 |
| --- | --- | --- |
| size1 | const Size\& | 減算対象の [Size](../) オブジェクト |
| size2 | const Size\& | 減算する [Size](../) オブジェクト |

### ReturnValue

**size1** から **size2** を減算した結果の [Size](../) オブジェクト
