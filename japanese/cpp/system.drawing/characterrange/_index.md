## CharacterRange class


文字列内の文字位置の範囲を表します。この型はスタック上に割り当て、値渡しまたは参照渡しで関数に渡すべきです。この型のオブジェクトを管理するために [System::SmartPtr](../../system/smartptr/) クラスを使用しないでください。

```cpp
class CharacterRange
```
