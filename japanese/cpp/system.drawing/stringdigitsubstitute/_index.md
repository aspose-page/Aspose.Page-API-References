## StringDigitSubstitute enum


文字列内の数字がロケールまたは言語に従ってどのように置換されるかを指定します。

```cpp
enum class StringDigitSubstitute
```

### 値

| 名前 | 値 | 説明 |
| --- | --- | --- |
| User | 0 | ユーザー定義の置換です。 |
| None | 1 | 置換は無効です。 |
| National | 2 | ユーザーのロケールの公式な国語に対応する数字を置換します。 |
| Traditional | 3 | ユーザーの母語またはスクリプトに対応する数字を置換します。これはユーザーのロケールの公式な国語と同じでない場合があります。 |
