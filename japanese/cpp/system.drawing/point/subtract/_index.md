## Point::Subtract method


指定された [Size](../../size/) オブジェクトの幅と高さの値を、指定された [Point](../) オブジェクトの X および Y 座標の値からそれぞれ減算します。

```cpp
static Point System::Drawing::Point::Subtract(const Point &point, const Size &size)
```


| パラメーター | 型 | 説明 |
| --- | --- | --- |
| ポイント | const Point\& | 変換する点 |
| size | const Size\& | 座標値から減算する値を指定する [Size](../../size/) オブジェクト **point** |

### ReturnValue

幅の値を **size** から **point** の X 座標の値から減算した結果と等しい X 座標値、そして高さの値を **size** から **point** の Y 座標の値から減算した結果と等しい Y 座標値を持つ新しい [Point](../) オブジェクト
