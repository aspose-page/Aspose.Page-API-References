## SizeF::Subtract method


新しい [SizeF](../) オブジェクトを返します。これは **size2** を **size1** から減算した結果であり、つまり幅の値は **size1** の幅から **size2** の幅を減算した結果、そして高さの値は **size1** の高さから **size2** の高さを減算した結果です。

```cpp
static SizeF System::Drawing::SizeF::Subtract(const SizeF &size1, const SizeF &size2)
```

| size2 | const SizeF\& | 減算対象の [SizeF](../) オブジェクト |

### ReturnValue
