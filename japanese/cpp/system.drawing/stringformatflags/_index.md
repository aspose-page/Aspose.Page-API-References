## StringFormatFlags enum


テキスト文字列の表示およびレイアウト情報を指定します。

```cpp
enum class StringFormatFlags
```

### 値

| 名前 | 値 | 説明 |
| --- | --- | --- |
| DirectionRightToLeft | 1 | 右から左へのテキスト方向です。 |
| DirectionVertical | 2 | [Text](../../system.drawing.text/) は縦方向に揃えられています。 |
| FitBlackBox | 4 | 文字の一部がレイアウトボックスからはみ出すことがあります。 |
| DisplayFormatControl | 32 | 制御文字が表示されます。 |
| NoFontFallback | 1024 | [Font](../font/) のフォールバックが無効になっています。 |
| MeasureTrailingSpaces | 2048 | 各行の末尾に末尾スペースを挿入します。 |
| NoWrap | 4096 | [Text](../../system.drawing.text/) の折り返しが無効になっています。 |
| LineLimit | 8192 | 全行が書式設定矩形内に配置されます。 |
| NoClip | 16384 | グリフのはみ出した部分の表示が許可されています。 |
