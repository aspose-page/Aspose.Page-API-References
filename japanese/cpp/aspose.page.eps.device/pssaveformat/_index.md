## PsSaveFormat enum


この列挙型は保存形式の利用可能なオプションを含みます。PS または [EPS](../../aspose.page.eps/) を指定できます。[EPS](../../aspose.page.eps/) は 1 ページのドキュメントのみで使用され、PS ファイルは任意のページ数を含むことができます。

```cpp
enum class PsSaveFormat
```

### 値

| 名前 | 値 | 説明 |
| --- | --- | --- |
| PS | 0 | このオプションは、結果のドキュメントが PostScript (PS) または Encapsulated PostScript ([EPS](../../aspose.page.eps/)) ファイルである必要があることを示します。 |
| EPS | 1 | このオプションは、結果のドキュメントが Encapsulated PostScript ([EPS](../../aspose.page.eps/)) ファイルである必要があることを示します。これは 1 ページのドキュメントでのみ使用されます。 |
