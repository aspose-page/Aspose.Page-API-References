## クラス

| クラス | 説明 |
| --- | --- |
| [IXmlSerializable](./ixmlserializable/) | XML のシリアライズおよびデシリアライズのためのカスタム書式設定を提供します。このクラスのオブジェクトは、[System::MakeObject()](../system/makeobject/) 関数を使用してのみ割り当てるべきです。スタック上や operator new を使用してこの型のインスタンスを作成しないでください。そうしないと実行時エラーやアサーション違反が発生します。常にこのクラスを [System::SmartPtr](../system/smartptr/) ポインタでラップし、そのポインタを関数への引数として渡すようにしてください。 |
| [XmlAttributeOverrides](./xmlattributeoverrides/) | オブジェクトのシリアライズまたはデシリアライズに [XmlSerializer](./xmlserializer/) を使用する際に属性を上書きできるようにします。このクラスのオブジェクトは、[System::MakeObject()](../system/makeobject/) 関数を使用してのみ割り当てるべきです。スタック上や operator new を使用してこの型のインスタンスを作成しないでください。そうしないと実行時エラーやアサーション違反が発生します。常にこのクラスを [System::SmartPtr](../system/smartptr/) ポインタでラップし、そのポインタを関数への引数として渡すようにしてください。 |
