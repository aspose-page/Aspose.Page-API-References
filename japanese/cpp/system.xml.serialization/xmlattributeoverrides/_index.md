## XmlAttributeOverrides class


[XmlSerializer](../xmlserializer/) がオブジェクトのシリアライズまたはデシリアライズに使用される場合、属性をオーバーライドできます。このクラスのオブジェクトは [System::MakeObject()](../../system/makeobject/) 関数を使用してのみ割り当てるべきです。スタック上や operator new でインスタンスを作成しないでください。そうしないと実行時エラーやアサーション障害が発生します。常にこのクラスを [System::SmartPtr](../../system/smartptr/) ポインタでラップし、そのポインタを関数の引数として渡してください。

```cpp
class XmlAttributeOverrides : public System::Object
```
