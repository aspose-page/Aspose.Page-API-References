## AssemblyConstants class


コンポーネントのライセンスチェックに参加する定数を定義します。これらは以前はアセンブリ属性として直接定義されていましたが、.NET Compact Framework ではアセンブリ属性にアクセスできないため、別のクラスに移動しました。.NET Compact Framework 用にコンパイルされたライセンスコードは、アセンブリ属性の代わりにこれらの定数を使用します。

```cpp
class AssemblyConstants : public System::Object
```
