## Array::rbegin() const method


逆転したコンテナの最初の要素へのリバースイテレータを返します。これは、逆転していないコンテナの最後の要素に対応します。コンテナが空の場合、返されるイテレータは [rend()](../rend/) と等価です。

```cpp
const_reverse_iterator System::Array<T>::rbegin() const noexcept
```


### ReturnValue

const 修飾されたコンテナの最後の要素を指すイテレータです。
