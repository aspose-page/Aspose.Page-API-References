## Array::cbegin method


コンテナの最初の const 修飾要素へのイテレータを返します。コンテナが空の場合、返されるイテレータは [cend()](../cend/) と同じになります。

```cpp
const_iterator System::Array<T>::cbegin() const noexcept
```

コンテナの最初の const 修飾要素を指すイテレータ。
