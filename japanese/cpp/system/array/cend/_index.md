## Array::cend method


コンテナの最後の要素の次の要素へのイテレータを返します。この要素はプレースホルダーとして機能し、アクセスしようとすると未定義の動作になります。

```cpp
const_iterator System::Array<T>::cend() const noexcept
```


### ReturnValue

コンテナの最後の要素の後にある理論上の const 修飾要素を指すイテレータ。
