## Array::rend() const method


逆順コンテナの最後の要素の次の要素への逆イテレータを返します。これは逆順でないコンテナの最初の要素の前の要素に対応します。この要素はプレースホルダーとして機能し、アクセスしようとすると未定義の動作になります。

```cpp
const_reverse_iterator System::Array<T>::rend() const noexcept
```
