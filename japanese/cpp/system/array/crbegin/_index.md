## Array::crbegin method


逆転されたコンテナの最初の要素への逆イテレータを返します。これは、逆転されていないコンテナの最後の要素に対応します。コンテナが空の場合、返されるイテレータは [crend()](../crend/) と同じになります。

```cpp
const_reverse_iterator System::Array<T>::crbegin() const noexcept
```


### ReturnValue

コンテナの最後の const 修飾要素を指すイテレータ。
