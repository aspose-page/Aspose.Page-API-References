## WeakReference class


オブジェクトを参照しつつ、対象オブジェクトが削除されることを許容する弱参照を表します。

```cpp
template<typename...>class WeakReference
```


| パラメーター | 説明 |
| --- | --- |
| 引数 | 弱参照のテンプレート引数は参照対象オブジェクトの型を宣言し、引数の数は 0 または 1 にすることができます。 |
