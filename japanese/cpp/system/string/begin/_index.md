## String::begin method


実際の文字列バッファの先頭へのポインタを返します。何も再割り当てしません。バッファがヌル終端であることは保証しません。

```cpp
const UChar * System::String::begin() const
```


### ReturnValue

実際の文字列バッファへのポインタ、ヌル終端かどうかは問わない。
