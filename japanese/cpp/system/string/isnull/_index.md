## String::IsNull method


文字列が null と見なされるかをチェックします。[String](../) は、[String()](../string/) コンストラクタで作成された場合、または null 文字列からムーブ、コピー、代入された場合、もしくは [reset()](../reset/) メソッドが呼び出された場合にのみ null です。

```cpp
bool System::String::IsNull() const
```


### ReturnValue

文字列が null と見なされる場合は true、そうでない場合は false。
