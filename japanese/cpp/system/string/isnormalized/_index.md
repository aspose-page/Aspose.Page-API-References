## String::IsNormalized method


Unicode 文字列が指定された正規化形式で正規化されているかどうかをチェックします。

```cpp
bool System::String::IsNormalized(System::Text::NormalizationForm form) const
```


| パラメーター | 型 | 説明 |
| --- | --- | --- |
| form | System::Text::NormalizationForm | 正規化形式。詳細は [System::Text::NormalizationForm](../../../system.text/normalizationform/) を参照してください。 |

### ReturnValue

文字列がすでに正規化されている場合は true、そうでない場合は false。
