## String::end method


実際の文字列バッファの末尾へのポインタを返します。何も再割り当てしません。バッファがヌル終端であることは保証しません。

```cpp
const UChar * System::String::end() const
```


### ReturnValue

実際の文字列バッファの直後の最初の要素へのポインタ、null 終端かどうかは問わない。
