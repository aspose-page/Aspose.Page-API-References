## StringComparison enum


文字列比較のスタイルを定義します。

```cpp
enum class StringComparison
```

### 値

| 名前 | 値 | 説明 |
| --- | --- | --- |
| CurrentCulture | 0 | 現在のカルチャを使用します。 |
| CurrentCultureIgnoreCase | 1 | 現在のカルチャを使用しますが、大文字小文字は無視します。 |
| InvariantCulture | 2 | 不変カルチャーを使用します。 |
| InvariantCultureIgnoreCase | 3 | 不変カルチャーを使用しますが、ケースを無視します。 |
| Ordinal | 4 | 順序ソート規則を使用します。 |
| OrdinalIgnoreCase | 5 | 順序ソート規則を使用しますが、ケースを無視します。 |
