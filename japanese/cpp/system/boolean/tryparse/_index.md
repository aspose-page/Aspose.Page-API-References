## Boolean::TryParse method


指定された文字列を bool 型の値に変換します。

```cpp
static bool System::Boolean::TryParse(const String &value, bool &result)
```


| パラメーター | 型 | 説明 |
| --- | --- | --- |
| value | const String\& | 変換する文字列。 |
| 結果 | bool\& | 変換結果が格納される bool 変数への参照；指定された文字列が "True" と等しい場合は true、"False" と等しい場合は false になります。 |

### ReturnValue

指定された文字列が "True" または "False" のいずれかと等しい場合は true、そうでなければ false。
