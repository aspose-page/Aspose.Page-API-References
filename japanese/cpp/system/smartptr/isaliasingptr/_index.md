## SmartPtr::IsAliasingPtr method


ポインタが所有しているオブジェクト以外（エイリアシングコンストラクタで作成された）に指し示されているかどうかをチェックします。

```cpp
bool System::SmartPtr<T>::IsAliasingPtr() const
```


### ReturnValue

ポインタが指すオブジェクトと所有オブジェクトに差異がある場合は true、そうでなければ false。
