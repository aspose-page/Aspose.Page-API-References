## SmartPtr::get_shared_count method


参照対象オブジェクトに存在する共有ポインタの数（現在のポインタを含む）を取得します。現在のポインタが共有モードであることをアサートします。

```cpp
int System::SmartPtr<T>::get_shared_count() const
```


### ReturnValue

参照先オブジェクトに存在する共有ポインタの数（存在する場合）。ポインタが null の場合は 0 を返します。
