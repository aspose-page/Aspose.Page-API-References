## System::operator/ method

```cpp
template<typename T,typename _> Decimal System::operator/(const T &x, const Decimal &d)
```


| パラメーター | 型 | 説明 |
| --- | --- | --- |
| x | const T\& | 除算する値 |
| d | const Decimal\& | 除数を表す [Decimal](../decimal/) オブジェクト |

### ReturnValue

[Decimal](../decimal/) クラスの新しいインスタンスで、**x** を **d** が表す値で除算した結果の値を表します。
