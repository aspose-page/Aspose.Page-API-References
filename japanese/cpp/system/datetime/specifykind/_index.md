## DateTime::SpecifyKind method


指定された [DateTime](../) オブジェクトと同じティック数を持ち、引数 **kind** で指定されたローカル時間、UTC 時間、またはそのどちらでもない時間を表す新しい [DateTime](../) オブジェクトを作成します。

```cpp
static DateTime System::DateTime::SpecifyKind(DateTime value, DateTimeKind kind)
```


| パラメーター | 型 | 説明 |
| --- | --- | --- |
| value | DateTime | ティック数をコピーする元となる [DateTime](../) オブジェクト |
| 種類 | DateTimeKind | 新しいオブジェクトがローカル時間、UTC 時間、またはそのどちらでもない時間を表すかを指定します。 |

### ReturnValue

**value** と同じティック数を持ち、**kind** で指定された [DateTimeKind](../../datetimekind/) の値を表す新しい [DateTime](../) オブジェクト。
