## DateTime::Compare method


指定された [DateTime](../) クラスのインスタンスが表す 2 つの値を比較し、時間軸上での相対的な位置を示す値を返します。

```cpp
static constexpr int System::DateTime::Compare(DateTime t1, DateTime t2)
```


| パラメーター | 型 | 説明 |
| --- | --- | --- |
| t1 | DateTime | 最初の比較対象 |
| t2 | DateTime | 2 番目の比較対象 |

### ReturnValue

**t1** が **t2** より早い場合は 0 未満の値、**t1** が **t2** と同じ場合は 0、**t1** が **t2** より遅い場合は 0 超の値を返します。
