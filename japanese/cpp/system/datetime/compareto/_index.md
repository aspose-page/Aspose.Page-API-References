## DateTime::CompareTo method


現在のオブジェクトと指定された [DateTime](../) クラスのインスタンスが表す 2 つの日付時刻値を比較し、時間軸上での相対的な位置を示す値を返します。

```cpp
int System::DateTime::CompareTo(DateTime value) const
```


| パラメーター | 型 | 説明 |
| --- | --- | --- |
| value | DateTime | 現在のオブジェクトと比較するための [DateTime](../) クラスのインスタンス |

### ReturnValue

現在のオブジェクトが **value** が表す値よりも早い場合は 0 未満の値、両オブジェクトが同じ値を表す場合は 0、現在のオブジェクトが **value** が表す値よりも遅い場合は 0 超の値を返します。
