## ValueType class


パフォーマンス上の理由で [Object](../object/) 継承が省略された値型の基底クラスです。この型はスタック上に割り当て、関数に値渡しまたは参照渡しで渡すべきです。この型のオブジェクトを管理するために [System::SmartPtr](../smartptr/) クラスを使用しないでください。

```cpp
class ValueType
```
