## Char::IsControl(char_t) method


指定された文字が Unicode 制御文字として分類されるかどうかを判定します。

```cpp
static bool System::Char::IsControl(char_t c)
```


| パラメーター | 型 | 説明 |
| --- | --- | --- |
| c | char_t | テスト対象の文字 |

### ReturnValue

指定された文字が Unicode の制御文字である場合は true、そうでない場合は - false
