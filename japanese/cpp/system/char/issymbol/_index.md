## Char::IsSymbol(char_t) method


指定された文字が記号文字として分類されるかどうかを判断します。

```cpp
static bool System::Char::IsSymbol(char_t c)
```


| パラメーター | 型 | 説明 |
| --- | --- | --- |
| c | char_t | テスト対象の文字 |

### ReturnValue

指定された文字が記号文字である場合は true、そうでない場合は - false
