## クラス

| クラス | 説明 |
| --- | --- |
| [AuthenticatedStream](./authenticatedstream/) | ストリーム間で資格情報を渡すためのメソッドを含みます。このクラスのオブジェクトは [System::MakeObject()](../system/makeobject/) 関数を使用してのみ割り当てるべきです。スタック上や operator new を使用してこの型のインスタンスを作成しないでください。実行時エラーやアサーション違反が発生します。このクラスは常に [System::SmartPtr](../system/smartptr/) ポインタでラップし、そのポインタを関数への引数として使用してください。 |
| [SslStream](./sslstream/) | サーバーと（オプションで）クライアントを認証するために SSL プロトコルを使用するストリームです。 |
