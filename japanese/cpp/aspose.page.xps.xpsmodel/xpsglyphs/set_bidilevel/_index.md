## XpsGlyphs::set_BidiLevel method


Unicode アルゴリズムの双方向入れ子レベルを指定する値を取得/設定します。偶数値は左から右へのレイアウトを意味し、奇数値は右から左へのレイアウトを意味します。右から左へのレイアウトでは、最初のグリフの右側にランの起点が置かれ、正の前進幅（左方向への進みを表す）により、後続のグリフが前のグリフの左側に配置されます。
