## FlatStyle enum


コントロールがフラットに描画されるか、立体的に描画されるかを定義します。

```cpp
enum class FlatStyle
```

### 値

| 名前 | 値 | 説明 |
| --- | --- | --- |
| Flat | 0 | [Control](../control/) はフラットに描画されます。 |
| Popup | 1 | [Control](../control/) は、マウスが上に乗って 3D のように描画されている場合を除き、フラットに描画されます。 |
| Standard | 2 | [Control](../control/) は 3D のように描画されます。 |
| System | 3 | スタイルは OS によって制御されます。 |
