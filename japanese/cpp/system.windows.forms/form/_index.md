## Form class


[Form](./) クラスを使用する翻訳コードをコンパイル可能にするためのダミークラスです。メソッドは実装されていません。このクラスのオブジェクトは [System::MakeObject()](../../system/makeobject/) 関数を使用してのみ割り当てるべきです。スタック上や operator new を使用してインスタンスを作成しないでください。そうしないと実行時エラーやアサーション失敗が発生します。常にこのクラスを [System::SmartPtr](../../system/smartptr/) ポインタでラップし、関数の引数としてこのポインタを渡すようにしてください。

```cpp
class Form : public System::Windows::Forms::Control
```
