## LineJoin enum


連続する直線や曲線がどのように結合されるかを指定します。

```cpp
enum class LineJoin
```

### 値

| 名前 | 値 | 説明 |
| --- | --- | --- |
| Miter | 0 | マイタージョインは、マイターの長さがマイターリミットを超えるかどうかに応じて、鋭いまたは切り取られた角を生成します。 |
| Bevel | 1 | ベベルジョインは、対角線の角を生成します。 |
| 丸め | 2 | ラウンドジョインは、線間に滑らかな円弧を生成します。 |
| MiterClipped | 3 | マイタージョインは、マイターの長さがマイターリミットを超えるかどうかに応じて、鋭いまたはベベル加工された角を生成します。 |
