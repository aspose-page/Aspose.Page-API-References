## PixelOffsetMode enum


レンダリング時にピクセルがどのようにオフセットされるかを指定します。

```cpp
enum class PixelOffsetMode
```

### 値

| 名前 | 値 | 説明 |
| --- | --- | --- |
| Invalid | n/a | 無効な値です。 |
| Default | n/a | デフォルトモードです。 |
| HighSpeed | n/a | 低品質（高パフォーマンス）モード。 |
| 高品質 | n/a | 高品質（低パフォーマンス）モード。 |
| None | n/a | ピクセルオフセットなし。 |
| Half | n/a | 高パフォーマンスのアンチエイリアスのために、ピクセルは水平方向および垂直方向に -0.5 ユニットだけオフセットされます。 |
