## NormalizationForm enum


Unicode 文字列の正規化方法を定義します。

```cpp
enum class NormalizationForm
```

### 値

| 名前 | 値 | 説明 |
| --- | --- | --- |
| FormC | 1 | 完全な正規分解を行い、その後シーケンスを一次合成文字に置き換えます。 |
| FormD | 2 | 完全な正規分解を使用します。 |
| FormKC | 5 | 完全な互換分解を行い、その後シーケンスを一次合成文字に置き換えます。 |
| FormKD | 6 | 完全な互換分解を使用します。 |
