## FileShare enum


他の [FileStream](../filestream/) オブジェクトが開かれるファイルに対して持つことのできるアクセスの種類を指定します。

```cpp
enum class FileShare
```

### 値

| 名前 | 値 | 説明 |
| --- | --- | --- |
| None | 0 | アクセスなし。 |
| 読み取り | 1 | 読み取り専用アクセス。 |
| Write | 2 | 書き込み専用アクセス。 |
| ReadWrite | 3 | 読み取りおよび書き込みアクセス。 |
| 削除 | 4 | ファイルを削除できます。 |
| 継承可能 | 16 | ファイルハンドルを子プロセスが継承できるようにします。 |
