## SearchOption enum


検索を現在のディレクトリのみで実行するか、現在のディレクトリとすべてのサブディレクトリで実行するかを指定します。

```cpp
enum class SearchOption
```

### 値

| 名前 | 値 | 説明 |
| --- | --- | --- |
| AllDirectories | 0 | 現在のディレクトリとすべてのサブディレクトリを検索します。 |
| TopDirectoryOnly | 1 | 現在のディレクトリのみを検索します。 |
