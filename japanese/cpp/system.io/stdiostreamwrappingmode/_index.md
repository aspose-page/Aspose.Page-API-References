## STDIOStreamWrappingMode enum


ラッパーが std::iostream のようなストリームで実行する I/O 操作のモードを指定します。

```cpp
enum class STDIOStreamWrappingMode
```

### 値

| 名前 | 値 | 説明 |
| --- | --- | --- |
| Binary | 0 | 入力操作が char_type 型のストリームデータをバイトにデコードし、出力操作がバイトを char_type データにエンコードできるモードです。 |
| Conversion | 1 | 入力操作が char_type 型のストリームデータを uint8_t 型に変換し、出力操作がその逆（uint8_t から char_type）に変換できるモードです。 |
