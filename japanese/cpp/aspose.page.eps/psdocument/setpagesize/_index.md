## PsDocument::SetPageSize method


ページサイズを設定します。1つのドキュメント内でサイズが異なるページを作成するには、このメソッドの直後に [SetPageDevice](../setpagedevice/) メソッドを使用します。

```cpp
void Aspose::Page::EPS::PsDocument::SetPageSize(float width, float height)
```


| パラメーター | 型 | 説明 |
| --- | --- | --- |
| width | 単精度浮動小数点数 | 生成された PostScript ファイルにおけるページの幅。 |
| height | 単精度浮動小数点数 | 生成された PostScript ファイルにおけるページの高さ。 |
