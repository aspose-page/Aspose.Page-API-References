## XmlNodeOrder enum


ノードと別のノードとの文書内順序を記述します。

```cpp
enum class XmlNodeOrder
```

### 値

| 名前 | 値 | 説明 |
| --- | --- | --- |
| Before | 0 | このナビゲーターの現在のノードは、提供されたナビゲーターの現在のノードの前にあります。 |
| 後 | 1 | このナビゲーターの現在のノードは、提供されたナビゲーターの現在のノードの後にあります。 |
| 同じ | 2 | 2つのナビゲーターは同じノードに位置しています。 |
| 不明 | 3 | ノードの位置は文書順で相互に決定できません。これは、2つのノードが異なるツリーに存在する場合に起こる可能性があります。 |
