## XmlElement::set_IsEmpty method


要素のタグ形式を設定します。

```cpp
void System::Xml::XmlElement::set_IsEmpty(bool value)
```


| パラメーター | 型 | 説明 |
| --- | --- | --- |
| value | bool | 設定する値です。この値を設定する際、**true** に設定すると要素の子要素が削除され、要素は短いタグ形式でシリアライズされます。**false** に設定すると、要素の内容の有無に関係なく値が変更されます。要素が空の場合、長いタグ形式でシリアライズされます。 |
