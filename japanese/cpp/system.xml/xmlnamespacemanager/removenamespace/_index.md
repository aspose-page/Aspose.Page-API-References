## XmlNamespaceManager::RemoveNamespace method


指定されたプレフィックスに対する指定された名前空間を削除します。

```cpp
virtual void System::Xml::XmlNamespaceManager::RemoveNamespace(String prefix, String uri)
```


| パラメーター | 型 | 説明 |
| --- | --- | --- |
| prefix | String | 名前空間のプレフィックスです。 |
| uri | String | 指定されたプレフィックスに対して削除する名前空間です。削除される名前空間は現在の名前空間スコープからのものです。現在のスコープ外の名前空間は無視されます。 |
