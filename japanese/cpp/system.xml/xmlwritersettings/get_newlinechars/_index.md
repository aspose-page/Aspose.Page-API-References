## XmlWriterSettings::get_NewLineChars method


改行に使用する文字列を返します。

```cpp
String System::Xml::XmlWriterSettings::get_NewLineChars()
```


### ReturnValue

改行に使用する文字列です。任意の文字列値に設定できます。ただし、XML の有効性を保つため、スペース文字、タブ、キャリッジリターン、改行などの有効な空白文字のみを指定すべきです。デフォルトは **\\r\\n**（キャリッジリターン、改行）です。
