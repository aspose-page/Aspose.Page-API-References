## XmlWriterSettings::get_NamespaceHandling method


[XmlWriter](../../xmlwriter/) が XML コンテンツを書き込む際に重複する名前空間宣言を削除すべきかどうかを示す値を返します。既定の動作は、ライターがライターの名前空間リゾルバに存在するすべての名前空間宣言を出力することです。

```cpp
System::Xml::NamespaceHandling System::Xml::XmlWriterSettings::get_NamespaceHandling()
```


### ReturnValue

重複する名前空間宣言を削除するかどうかを指定するために使用される [NamespaceHandling](../../namespacehandling/) 列挙体。対象は [XmlWriter](../../xmlwriter/) です。
