## XmlReaderSettings::get_MaxCharactersInDocument method


XML ドキュメントで許容される最大文字数を示す値を返します。0 の値は XML ドキュメントのサイズに制限がないことを意味します。0 以外の値は文字数で最大サイズを指定します。

```cpp
int64_t System::Xml::XmlReaderSettings::get_MaxCharactersInDocument()
```


### ReturnValue

XML ドキュメントで許容される最大文字数です。デフォルトは 0 です。
