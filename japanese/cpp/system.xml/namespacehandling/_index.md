## NamespaceHandling enum


重複する名前空間宣言を [XmlWriter](../xmlwriter/) で削除するかどうかを指定します。

```cpp
enum class NamespaceHandling
```

### 値

| 名前 | 値 | 説明 |
| --- | --- | --- |
| Default | 0 | 重複する名前空間宣言は削除されないことを指定します。 |
| OmitDuplicates | 1 | 重複する名前空間宣言が削除されることを指定します。削除されるためには、プレフィックスと名前空間が一致している必要があります。 |
