## ParameterRef class


共通の [PrintTicket](../printticket/) パラメータ参照を実装するクラスです。**[ParameterRef](./)** 要素は [ParameterInit](../parameterinit/) 要素への参照を定義します。[ParameterRef](./) 要素を含む [ScoredProperty](../scoredproperty/) 要素は明示的に設定された [Value](../value/) 要素を持ちません。その代わり、[ScoredProperty](../scoredproperty/) 要素は **[ParameterRef](./)** 要素が参照する [ParameterInit](../parameterinit/) 要素から値を取得します。 [https://docs.microsoft.com/en-us/windows/win32/printdocs/parameterref](https://docs.microsoft.com/en-us/windows/win32/printdocs/parameterref)。

```cpp
class ParameterRef : public Aspose::Page::XPS::XpsMetadata::PrintTicketElement,
                     public Aspose::Page::XPS::XpsMetadata::IPrintTicketItem
```
