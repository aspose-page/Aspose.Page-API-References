## JobPrimaryCoverBack class


裏表紙（終了）カバーシートを記述します。各ジョブは個別のプライマリシートを持ちます。カバーシートはジョブの最終ページに使用される [PageMediaSize](../pagemediasize/) と [PageMediaType](../pagemediatype/) で印刷されるべきです。カバーシートは指定された [Option](../option/) によって示されるように、処理オプション（例：[JobDuplexAllDocumentsContiguously](../jobduplexalldocumentscontiguously/)、[JobNUpAllDocumentsContiguously](../jobnupalldocumentscontiguously/)）に統合されるべきです。[https://docs.microsoft.com/en-us/windows/win32/printdocs/jobprimarycoverback](https://docs.microsoft.com/en-us/windows/win32/printdocs/jobprimarycoverback)。
