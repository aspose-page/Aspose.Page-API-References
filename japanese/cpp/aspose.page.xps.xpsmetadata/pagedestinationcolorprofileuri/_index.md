## PageDestinationColorProfileURI class


相対 URI 参照を、[XPS](../../aspose.page.xps/)[Document](../../aspose.page/document/) に含まれる ICC プロファイルに対して指定します。このオプションの処理は、[PageDeviceColorSpaceUsage](../pagedevicecolorspaceusage/) 機能の設定に依存します。そのプロファイルを使用するすべての要素は、既に適切なデバイス カラースペースにあるとみなされ、ドライバーやデバイスでカラー管理されません。[https://docs.microsoft.com/en-us/windows/win32/printdocs/pagedestinationcolorprofileuri](https://docs.microsoft.com/en-us/windows/win32/printdocs/pagedestinationcolorprofileuri)。
