## JobPrimaryCoverFront class


表紙（開始）カバーシートを記述します。ジョブ全体は単一のプライマリシートを持ちます。カバーシートはジョブの最初のページに使用される [PageMediaSize](../pagemediasize/) と [PageMediaType](../pagemediatype/) で印刷されるべきです。カバーシートは指定された [Option](../option/) によって示されるように、処理オプション（例：[JobDuplexAllDocumentsContiguously](../jobduplexalldocumentscontiguously/)、[JobNUpAllDocumentsContiguously](../jobnupalldocumentscontiguously/)）に統合されるべきです。[https://docs.microsoft.com/en-us/windows/win32/printdocs/jobprimarycoverfront](https://docs.microsoft.com/en-us/windows/win32/printdocs/jobprimarycoverfront)。
