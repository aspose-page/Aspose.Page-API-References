## KeyInfoName class


XMLDSIG の 'KeyName' サブ要素または XML 暗号化の '[KeyInfo](../keyinfo/)' 要素を表します。このクラスのオブジェクトは [System::MakeObject()](../../system/makeobject/) 関数を使用してのみ割り当てるべきです。スタック上や operator new を使用してインスタンスを作成しないでください。そうしないと実行時エラーやアサーション障害が発生します。常にこのクラスを [System::SmartPtr](../../system/smartptr/) ポインタでラップし、関数の引数としてこのポインタを使用してください。

```cpp
class KeyInfoName : public System::Security::Cryptography::Xml::KeyInfoClause
```
