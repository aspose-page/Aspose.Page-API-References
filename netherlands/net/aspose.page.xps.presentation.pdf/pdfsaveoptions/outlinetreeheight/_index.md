## PdfSaveOptions.OutlineTreeHeight property

Specificeert de hoogte van de omtrekstructuur van het document die moet worden opgeslagen. 0 - de omtrekstructuur wordt niet geconverteerd, 1 - alleen de overzichtsitems op het eerste niveau worden geconverteerd, enzovoort. Standaard is 10.

public int OutlineTreeHeight { get; set; }
