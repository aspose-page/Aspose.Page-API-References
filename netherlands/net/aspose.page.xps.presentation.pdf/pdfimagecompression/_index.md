## PdfImageCompression enumeration

Specificeert het type compressie dat wordt toegepast op afbeeldingen in het PDF-bestand.

```csharp
public enum PdfImageCompression
```

### Waarden

| Naam | Waarde | Beschrijving |
| --- | --- | --- |
| Auto | `0` | Selecteert automatisch de meest geschikte compressie voor elke afbeelding. |
| None | `1` | Slaat onbewerkte afbeeldingsbytes op, wat resulteert in grotere PDF-bestandsgroottes. |
| Rle | `2` | Looplengte compressie. |
| Flate | `3` | Vlakke compressie. |
| LzwBaselinePredictor | `4` | Voorspellerselectie is beperkt tot PNG Paeth-voorspeller om het proces te versnellen. In de praktijk presteert verrassend goed. Beter danLzwOptimizedPredictor . |
| LzwOptimizedPredictor | `5` | Predictorselectie is ingewikkelder en zou moeten resulteren in kleinere afbeeldingsformaten, maar neemt meer tijd in beslag. |
| Jpeg | `6` | JPEG-compressie. Ondersteunt geen transparantie. |
