## ParameterRef class

De klasse die een algemene PrintTicket-parameterreferentie implementeert. A element definieert een verwijzing naar een[`ParameterInit`](../parameterinit/)element. A[`ScoredProperty`](../scoredproperty/) element dat een ParameterRef-element bevat, heeft geen een expliciet ingestelde[`Value`](../value/) element. In plaats daarvan, de[`ScoredProperty`](../scoredproperty/) element ontvangt zijn waarde van de[`ParameterInit`](../parameterinit/) element waarnaar wordt verwezen door een element. https://docs.microsoft.com/en-us/windows/win32/printdocs/parameterref
