## DocumentCoverFront class

Beschrijft het voorblad (beginblad). Elk document heeft een apart vel. Het voorblad moet worden afgedrukt op de[`PageMediaSize`](../pagemediasize/) En[`PageMediaType`](../pagemediatype/) gebruikt voor de eerste pagina van het document. Het voorblad moet worden geïntegreerd in processing options (zoals[`DocumentDuplex`](../documentduplex/) ,[`DocumentNUp`](../documentnup/) ) zoals aangegeven door de opgegeven optie. https://docs.microsoft.com/en-us/windows/win32/printdocs/documentcoverfront
