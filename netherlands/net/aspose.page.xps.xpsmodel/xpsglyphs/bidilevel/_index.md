## XpsGlyphs.BidiLevel property

Retourneert/stelt de waarde in die het bidirectionele nestniveau van het Unicode-algoritme specificeert. Even waarden impliceren een lay-out van links naar rechts, oneven waarden impliceren een lay-out van rechts naar links. Lay-out van rechts naar links plaatst de start van de run aan de rechterkant van de eerste glyph, met positieve voortgangsbreedten (die vorderingen naar links vertegenwoordigen) waarbij volgende glyphs links van de vorige glyph worden geplaatst.

```csharp
public int BidiLevel { get; set; }
```
