## FillAndStrokeText(string, Font, float, float, Brush, Pen) {#fillandstroketext_3}

Voegt een tekenreeks toe door de binnenkant van glyphs te vullen en contouren van glyphs te tekenen.

```csharp
public void FillAndStrokeText(string text, Font font, float x, float y, Brush fillPaint, Pen stroke)
```

| Parameter | Type | Beschrijving |
| --- | --- | --- |
| text | String | De tekst die moet worden toegevoegd. |
| font | Font | Systeemlettertype dat zal worden gebruikt om tekst te tekenen. |
| x | Single | X-coördinaat voor tekstoorsprong. |
| y | Single | Y-coördinaat voor tekstoorsprong. |
| fillPaint | Brush | De vulling die wordt gebruikt voor het schilderen van glyphs-interieur. |
| stroke | Pen | De streek die wordt gebruikt voor het tekenen van contouren van glyphs. |
