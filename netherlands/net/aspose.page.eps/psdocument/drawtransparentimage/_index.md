## PsDocument.DrawTransparentImage method

Teken getransformeerde transparante afbeelding. Als de afbeelding geen alfakanaal heeft, wordt deze getekend als dekkende afbeelding

```csharp
public void DrawTransparentImage(Bitmap image, Matrix transform, int transparencyThreshold)
```

| Parameter | Type | Beschrijving |
| --- | --- | --- |
| image | Bitmap | De afbeelding om te tekenen. |
| transform | Matrix | De matrix om het beeld te transformeren. |
| transparencyThreshold | Int32 | Een drempelwaarde die bepaalt vanaf welke waarde van de transparantiepixel wordt geïnterpreteerd als volledig transparant. Alle waarden onder deze drempel worden geïnterpreteerd als volledig ondoorzichtig. |
