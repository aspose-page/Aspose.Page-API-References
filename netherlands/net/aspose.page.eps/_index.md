De **Aspose.Page.EPS** is een hoofdnaamruimte voor alle klassen die te maken hebben met PS/EPS-bestanden.

## Klassen

| Klas | Beschrijving |
| --- | --- |
| [FontConstants](./fontconstants/) | Deze klasse definieert een set constanten voor het opslaan van lettertypen. |
| [GradientBrush](./gradientbrush/) | Deze klasse wordt gebruikt voor het inkapselen van LinearGradientBrush en PathGradientBrush met de mogelijkheid om de wrap-modus in te stellen op clamp. Native verlooppenselen genereren altijd een uitzondering wanneer iemand de eigenschap WrapMode probeert in te stellen op WrapMode.Clamp. |
| [LoadOptions](./loadoptions/) | Basisklasse voor laadopties voor documenten. |
| [PageConstants](./pageconstants/) | Deze klasse definieert een set constanten die een pagina beschrijven. Gemaksobjecten zijn beschikbaar voor verschillende marges, richtingen, herschaling en standaard paginaformaten. |
| [PsConverterException](./psconverterexception/) | Deze klasse bevat informatie over een fout die is optreedt terwijl het PS-bestand wordt geconverteerd naar een PDF-document. |
| [PsDocument](./psdocument/) | Deze klasse kapselt PS/EPS-documenten in. |
| [PsDocumentException](./psdocumentexception/) | Deze klasse bevat informatie over een fout die is optreedt terwijl het PostScript-bestand wordt gemaakt, bewerkt en opgeslagen. |
| [PsLoadOptions](./psloadoptions/) | Laadopties voor PS-documenten. |
