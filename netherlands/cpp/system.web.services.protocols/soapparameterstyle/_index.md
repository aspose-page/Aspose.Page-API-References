## SoapParameterStyle enum


Enumereert de parameterformaten in een SOAP‑bericht.

```cpp
enum class SoapParameterStyle
```

### Waarden

| Naam | Waarde | Beschrijving |
| --- | --- | --- |
| Default | 0 | Als '[SoapDocumentServiceAttribute](../soapdocumentserviceattribute/)' niet op de klasse wordt toegepast, dan is de standaardwaarde 'Wrapped'. |
| Bare | 1 | De parameters worden geplaatst in XML-elementen die volgen op het 'Body'-element. |
| Ingesloten | 2 | De parameters zijn ingekapseld binnen één XML-element dat volgt op het 'Body'-element. |
