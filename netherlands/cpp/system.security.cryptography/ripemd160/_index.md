## RIPEMD160 class


Berekent [RIPEMD160](./)-hash voor invoergegevens. In C++ is [RIPEMD160](./) geen abstracte klasse. Objecten van deze klasse mogen alleen worden gealloceerd met de functie [System::MakeObject()](../../system/makeobject/). Maak nooit een instantie van dit type op de stack of met operator new, omdat dit leidt tot runtime-fouten en/of assertiefouten. Wikkel deze klasse altijd in een [System::SmartPtr](../../system/smartptr/)-pointer en gebruik deze pointer om deze als argument aan functies door te geven.

```cpp
class RIPEMD160 : public System::Security::Cryptography::HashAlgorithm
```
