## DataGrid class


Dummklasse die het mogelijk maakt vertaalde code te koppelen aan [DataGrid](./) referenties, maar deze niet uitvoert. Bevat geen correct geïmplementeerde leden. Objecten van deze klasse mogen alleen worden gealloceerd met de functie [System::MakeObject()](../../system/makeobject/). Maak nooit een instantie van dit type op de stack of met operator new, omdat dit runtime‑fouten en/of assertiefouten veroorzaakt. Wikkel deze klasse altijd in een [System::SmartPtr](../../system/smartptr/) pointer en gebruik deze pointer om deze als argument door te geven aan functies.
