## XmlSchemaComplexType::get_BlockResolved method


Retourneert de waarde nadat het type is gecompileerd naar de post-schema-validatie-informatieset (infoset). Deze waarde geeft aan hoe het type wordt afgedwongen wanneer **xsi:type** wordt gebruikt in het instantiedocument.

```cpp
XmlSchemaDerivationMethod System::Xml::Schema::XmlSchemaComplexType::get_BlockResolved()
```


### ReturnValue

De post-schema-geverifieerde infoset-waarde. De standaard is **BlockDefault** waarde op het **schema** element.
