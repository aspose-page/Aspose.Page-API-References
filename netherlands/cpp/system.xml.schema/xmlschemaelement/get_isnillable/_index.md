## XmlSchemaElement::get_IsNillable method


Retourneert informatie die aangeeft of **xsi:nil** kan voorkomen in de instantie‑gegevens. Geeft aan of een expliciete nil‑waarde aan het element kan worden toegewezen.

```cpp
bool System::Xml::Schema::XmlSchemaElement::get_IsNillable()
```


### ReturnValue

Als nillable **true** is, maakt dit het mogelijk dat een instantie van het element het **nil**‑attribuut op **true** zet. Het **nil**‑attribuut is gedefinieerd als onderdeel van de XML [Schema](../../) namespace voor instanties. Standaard is **false**. Optioneel.
