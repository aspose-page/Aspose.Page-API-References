## XmlSchemaContentProcessing enum


Biedt informatie over de validatiemodus van vervangingen van **any**‑ en **anyAttribute**‑elementen.

```cpp
enum class XmlSchemaContentProcessing
```

### Waarden

| Naam | Waarde | Beschrijving |
| --- | --- | --- |
| None | 0 | Documentitems worden niet gevalideerd. |
| Overslaan | 1 | Documentitems moeten bestaan uit goed gevormde XML en worden niet gevalideerd door het schema. |
| Lax | 2 | Als het gekoppelde schema wordt gevonden, worden de documentitems gevalideerd. Anders worden er geen fouten gegenereerd. |
| Strict | 3 | De schemaprocessor moet een schema vinden dat is gekoppeld aan de aangegeven namespace om de documentitems te valideren. |
