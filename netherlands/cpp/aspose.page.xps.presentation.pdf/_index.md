De **Aspose.Page.Xps.Presentation.Pdf**-namespace biedt klassen voor het renderen van [XPS](../aspose.page.xps/) document naar PDF-document.

| Klasse | Beschrijving |
| --- | --- |
| [PdfDigitalSignatureDetails](./pdfdigitalsignaturedetails/) | Bevat details voor een digitale PDF-handtekening. |
| [PdfEncryptionDetails](./pdfencryptiondetails/) | Bevat details voor een pdf-encryptie. |
| [PdfSaveOptions](./pdfsaveoptions/) | Klasse voor XPS-als-PDF-opslagopties. |
