## AssemblyConstants class


Definieert de constanten die deelnemen aan de licentiecontrole voor het component. Deze werden vroeger direct gedefinieerd als assembly-attributen, maar ik heb ze naar een aparte klasse verplaatst omdat ik in .NET Compact Framework geen toegang heb tot assembly-attributen. Nu gebruikt de licentiecode, wanneer gecompileerd voor .NET Compact Framework, deze constanten in plaats van de assembly-attributen.

```cpp
class AssemblyConstants : public System::Object
```
