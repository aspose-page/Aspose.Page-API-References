## MemoryStream::TryGetBuffer method


Retourneert de array van ongetekende bytes waaruit deze stream is gemaakt.

```cpp
bool System::IO::MemoryStream::TryGetBuffer(ArraySegment<uint8_t> &buffer)
```


| Parameter | Type | Beschrijving |
| --- | --- | --- |
| buffer | ArraySegment\<uint8_t\>\& | byte array - out-parameter. Wanneer deze methode true retourneert, het byte array-segment waaruit deze stream is gemaakt; wanneer deze methode false retourneert, wordt deze parameter op de standaardwaarde gezet. |

### ReturnValue

True als de conversie is geslaagd.
