## File::Copy method


Kopieert het opgegeven bestand naar de opgegeven locatie. Als het bestemmingsbestand al bestaat, geeft een parameter aan of het moet worden overschreven.

```cpp
static void System::IO::File::Copy(const String &sourceFileName, const String &destFileName, bool overwrite=false)
```


| Parameter | Type | Beschrijving |
| --- | --- | --- |
| sourceFileName | const String\& | Een pad naar het te kopiëren bestand |
| destFileName | const String\& | Een pad naar de nieuwe locatie van het te kopiëren bestand |
| overschrijven | bool | True als het bestaande doelbestand moet worden overschreven, false als het kopiëren moet mislukken wanneer het doelbestand al bestaat |
