## ApsLoadOptions::get_TransparencyThreshold method


Een gehele waarde van 0 tot 255 waaronder een pixel van de afbeelding die alfa-waarden bevat, wordt beschouwd als volledig transparant. PostScript ondersteunt geen transparantie, maar kan een expliciete maskering boven een kleurenafbeelding toepassen waarbij sommige pixels volledig ondoorzichtig zijn en sommige volledig transparant. De transparantiedrempel wordt gebruikt om de beste gelijkenis tussen het origineel en het PostScript-resultaat te vinden. Standaardwaarde is 255.

int32_t Aspose::Page::XPS::ApsLoadOptions::get_TransparencyThreshold() const
