## StringCollection::crend method


Retourneert een reverse-iterator naar het element dat volgt op het laatste element van de omgekeerde container. Het correspondeert met het element dat voorafgaat aan het eerste element van de niet-omgekeerde container. Dit element fungeert als een placeholder; proberen er toegang toe te krijgen resulteert in ongedefinieerd gedrag.

```cpp
const_reverse_iterator System::Collections::Specialized::StringCollection::crend() const noexcept
```


### ReturnValue

Een iterator die wijst naar het theoretische const-gekwalificeerde element dat voorafgaat aan het eerste element van de container.
