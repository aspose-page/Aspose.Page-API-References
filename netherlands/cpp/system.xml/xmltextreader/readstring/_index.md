## XmlTextReader::ReadString method


Leest de inhoud van een element of een tekstknooppunt als een string.

```cpp
String System::Xml::XmlTextReader::ReadString() override
```


### ReturnValue

De inhoud van het element of tekstknooppunt. Dit kan een lege string zijn als de lezer zich bevindt op iets anders dan een element of tekstknooppunt, of als er geen tekstinhoud meer is om terug te geven in de huidige context. **Opmerking:** Het tekstknooppunt kan zowel een element als een attribuut-tekstknooppunt zijn.
