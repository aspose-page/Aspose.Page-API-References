## XmlDeclaration::get_Standalone method


Retourneert de waarde van het standalone‑attribuut.

```cpp
String System::Xml::XmlDeclaration::get_Standalone()
```


### ReturnValue

Geldige waarden zijn **yes** als alle entiteitsdeclaraties die door het XML‑document vereist zijn, in het document zijn opgenomen, of **no** als een externe documenttype‑definitie (DTD) vereist is. Als een standalone‑attribuut niet aanwezig is in de XML‑declaratie, retourneert deze methode [String::Empty](../../../system/string/empty/).
