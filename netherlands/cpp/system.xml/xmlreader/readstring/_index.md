## XmlReader::ReadString method


Wanneer overschreven in een afgeleide klasse, leest het de inhoud van een element of tekstknooppunt als een tekenreeks. Het wordt echter aanbevolen om in plaats daarvan de [XmlReader::ReadElementContentAsString](../readelementcontentasstring/) methode te gebruiken, omdat deze een eenvoudigere manier biedt om deze bewerking af te handelen.

```cpp
virtual String System::Xml::XmlReader::ReadString()
```


### ReturnValue

De inhoud van het element of een lege tekenreeks.
