## XmlOutputMethod enum


Specificeert de methode die wordt gebruikt om de [XmlWriter](../xmlwriter/) uitvoer te serialiseren.

```cpp
enum class XmlOutputMethod
```

### Waarden

| Naam | Waarde | Beschrijving |
| --- | --- | --- |
| Xml | 0 | Serialiseer volgens de XML 1.0-regels. |
| Html | 1 | Serialiseer volgens de HTML-regels die door XSLT zijn gespecificeerd. |
| Text | 2 | Serialiseer alleen tekstblokken. |
| AutoDetect | 3 | Gebruik de XSLT-regels om tijdens runtime te kiezen tussen de [XmlOutputMethod::Xml](./) en [XmlOutputMethod::Html](./) uitvoermethoden. |
