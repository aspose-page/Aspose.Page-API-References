## XmlAttribute::get_BaseURI method


Retourneert de basis Uniform Resource Identifier (URI) van het knooppunt.

```cpp
String System::Xml::XmlAttribute::get_BaseURI() override
```


### ReturnValue

De locatie waar het knooppunt is geladen of [String::Empty](../../../system/string/empty/) als het knooppunt geen basis-URI heeft. [Attribute](../../../system/attribute/) knooppunten hebben dezelfde basis-URI als hun eigenaarselement. Als een attribuutknooppunt geen eigenaarselement heeft, retourneert get_BaseURI [String::Empty](../../../system/string/empty/).
