## XmlNodeOrder enum


Beschrijft de documentvolgorde van een knooppunt ten opzichte van een tweede knooppunt.

```cpp
enum class XmlNodeOrder
```

### Waarden

| Naam | Waarde | Beschrijving |
| --- | --- | --- |
| Before | 0 | Het huidige knooppunt van deze navigator staat vóór het huidige knooppunt van de opgegeven navigator. |
| Na | 1 | De huidige node van deze navigator bevindt zich na de huidige node van de opgegeven navigator. |
| Hetzelfde | 2 | De twee navigators staan op dezelfde node. |
| Onbekend | 3 | De nodeposities kunnen niet worden bepaald in documentvolgorde ten opzichte van elkaar. Dit kan gebeuren als de twee nodes zich in verschillende bomen bevinden. |
