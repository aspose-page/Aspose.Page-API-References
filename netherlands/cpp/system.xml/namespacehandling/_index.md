## NamespaceHandling enum


Geeft aan of dubbele namespace‑declaraties moeten worden verwijderd in de [XmlWriter](../xmlwriter/).

```cpp
enum class NamespaceHandling
```

### Waarden

| Naam | Waarde | Beschrijving |
| --- | --- | --- |
| Default | 0 | Geeft aan dat dubbele namespace‑declaraties niet worden verwijderd. |
| OmitDuplicates | 1 | Geeft aan dat dubbele namespace‑declaraties worden verwijderd. Om een dubbele namespace te verwijderen, moeten het voorvoegsel en de namespace overeenkomen. |
