## XmlNodeReader::ReadString method


Leest de inhoud van een element of tekstknooppunt als een string.

```cpp
String System::Xml::XmlNodeReader::ReadString() override
```


### ReturnValue

De inhoud van het element of tekstachtige knooppunt (Dit kan CDATA, [Text](../../../system.text/) knooppunten, enzovoort omvatten). Dit kan een lege string zijn als de lezer zich bevindt op iets anders dan een element of tekstknooppunt, of als er geen tekstinhoud meer is om terug te geven in de huidige context. Opmerking: Het tekstknooppunt kan een element of een attribuut-tekstknooppunt zijn.
