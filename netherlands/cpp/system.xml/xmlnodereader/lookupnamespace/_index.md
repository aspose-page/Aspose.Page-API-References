

| Parameter | Type | Beschrijving |
| --- | --- | --- |
| prefix | const String\& | De prefix waarvan u de namespace-URI wilt oplossen. Om overeen te komen met de standaardnamespace, geeft u een lege string door. Deze string hoeft niet geatomiseerd te worden. |
