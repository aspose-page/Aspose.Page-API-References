## XmlConvert::ToDateTimeOffset(const String\&) method


Converteert de opgegeven [String](../../../system/string/) naar een [DateTimeOffset](../../../system/datetimeoffset/) equivalent.



| Parameter | Type | Beschrijving |
| --- | --- | --- |
| s | const String\& | De string om te converteren. De string moet voldoen aan een subset van de W3C-aanbeveling voor het XML dateTime-type. Voor meer informatie, zie de [dateTime](https://www.w3.org/TR/xmlschema-2/#dateTime) sectie van de XML [Schema](../../../system.xml.schema/) specificatie. |

### ReturnValue
