## XmlElement::set_IsEmpty method


Stelt het tagformaat van het element in.

```cpp
void System::Xml::XmlElement::set_IsEmpty(bool value)
```


| Parameter | Type | Beschrijving |
| --- | --- | --- |
| value | bool | De waarde om in te stellen. Bij het instellen van deze waarde, als deze op **true** wordt gezet, worden de kinderen van het element verwijderd en wordt het element geserialiseerd in het korte tagformaat. Als deze op **false** wordt gezet, wordt de waarde gewijzigd (ongeacht of het element inhoud heeft of niet); als het element leeg is, wordt het geserialiseerd in het lange formaat. |
