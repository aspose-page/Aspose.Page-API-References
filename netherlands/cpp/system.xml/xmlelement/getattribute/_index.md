## XmlElement::GetAttribute(String, String) method


Retourneert de waarde voor het attribuut met de opgegeven lokale naam en namespace-URI.

```cpp
virtual String System::Xml::XmlElement::GetAttribute(String localName, String namespaceURI)
```


| Parameter | Type | Beschrijving |
| --- | --- | --- |
| localName | String | De lokale naam van het attribuut om op te halen. |
| namespaceURI | String | De namespace-URI van het attribuut om op te halen. |

### ReturnValue

De waarde van het opgegeven attribuut. Een lege tekenreeks wordt geretourneerd als een overeenkomend attribuut niet wordt gevonden of als het attribuut geen opgegeven of standaardwaarde heeft.
