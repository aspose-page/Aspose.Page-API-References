## XmlElement::RemoveAttributeAt method


Verwijdert het attribuutknooppunt met de opgegeven index uit het element. (Als het verwijderde attribuut een standaardwaarde heeft, wordt deze onmiddellijk vervangen).

```cpp
virtual SharedPtr<XmlNode> System::Xml::XmlElement::RemoveAttributeAt(int32_t i)
```


| Parameter | Type | Beschrijving |
| --- | --- | --- |
| i | int32_t | De index van het knooppunt dat verwijderd moet worden. Het eerste knooppunt heeft index 0. |

### ReturnValue

Het verwijderde attribuutknooppunt of **nullptr** als er geen knooppunt op de opgegeven index bestaat.
