## XmlWriterSettings::get_NewLineChars method


Retourneert de tekenreeks die moet worden gebruikt voor regeleinden.

```cpp
String System::Xml::XmlWriterSettings::get_NewLineChars()
```


### ReturnValue

De tekenreeks die gebruikt wordt voor regeleinden. Deze kan op elke tekenreekswaarde worden ingesteld. Om echter geldige XML te waarborgen, moet u alleen geldige witruimte‑tekens opgeven, zoals spaties, tabs, carriage returns of line feeds. De standaardwaarde is **\r\n** (carriage return, nieuwe regel).
