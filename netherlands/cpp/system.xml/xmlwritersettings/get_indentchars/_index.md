## XmlWriterSettings::get_IndentChars method


Retourneert de tekenreeks die moet worden gebruikt bij inspringen. Deze instelling wordt gebruikt wanneer de [XmlWriterSettings::set_Indent](../set_indent/) waarde is ingesteld op **true**.

```cpp
String System::Xml::XmlWriterSettings::get_IndentChars()
```


### ReturnValue

De tekenreeks die moet worden gebruikt bij inspringen. Deze kan worden ingesteld op elke tekenreekswaarde. Om echter geldige XML te garanderen, moet u alleen geldige witruimtetekens opgeven, zoals spaties, tabs, regeleinden of nieuwe regels. Standaard zijn twee spaties.
