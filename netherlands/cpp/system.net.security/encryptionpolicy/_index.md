## EncryptionPolicy enum


Somt de encryptie‑beleidsregels op.

```cpp
enum class EncryptionPolicy
```

### Waarden

| Naam | Waarde | Beschrijving |
| --- | --- | --- |
| RequireEncryption | 0 | Vereis encryptie en sta nooit een 'Null' cipher toe. |
| AllowNoEncryption | 1 | Geef de voorkeur aan volledige encryptie, maar een 'Null' cipher kan worden gebruikt als de server akkoord gaat. |
| NoEncryption | 2 | Sta geen encryptie toe en vraag dat een 'Null' cipher wordt gebruikt als het andere eindpunt een 'Null' cipher kan verwerken. |
