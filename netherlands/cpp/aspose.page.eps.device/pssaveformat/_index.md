## PsSaveFormat enum


Deze enumeratie bevat beschikbare opties voor het opslagformaat. Het kan PS of [EPS](../../aspose.page.eps/) zijn. [EPS](../../aspose.page.eps/) wordt alleen gebruikt voor éénpagina‑documenten, terwijl een PS‑bestand een willekeurig aantal pagina's kan bevatten.

```cpp
enum class PsSaveFormat
```

### Waarden

| Naam | Waarde | Beschrijving |
| --- | --- | --- |
| PS | 0 | Deze optie geeft aan dat het resulterende document een PostScript (PS) of Encapsulated PostScript ([EPS](../../aspose.page.eps/)) bestand moet zijn. |
| EPS | 1 | Deze optie geeft aan dat het resulterende document een Encapsulated PostScript ([EPS](../../aspose.page.eps/)) bestand moet zijn. Het wordt alleen gebruikt voor éénpagina‑documenten;. |
