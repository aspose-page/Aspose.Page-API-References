## TimeSpanStyles enum


Specificeert de opmaakopties die het parseren van strings aanpassen voor methoden die een tekenreeksrepresentatie van een tijdsinterval omzetten naar een Timespan‑object.

```cpp
enum class TimeSpanStyles : int32_t
```

### Waarden

| Naam | Waarde | Beschrijving |
| --- | --- | --- |
| None | 0 | Het tijdsinterval dat wordt weergegeven door de inptut-tekenreeks wordt alleen als een negatief tijdsinterval geïnterpreteerd als er een minteken aanwezig is. |
| AssumeNegative | 1 | Het tijdsinterval dat wordt weergegeven door de inptut-tekenreeks wordt altijd als een negatief tijdsinterval geïnterpreteerd. |
