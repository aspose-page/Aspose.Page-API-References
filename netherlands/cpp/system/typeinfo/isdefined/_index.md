## TypeInfo::IsDefined method


NIET GEREALISEERD. Geeft aan of één of meer attributen van het opgegeven type of van afgeleide types op dit lid zijn toegepast.

```cpp
bool System::TypeInfo::IsDefined(const TypeInfo &attributeType, bool inherit) const
```


| Parameter | Type | Beschrijving |
| --- | --- | --- |
| attributeType | const TypeInfo\& | Het type aangepaste attribuut om naar te zoeken. De zoekopdracht omvat afgeleide types. |
| inherit | bool | true om de overervingsketen van dit lid te doorzoeken om de attributen te vinden; anders false. Deze parameter wordt genegeerd voor eigenschappen en gebeurtenissen. |

### ReturnValue

true als één of meer instanties van attributeType of een van zijn afgeleide types op dit lid zijn toegepast; anders false.
