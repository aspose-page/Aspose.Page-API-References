## Array::TrueForAll method



| Parameter | Type | Beschrijving |
| --- | --- | --- |
| arr | System::ArrayPtr\<T\> | [Array](../) elementen die overeenkomen met de voorwaarden |
| overeenkomst | System::Predicate\<T\> | Een predicaat dat de voorwaarden definieert waaraan array‑elementen moeten voldoen |
