## SmartPtr::get_shared_count method


Haalt het aantal gedeelde pointers op dat bestaat naar het gerefereerde object, inclusief de huidige. Stelt dat de huidige pointer in gedeelde modus is.

```cpp
int System::SmartPtr<T>::get_shared_count() const
```


### ReturnValue

Aantal gedeelde pointers dat bestaat naar het gerefereerde object, indien aanwezig. Als de pointer null is, wordt 0 geretourneerd.
