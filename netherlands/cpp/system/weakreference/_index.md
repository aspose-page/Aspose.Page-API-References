## WeakReference class


Stelt een zwakke referentie voor, die een object referereert terwijl dat object nog steeds kan worden verwijderd.

```cpp
template<typename...>class WeakReference
```


| Parameter | Beschrijving |
| --- | --- |
| Argumenten | Sjabloonargumenten van een zwakke referentie die een type van een verwezen object declareren, het aantal argumenten kan nul of één zijn. |
