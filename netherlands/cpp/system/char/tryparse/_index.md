## Char::TryParse method


Probeert een tekenreeks die uit één enkel teken bestaat omzetten naar een UTF-16‑teken. De functie slaagt alleen wanneer de invoertekenreeks niet null is en precies één teken lang is.

static bool System::Char::TryParse(const System::String &s, char_t &result)

| result | char_t\\& | De uitvoervariabele die het resultaat van de conversie bevat als de conversie slaagt |

True als de conversie geslaagd is, anders - false
