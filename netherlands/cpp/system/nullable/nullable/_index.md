## Nullable::Nullable(const Nullable\<T1\>\&) constructor


Construeert een instantie die een waarde vertegenwoordigt die wordt weergegeven door het opgegeven [Nullable](../) object. Het opgegeven nullable‑object kan een waarde van een ander type vertegenwoordigen dan het onderliggende type van de geconstrueerde instantie; in dat geval wordt de weergegeven waarde geconverteerd naar een waarde van type T.

```cpp
template<typename T1> System::Nullable<T>::Nullable(const Nullable<T1> &value)
```


| Parameter | Beschrijving |
| --- | --- |
| T1 | Het type van de waarde die wordt weergegeven door het opgegeven [Nullable](../) object |
