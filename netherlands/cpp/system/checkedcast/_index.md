## System::CheckedCast method

```cpp
template<typename TTo,typename TFrom> TTo System::CheckedCast(TFrom value)
```


| Parameter | Beschrijving |
| --- | --- |
| TTo | Het type waarnaar de opgegeven waarde moet worden gecast |
| TFrom | Het type van de opgegeven waarde |

| Parameter | Type | Beschrijving |
| --- | --- | --- |
| value | TFrom | De waarde om te casten |

### ReturnValue
