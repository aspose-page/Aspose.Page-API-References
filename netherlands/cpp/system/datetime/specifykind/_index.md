## DateTime::SpecifyKind method


Construeert een nieuw [DateTime](../) object dat hetzelfde aantal ticks vertegenwoordigt als het opgegeven [DateTime](../) object en lokale tijd, UTC‑tijd of geen van beide weergeeft, zoals gespecificeerd door het argument **kind**.

```cpp
static DateTime System::DateTime::SpecifyKind(DateTime value, DateTimeKind kind)
```


| Parameter | Type | Beschrijving |
| --- | --- | --- |
| value | DateTime | Het [DateTime](../) object waarvan het aantal ticks moet worden gekopieerd. |
| type | DateTimeKind | Specificeert of het nieuwe object lokale tijd, UTC‑tijd of geen van beide moet weergeven. |

### ReturnValue

Een nieuw [DateTime](../) object dat hetzelfde aantal ticks vertegenwoordigt als **value** en de [DateTimeKind](../../datetimekind/)‑waarde gespecificeerd door **kind**.
