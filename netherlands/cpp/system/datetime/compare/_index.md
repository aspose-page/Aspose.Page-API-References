## DateTime::Compare method


Vergelijkt twee waarden die worden weergegeven door de opgegeven instanties van de [DateTime](../) klasse en retourneert de waarde die de relatieve posities van de waarden op de tijdlijn aangeeft.

```cpp
static constexpr int System::DateTime::Compare(DateTime t1, DateTime t2)
```


| Parameter | Type | Beschrijving |
| --- | --- | --- |
| t1 | DateTime | De eerste operand |
| t2 | DateTime | De tweede operand |

### ReturnValue

Een waarde die kleiner is dan 0 als **t1** eerder is dan **t2**; 0 als **t1** gelijk is aan **t2**; een waarde die groter is dan 0 als **t1** later is dan **t2**
