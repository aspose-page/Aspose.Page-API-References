## DateTime::CompareTo method


Vergelijkt twee datum‑ en tijdwaarden die worden weergegeven door het huidige object en de opgegeven instantie van de [DateTime](../)‑klasse en retourneert de waarde die de relatieve posities van de waarden op de tijdlijn aangeeft.

```cpp
int System::DateTime::CompareTo(DateTime value) const
```


| Parameter | Type | Beschrijving |
| --- | --- | --- |
| value | DateTime | Een instantie van de [DateTime](../)‑klasse om het huidige object mee te vergelijken |

### ReturnValue

Een waarde die kleiner is dan 0 als het huidige object een waarde vertegenwoordigt die eerder is dan die vertegenwoordigd door **value**; 0 als de waarden die door beide objecten worden weergegeven gelijk zijn; een waarde die groter is dan 0 als de waarde die door het huidige object wordt weergegeven later is dan die vertegenwoordigd door **value**
