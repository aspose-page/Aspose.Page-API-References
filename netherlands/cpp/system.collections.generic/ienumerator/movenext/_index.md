## IEnumerator::MoveNext method


Verplaatst de enumerator naar het volgende element. Als er eerder geen element werd gerefereerd, wordt de referentie ingesteld op het eerste beschikbare element. Als het einde van de container is bereikt, gebeurt er niets.

```cpp
virtual bool System::Collections::Generic::IEnumerator<T>::MoveNext()=0
```


### ReturnValue

True als er een element beschikbaar is na de aanroep, false anders.
