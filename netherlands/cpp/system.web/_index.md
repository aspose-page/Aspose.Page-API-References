## Klassen

| Klasse | Beschrijving |
| --- | --- |
| [HttpContext](./httpcontext/) | Dummy‑klasse die het mogelijk maakt vertaalde code te koppelen aan [HttpContext](./httpcontext/) referenties, maar deze niet uit te voeren. Bevat geen correct geïmplementeerde leden. Objecten van deze klasse mogen alleen worden gealloceerd met de functie [System::MakeObject()](../system/makeobject/). Maak nooit een instantie van dit type op de stack of met operator new, omdat dit leidt tot runtime‑fouten en/of assertiefouten. Wikkel deze klasse altijd in een [System::SmartPtr](../system/smartptr/) pointer en gebruik deze pointer om deze als argument aan functies door te geven. |
| [HttpResponse](./httpresponse/) | Dummy‑klasse die het mogelijk maakt vertaalde code te koppelen aan [HttpResponse](./httpresponse/) referenties, maar deze niet uit te voeren. Maak nooit een instantie van dit type op de stack of met operator new, omdat dit leidt tot runtime‑fouten en/of assertiefouten. Wikkel deze klasse altijd in een [System::SmartPtr](../system/smartptr/) pointer en gebruik deze pointer om deze als argument aan functies door te geven. |
