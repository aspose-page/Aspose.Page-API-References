## ParameterRef class


De klasse die een algemene [PrintTicket](../printticket/) parameterreferentie implementeert. Een **[ParameterRef](./)** element definieert een verwijzing naar een [ParameterInit](../parameterinit/) element. Een [ScoredProperty](../scoredproperty/) element dat een [ParameterRef](./) element bevat, heeft geen expliciet ingesteld [Value](../value/) element. In plaats daarvan ontvangt het [ScoredProperty](../scoredproperty/) element zijn waarde van het [ParameterInit](../parameterinit/) element waarnaar wordt verwezen door een **[ParameterRef](./)** element. [https://docs.microsoft.com/en-us/windows/win32/printdocs/parameterref](https://docs.microsoft.com/en-us/windows/win32/printdocs/parameterref).
