## ScoredProperty class


De klasse die een algemene [PrintTicket](../printticket/)**[ScoredProperty](./)** implementeert. De basisklasse voor alle schema‑gedefinieerde scored properties. Een **[ScoredProperty](./)** element declareert een eigenschap die intrinsiek is aan een [Option](../option/) definitie. Dergelijke eigenschappen moeten worden vergeleken bij het evalueren hoe nauw een aangevraagde [Option](../option/) overeenkomt met een apparaat‑ondersteunde [Option](../option/).
