## XPathNavigator::AppendChildElement method


Maakt een nieuw kindelementknooppunt aan het einde van de lijst met kindknooppunten van het huidige knooppunt met behulp van het opgegeven namespace‑voorvoegsel, de lokale naam, de opgegeven namespace‑URI en de opgegeven waarde.

```cpp
virtual void System::Xml::XPath::XPathNavigator::AppendChildElement(String prefix, String localName, String namespaceURI, String value)
```


| Parameter | Type | Beschrijving |
| --- | --- | --- |
| prefix | String | Het namespace‑voorvoegsel van het nieuwe kind‑elementknooppunt (indien aanwezig). |
| localName | String | De lokale naam van het nieuwe kind‑elementknooppunt (indien aanwezig). |
| namespaceURI | String | De namespace‑URI van het nieuwe kind‑elementknooppunt (indien aanwezig). [String::Empty](../../../system/string/empty/) en **nullptr** zijn equivalent. |
| value | String | De waarde van het nieuwe kind‑elementknooppunt. Als [String::Empty](../../../system/string/empty/) of **nullptr** wordt doorgegeven, wordt een leeg element aangemaakt. |
