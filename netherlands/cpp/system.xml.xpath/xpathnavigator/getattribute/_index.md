## XPathNavigator::GetAttribute method


Geeft de waarde van het attribuut met de opgegeven lokale naam en namespace-URI terug.

```cpp
virtual String System::Xml::XPath::XPathNavigator::GetAttribute(String localName, String namespaceURI)
```


| Parameter | Type | Beschrijving |
| --- | --- | --- |
| localName | String | De lokale naam van het attribuut. **localName** is hoofdlettergevoelig. |
| namespaceURI | String | De namespace-URI van het attribuut. |

### ReturnValue

Een [String](../../../system/string/) die de waarde van het opgegeven attribuut bevat; [String::Empty](../../../system/string/empty/) als een overeenkomend attribuut niet wordt gevonden, of als de [XPathNavigator](../) niet op een elementknooppunt staat.
