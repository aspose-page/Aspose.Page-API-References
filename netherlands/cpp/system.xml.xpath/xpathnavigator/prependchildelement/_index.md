## XPathNavigator::PrependChildElement method


Maakt een nieuw kindelement aan het begin van de lijst met kindknooppunten van het huidige knooppunt met behulp van het opgegeven namespace‑voorvoegsel, lokale naam en namespace‑URI met de opgegeven waarde.

```cpp
virtual void System::Xml::XPath::XPathNavigator::PrependChildElement(String prefix, String localName, String namespaceURI, String value)
```


| Parameter | Type | Beschrijving |
| --- | --- | --- |
| prefix | String | Het namespace‑voorvoegsel van het nieuwe kindelement (indien aanwezig). |
| localName | String | De lokale naam van het nieuwe kindelement (indien aanwezig). |
| namespaceURI | String | De namespace‑URI van het nieuwe kindelement (indien aanwezig). [String::Empty](../../../system/string/empty/) en **nullptr** zijn equivalent. |
| value | String | De waarde van het nieuwe kindelement. Als [String::Empty](../../../system/string/empty/) of **nullptr** wordt doorgegeven, wordt een leeg element gemaakt. |
