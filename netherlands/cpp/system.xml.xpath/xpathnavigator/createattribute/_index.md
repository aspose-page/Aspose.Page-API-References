## XPathNavigator::CreateAttribute method


Maakt een attribuutknooppunt op het huidige elementknooppunt met behulp van het opgegeven namespace‑voorvoegsel, de lokale naam, de opgegeven namespace‑URI en de opgegeven waarde.

```cpp
virtual void System::Xml::XPath::XPathNavigator::CreateAttribute(String prefix, String localName, String namespaceURI, String value)
```


| Parameter | Type | Beschrijving |
| --- | --- | --- |
| prefix | String | Het namespace-voorvoegsel van het nieuwe attribuut-node (indien aanwezig). |
| localName | String | De lokale naam van het nieuwe attribuut-node die niet [String::Empty](../../../system/string/empty/) of **nullptr** mag zijn. |
| namespaceURI | String | De namespace-URI voor het nieuwe attribuut-node (indien aanwezig). |
| value | String | De waarde van het nieuwe attribuut-node. Als [String::Empty](../../../system/string/empty/) of **nullptr** wordt doorgegeven, wordt een leeg attribuut-node aangemaakt. |
