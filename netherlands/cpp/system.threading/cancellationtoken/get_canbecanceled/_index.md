## CancellationToken::get_CanBeCanceled method


Bepaalt of dit token in de geannuleerde status kan verkeren.

```cpp
bool System::Threading::CancellationToken::get_CanBeCanceled() const
```


### ReturnValue

true als dit token in staat is om zich in de geannuleerde status te bevinden; anders false.
## Opmerkingen



Tokens die zijn gemaakt vanuit [CancellationTokenSource](../../cancellationtokensource/) zullen true retourneren, terwijl het None-token altijd false retourneert.
