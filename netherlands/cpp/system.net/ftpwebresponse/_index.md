## FtpWebResponse class


Dummy‑klasse die het mogelijk maakt om vertaalde code te koppelen aan [FtpWebResponse](./)-referenties, maar deze niet uit te voeren. Bevat geen correct geïmplementeerde leden. Objecten van deze klasse mogen alleen worden gealloceerd met de functie [System::MakeObject()](../../system/makeobject/). Maak nooit een instantie van dit type op de stack of met operator new, omdat dit zal leiden tot runtime‑fouten en/of assertiefouten. Wrap deze klasse altijd in een [System::SmartPtr](../../system/smartptr/)-pointer en gebruik deze pointer om deze als argument door te geven aan functies.
