## StringTrimming enum

```cpp
enum class StringTrimming
```

### Waarden

| Naam | Waarde | Beschrijving |
| --- | --- | --- |
| None | 0 | Geen bijsnijden. |
| Teken | 1 | Snijd af tot het dichtstbijzijnde teken. |
| Woord | 2 | Snijd af tot het dichtstbijzijnde woord. |
| EllipsisCharacter | 3 | Snijd af tot het dichtstbijzijnde teken en voeg een ellipsis toe aan het einde van de tekenreeks. |
| EllipsisWord | 4 | Snijd af tot het dichtstbijzijnde woord en voeg een ellipsis toe aan het einde van de tekenreeks. |
