## Point::Add method


Voegt de breedte- en hoogtewaarden van het opgegeven [Size](../../size/) object toe aan de X- en Y-coördinaatwaarden van het opgegeven [Point](../) object overeenkomstig.

```cpp
static Point System::Drawing::Point::Add(const Point &point, const Size &size)
```


| Parameter | Type | Beschrijving |
| --- | --- | --- |
| punt | const Point\& | Het punt om te vertalen |
| size | const Size\& | Het [Size](../../size/) object dat de waarden specificeert die moeten worden toegevoegd aan de coördinaatwaarden van het **point** |

### ReturnValue

Een nieuw [Point](../) object waarvan de X-coördinaatwaarde gelijk is aan de som van de X-coördinaatwaarde van **point** en de breedtewaarde van **size**, en waarvan de Y-coördinaatwaarde gelijk is aan de som van de Y-coördinaatwaarde van **point** en de hoogtewaarde van **size**
