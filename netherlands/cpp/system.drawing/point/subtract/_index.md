## Point::Subtract method


Trekt de breedte- en hoogtewaarden van het opgegeven [Size](../../size/) object af van de X- en Y-coördinaatwaarden van het opgegeven [Point](../) object, respectievelijk.

```cpp
static Point System::Drawing::Point::Subtract(const Point &point, const Size &size)
```


| Parameter | Type | Beschrijving |
| --- | --- | --- |
| punt | const Point\& | Het punt om te vertalen |
| size | const Size\& | Het [Size](../../size/)-object dat de waarden specificeert die van de coördinaatwaarden van de **point** moeten worden afgetrokken |

### ReturnValue

Een nieuw [Point](../) object waarvan de X-coördinaatwaarde gelijk is aan het resultaat van het aftrekken van de breedtewaarde van **size** van de X-coördinaatwaarde van **point**, en waarvan de Y-coördinaatwaarde gelijk is aan het resultaat van het aftrekken van de hoogtewaarde van **size** van de Y-coördinaatwaarde van **point**.
