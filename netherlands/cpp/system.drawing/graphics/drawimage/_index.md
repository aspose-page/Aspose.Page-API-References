## Graphics::DrawImage(const SharedPtr\<Image\>\&, Rectangle, int, int, int, int, GraphicsUnit, const Imaging::ImageAttributesPtr\&) method

```cpp
void System::Drawing::Graphics::DrawImage(const SharedPtr<Image> &image, Rectangle destRect, int srcX, int srcY, int srcWidth, int srcHeight, GraphicsUnit srcUnit, const Imaging::ImageAttributesPtr &imgAttributes)
```


| Parameter | Type | Beschrijving |
| --- | --- | --- |
| afbeelding | const SharedPtr\<Image\>\& | De afbeelding om te tekenen |
| destRect | Rechthoek | Een rechthoek om de afbeelding naartoe te tekenen |
| srcX | int | De X-coördinaat van de linkerbovenhoek van de rechthoek die het gedeelte van de afbeelding specificeert om te tekenen |
| srcY | int | De Y-coördinaat van de linkerbovenhoek van de rechthoek die het gedeelte van de afbeelding specificeert om te tekenen |
| srcWidth | int | De breedte van de linkerbovenhoek van de rechthoek die het gedeelte van de afbeelding specificeert dat getekend moet worden |
| srcHeight | int | De hoogte van de linkerbovenhoek van de rechthoek die het gedeelte van de afbeelding specificeert dat getekend moet worden |
| srcUnit | GraphicsUnit | De meeteenheden waarin de parameters **srcX**, **srcY**, **srcWidth** en **srcHeight** worden gespecificeerd |
| imgAttributes | const Imaging::ImageAttributesPtr\& | Specificeert kleur- en gamma-informatie voor de afbeelding |
