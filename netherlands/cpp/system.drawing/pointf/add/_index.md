## PointF::Add(const PointF\&, const Size\&) method

```cpp
static PointF System::Drawing::PointF::Add(const PointF &point, const Size &size)
```


| Parameter | Type | Beschrijving |
| --- | --- | --- |

| size | const Size\& | Het [Size](../../size/) object dat de waarden specificeert die moeten worden toegevoegd aan de coördinaatwaarden van het **point** |

### ReturnValue
