## PointF::Subtract(const PointF\&, const Size\&) method

```cpp

```


| Parameter | Type | Beschrijving |
| --- | --- | --- |
| punt | const PointF\& | Het punt om te vertalen |
| size | const Size\& | Het [Size](../../size/)-object dat de waarden specificeert die van de coördinaatwaarden van de **point** moeten worden afgetrokken |

### ReturnValue
