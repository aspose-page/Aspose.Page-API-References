## Klassen

| Klasse | Beschrijving |
| --- | --- |
| [SecurityPermission](./securitypermission/) | Klasse die een beveiligingsmachtiging beschrijft. Objecten van deze klasse mogen alleen worden toegewezen met de functie [System::MakeObject()](../system/makeobject/). Maak nooit een instantie van dit type op de stack of met operator new, omdat dit leidt tot runtime‑fouten en/of assertiefouten. Wikkel deze klasse altijd in een [System::SmartPtr](../system/smartptr/) pointer en gebruik deze pointer om het als argument aan functies door te geven. |
