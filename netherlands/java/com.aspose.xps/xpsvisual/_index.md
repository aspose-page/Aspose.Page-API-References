Klasse die visuele elementeigenschappen omvat. Dit eigenschapselement bevat markup die de inhoud van een enkele visuele penseel-tegel definieert. De tegel kan worden gebruikt om het geometrische gebied te vullen waarop de visuele penseel wordt toegepast.
