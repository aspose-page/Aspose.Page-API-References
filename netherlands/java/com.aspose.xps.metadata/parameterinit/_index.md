Klasse die een printticket-parameterinitialisator implementeert. De klasse die een algemene PrintTicket-parameterinitialisator implementeert. De basisklasse voor alle schema‑gedefinieerde parameterinitialisatoren. Definieert een waarde voor een instantie van een ParameterDef‑element. Een ParameterInit‑element is het doel van de referentie die wordt gemaakt door een ParameterRef‑element. https://docs.microsoft.com/en-us/windows/win32/printdocs/parameterinit
