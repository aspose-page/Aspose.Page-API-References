De klasse die een algemene PrintTicket-parameterreferentie implementeert. Een  ParameterRef  element definieert een referentie naar een  ParameterInit  element. Een  ScoredProperty  element dat een ParameterRef‑element bevat, heeft geen expliciet ingestelde  Value  element. In plaats daarvan ontvangt het  ScoredProperty  element zijn waarde van het  ParameterInit  element waarnaar wordt verwezen door een  ParameterRef  element. https://docs.microsoft.com/en-us/windows/win32/printdocs/parameterref
