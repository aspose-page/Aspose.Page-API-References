De klasse die een algemene PrintTicket  ScoredProperty implementeert. De basisklasse voor alle schema‑gedefinieerde gescoorde eigenschappen. Een  ScoredProperty  element declareert een eigenschap die intrinsiek is aan een  Option  definitie. Dergelijke eigenschappen moeten worden vergeleken bij het evalueren hoe nauw een aangevraagde  Option  overeenkomt met een apparaat‑ondersteunde  Option . https://docs.microsoft.com/en-us/windows/win32/printdocs/scoredproperty
