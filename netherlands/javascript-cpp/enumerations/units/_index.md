## Units enumeration

Specificeert type van grootte.

```js
enum Units
```

### Waarden

| Naam | Waarde | Beschrijving |
| ---        | ---   | --- |
| Points | `0` | Punten (1/72 van een inch). |
| Inches | `1` | Inches. |
| Millimeters | `2` | Millimeters. |
| Centimeters | `3` | Centimeters. |
| Percents | `4` | Percenten van bestaande grootte. |
