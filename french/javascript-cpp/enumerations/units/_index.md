## Units enumeration

Spécifie le type de taille.

```js
enum Units
```

### Valeurs

| Nom | Valeur | Description |
| ---        | ---   | --- |
| Points | `0` | Points (1/72 de pouce). |
| Inches | `1` | Pouces. |
| Millimeters | `2` | Millimètres. |
| Centimeters | `3` | Centimètres. |
| Percents | `4` | Pourcentages de la taille existante. |
