public class ScoredProperty extends CompositePrintTicketElement implements IOptionItem, IScoredPropertyItem

La classe qui implémente une ScoredProperty de PrintTicket commune. La classe de base pour toutes les propriétés notées définies par le schéma. Un élément ScoredProperty déclare une propriété qui est intrinsèque à une définition Option. Ces propriétés doivent être comparées lors de l'évaluation de la proximité entre une Option demandée et une Option prise en charge par le dispositif. https://docs.microsoft.com/en-us/windows/win32/printdocs/scoredproperty
