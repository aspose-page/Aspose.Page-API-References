Classe implémentant l'initialiseur de paramètres de ticket d'impression. La classe qui implémente un initialiseur de paramètres PrintTicket commun. La classe de base pour tous les initialiseurs de paramètres définis par le schéma. Définit une valeur pour une instance d'un élément  ParameterDef  . Un élément  ParameterInit  est la cible de la référence faite par un élément  ParameterRef . https://docs.microsoft.com/en-us/windows/win32/printdocs/parameterinit
