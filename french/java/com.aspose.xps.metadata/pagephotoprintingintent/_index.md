Indique une intention de haut niveau pour le pilote afin de remplir les paramètres d'impression photo. Ces paramètres concernent la qualité de sortie attendue qu'un utilisateur peut spécifier lors de l'impression de photos. https://docs.microsoft.com/en-us/windows/win32/printdocs/pagephotoprintingintent
