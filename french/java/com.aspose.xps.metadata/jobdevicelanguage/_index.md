Décrit les langages d'appareil pris en charge pour l'envoi de données du pilote vers le dispositif physique. Cela est souvent appelé « Langage de description de page ». Ce mot‑clé définit quel langage de description de page est pris en charge par le pilote et le dispositif physique. https://docs.microsoft.com/en-us/windows/win32/printdocs/jobdevicelanguage
