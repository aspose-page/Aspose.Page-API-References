La classe qui implémente une référence de paramètre PrintTicket commune. Un élément  ParameterRef  définit une référence à un élément  ParameterInit . Un élément  ScoredProperty  qui contient un élément ParameterRef n'a pas d'élément  Value  explicitement défini. Au lieu de cela, l'élément  ScoredProperty  reçoit sa valeur de l'élément  ParameterInit  référencé par un élément  ParameterRef . https://docs.microsoft.com/en-us/windows/win32/printdocs/parameterref
