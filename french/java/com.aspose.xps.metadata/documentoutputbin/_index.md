Décrit la liste complète des bacs pris en charge pour le dispositif. Permet la spécification du bac de sortie par document. Les mots‑clés JobOutputBin, DocumentOutputBin et PageOutputBin sont mutuellement exclusifs ; un seul doit être spécifié dans un document PrintTicket ou Print Capabilities. https://docs.microsoft.com/en-us/windows/win32/printdocs/documentoutputbin
