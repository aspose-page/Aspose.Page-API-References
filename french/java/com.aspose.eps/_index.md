Le **com.aspose.eps** est un package racine pour toutes les classes qui traitent des fichiers PS/EPS.


## Classes

| Classe | Description |
| --- | --- |
| [FontConstants](../com.aspose.eps/fontconstants) | Cette classe définit un ensemble de constantes pour l'enregistrement des polices. |
| [HatchPaintLibrary](../com.aspose.eps/hatchpaintlibrary) | Convertit les brosses hachurées GDI+ en images adaptées à l'écriture dans XPS et PDF. |
| [LoadOptions](../com.aspose.eps/loadoptions) | Classe de base pour les options de chargement de document. |
| [PageConstants](../com.aspose.eps/pageconstants) | Cette classe définit un ensemble de constantes qui décrivent une page. |
| [PsConverterException](../com.aspose.eps/psconverterexception) | Cette classe contient des informations sur une erreur qui est levée lors de la conversion d'un fichier PS en document PDF. |
| [PsConverterLicenseException](../com.aspose.eps/psconverterlicenseexception) | Cette classe contient des informations sur une erreur liée à la licence du produit Aspose.Page. |
| [PsDocument](../com.aspose.eps/psdocument) | Cette classe encapsule les documents PS/EPS. |
| [PsDocumentException](../com.aspose.eps/psdocumentexception) | Cette classe contient des informations sur une erreur qui est levée pendant la création, la finalisation et l'enregistrement du fichier PS. |
| [PsLoadOptions](../com.aspose.eps/psloadoptions) | Options de chargement du document PS. |
