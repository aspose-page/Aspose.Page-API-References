public enum PsSaveFormat extends Enum<PsSaveFormat>

Cette énumération contient les options disponibles du format d'enregistrement. Il peut être PS ou EPS. EPS est utilisé uniquement pour les documents d'une seule page, tandis qu'un fichier PS peut contenir un nombre quelconque de pages.
## Champs

| Champ | Description |
| --- | --- |
| [EPS](#EPS) | Cette option indique que le document résultant doit être un fichier Encapsulated PostScript (EPS). |
| [PS](#PS) | Cette option indique que le document résultant doit être un fichier PostScript (PS) ou Encapsulated PostScript (EPS). |
