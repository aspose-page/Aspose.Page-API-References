Classe encapsulant les fonctionnalités de l'élément Visual. Cet élément de propriété contient le balisage qui définit le contenu d'une seule tuile de pinceau visuel. La tuile peut être utilisée pour remplir la région géométrique à laquelle le pinceau visuel est appliqué.
