Le package **com.aspose.xps.rendering** fournit des classes de base pour le rendu du document XPS vers d'autres formats.

| Classe | Description |
| --- | --- |
| [BmpSaveOptions](../com.aspose.xps.rendering/bmpsaveoptions) | Classe pour les options d'enregistrement XPS-as-BMP. |
| [ImageSaveOptions](../com.aspose.xps.rendering/imagesaveoptions) | Classe de base pour les options d'enregistrement XPS-as-image. |
| [JpegSaveOptions](../com.aspose.xps.rendering/jpegsaveoptions) | Classe pour les options d'enregistrement XPS-as-JPEG. |
| [PdfEncryptionDetails](../com.aspose.xps.rendering/pdfencryptiondetails) | Contient les détails d'un chiffrement pdf. |
| [PdfSaveOptions](../com.aspose.xps.rendering/pdfsaveoptions) | Classe pour les options d'enregistrement XPS-as-PDF. |
| [PngSaveOptions](../com.aspose.xps.rendering/pngsaveoptions) | Classe pour les options d'enregistrement XPS-as-PNG. |
| [TiffSaveOptions](../com.aspose.xps.rendering/tiffsaveoptions) | Classe pour les options d'enregistrement XPS-as-TIFF. |
