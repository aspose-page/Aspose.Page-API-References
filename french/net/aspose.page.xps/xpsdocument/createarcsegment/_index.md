## XpsDocument.CreateArcSegment method

Crée un nouveau segment d'arc elliptique.

```csharp
public XpsArcSegment CreateArcSegment(PointF point, SizeF size, float rotationAngle, 
    bool isLargeArc, XpsSweepDirection sweepDirection, bool isStroked = true)
```

| Paramètre | Taper | La description |
| --- | --- | --- |
| point | PointF | L'extrémité de l'arc elliptique. |
| size | SizeF | Les rayons x et y de l'arc elliptique sous forme de paire x,y. |
| rotationAngle | Single | Indique comment l'ellipse est tournée par rapport au système de coordonnées actuel. |
| isLargeArc | Boolean | Détermine si l'arc est dessiné avec un balayage de 180 ou plus. |
| sweepDirection | XpsSweepDirection | Direction dans laquelle l'arc est tracé. |
| isStroked | Boolean | Spécifie si le trait de ce segment du chemin est dessiné. |

### Return_Value

Nouveau segment d'arc elliptique.
