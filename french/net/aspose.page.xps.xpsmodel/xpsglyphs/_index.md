## XpsGlyphs class

Class incapsulating Glyphs element features. Cet élément représente une série de texte au format uniforme à partir d'une seule police. Il fournit les informations nécessaires pour un rendu précis et prend en charge les fonctionnalités de recherche et de sélection dans l'affichage des consommateurs.

```csharp
public sealed class XpsGlyphs : XpsContentElement
```
