## XpsGlyphs.BidiLevel property

Renvoie/définit la valeur spécifiant le niveau d'imbrication bidirectionnelle de l'algorithme Unicode. Les valeurs paires impliquent une disposition de gauche à droite, les valeurs impaires impliquent une disposition de droite à gauche. La disposition de droite à gauche place l'origine de l'exécution sur le côté droit du premier glyphe, avec des largeurs d'avance positives (représentant les avancées vers la gauche) plaçant les glyphes suivants à gauche du glyphe précédent.

```csharp
public int BidiLevel { get; set; }
```
