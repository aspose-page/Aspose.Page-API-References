## PdfSaveOptions.OutlineTreeHeight property

Spécifie la hauteur de l'arborescence du document à enregistrer. 0 - l'arborescence ne sera pas convertie, 1 - seuls les éléments de plan de premier niveau seront convertis, et ainsi de suite. La valeur par défaut est 10.

public int OutlineTreeHeight { get; set; }
