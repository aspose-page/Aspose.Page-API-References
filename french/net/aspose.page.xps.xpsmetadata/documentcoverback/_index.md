## DocumentCoverBack class

Décrit la couverture arrière (de fin). Chaque document aura une feuille séparée. La page de garde doit être imprimée sur le[`PageMediaSize`](../pagemediasize/) et[`PageMediaType`](../pagemediatype/) utilisé pour la dernière page du document. La page de garde doit être intégrée dans les options de traitement (telles que[`DocumentDuplex`](../documentduplex/) ,[`DocumentNUp`](../documentnup/) ) comme indiqué par l'option spécifiée. https://docs.microsoft.com/en-us/windows/win32/printdocs/documentcoverback
