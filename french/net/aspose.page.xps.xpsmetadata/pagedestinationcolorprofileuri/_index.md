## PageDestinationColorProfileURI class

Spécifie une référence URI relative à un profil ICC contenu dans un document XPS. Le traitement de cette option dépend du réglage de la fonctionnalité PageDeviceColorSpaceUsage. Tous les éléments utilisant ce profil sont supposés être déjà dans l'espace colorimétrique approprié du périphérique, et ne sera pas géré en couleur dans le pilote ou le périphérique. https://docs.microsoft.com/en-us/windows/win32/printdocs/pagedestinationcolorprofileuri
