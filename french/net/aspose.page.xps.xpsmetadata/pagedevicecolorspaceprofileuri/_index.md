## PageDeviceColorSpaceProfileURI class

Spécifie un URI relatif à la racine du package d'un profil ICC contenu dans un document XPS. Le traitement de cette option dépend du paramètre de la fonctionnalité PageDeviceColorSpaceUsage. Tous les éléments utilisant ce profil sont supposés être déjà dans la couleur de périphérique appropriée espace, et ne sera pas géré en couleur dans le pilote ou le périphérique. https://docs.microsoft.com/en-us/windows/win32/printdocs/pagedevicecolorspaceprofileuri
