## PsDocument.DrawTransparentImage method

Dessine une image transparente transformée. Si l'image n'a pas de canal Alpha, elle sera dessinée comme opaque image

```csharp
public void DrawTransparentImage(Bitmap image, Matrix transform, int transparencyThreshold)
```

| Paramètre | Taper | La description |
| --- | --- | --- |
| image | Bitmap | L'image à dessiner. |
| transform | Matrix | La matrice pour transformer l'image. |
| transparencyThreshold | Int32 | Un seuil qui définit à partir de quelle valeur de transparence le pixel sera interprété comme totalement transparent. Toutes les valeurs inférieures à ce seuil seront interprétées comme totalement opaques. |
