Le **Aspose.Page** est un espace de noms racine pour toutes les classes de la bibliothèque Aspose.Page qui y sont soit directement comme **Appareil** ou indirectement via plusieurs sous-espaces de noms.

## Des classes

| Classer | La description |
| --- | --- |
| [BuildVersionInfo](./buildversioninfo/) | Cette classe fournit des informations sur la version actuelle du produit. |
| [Device](./device/) | Cette classe encapsule le rendu du document sur un périphérique abstrait. Le rendu du document est effectué page par page. |
| [Document](./document/) | Une superclasse pour tous les documents encapsulés. |
| [ExternalFontCache](./externalfontcache/) | Utilisez cette classe pour obtenir l'encapsulation des polices sous une forme acceptée par[`Device`](../aspose.page/device/) . |
| [License](./license/) | Fournit des méthodes pour autoriser le composant. |
| [Margins](./margins/) | Cette classe encapsule les marges supérieure, gauche, inférieure et droite. |
| [Metered](./metered/) | Fournit des méthodes pour définir la clé mesurée. |
| [SaveOptions](./saveoptions/) | Cette classe contient les options nécessaires à la gestion du processus de conversion. |
| [UserProperties](./userproperties/) | Classe de propriété spéciale qui permet de définir des propriétés typées et de les renvoyer . Il permet également de rechercher le raccordement de deux objets de propriété par défaut si cet objet de propriété ne contient pas la propriété. |

| [IStreamable](./istreamable/) | Cette interface est utilisée pour les appareils qui rendent le document dans un flux. |
| [ITrFont](./itrfont/) | Cette interface donne accès aux principaux paramètres de font. |
