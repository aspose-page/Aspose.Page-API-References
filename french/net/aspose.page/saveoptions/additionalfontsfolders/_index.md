## SaveOptions.AdditionalFontsFolders property

Spécifie des dossiers supplémentaires où le convertisseur doit trouver les polices pour le document d'entrée. Le dossier par défaut est le dossier de polices standard où le système d'exploitation trouve les polices pour les besoins internes.

public string[] AdditionalFontsFolders { get; set; }
