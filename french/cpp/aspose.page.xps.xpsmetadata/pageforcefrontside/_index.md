## PageForceFrontSide class


Force la sortie à apparaître sur le devant d’une feuille de support. Pertinent pour les feuilles de support avec des surfaces différentes de chaque côté. Dans les cas où cette fonctionnalité interfère avec les options de traitement (telles que [DocumentDuplex](../documentduplex/)), **[PageForceFrontSide](./)** prend le pas sur la page spécifique à laquelle la fonctionnalité s’applique.

```cpp
class PageForceFrontSide : public Aspose::Page::XPS::XpsMetadata::Feature,
                           public Aspose::Page::XPS::XpsMetadata::IJobPrintTicketItem,
                           public Aspose::Page::XPS::XpsMetadata::IDocumentPrintTicketItem,
```
