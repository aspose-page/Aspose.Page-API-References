## Champs

| Champ | Description |
| --- | --- |
| static [BlankCover](./blankcover/) | Spécifie qu’une feuille de couverture vierge doit être imprimée. |
| static [NoCover](./nocover/) | Spécifie qu’aucune couverture ne sera produite. |
| static [PrintBack](./printback/) | Spécifie que la couverture indiquée par "CoverBackSource" doit être imprimée sur le côté arrière de la feuille de couverture. Si un élément [DocumentCoverBackSource](../../documentcoverbacksource/)[ParameterInit](../../parameterinit/) n'est pas spécifié, cette [Option](../../option/) doit être ignorée. |
| static [PrintBoth](./printboth/) | Spécifie que la couverture indiquée par "CoverBackSource" peut être imprimée sur l'un ou l'autre côté de la feuille de couverture. Si un élément [DocumentCoverBackSource](../../documentcoverbacksource/)[ParameterInit](../../parameterinit/) n'est pas spécifié, cette [Option](../../option/) doit être ignorée. |
| static [PrintFront](./printfront/) | Spécifie que la couverture indiquée par "CoverBackSource" doit être imprimée sur le côté avant de la feuille de couverture. Si un élément [DocumentCoverBackSource](../../documentcoverbacksource/)[ParameterInit](../../parameterinit/) n'est pas spécifié, cette [Option](../../option/) doit être ignorée. |
