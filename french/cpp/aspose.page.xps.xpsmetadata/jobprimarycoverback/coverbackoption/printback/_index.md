## PrintBack field


Spécifie que la couverture indiquée par "CoverBackSource" doit être imprimée sur le côté arrière de la feuille de couverture. Si un élément [JobPrimaryCoverBackSource](../../../jobprimarycoverbacksource/)[ParameterInit](../../../parameterinit/) n'est pas spécifié, cette [Option](../../../option/) doit être ignorée.

```cpp
static System::SharedPtr<JobPrimaryCoverBack::CoverBackOption> Aspose::Page::XPS::XpsMetadata::JobPrimaryCoverBack::CoverBackOption::PrintBack
```
