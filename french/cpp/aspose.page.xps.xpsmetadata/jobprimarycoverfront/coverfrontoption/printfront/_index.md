## PrintFront field


Spécifie que la couverture indiquée par "CoverFrontSource" doit être imprimée sur le côté avant de la feuille de couverture. Si un élément [JobPrimaryCoverFrontSource](../../../jobprimarycoverfrontsource/)[ParameterInit](../../../parameterinit/) n'est pas spécifié, cette [Option](../../../option/) doit être ignorée.

```cpp
static System::SharedPtr<JobPrimaryCoverFront::CoverFrontOption> Aspose::Page::XPS::XpsMetadata::JobPrimaryCoverFront::CoverFrontOption::PrintFront
```
