## ParameterRef class


La classe qui implémente une référence de paramètre [PrintTicket](../printticket/) commune. Un élément **[ParameterRef](./)** définit une référence à un élément [ParameterInit](../parameterinit/). Un élément [ScoredProperty](../scoredproperty/) qui contient un élément [ParameterRef](./) n'a pas d'élément [Value](../value/) explicitement défini. Au lieu de cela, l'élément [ScoredProperty](../scoredproperty/) reçoit sa valeur de l'élément [ParameterInit](../parameterinit/) référencé par un élément **[ParameterRef](./)**. [https://docs.microsoft.com/en-us/windows/win32/printdocs/parameterref](https://docs.microsoft.com/en-us/windows/win32/printdocs/parameterref).

```cpp
class ParameterRef : public Aspose::Page::XPS::XpsMetadata::PrintTicketElement,
                     public Aspose::Page::XPS::XpsMetadata::IPrintTicketItem
```
