## PageDestinationColorProfileURI class


Spécifie une référence URI relative à un profil ICC contenu dans un [XPS](../../aspose.page.xps/)[Document](../../aspose.page/document/). Le traitement de cette option dépend du paramètre de la fonctionnalité [PageDeviceColorSpaceUsage](../pagedevicecolorspaceusage/). Tous les éléments utilisant ce profil sont supposés être déjà dans l'espace colorimétrique du dispositif approprié, et ne seront pas gérés en couleur par le pilote ou le dispositif. [https://docs.microsoft.com/en-us/windows/win32/printdocs/pagedestinationcolorprofileuri](https://docs.microsoft.com/en-us/windows/win32/printdocs/pagedestinationcolorprofileuri).

class PageDestinationColorProfileURI : public Aspose::Page::XPS::XpsMetadata::StringParameterInit,
