## JobDeviceLanguage class


Décrit les langages d'appareil pris en charge pour l'envoi de données du pilote vers le dispositif physique. On l'appelle souvent "Page Description Language". Ce mot-clé définit quel langage de description de page est pris en charge par le pilote et le dispositif physique.
