## PageICMRenderingIntent class


Décrit l'intention de rendu telle que définie par la spécification ICC v2. Cette valeur doit être ignorée si une image ou un élément graphique possède un profil intégré qui spécifie l'intention **Rendering**. [https://docs.microsoft.com/en-us/windows/win32/printdocs/pageicmrenderingintent](https://docs.microsoft.com/en-us/windows/win32/printdocs/pageicmrenderingintent).
