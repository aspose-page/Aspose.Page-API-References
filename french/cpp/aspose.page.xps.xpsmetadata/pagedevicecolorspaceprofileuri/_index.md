## PageDeviceColorSpaceProfileURI class


Spécifie un URI relatif depuis la racine du package vers un profil ICC contenu dans un [XPS](../../aspose.page.xps/)[Document](../../aspose.page/document/). Le traitement de cette option dépend du paramètre de la fonctionnalité [PageDeviceColorSpaceUsage](../pagedevicecolorspaceusage/). Tous les éléments utilisant ce profil sont supposés être déjà dans l'espace colorimétrique du dispositif approprié, et ne seront pas gérés couleur dans le pilote ou le dispositif. [https://docs.microsoft.com/en-us/windows/win32/printdocs/pagedevicecolorspaceprofileuri](https://docs.microsoft.com/en-us/windows/win32/printdocs/pagedevicecolorspaceprofileuri).
