## MatchToDefault field


Si l'appareil détermine que le profil spécifié par le paramètre [PageDeviceColorSpaceProfileURI](../../../pagedevicecolorspaceprofileuri/) peut être utilisé comme profil d'espace couleur de l'appareil, tous les éléments utilisant le même profil sont considérés comme déjà spécifiés dans l'espace couleur de l'appareil. Tous les autres éléments MUST utiliser le profil spécifié pour cet élément. Si le profil ne peut pas être utilisé comme profil d'espace couleur de l'appareil, les éléments utilisant le profil MUST être gérés colorimétriquement comme tout autre élément utilisant le profil couleur.
