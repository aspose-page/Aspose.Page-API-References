| Champ | Description |

| static [BlankCover](./blankcover/) | Spécifie qu’une feuille de couverture vierge doit être imprimée. |
| static [NoCover](./nocover/) | Spécifie qu’aucune couverture ne sera produite. |

| static [PrintFront](./printfront/) | Spécifie que la couverture indiquée par \"CoverFrontSource\" doit être imprimée sur le recto de la feuille de couverture.
