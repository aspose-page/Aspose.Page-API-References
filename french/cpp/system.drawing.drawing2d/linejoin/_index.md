## LineJoin enum


Spécifie comment les lignes ou courbes consécutives sont jointes.

```cpp
enum class LineJoin
```

### Valeurs

| Nom | Valeur | Description |
| --- | --- | --- |
| Miter | 0 | Joint en onglet qui produit un coin aigu ou tronqué selon que la longueur de l'onglet dépasse la limite d'onglet. |
| Bevel | 1 | Joint biseauté qui produit un coin diagonal. |
| Arrondir | 2 | Joint arrondi qui produit un arc circulaire lisse entre les lignes. |
| MiterClipped | 3 | Joint en onglet qui produit un coin aigu ou biseauté selon que la longueur de l'onglet dépasse la limite d'onglet. |
