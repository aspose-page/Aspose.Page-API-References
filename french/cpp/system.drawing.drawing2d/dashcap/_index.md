## DashCap enum


Spécifie le type d'embout utilisé aux deux extrémités d'un tiret dans une ligne pointillée.

```cpp
enum class DashCap
```

### Valeurs

| Nom | Valeur | Description |
| --- | --- | --- |
| Plat | 0 | Un bouchon carré qui rend carrées les deux extrémités d’un tiret. |
| Arrondir | 2 | Un bouchon circulaire qui arrondit les deux extrémités d’un tiret. |
| Triangle | 3 | Un bouchon triangulaire qui pointe les deux extrémités d’un tiret. |
