## X500DistinguishedNameFlags enum


Règles de formatage du nom distinctif du certificat X509.

```cpp
enum class X500DistinguishedNameFlags
```

### Valeurs

| Nom | Valeur | Description |
| --- | --- | --- |
| None | 0 | Aucune caractéristique spéciale. |
| Inversé | 1 | Le nom est réservé. |
| UseSemicolons | 16 | Utilisez des points-virgules. |
| DoNotUsePlusSign | 32 | Le nom n\'utilise pas le signe plus. |
| DoNotUseQuotes | 64 | Désactive les guillemets dans le nom. |
| UseCommas | 128 | Active l\'utilisation de virgules. |
| UseNewLines | 256 | Active l\'utilisation de nouvelles lignes. |
| UseUTF8Encoding | 4096 | Passe de l\'utilisation d\'Unicode à l\'utilisation du codage UTF-8. |
| UseT61Encoding | 8192 | Passe au codage T61. |
| ForceUTF8Encoding | 16384 | Force l\'utilisation de UTF-8 lors du codage de clés X500 spécifiques. |
