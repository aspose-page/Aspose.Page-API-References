## Timer::set_Enabled method


Démarre ou arrête le minuteur. Démarrer le minuteur ne redémarre pas le comptage du temps si le minuteur est déjà en cours d'exécution.

```cpp
void System::Timers::Timer::set_Enabled(bool value)
```


| Paramètre | Type | Description |
| --- | --- | --- |
| value | bool | true pour démarrer le minuteur, false pour l'arrêter. |
