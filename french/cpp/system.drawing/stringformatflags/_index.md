## StringFormatFlags enum


Spécifie les informations d'affichage et de mise en page pour les chaînes de texte.

```cpp
enum class StringFormatFlags
```

### Valeurs

| Nom | Valeur | Description |
| --- | --- | --- |
| DirectionRightToLeft | 1 | Direction du texte de droite à gauche. |
| DirectionVertical | 2 | [Text](../../system.drawing.text/) est aligné verticalement. |
| FitBlackBox | 4 | Les parties des caractères peuvent dépasser la boîte de mise en page. |
| DisplayFormatControl | 32 | Les caractères de contrôle sont affichés. |
| NoFontFallback | 1024 | [Font](../font/) de secours est désactivé. |
| MeasureTrailingSpaces | 2048 | Insère l'espace de fin à la fin de chaque ligne. |
| NoWrap | 4096 | [Text](../../system.drawing.text/) le renvoi à la ligne est désactivé. |
| LineLimit | 8192 | Les lignes entières sont disposées dans le rectangle de formatage. |
| NoClip | 16384 | Les parties dépassantes des glyphes sont autorisées à s'afficher. |
