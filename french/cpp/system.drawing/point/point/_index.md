## Point::Point(int) constructor


| Paramètre | Type | Description |
| --- | --- | --- |
| dw | int | La valeur entière 32 bits dont les 16 bits supérieurs spécifient la valeur de la coordonnée X et les 16 bits inférieurs spécifient la valeur de la coordonnée Y de l'objet en cours de création. |
