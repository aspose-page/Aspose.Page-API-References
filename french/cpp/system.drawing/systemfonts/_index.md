## SystemFonts class


Fournit un ensemble d'objets [Font](../font/) pré‑créés qui représentent les polices utilisées pour afficher du texte dans les éléments d'affichage [Windows](../../system.windows/). Il s'agit d'un type statique sans services d'instance. Vous ne devez jamais créer d'instances de celui‑ci par quelque moyen que ce soit.

```cpp
class SystemFonts : public System::Object
```
