## RotateFlipType enum


Spécifie le type d'opération de rotation et/ou de retournement.

```cpp
enum class RotateFlipType
```

### Valeurs

| Nom | Valeur | Description |
| --- | --- | --- |
| RotateNoneFlipNone | 0 | Aucune rotation et aucun retournement. |
| Rotate90FlipNone | 1 | Rotation de 90 degrés dans le sens des aiguilles d'une montre et aucun retournement. |
| Rotate180FlipNone | 2 | Rotation de 180 degrés dans le sens des aiguilles d'une montre et aucun retournement. |
| Rotate270FlipNone | 3 | Rotation de 270 degrés dans le sens des aiguilles d'une montre et aucun retournement. |
| RotateNoneFlipX | 4 | Retournement horizontal et aucune rotation. |
| Rotate90FlipX | 5 | Rotation de 90 degrés dans le sens des aiguilles d'une montre et retournement horizontal. |
| Rotate180FlipX | 6 | Rotation de 180 degrés dans le sens des aiguilles d'une montre et retournement horizontal. |
| Rotate270FlipX | 7 | Rotation de 270 degrés dans le sens des aiguilles d'une montre et retournement horizontal. |
| RotateNoneFlipY | n/a | Retournement vertical et aucune rotation. |
| Rotate90FlipY | n/a | Rotation de 90 degrés dans le sens des aiguilles d'une montre et retournement vertical. |
| Rotate180FlipY | n/a | Rotation de 180 degrés dans le sens des aiguilles d'une montre et retournement vertical. |
| Rotate270FlipY | n/a | Faire pivoter de 270 degrés dans le sens des aiguilles d'une montre et retourner verticalement. |
| RotateNoneFlipXY | n/a | Retourner horizontalement et verticalement sans rotation. |
| Rotate90FlipXY | n/a | Faire pivoter de 90 degrés dans le sens des aiguilles d'une montre et retourner horizontalement et verticalement. |
| Rotate180FlipXY | n/a | Faire pivoter de 180 degrés dans le sens des aiguilles d'une montre et retourner horizontalement et verticalement. |
| Rotate270FlipXY | n/a | Faire pivoter de 270 degrés dans le sens des aiguilles d'une montre et retourner horizontalement et verticalement. |
