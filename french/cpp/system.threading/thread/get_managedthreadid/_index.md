## Thread::get_ManagedThreadId method


Obtient l’identifiant du thread. Il peut être récupéré auprès du système d’exploitation, mais si l’identifiant du thread du système dépasse les limites d’un int, les identifiants des threads peuvent se chevaucher.

```cpp
int System::Threading::Thread::get_ManagedThreadId() const
```


### ReturnValue

Identifiant de thread pseudo‑unique.
