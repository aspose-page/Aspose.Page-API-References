## CancellationTokenRegistration::Dispose method


Libère l'enregistrement et supprime le rappel du [CancellationTokenSource](../../cancellationtokensource/) associé. Après avoir appelé cette méthode, le rappel enregistré ne sera plus invoqué lorsque le [CancellationTokenSource](../../cancellationtokensource/) associé est annulé.

```cpp
void System::Threading::CancellationTokenRegistration::Dispose()
```

## Remarques



Il est sûr d'appeler cette méthode plusieurs fois - les appels ultérieurs n'auront aucun effet.
