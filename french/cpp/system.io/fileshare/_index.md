## FileShare enum


Spécifie le type d'accès que d'autres objets [FileStream](../filestream/) peuvent avoir à un fichier ouvert.

```cpp
enum class FileShare
```

### Valeurs

| Nom | Valeur | Description |
| --- | --- | --- |
| None | 0 | Aucun accès. |
| Lire | 1 | Accès en lecture seule. |
| Write | 2 | Accès en écriture seule. |
| ReadWrite | 3 | Accès en lecture et écriture. |
| Supprimer | 4 | Le fichier peut être supprimé. |
| Héritable | 16 | Rend le handle de fichier héritable par les processus enfants. |
