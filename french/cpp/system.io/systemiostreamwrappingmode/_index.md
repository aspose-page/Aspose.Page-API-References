## SystemIOStreamWrappingMode enum


Spécifie le mode des opérations d'E/S que les wrappers exécuteront sur les flux de type [System::IO::Stream](../stream/).

```cpp
enum class SystemIOStreamWrappingMode
```

### Valeurs

| Nom | Valeur | Description |
| --- | --- | --- |
| Binary | 0 | Un mode qui permet aux opérations d'entrée d'encoder les octets du flux en données de type char_type et de décoder les données de type char_type en octets du flux pour les opérations de sortie. |
| Conversion | 1 | Un mode qui permet aux opérations d'entrée de convertir les octets du flux du type uint8_t vers le type char_type et inversement pour les opérations de sortie. |
