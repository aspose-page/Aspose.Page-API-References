## Directory::Delete method


Supprime le fichier ou le répertoire spécifié. Ne lève pas d'exception.

```cpp
static void System::IO::Directory::Delete(const String &path, bool recursive=false)
```


| Paramètre | Type | Description |
| --- | --- | --- |
| chemin | const String\& | Le chemin du répertoire ou du fichier à supprimer |
| récursif | bool | Si **path** spécifie un répertoire non vide, alors **recursive** indique si tout le contenu du répertoire doit être supprimé de manière récursive ; si le répertoire spécifié par **path** n'est pas vide et que **recursive** vaut 'false', l'opération échoue |
