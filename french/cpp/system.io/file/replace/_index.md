## File::Replace method


Remplace le contenu d'un fichier par un autre et crée une sauvegarde du fichier remplacé.

```cpp
static void System::IO::File::Replace(const String &sourceFileName, const String &destinationFileName, const String &destinationBackupFileName, bool ignoreMetadataErrors=true)
```


| Paramètre | Type | Description |
| --- | --- | --- |
| sourceFileName | const String\& | Un nom du fichier avec lequel remplacer |
| destinationFileName | const String\& | Le nom du fichier à remplacer |
| destinationBackupFileName | const String\& | Le nom du fichier de sauvegarde |
| ignoreMetadataErrors | bool | Spécifie si les erreurs de fusion du fichier remplacé vers le fichier de remplacement doivent être ignorées (true) ou non (false) |
