## ColorMatrixFlag enum


Spécifie les types d'images et de couleurs qui seront affectés par les paramètres d'ajustement des couleurs et du niveau de gris d'un objet [ImageAttributes](../imageattributes/).

```cpp
enum class ColorMatrixFlag
```

### Valeurs

| Nom | Valeur | Description |
| --- | --- | --- |
| Default | 0 | Les couleurs et les valeurs de niveau de gris sont toutes deux ajustées en couleur. |
| SkipGrays | 1 | Les valeurs de niveau de gris ne sont pas ajustées en couleur. |
| AltGrays | 2 | Seules les valeurs de niveau de gris sont ajustées en couleur. |
