## EmfType enum


Spécifie les types d'enregistrements placés dans un fichier EMF. Les objets de cette classe ne doivent être alloués qu'en utilisant la fonction [System::MakeObject()](../system/makeobject/). Ne créez jamais d'instance de ce type sur la pile ou en utilisant l'opérateur new, car cela entraînera des erreurs d'exécution et/ou des échecs d'assertion. Enveloppez toujours cette classe dans un pointeur [System::SmartPtr](../system/smartptr/) et utilisez ce pointeur pour le transmettre aux fonctions en tant qu'argument.

```cpp
enum class EmfType
```

### Valeurs

| Nom | Valeur | Description |
| --- | --- | --- |
| EmfOnly | n/a | [Windows](../../system.windows/) métafile amélioré. Contient des commandes GDI. Les métafichiers de ce type sont appelés un fichier EMF. |
| EmfPlusOnly | n/a | [Windows](../../system.windows/) métafile amélioré plus. Contient des commandes GDI+. Les métafichiers de ce type sont appelés un fichier EMF+. |
| EmfPlusDual | n/a | Double [Windows](../../system.windows/) métafile amélioré. Contient des commandes GDI et GDI+ équivalentes. Les métafichiers de ce type sont appelés un fichier EMF+. |
