## AutoSizeMode enum


Définit le fonctionnement de l'autodimensionnement.

```cpp
enum class AutoSizeMode
```

### Valeurs

| Nom | Valeur | Description |
| --- | --- | --- |
| GrowAndShrink | 0 | [Control](../control/) croît et rétrécit selon les besoins. Il ne peut pas être redimensionné manuellement. |
| GrowOnly | 1 | [Control](../control/) croît selon les besoins mais ne peut pas rétrécir en dessous de la taille minimale. Il peut être redimensionné manuellement. |
