## TimeSpanStyles enum


Spécifie les options de formatage qui personnalisent l'analyse de chaîne pour les méthodes qui convertissent une représentation sous forme de chaîne d'un intervalle de temps en objet Timespan.

```cpp
enum class TimeSpanStyles : int32_t
```

### Valeurs

| Nom | Valeur | Description |
| --- | --- | --- |
| None | 0 | L'intervalle de temps représenté par la chaîne inptut est interprété comme un intervalle de temps négatif uniquement si un signe négatif est présent. |
| AssumeNegative | 1 | L'intervalle de temps représenté par la chaîne inptut est toujours interprété comme un intervalle de temps négatif. |
