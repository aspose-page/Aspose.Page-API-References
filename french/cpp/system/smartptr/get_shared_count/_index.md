## SmartPtr::get_shared_count method


Obtient le nombre de pointeurs partagés existants vers l'objet référencé, y compris le pointeur actuel. Affirme que le pointeur actuel est en mode partagé.

```cpp
int System::SmartPtr<T>::get_shared_count() const
```


### ReturnValue

Nombre de pointeurs partagés existants vers l'objet référencé, le cas échéant. Si le pointeur est nul, renvoie 0.
