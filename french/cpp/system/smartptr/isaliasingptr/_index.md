## SmartPtr::IsAliasingPtr method


Vérifie si le pointeur pointe vers un autre objet que celui possédé (créé par un constructeur d'alias).

```cpp
bool System::SmartPtr<T>::IsAliasingPtr() const
```


### ReturnValue

Vrai si le pointeur a une différence entre les objets pointé et possédé, faux sinon.
