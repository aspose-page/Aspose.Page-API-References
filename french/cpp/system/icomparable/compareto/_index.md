## IComparable::CompareTo method


Compare l'objet actuel avec l'objet spécifié.

```cpp
virtual int System::IComparable<T>::CompareTo(T other)=0
```


| Paramètre | Type | Description |
| --- | --- | --- |
| autre | T | L'objet à comparer avec l'objet actuel |

### ReturnValue

Un nombre négatif si l'objet actuel précède l'objet spécifié dans l'ordre de tri ; 0 si les objets comparés sont égaux ; un nombre positif si l'objet actuel suit l'objet spécifié dans l'ordre de tri
