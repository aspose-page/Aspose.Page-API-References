## Char::TryParse method


Tente de convertir une chaîne composée d'un seul caractère en caractère UTF-16. La fonction réussit uniquement lorsque la chaîne d'entrée n'est pas nulle et a une longueur d'exactement un caractère.

```cpp
static bool System::Char::TryParse(const System::String &s, char_t &result)
```


| Paramètre | Type | Description |
| --- | --- | --- |
| s | const System::String\& | [String](../../string/) à convertir |
| résultat | char_t\& | La variable de sortie qui contiendra le résultat de la conversion si la conversion réussit |

### ReturnValue

Vrai si la conversion a réussi, sinon - faux
