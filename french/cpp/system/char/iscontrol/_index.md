## Char::IsControl(char_t) method


Détermine si le caractère spécifié est classé comme un caractère de contrôle Unicode.

```cpp
static bool System::Char::IsControl(char_t c)
```


| Paramètre | Type | Description |
| --- | --- | --- |
| c | char_t | Le caractère à tester |

### ReturnValue

Vrai si le caractère spécifié est un caractère de contrôle Unicode, sinon - false
