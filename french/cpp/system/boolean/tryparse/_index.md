## Boolean::TryParse method


Convertit la chaîne spécifiée en une valeur de type bool.

```cpp
static bool System::Boolean::TryParse(const String &value, bool &result)
```


| Paramètre | Type | Description |
| --- | --- | --- |
| value | const String\& | La chaîne à convertir. |
| résultat | bool\& | La référence à une variable bool où le résultat de la conversion est placé ; le résultat est vrai si la chaîne spécifiée est égale à "True" et faux si la chaîne spécifiée est égale à "False". |

### ReturnValue

Vrai si la chaîne spécifiée est égale soit à "True" soit à "False", sinon - faux.
