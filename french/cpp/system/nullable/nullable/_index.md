## Nullable::Nullable(const Nullable\<T1\>\&) constructor


Construit une instance qui représente une valeur représentée par l'objet [Nullable](../) spécifié. L'objet nullable spécifié peut représenter une valeur d'un type différent de celui du type sous‑jacent de l'instance construite, auquel cas la valeur représentée est convertie en une valeur de type T.

```cpp
template<typename T1> System::Nullable<T>::Nullable(const Nullable<T1> &value)
```
