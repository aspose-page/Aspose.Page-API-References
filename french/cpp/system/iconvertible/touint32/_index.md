## IConvertible::ToUInt32 method


Convertit la valeur de cette instance en un entier 32 bits équivalent uint32_teger en utilisant les informations de formatage spécifiques à la culture spécifiées.

```cpp
virtual uint32_t System::IConvertible::ToUInt32(System::SharedPtr<System::IFormatProvider> provider)=0
```


| Paramètre | Type | Description |
| --- | --- | --- |
| provider | System::SharedPtr\<System::IFormatProvider\> | Une implémentation d'interface [System::IFormatProvider](../../iformatprovider/) qui fournit des informations de formatage spécifiques à la culture. |

### ReturnValue

Une valeur entière non signée de 32 bits équivalente à la valeur de cette instance.
