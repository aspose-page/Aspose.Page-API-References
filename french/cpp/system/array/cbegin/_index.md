## Array::cbegin method


Retourne un itérateur vers le premier élément const-qualifié du conteneur. Si le conteneur est vide, l'itérateur retourné sera égal à [cend()](../cend/).

```cpp
const_iterator System::Array<T>::cbegin() const noexcept
```


### ReturnValue

Un itérateur pointant vers le premier élément const-qualifié du conteneur.
