## Array::Resize method


Modifie la taille du tableau spécifié à la valeur spécifiée ou crée un nouveau tableau avec la taille spécifiée.

```cpp
template<typename Type> static void System::Array<T>::Resize(ArrayPtr<Type> &arr, int new_size)
```


| Paramètre | Type | Description |
| --- | --- | --- |
| arr | ArrayPtr\<Type\>\& | [Array](../) à redimensionner. Si **arr** est un pointeur nul, le nouveau tableau sera créé |
| new_size | int | La nouvelle taille du tableau, ou la taille du nouveau tableau de **arr** est nulle |
