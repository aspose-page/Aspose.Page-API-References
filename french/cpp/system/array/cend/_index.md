## Array::cend method


Renvoie un itérateur vers l'élément suivant le dernier élément du conteneur. Cet élément sert de placeholder ; tenter d'y accéder entraîne un comportement indéfini.

```cpp
const_iterator System::Array<T>::cend() const noexcept
```


### ReturnValue

Un itérateur pointant vers l'élément théorique const-qualifié placé après le dernier élément du conteneur.
