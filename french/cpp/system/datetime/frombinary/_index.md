## DateTime::FromBinary method


Désérialise la valeur de date et d'heure à partir de l'entier non signé de 64 bits spécifié et définit la nouvelle instance de la classe [DateTime](../) à cette valeur.

```cpp
static DateTime System::DateTime::FromBinary(int64_t value)
```


| Paramètre | Type | Description |
| --- | --- | --- |
| value | int64_t | L'entier de 64 bits à partir duquel désérialiser la valeur de date et d'heure |

### ReturnValue

Une nouvelle instance de la classe [DateTime](../) représentant la valeur de date et d'heure désérialisée à partir de l'entier non signé de 64 bits spécifié.
