## DateTime::CompareTo method


Compare deux valeurs de date et d'heure représentées par l'objet actuel et l'instance spécifiée de la classe [DateTime](../) et renvoie la valeur indiquant les positions relatives des valeurs sur la ligne du temps.

```cpp
int System::DateTime::CompareTo(DateTime value) const
```


| Paramètre | Type | Description |
| --- | --- | --- |
| value | DateTime | Une instance de la classe [DateTime](../) avec laquelle comparer l'objet actuel |

### ReturnValue

Une valeur inférieure à 0 si l'objet actuel représente une valeur antérieure à celle représentée par **value** ; 0 si les valeurs représentées par les deux objets sont identiques ; une valeur supérieure à 0 si la valeur représentée par l'objet actuel est postérieure à celle représentée par **value**
