## DateTime::operator-(DateTime) const method


Retourne une instance de la classe [TimeSpan](../../timespan/) qui représente l'intervalle de temps entre les valeurs de date et d'heure représentées par l'objet actuel et les objets spécifiés.



| Paramètre | Type | Description |
| --- | --- | --- |
| value | DateTime | Une instance de la classe [DateTime](../) qui marque une extrémité de l’intervalle à calculer |

### ReturnValue
