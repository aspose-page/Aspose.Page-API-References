## GC class


Représente une collecte de déchets émulée qui agit davantage comme un stub et ne fait effectivement rien. Il s’agit d’un type statique sans services d’instance. Vous ne devez jamais créer d’instances de celui‑ci, quels que soient les moyens.

```cpp
class GC : public System::Object
```
