## MulticastDelegate< ReturnType(ArgumentTypes...)>::operator() method


Invoque tous les délégués actuellement présents dans la collection de délégués. Les délégués sont invoqués dans le même ordre que celui dans lequel ils ont été ajoutés à la collection. L'opérateur se bloque pendant l'exécution des délégués.

```cpp
ReturnType System::MulticastDelegate<ReturnType(ArgumentTypes...)>::operator()(ArgumentTypes... args) const
```


| Paramètre | Type | Description |
| --- | --- | --- |
| args | ArgumentTypes... | Arguments à passer aux délégués à invoquer |

### ReturnValue

Valeur de retour du dernier délégué invoqué
