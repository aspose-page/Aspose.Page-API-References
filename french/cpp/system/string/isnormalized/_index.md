## String::IsNormalized method


Vérifie si la chaîne Unicode est normalisée en utilisant la forme de normalisation spécifiée.

```cpp
bool System::String::IsNormalized(System::Text::NormalizationForm form) const
```


| Paramètre | Type | Description |
| --- | --- | --- |
| form | System::Text::NormalizationForm | Forme de normalisation, voir [System::Text::NormalizationForm](../../../system.text/normalizationform/) pour plus de détails. |

### ReturnValue
