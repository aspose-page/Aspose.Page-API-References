## Decimal::Truncate method


Renvoie l'objet [Decimal](../) représentant une valeur dont la partie entière est égale à celle de la valeur représentée par l'objet [Decimal](../) spécifié, avec tous les chiffres fractionnaires supprimés.

```cpp
static Decimal System::Decimal::Truncate(const Decimal &d)
```


| Paramètre | Type | Description |
| --- | --- | --- |
| d | const Decimal\& | Un nombre décimal |

### ReturnValue

Un objet [Decimal](../) représentant une valeur décimale dont la partie entière est égale à celle de la valeur spécifiée et dont les chiffres fractionnaires sont tous à 0.
