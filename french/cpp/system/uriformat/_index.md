## UriFormat enum


Spécifie comment l'URI est échappé.

```cpp
enum class UriFormat
```

### Valeurs

| Nom | Valeur | Description |
| --- | --- | --- |
| UriEscaped | 1 | Échappement selon la RFC-2396. |
| Non échappé | 2 | Pas d'échappement. |
| SafeUnescaped | 3 | Seuls les caractères ayant une signification réservée dans l'URI sont échappés. |
