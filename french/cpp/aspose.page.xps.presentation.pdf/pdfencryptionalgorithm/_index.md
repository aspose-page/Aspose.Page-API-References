## PdfEncryptionAlgorithm enum


Énumération du mode de chiffrement. Décrit l'utilisation de l'algorithme et de la longueur de clé. Cette énumération est étendue afin de pouvoir augmenter davantage les fonctionnalités. Cette énumération implémente le modèle "Base-to-Core".

```cpp
enum class PdfEncryptionAlgorithm
```

### Valeurs

| Nom | Valeur | Description |
| --- | --- | --- |
| RC4_40 | 0 | Algorithme, avec une longueur de clé de chiffrement RC4 de 40 bits ;. |
| RC4_128 | 1 | Algorithme, avec une longueur de clé de chiffrement RC4 de 128 bits et un jeu d'autorisations avancées ;. |
