## PdfSaveOptions::get_TextCompression method


Spécifie à quel niveau du plan du document afficher les objets [ApsBookmark](../). 0 - non affiché. 1 au premier niveau, etc. La valeur par défaut est 0.

```cpp
PdfTextCompression Aspose::Page::XPS::Presentation::Pdf::PdfSaveOptions::get_TextCompression() const
```

## Remarques


Spécifie le type de compression à utiliser pour tous les flux de contenu, à l'exception des images. La valeur par défaut est [PdfTextCompression::Flate](../../pdftextcompression/).
