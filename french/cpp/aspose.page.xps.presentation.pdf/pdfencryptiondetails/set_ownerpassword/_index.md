## Remarques


Le mot de passe propriétaire.

L'ouverture du document avec le mot de passe propriétaire correct (en supposant qu'il ne soit pas identique au mot de passe utilisateur) permet un accès complet (propriétaire) au document. Cet accès illimité comprend la possibilité de modifier les mots de passe du document ainsi que les autorisations d'accès.
