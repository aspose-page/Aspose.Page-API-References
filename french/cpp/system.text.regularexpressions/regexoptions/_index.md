## RegexOptions enum


[Regex](../regex/) options.

```cpp
enum class RegexOptions
```

### Valeurs

| Nom | Valeur | Description |
| --- | --- | --- |
| None | 0 | Comportement par défaut. |
| Compiled | 1 | Compile regex pour les performances. Toujours fait par défaut. |
| CultureInvariant | 2 | Utilise la correspondance indépendante de la culture. Ignoré. |
| ECMAScript | 4 | Utilisez la syntaxe ECMAScript. Ignoré. |
| ExplicitCapture | 8 | Capture explicite uniquement. Ignoré. |
| IgnoreCase | 16 | Ignorer la casse lors de la correspondance. |
| IgnorePatternWhitespace | 32 | Ignorer les espaces blancs dans le motif. Non pris en charge. |
| Multiline | 64 | Traiter '^' et '$' comme le début et la fin de la ligne, pas de la chaîne entière. |
| RightToLeft | 128 | Correspondance de droite à gauche. Non pris en charge. |
| Singleline | 256 | Fait que '.' corresponde à n'importe quel caractère sans exceptions (normalement, les caractères de nouvelle ligne ne sont pas correspondus). |
