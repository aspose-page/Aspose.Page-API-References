## MatchCollection class


Collection de correspondances obtenues en appliquant de manière répétée une expression régulière à une chaîne. Les objets de cette classe ne doivent être alloués qu'en utilisant la fonction [System::MakeObject()](../../system/makeobject/). Ne créez jamais d'instance de ce type sur la pile ou en utilisant l'opérateur new, car cela entraînera des erreurs d'exécution et/ou des échecs d'assertion. Enveloppez toujours cette classe dans le pointeur [System::SmartPtr](../../system/smartptr/) et utilisez ce pointeur pour le transmettre aux fonctions en argument.

```cpp
class MatchCollection : public System::Collections::Generic::List<MatchPtr>
```
