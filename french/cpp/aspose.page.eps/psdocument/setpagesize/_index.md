## PsDocument::SetPageSize method


Définit la taille de la page. Pour créer des pages de tailles différentes dans un même document, utilisez la méthode [SetPageDevice](../setpagedevice/) immédiatement après cette méthode.

```cpp
void Aspose::Page::EPS::PsDocument::SetPageSize(float width, float height)
```


| Paramètre | Type | Description |
| --- | --- | --- |
| width | float | La largeur de la page dans le fichier PostScript résultant. |
| height | float | La hauteur de la page dans le fichier PostScript résultant. |
