## PsSaveFormat enum


Cette énumération contient les options disponibles du format d'enregistrement. Elle peut être PS ou [EPS](../../aspose.page.eps/). [EPS](../../aspose.page.eps/) est utilisé uniquement pour les documents d'une seule page, tandis que le fichier PS peut contenir un nombre quelconque de pages.

```cpp
enum class PsSaveFormat
```

### Valeurs

| Nom | Valeur | Description |
| --- | --- | --- |
| PS | 0 | Cette option indique que le document résultant doit être un fichier PostScript (PS) ou Encapsulated PostScript ([EPS](../../aspose.page.eps/)). |
| EPS | 1 | Cette option indique que le document résultant doit être un fichier Encapsulated PostScript ([EPS](../../aspose.page.eps/)). Il est utilisé uniquement pour les documents d'une seule page ;. |
