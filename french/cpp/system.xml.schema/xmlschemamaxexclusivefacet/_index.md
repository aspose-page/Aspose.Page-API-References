## XmlSchemaMaxExclusiveFacet class


Représente l'élément **maxExclusive** du XML [Schéma](../) tel que spécifié par le Consortium du World Wide [Web](../../system.web/) (W3C). Cette classe peut être utilisée pour spécifier une restriction sur la valeur maximale d'un élément **simpleType**. La valeur de l'élément doit être inférieure à la valeur de l'élément **maxExclusive**.

```cpp
class XmlSchemaMaxExclusiveFacet : public System::Xml::Schema::XmlSchemaFacet
```
