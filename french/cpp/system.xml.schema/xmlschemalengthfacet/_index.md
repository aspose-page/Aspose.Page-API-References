## XmlSchemaLengthFacet class


Représente le facet **length** du XML [Schema](../) tel que spécifié par le World Wide [Web](../../system.web/) Consortium (W3C). Cette classe peut être utilisée pour spécifier une restriction sur la longueur d'un élément **simpleType** du type de données.

```cpp
class XmlSchemaLengthFacet : public System::Xml::Schema::XmlSchemaNumericFacet
```
