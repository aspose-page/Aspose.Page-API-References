## XmlSchemaMaxLengthFacet class


Représente l'élément **maxLength** du [Schéma](../) XML tel que spécifié par le [Web](../../system.web/) Consortium (W3C). Cette classe peut être utilisée pour spécifier une restriction sur la longueur maximale de la valeur de données d'un élément **simpleType**. La longueur doit être inférieure à la valeur de l'élément **maxLength**.

```cpp
class XmlSchemaMaxLengthFacet : public System::Xml::Schema::XmlSchemaNumericFacet
```
