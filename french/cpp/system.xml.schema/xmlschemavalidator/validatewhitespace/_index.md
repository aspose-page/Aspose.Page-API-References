## XmlSchemaValidator::ValidateWhitespace(const String\&) method


Valide si l'espace blanc dans la **string** spécifiée est autorisé dans le contexte de l'élément actuel, et accumule l'espace blanc pour la validation si l'élément actuel a un contenu simple.

```cpp
void System::Xml::Schema::XmlSchemaValidator::ValidateWhitespace(const String &elementValue)
```


| Paramètre | Type | Description |
| --- | --- | --- |
| elementValue | const String\& | Une **string** d'espace blanc à valider dans le contexte actuel de l'élément. |
