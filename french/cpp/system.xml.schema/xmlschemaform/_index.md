## XmlSchemaForm enum


Indique si les attributs ou les éléments doivent être qualifiés avec un préfixe d'espace de noms.

```cpp
enum class XmlSchemaForm
```

### Valeurs

| Nom | Valeur | Description |
| --- | --- | --- |
| None | 0 | La forme des éléments et des attributs n'est pas spécifiée dans le schéma. |
| Qualifié | 1 | Les éléments et les attributs doivent être qualifiés avec un préfixe d'espace de noms. |
| Non qualifié | 2 | Les éléments et les attributs ne sont pas obligés d'être qualifiés avec un préfixe d'espace de noms. |
