## XmlSchemaComplexContent::get_IsMixed method


Renvoie les informations qui déterminent si le type possède un modèle de contenu mixte.

```cpp
bool System::Xml::Schema::XmlSchemaComplexContent::get_IsMixed()
```


### ReturnValue

Si cette valeur est **true**, les données de caractères sont autorisées à apparaître entre les éléments enfants du type complexe (modèle de contenu mixte). La valeur par défaut est **false**. Optionnel.
