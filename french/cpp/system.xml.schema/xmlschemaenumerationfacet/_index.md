## XmlSchemaEnumerationFacet class


Représente la facette **enumeration** du XML [Schema](../) telle que spécifiée par le World Wide [Web](../../system.web/) Consortium (W3C). Cette classe spécifie une liste de valeurs valides pour un élément simpleType. La déclaration est contenue dans une déclaration **restriction**.
