## XmlSchemaContentProcessing enum


Fournit des informations sur le mode de validation des remplacements d'éléments **any** et **anyAttribute**.

```cpp
enum class XmlSchemaContentProcessing
```

### Valeurs

| Nom | Valeur | Description |
| --- | --- | --- |
| None | 0 | Les éléments de document ne sont pas validés. |
| Skip | 1 | Les éléments de document doivent être constitués de XML bien formé et ne sont pas validés par le schéma. |
| Souple | 2 | Si le schéma associé est trouvé, les éléments de document seront validés. Aucune erreur ne sera levée sinon. |
| Strict | 3 | Le processeur de schéma doit trouver un schéma associé à l'espace de noms indiqué pour valider les éléments de document. |
