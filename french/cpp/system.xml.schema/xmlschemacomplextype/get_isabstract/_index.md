## XmlSchemaComplexType::get_IsAbstract method


Renvoie l'information qui détermine si l'élément **complexType** peut être utilisé dans le document d'instance.

```cpp
bool System::Xml::Schema::XmlSchemaComplexType::get_IsAbstract()
```


### ReturnValue

Si **true**, un élément ne peut pas utiliser cet élément **complexType** directement et doit utiliser un type complexe dérivé de cet élément **complexType**. La valeur par défaut est **false**. Optionnel.
