## XmlWriter::LookupPrefix method


Lorsqu'il est remplacé dans une classe dérivée, renvoie le préfixe le plus proche défini dans la portée de l'espace de noms actuel pour l'URI de l'espace de noms.

```cpp
virtual String System::Xml::XmlWriter::LookupPrefix(String ns)=0
```


| Paramètre | Type | Description |
| --- | --- | --- |
| ns | String | L'URI d'espace de noms dont vous souhaitez trouver le préfixe. |

### ReturnValue

Le préfixe correspondant ou **nullptr** si aucun URI d'espace de noms correspondant n'est trouvé dans la portée actuelle.
