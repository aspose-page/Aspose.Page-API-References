## XmlWriter::WriteAttributes method



| Paramètre | Type | Description |
| --- | --- | --- |
| reader | SharedPtr\<XmlReader\> | Le [XmlReader](../../xmlreader/) à partir duquel copier les attributs. |
| defattr | bool | **true** pour copier les attributs par défaut du [XmlReader](../../xmlreader/); sinon, **false**. |
