## NamespaceHandling enum


Indique s'il faut supprimer les déclarations d'espace de noms en double dans le [XmlWriter](../xmlwriter/).

```cpp
enum class NamespaceHandling
```

### Valeurs

| Nom | Valeur | Description |
| --- | --- | --- |
| Default | 0 | Indique que les déclarations d'espace de noms en double ne seront pas supprimées. |
| OmitDuplicates | 1 | Indique que les déclarations d'espace de noms en double seront supprimées. Pour qu'une déclaration d'espace de noms en double soit supprimée, le préfixe et l'espace de noms doivent correspondre. |
