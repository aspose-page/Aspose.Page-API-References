## XmlWriterSettings::set_NamespaceHandling method


Définit une valeur qui indique si le [XmlWriter](../../xmlwriter/) doit supprimer les déclarations d'espaces de noms en double lors de l'écriture du contenu XML. Le comportement par défaut est que le writer écrive toutes les déclarations d'espaces de noms présentes dans le résolveur d'espaces de noms du writer.

```cpp
void System::Xml::XmlWriterSettings::set_NamespaceHandling(System::Xml::NamespaceHandling value)
```


| Paramètre | Type | Description |
| --- | --- | --- |
| value | System::Xml::NamespaceHandling | La valeur à définir. |
