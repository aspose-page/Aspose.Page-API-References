## XmlWriterSettings::get_WriteEndDocumentOnClose method


Renvoie une valeur indiquant si le [XmlWriter](../../xmlwriter/) ajoutera des balises de fermeture à toutes les balises d'élément non fermées lorsque la méthode [XmlWriter::Close](../../xmlwriter/close/) est appelée.

```cpp
bool System::Xml::XmlWriterSettings::get_WriteEndDocumentOnClose()
```


### ReturnValue

**true** if all unclosed element tags will be closed out; otherwise, **false**. The default value is **true**.
