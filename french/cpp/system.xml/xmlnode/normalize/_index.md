## XmlNode::Normalize method


Place tous les nœuds [XmlText](../../xmltext/) à la profondeur maximale du sous‑arbre sous ce [XmlNode](../) dans une forme « normale » où seul le balisage (c’est‑à‑dire les balises, les commentaires, les instructions de traitement, les sections CDATA et les références d’entité) sépare les nœuds [XmlText](../../xmltext/), c’est‑à‑dire qu’il n’y a aucun nœud [XmlText](../../xmltext/) adjacent.
