## XmlReaderSettings::get_MaxCharactersInDocument method


Renvoie une valeur indiquant le nombre maximal de caractères autorisé dans un document XML. Une valeur zéro (0) signifie qu'il n'y a aucune limite à la taille du document XML. Une valeur non nulle spécifie la taille maximale, en caractères.

```cpp
int64_t System::Xml::XmlReaderSettings::get_MaxCharactersInDocument()
```


### ReturnValue

Le nombre maximal de caractères autorisé dans un document XML. La valeur par défaut est 0.
