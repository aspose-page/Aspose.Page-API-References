## Formatting enum


Spécifie les options de mise en forme pour le [XmlTextWriter](../xmltextwriter/).

```cpp
enum class Formatting
```

### Valeurs

| Nom | Valeur | Description |
| --- | --- | --- |
| None | 0 | Aucune mise en forme spéciale n'est appliquée. C'est la valeur par défaut. |
| Indented | 1 | Indente les éléments enfants selon les paramètres [XmlTextWriter::set_Indentation](../xmltextwriter/set_indentation/) et [XmlTextWriter::set_IndentChar](../xmltextwriter/set_indentchar/). |
