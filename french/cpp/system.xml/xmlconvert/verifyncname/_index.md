## XmlConvert::VerifyNCName method


Vérifie que le nom est un **NCName** valide selon la recommandation du langage de balisage étendu W3C. Un **NCName** est un nom qui ne peut pas contenir de deux-points.

```cpp
static String System::Xml::XmlConvert::VerifyNCName(const String &name)
```


| Paramètre | Type | Description |
| --- | --- | --- |
| name | const String\& | Le nom à vérifier. |

### ReturnValue

Le nom, s'il s'agit d'un NCName valide.
