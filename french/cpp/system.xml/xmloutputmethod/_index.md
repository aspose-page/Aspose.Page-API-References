## XmlOutputMethod enum


Spécifie la méthode utilisée pour sérialiser la sortie de [XmlWriter](../xmlwriter/).

```cpp
enum class XmlOutputMethod
```

### Valeurs

| Nom | Valeur | Description |
| --- | --- | --- |
| Xml | 0 | Sérialise selon les règles XML 1.0. |
| Html | 1 | Sérialise selon les règles HTML spécifiées par XSLT. |
| Text | 2 | Sérialise uniquement les blocs de texte. |
| AutoDetect | 3 | Utilisez les règles XSLT pour choisir entre les méthodes de sortie [XmlOutputMethod::Xml](./) et [XmlOutputMethod::Html](./) au moment de l'exécution. |
