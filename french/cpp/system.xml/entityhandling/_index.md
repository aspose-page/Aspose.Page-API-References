## EntityHandling enum


Spécifie comment le [XmlTextReader](../xmltextreader/) ou le [XmlValidatingReader](../xmlvalidatingreader/) gèrent les entités.

```cpp
enum class EntityHandling
```

### Valeurs

| Nom | Valeur | Description |
| --- | --- | --- |
| ExpandEntities | 1 | Développe toutes les entités et renvoie les nœuds développés. |
| ExpandCharEntities | 2 | Développe les entités de caractères et renvoie les entités générales sous forme de nœuds [XmlNodeType::EntityReference](../xmlnodetype/). |
