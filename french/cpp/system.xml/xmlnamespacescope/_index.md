## XmlNamespaceScope enum


Définit la portée de l'espace de noms.

```cpp
enum class XmlNamespaceScope
```

### Valeurs

| Nom | Valeur | Description |
| --- | --- | --- |
| Tous | 0 | Tous les espaces de noms définis dans la portée du nœud actuel. Cela inclut l'espace de noms xmlns:xml qui est toujours déclaré implicitement. L'ordre des espaces de noms retournés n'est pas défini. |
| ExcludeXml | 1 | Tous les espaces de noms définis dans la portée du nœud actuel, à l'exception de l'espace de noms xmlns:xml, qui est toujours déclaré implicitement. L'ordre des espaces de noms retournés n'est pas défini. |
| Local | 2 | Tous les espaces de noms qui sont définis localement au nœud actuel. |
