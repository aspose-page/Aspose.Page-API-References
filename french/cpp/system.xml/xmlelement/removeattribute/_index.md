## XmlElement::RemoveAttribute(String, String) method


Supprime un attribut dont le nom local et l'URI d'espace de noms sont spécifiés. (Si l'attribut supprimé possède une valeur par défaut, elle est immédiatement remplacée).

```cpp
virtual void System::Xml::XmlElement::RemoveAttribute(String localName, String namespaceURI)
```


| Paramètre | Type | Description |
| --- | --- | --- |
| localName | String | Le nom local de l'attribut à supprimer. |
| namespaceURI | String | L'URI d'espace de noms de l'attribut à supprimer. |
