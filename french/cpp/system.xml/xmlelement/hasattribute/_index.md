## XmlElement::HasAttribute(String, String) method


Détermine si le nœud actuel possède un attribut portant le nom local et l'URI d'espace de noms spécifiés.

```cpp
virtual bool System::Xml::XmlElement::HasAttribute(String localName, String namespaceURI)
```


| Paramètre | Type | Description |
| --- | --- | --- |
| localName | String | Le nom local de l'attribut à trouver. |
| namespaceURI | String | L'URI d'espace de noms de l'attribut à trouver. |

### ReturnValue

**true** if the current node has the specified attribute; otherwise, **false**.
