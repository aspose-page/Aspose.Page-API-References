## XmlElement::set_IsEmpty method


Définit le format de balise de l'élément.

```cpp
void System::Xml::XmlElement::set_IsEmpty(bool value)
```


| Paramètre | Type | Description |
| --- | --- | --- |
| value | bool | La valeur à définir. Lors de la définition de cette valeur, si elle est réglée sur **true**, les enfants de l'élément sont supprimés et l'élément est sérialisé au format de balise courte. Si elle est réglée sur **false**, la valeur est modifiée (indépendamment du fait que l'élément contienne du contenu ou non) ; si l'élément est vide, il est sérialisé au format long. |
