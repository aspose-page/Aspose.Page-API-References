## XpsGradientStop class


Classe encapsulant les fonctionnalités de l'élément GradientStop. Cet élément est utilisé par les éléments LinearGradientBrush et RadialGradientBrush pour définir l'emplacement et la portée de la progression de couleur lors du rendu d'un dégradé.

```cpp
class XpsGradientStop : public Aspose::Page::XPS::XpsModel::XpsObject
```
