## IXsltContextVariable::get_IsParam method


Renvoie une valeur indiquant si la variable est un paramètre Extensible Stylesheet Language Transformations (XSLT). Cela peut être un paramètre d'une feuille de style ou d'un modèle.

```cpp
virtual bool System::Xml::Xsl::IXsltContextVariable::get_IsParam()=0
```


### ReturnValue

**true** if the variable is an XSLT parameter; otherwise, **false**.
