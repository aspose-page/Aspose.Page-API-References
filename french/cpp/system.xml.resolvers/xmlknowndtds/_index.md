## XmlKnownDtds enum


L'énumération [Resolvers::XmlKnownDtds](./) est utilisée par le [Resolvers::XmlPreloadedResolver](../xmlpreloadedresolver/) et définit quels DTD bien connus sont reconnus par le [Resolvers::XmlPreloadedResolver](../xmlpreloadedresolver/).

```cpp
enum class XmlKnownDtds
```

### Valeurs

| Nom | Valeur | Description |
| --- | --- | --- |
| None | 0 | Spécifie que le [Resolvers::XmlPreloadedResolver](../xmlpreloadedresolver/) ne reconnaîtra aucun des DTD prédéfinis. |
| Xhtml10 | 1 | Spécifie que le [Resolvers::XmlPreloadedResolver](../xmlpreloadedresolver/) reconnaîtra les DTD et entités définis dans XHTML 1.0. |
| Rss091 | 2 | Spécifie que le [Resolvers::XmlPreloadedResolver](../xmlpreloadedresolver/) reconnaîtra les DTD et entités définis dans RSS 0.91. |
| All | 65535 | Spécifie que le [Resolvers::XmlPreloadedResolver](../xmlpreloadedresolver/) reconnaîtra tous les DTD actuellement pris en charge. C'est le comportement par défaut. |
