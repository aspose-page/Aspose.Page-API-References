## IEnumerator::MoveNext method


Déplace l'énumérateur vers l'élément suivant. Si aucun élément n'avait été référencé auparavant, définit la référence sur le premier élément disponible. Si la fin du conteneur est atteinte, ne fait rien.

```cpp
virtual bool System::Collections::Generic::IEnumerator<T>::MoveNext()=0
```


### ReturnValue

Vrai si un élément est disponible après l'appel, faux sinon.
