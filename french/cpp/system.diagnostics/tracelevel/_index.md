## TraceLevel enum


Spécifie quels messages afficher pour les classes [System.Diagnostics.Debug](../debug/), [System.Diagnostics.Trace](../trace/) et System.Diagnostics.TraceSwitch.

```cpp
enum class TraceLevel
```

### Valeurs

| Nom | Valeur | Description |
| --- | --- | --- |
| Désactivé | 0 | N'affiche aucun message de traçage ni de débogage. |
| Erreur | 1 | Affiche les messages de gestion des erreurs. |
| Warning | 2 | Affiche les avertissements et les messages de gestion des erreurs. |
| Info | 3 | Afficher les messages d'information, les avertissements et les messages de gestion des erreurs. |
| Verbeux | 4 | Afficher tous les messages de débogage et de traçage. |
