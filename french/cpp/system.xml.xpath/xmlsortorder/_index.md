## XmlSortOrder enum


Spécifie l'ordre de tri.

```cpp
enum class XmlSortOrder
```

### Valeurs

| Nom | Valeur | Description |
| --- | --- | --- |
| Croissant | 1 | Les nœuds sont triés par ordre croissant. Par exemple, si les nombres 1,2,3 et 4 sont triés par ordre croissant, ils apparaissent comme 1,2,3,4. |
| Décroissant | 2 | Les nœuds sont triés par ordre décroissant. Par exemple, si les nombres 1,2,3 et 4 sont triés par ordre décroissant, ils apparaissent comme, 4,3,2,1. |
