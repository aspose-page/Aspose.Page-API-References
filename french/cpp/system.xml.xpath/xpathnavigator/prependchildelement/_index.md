## XPathNavigator::PrependChildElement method


Crée un nouvel élément enfant au début de la liste des nœuds enfants du nœud actuel en utilisant le préfixe d'espace de noms, le nom local et l'URI d'espace de noms spécifiés avec la valeur spécifiée.

```cpp
virtual void System::Xml::XPath::XPathNavigator::PrependChildElement(String prefix, String localName, String namespaceURI, String value)
```


| Paramètre | Type | Description |
| --- | --- | --- |
| préfixe | String | Le préfixe d'espace de noms du nouvel élément enfant (le cas échéant). |
| localName | String | Le nom local du nouvel élément enfant (le cas échéant). |
| namespaceURI | String | L'URI d'espace de noms du nouvel élément enfant (le cas échéant). [String::Empty](../../../system/string/empty/) et **nullptr** sont équivalents. |
| value | String | La valeur du nouvel élément enfant. Si [String::Empty](../../../system/string/empty/) ou **nullptr** sont fournis, un élément vide est créé. |
