## XPathNavigator::get_LocalName method


Lorsqu'elle est remplacée dans une classe dérivée, obtient le [XPathNavigator::get_Name](../get_name/) du nœud actuel sans aucun préfixe d'espace de noms.

```cpp
virtual String System::Xml::XPath::XPathNavigator::get_LocalName()=0
```


### ReturnValue

Une [String](../../../system/string/) qui contient le nom local du nœud actuel, ou [String::Empty](../../../system/string/empty/) si le nœud actuel n'a pas de nom (par exemple, des nœuds de texte ou de commentaire).
