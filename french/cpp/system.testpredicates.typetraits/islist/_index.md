## IsList typedef


Vérifie si le type est une spécialisation de [System::Collections::Generic::List](../../system.collections.generic/list/). Le cas échéant, le membre value est défini sur true, sinon il est défini sur false.

```cpp
using System::TestPredicates::TypeTraits::IsList =  std::is_same<T, System::Collections::Generic::List<typename T::ValueType>>
```
