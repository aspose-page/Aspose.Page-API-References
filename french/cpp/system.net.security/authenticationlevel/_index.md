## AuthenticationLevel enum


Drapeaux d'authentification spécifiques à WebRequest.

```cpp
enum class AuthenticationLevel
```

### Valeurs

| Nom | Valeur | Description |
| --- | --- | --- |
| None | 0 | Aucune authentification n'est requise pour le client et le serveur. |
| MutualAuthRequested | 1 | La requête ne échoue pas si le serveur n'est pas authentifié. |
| MutualAuthRequired | 2 | L'application actuelle recevra 'IOException' lorsque le serveur n'est pas authentifié. |
