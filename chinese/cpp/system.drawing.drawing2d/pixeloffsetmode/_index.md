## PixelOffsetMode enum

```cpp
enum class PixelOffsetMode
```

### 值

| 名称 | 值 | 描述 |
| --- | --- | --- |
| 无效 | n/a | 无效的值。 |
| Default | n/a | 默认模式。 |
| 高速 | n/a | 低质量（高性能）模式。 |

| None | n/a | 无像素偏移。 |
