## WrapMode enum


指定当纹理或渐变小于填充区域时的平铺方式。

```cpp
enum class WrapMode
```

### 值

| 名称 | 值 | 描述 |
| --- | --- | --- |
| Tile | 0 | 平铺渐变或纹理。 |
| TileFlipX | 1 | 水平翻转纹理或渐变，然后平铺该纹理或渐变。 |
| TileFlipY | 2 | 垂直翻转纹理或渐变，然后平铺纹理或渐变。 |
| TileFlipXY | 3 | 水平和垂直翻转纹理或渐变，然后平铺纹理或渐变。 |
| Clamp | 4 | 将纹理或渐变限制在对象边界内。 |
