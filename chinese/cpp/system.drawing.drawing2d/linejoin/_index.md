## LineJoin enum


指定后续直线或曲线的连接方式。

```cpp
enum class LineJoin
```

### 值

| 名称 | 值 | 描述 |
| --- | --- | --- |
| 斜接 | 0 | 斜接连接，根据斜接长度是否超过斜接限制，产生锐角或被裁剪的角。 |
| 斜角 | 1 | 斜面连接，产生对角角。 |
| 四舍五入 | 2 | 圆形连接，在直线之间产生平滑的圆弧。 |
| MiterClipped | 3 | 斜接连接，根据斜接长度是否超过斜接限制，产生锐角或斜面角。 |
