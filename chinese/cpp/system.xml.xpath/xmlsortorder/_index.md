## XmlSortOrder enum


指定排序顺序。

```cpp
enum class XmlSortOrder
```

### 值

| 名称 | 值 | 描述 |
| --- | --- | --- |
| 升序 | 1 | 节点按升序排序。例如，如果数字 1、2、3 和 4 按升序排序，它们将显示为 1、2、3、4。 |
| 降序 | 2 | 节点按降序排序。例如，如果数字 1、2、3 和 4 按降序排序，它们将显示为 4、3、2、1。 |
