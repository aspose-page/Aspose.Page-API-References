该 **Aspose.Page.Xps.Presentation.Pdf** 命名空间提供用于将 [XPS](../aspose.page.xps/) 文档渲染为 PDF 文档的类。

## 类

| 类 | 描述 |
| --- | --- |
| [PdfDigitalSignatureDetails](./pdfdigitalsignaturedetails/) | 包含 PDF 数字签名的详细信息。 |
| [PdfEncryptionDetails](./pdfencryptiondetails/) | 包含 PDF 加密的详细信息。 |
| [PdfSaveOptions](./pdfsaveoptions/) | 用于 XPS-as-PDF 保存选项的类。 |
## Enums

| 枚举 | 描述 |
| --- | --- |
| [PdfCompliance](./pdfcompliance/) | 指定输出文件的 PDF 合规级别。 |
| [PdfDigitalSignatureHashAlgorithm](./pdfdigitalsignaturehashalgorithm/) | 指定数字签名使用的数字哈希算法。 |
| [PdfEncryptionAlgorithm](./pdfencryptionalgorithm/) | 加密模式枚举。描述使用的算法和密钥长度。此枚举已扩展，以便进一步增强功能。此枚举实现了 “Base-to-Core” 模式。 |
| [PdfImageCompression](./pdfimagecompression/) | 指定 PDF 文件中图像使用的压缩类型。 |
| [PdfTextCompression](./pdftextcompression/) | 指定 PDF 文件中除图像外所有内容使用的压缩类型。 |
