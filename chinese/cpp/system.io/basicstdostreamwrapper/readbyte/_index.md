## BasicSTDOStreamWrapper::ReadByte method


如果包装模式为二进制，则从上一次解码的字符存储中读取单个字节；否则从流中读取单个字符并将其转换为 uint8_t 类型。不支持！

```cpp
virtual int System::IO::BasicSTDOStreamWrapper<T, typename>::ReadByte() override
```


### ReturnValue

一个等同于读取结果的 32 位整数值
