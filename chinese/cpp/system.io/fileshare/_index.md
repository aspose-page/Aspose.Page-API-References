## FileShare enum


指定其他 [FileStream](../filestream/) 对象在打开文件时可以拥有的访问类型。

```cpp
enum class FileShare
```

### 值

| 名称 | 值 | 描述 |
| --- | --- | --- |
| None | 0 | 无访问。 |
| 读取 | 1 | 只读访问。 |
| Write | 2 | 只写访问。 |
| 读写 | 3 | 读写访问。 |
| 删除 | 4 | 该文件可以被删除。 |
| 可继承的 | 16 | 使文件句柄对子进程可继承。 |
