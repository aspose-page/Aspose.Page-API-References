该 **[Aspose.Page](./)** 是所有 [Aspose.Page](./) 库的类的根命名空间，这些类要么直接位于其中，如 **Device**，要么通过多个子命名空间间接包含。

## 类

| 类 | 描述 |
| --- | --- |
| [AssemblyConstants](./assemblyconstants/) | 定义参与组件许可证检查的常量。这些常量过去直接定义为程序集属性，但我将它们移入单独的类，因为在 .NET Compact Framework 中无法访问程序集属性。现在，在为 .NET Compact Framework 编译时，许可证代码使用这些常量而不是程序集属性。 |
| [BuildVersionInfo](./buildversioninfo/) | 此类提供有关当前产品构建的信息。 |
| [CryptoProvider](./cryptoprovider/) |  |
| [Document](./document/) | 所有封装文档的超类。 |
| [ExternalFontCache](./externalfontcache/) | 使用此类获取以 [Device](../) 可接受的形式的字体封装。 |
| [IGlyph](./iglyph/) | 此接口提供对字形主要参数的访问。 |
| [IInteractiveDevice](./iinteractivedevice/) | 定义交互特性处理方法的接口。 |
| [IMultiPageDevice](./imultipagedevice/) | 此接口包含用于操作多页设备的方法。 |
| [IMultiPageSaveOptions](./imultipagesaveoptions/) | 定义用于设置转换页码的接口。 |
| [IMultiPartitionDevice](./imultipartitiondevice/) | 多分区设备的基本接口。 |
| [IStreamable](./istreamable/) | 此接口用于将文档渲染到流的设备。 |
| [License](./license/) | 提供对组件授权的方法。 |
| [Margins](./margins/) | 此类封装了上、左、下、右边距。 |
| [Metered](./metered/) | 提供设置计量密钥的方法。 |
| [SaveOptions](./saveoptions/) | 此类包含管理转换过程所需的选项。 |
| [UserProperties](./userproperties/) | 特殊属性类，允许设置和返回类型化属性。如果该属性对象不包含某属性，还允许挂接两个默认属性对象进行搜索。 |
