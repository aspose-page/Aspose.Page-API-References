## XmlSchemaDerivationMethod enum

### 值

| 名称 | 值 | 描述 |

| Empty | 0 | 覆盖默认的派生方法以允许任何派生。 |
| 替代 | 1 | 指的是通过 **Substitution** 的派生。 |
| 扩展 | 2 | 指的是通过 **Extension** 的派生。 |
| 限制 | 4 | 指的是通过 **Restriction** 的派生。 |
| 列表 | 8 | 指的是通过 **List** 的派生。 |
| 并集 | 16 | 指的是通过 **Union** 的派生。 |
| 全部 | 255 | **#all**。指所有派生方法。 |
| None | 256 | 接受默认的派生方法。 |
