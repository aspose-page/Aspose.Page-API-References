## XmlSchemaMinLengthFacet class


表示来自 XML [Schema](../) 的 **minLength** 元素，依据万维网联盟（[Web](../../system.web/)）的规定。此类可用于对 **simpleType** 元素的数据值的最小长度设置限制。该长度必须大于 **minLength** 元素的值。

```cpp
class XmlSchemaMinLengthFacet : public System::Xml::Schema::XmlSchemaNumericFacet
```
