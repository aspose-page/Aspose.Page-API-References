```cpp
bool System::Xml::Schema::XmlSchemaSet::get_IsCompiled()
```

**true** if the schemas in the [XmlSchemaSet](../) have been compiled since the last time a schema was added or removed from the [XmlSchemaSet](../); otherwise, **false**.
