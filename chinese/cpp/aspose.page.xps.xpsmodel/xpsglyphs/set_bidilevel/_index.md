## XpsGlyphs::set_BidiLevel method


返回/设置 指定 Unicode 算法双向嵌套级别的值。偶数值表示从左到右布局，奇数值表示从右到左布局。右到左布局将运行起点放在第一个字形的右侧，正的前进宽度（表示向左的前进）会将后续字形放置在前一个字形的左侧。
