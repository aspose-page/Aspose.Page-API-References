## 字段

| 字段 | 描述 |
| --- | --- |
| static [Automatic](./automatic/) | 指定自动输出的意图（允许客户端自动选择最佳设置）。 |
| static [Draft](./draft/) | 指定草稿输出的意图（在草稿质量的文本和图形之间取得平衡）。 |
| static [Fax](./fax/) | 指定传真输出的意图（在标准传真质量之间取得平衡）。 |
| static [High](./high/) | 指定高质量输出的意图（在高质量的文本和图形之间取得平衡）。 |
| static [Normal](./normal/) | 指定普通输出的意图（在良好质量的文本和图形之间取得平衡）。 |
| static [Photographic](./photographic/) | 指定摄影输出的意图（图像应具有最高质量）。 |
| static [Text](./text/) | 指定仅文本输出的意图（图形可能输出不佳或根本不输出）。 |
