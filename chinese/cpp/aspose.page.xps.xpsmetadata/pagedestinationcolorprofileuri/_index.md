## PageDestinationColorProfileURI class


指定对包含在 [XPS](../../aspose.page.xps/)[Document](../../aspose.page/document/) 中的 ICC 配置文件的相对 URI 引用。此选项的处理取决于 [PageDeviceColorSpaceUsage](../pagedevicecolorspaceusage/) 功能的设置。使用该配置文件的所有元素都被视为已经处于适当的设备色彩空间中，且不会在驱动程序或设备中进行颜色管理。 [https://docs.microsoft.com/en-us/windows/win32/printdocs/pagedestinationcolorprofileuri](https://docs.microsoft.com/en-us/windows/win32/printdocs/pagedestinationcolorprofileuri)。
