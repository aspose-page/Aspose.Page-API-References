## IEnumerator::MoveNext method


将枚举器移动到下一个元素。如果之前没有引用任何元素，则将引用设置为第一个可用元素。如果已到达容器末尾，则不执行任何操作。

```cpp
virtual bool System::Collections::Generic::IEnumerator<T>::MoveNext()=0
```


### ReturnValue

如果调用后有可用元素则为 true，否则为 false。
