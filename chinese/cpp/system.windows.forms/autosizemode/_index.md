## AutoSizeMode enum


定义自动大小的工作方式。

```cpp
enum class AutoSizeMode
```

### 值

| 名称 | 值 | 描述 |
| --- | --- | --- |
| GrowAndShrink | 0 | [Control](../control/) 根据需要增长和收缩。它不能手动调整大小。 |
| GrowOnly | 1 | [Control](../control/) 根据需要增长，但不能缩小到最小尺寸以下。它可以手动调整大小。 |
