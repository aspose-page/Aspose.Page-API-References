## DSASignatureFormatter class


用于创建 [DSA](../dsa/) 签名。此类的对象只能通过 [System::MakeObject()](../../system/makeobject/) 函数分配。切勿在栈上或使用 operator new 创建此类型的实例，因为会导致运行时错误和/或断言失败。始终将此类包装在 [System::SmartPtr](../../system/smartptr/) 指针中，并使用该指针将其作为参数传递给函数。

```cpp
class DSASignatureFormatter : public System::Security::Cryptography::AsymmetricSignatureFormatter
```
