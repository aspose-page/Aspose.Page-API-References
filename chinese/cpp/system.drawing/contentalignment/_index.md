## ContentAlignment enum

```cpp
enum class ContentAlignment
```

### 值

| 名称 | 值 | 描述 |
| --- | --- | --- |
| 左上 | 1 | 垂直对齐在顶部，水平对齐在左侧。 |
| 顶部居中 | 2 | 垂直对齐在顶部，水平对齐在中心。 |
| 右上 | 4 | 垂直对齐在顶部，水平对齐在右侧。 |
| 左中 | 16 | 垂直对齐在中部，水平对齐在左侧。 |
| 中间 | 32 | 垂直对齐在中部，水平对齐在中心。 |
| 右中 | 64 | 垂直对齐在中部，水平对齐在右侧。 |
| 左下 | 256 | 垂直对齐在底部，水平对齐在左侧。 |
| 底部居中 | 512 | 垂直对齐在底部，水平对齐在中心。 |
| 右下 | 1024 | 垂直对齐在底部，水平对齐在右侧。 |
