## StringDigitSubstitute enum


指定字符串中的数字如何根据区域设置或语言进行替换。

```cpp
enum class StringDigitSubstitute
```

### 值

| 名称 | 值 | 描述 |
| --- | --- | --- |
| User | 0 | 用户定义的替换。 |
| None | 1 | Subtitution 已禁用。 |
| National | 2 | 替换对应于用户区域设置的官方国家语言的数字。 |
| Traditional | 3 | 替换对应于用户本地文字或语言的数字，该语言可能与用户区域设置的官方国家语言不同。 |
