## XmlWriterSettings::get_NewLineChars method


返回用于换行的字符字符串。

```cpp
String System::Xml::XmlWriterSettings::get_NewLineChars()
```


### ReturnValue

用于换行的字符字符串。此值可以设置为任意字符串。但为确保 XML 有效，您应仅指定有效的空白字符，例如空格、制表符、回车或换行。默认值是 **\\r\\n**（回车，换行）。
