## XmlWriterSettings::get_NamespaceHandling method


返回一个值，指示在写入 XML 内容时 [XmlWriter](../../xmlwriter/) 是否应删除重复的命名空间声明。 默认行为是写入器输出 writer's 命名空间解析器中存在的所有命名空间声明。

```cpp
System::Xml::NamespaceHandling System::Xml::XmlWriterSettings::get_NamespaceHandling()
```


### ReturnValue

用于指定是否在 [XmlWriter](../../xmlwriter/) 中删除重复命名空间声明的 [NamespaceHandling](../../namespacehandling/) 枚举。
