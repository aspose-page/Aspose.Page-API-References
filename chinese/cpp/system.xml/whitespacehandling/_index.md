## WhitespaceHandling enum


指定如何处理空白字符。

```cpp
enum class WhitespaceHandling
```

### 值

| 名称 | 值 | 描述 |
| --- | --- | --- |
| 全部 | 0 | 返回 **Whitespace** 和 **SignificantWhitespace** 节点。这是默认值。 |
| 重要 | 1 | 仅返回 **SignificantWhitespace** 节点。 |
| None | 2 | 不返回 **Whitespace** 也不返回 **SignificantWhitespace** 节点。 |
