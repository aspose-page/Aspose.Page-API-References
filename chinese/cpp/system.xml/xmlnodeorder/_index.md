## XmlNodeOrder enum


描述节点相对于第二个节点的文档顺序。

```cpp
enum class XmlNodeOrder
```

### 值

| 名称 | 值 | 描述 |
| --- | --- | --- |
| Before | 0 | 此导航器的当前节点位于提供的导航器的当前节点之前。 |
| 之后 | 1 | 此导航器的当前节点位于提供的导航器的当前节点之后。 |
| 相同 | 2 | 两个导航器位于相同的节点上。 |
| 未知 | 3 | 无法在文档顺序中确定节点位置的相对关系。如果两个节点位于不同的树中，可能会出现这种情况。 |
