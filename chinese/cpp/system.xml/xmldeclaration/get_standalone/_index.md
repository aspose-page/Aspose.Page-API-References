## XmlDeclaration::get_Standalone method


返回 standalone 属性的值。

```cpp
String System::Xml::XmlDeclaration::get_Standalone()
```


### ReturnValue

有效值为 **yes**，表示 XML 文档所需的所有实体声明都包含在文档中；或 **no**，表示需要外部文档类型定义（DTD）。如果 XML 声明中不存在 standalone 属性，此方法返回 [String::Empty](../../../system/string/empty/)。
