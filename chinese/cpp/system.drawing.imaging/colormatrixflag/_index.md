## ColorMatrixFlag enum


指定在 [ImageAttributes](../imageattributes/) 对象的颜色和灰度调整设置将影响的图像和颜色类型。

```cpp
enum class ColorMatrixFlag
```

### 值

| 名称 | 值 | 描述 |
| --- | --- | --- |
| Default | 0 | 颜色和灰度值均进行颜色调整。 |
| SkipGrays | 1 | 灰度值不进行颜色调整。 |
| AltGrays | 2 | 仅对灰度值进行颜色调整。 |
