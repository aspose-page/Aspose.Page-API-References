## ImageLockMode enum


指定被锁定的图像区域的属性。

```cpp
enum class ImageLockMode
```

### 值

| 名称 | 值 | 描述 |
| --- | --- | --- |
| ReadOnly | 1 | 该区域仅锁定为只读。 |
| 仅写 | 2 | 该区域仅锁定为只写。 |
| 读写 | n/a | 该区域已同时锁定为读取和写入。 |
| 用户输入缓冲区 | 4 | 图像位于用户输入缓冲区中，其访问受用户控制。 |
