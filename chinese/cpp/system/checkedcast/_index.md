## System::CheckedCast method


确定指定值是否落在类型 **TTo** 的值范围内，如果是，则将其强制转换为类型 **TTo**。

```cpp
template<typename TTo,typename TFrom> TTo System::CheckedCast(TFrom value)
```


| Parameter | 描述 |
| --- | --- |
| TTo | 要将指定值强制转换到的类型 |
| TFrom | 指定值的类型 |

| Parameter | Type | 描述 |
| --- | --- | --- |
| value | TFrom | 要强制转换的值 |

### ReturnValue

类型为 **TTo** 且等价于 **value** 的值
