## SmartPtr::get_shared_count method


获取指向被引用对象的共享指针数量，包括当前指针。断言当前指针处于共享模式。

```cpp
int System::SmartPtr<T>::get_shared_count() const
```


### ReturnValue

指向被引用对象的共享指针数量（如果有）。如果指针为 null，则返回 0。
