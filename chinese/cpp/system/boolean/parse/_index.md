## Boolean::Parse method


将指定的字符串转换为 bool 类型的值。

```cpp
static bool System::Boolean::Parse(const String &value)
```


| Parameter | Type | 描述 |
| --- | --- | --- |
| value | const String\& | 要转换的字符串。 |

### ReturnValue

存放转换结果的 bool 变量；如果指定的字符串等于 "True" 则结果为 true，如果等于 "False" 则结果为 false。
