## System::Equals method

```cpp
template<typename TA,typename TB> bool System::Equals(const TA &a, const TB &b)
```


| Parameter | 描述 |
| --- | --- |
| TA | 第一个比较对象的类型 |
| TB | 第二个比较对象的类型 |

| Parameter | Type | 描述 |
| --- | --- | --- |
| 一个 | const TA\& | 第一个比较数 |
| b | const TB\& | 第二个比较数 |

### ReturnValue
