## DateTime::Compare method


比较由指定的 [DateTime](../) 类实例表示的两个值，并返回指示这些值在时间轴上相对位置的值。

```cpp
static constexpr int System::DateTime::Compare(DateTime t1, DateTime t2)
```


| Parameter | Type | 描述 |
| --- | --- | --- |
| t1 | DateTime | 第一个比较数 |
| t2 | DateTime | 第二个比较数 |

### ReturnValue

如果 **t1** 早于 **t2**，则返回小于 0 的值；如果 **t1** 与 **t2** 相同，则返回 0；如果 **t1** 晚于 **t2**，则返回大于 0 的值
