## DateTime::IsDaylightSavingTime method


确定当前对象表示的日期时间值是否位于当前时区的夏令时范围内。

```cpp
bool System::DateTime::IsDaylightSavingTime() const
```


### ReturnValue

如果当前对象指定的日期和时间值落在夏令时范围内，则为 true；否则为 false。
