## Array::crend method


返回指向反转容器最后一个元素之后的元素的逆向迭代器。它对应于未反转容器的第一个元素之前的元素。该元素充当占位符，尝试访问它会导致未定义行为。

```cpp
const_reverse_iterator System::Array<T>::crend() const noexcept
```


### ReturnValue

一个迭代器，指向容器中第一个元素之前的理论 const 限定元素。
