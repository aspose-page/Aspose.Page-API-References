## Enumerator::MoveNext method


检查当前元素的索引是否未指向数组中的最后一个元素，如果不是，则将其前进。

```cpp
virtual bool System::Array<T>::Enumerator::MoveNext() override
```


### ReturnValue

如果在前进之前当前元素的索引未指向数组的最后一个元素，则为 true；否则为 false。
