## Array::crbegin method


返回一个指向反向容器中第一个元素的逆向迭代器。它对应于非反向容器的最后一个元素。如果容器为空，返回的迭代器等于 [crend()](../crend/)。

```cpp
const_reverse_iterator System::Array<T>::crbegin() const noexcept
```


### ReturnValue

一个指向容器中最后一个 const 限定元素的迭代器。
