## Char::IsNumber(char_t) method


确定指定字符是否被归类为数字。

```cpp
static bool System::Char::IsNumber(char_t c)
```


| Parameter | Type | 描述 |
| --- | --- | --- |
| c | char_t | 要测试的字符 |

### ReturnValue

如果指定字符是数字，则为 true，否则为 false
