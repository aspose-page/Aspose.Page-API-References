## Char::IsSymbol(char_t) method


确定指定字符是否被分类为符号字符。

```cpp
static bool System::Char::IsSymbol(char_t c)
```


| Parameter | Type | 描述 |
| --- | --- | --- |
| c | char_t | 要测试的字符 |

### ReturnValue

如果指定字符是符号字符，则返回 true；否则返回 false
