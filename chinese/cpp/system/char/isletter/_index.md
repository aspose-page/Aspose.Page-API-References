## Char::IsLetter(char_t) method


确定指定字符是否被归类为 Unicode 字母。

```cpp
static bool System::Char::IsLetter(char_t c)
```


| Parameter | Type | 描述 |
| --- | --- | --- |
| c | char_t | 要测试的字符 |

### ReturnValue

如果指定字符是 Unicode 字母，则为 true，否则为 false
