## Char::IsPunctuation(char_t) method


确定指定字符是否被归类为标点符号。

```cpp
static bool System::Char::IsPunctuation(char_t c)
```


| Parameter | Type | 描述 |
| --- | --- | --- |
| c | char_t | 要测试的字符 |

### ReturnValue

如果指定字符是标点符号，则为 true；否则为 false
