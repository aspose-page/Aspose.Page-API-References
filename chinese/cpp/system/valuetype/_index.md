## ValueType class


作为值类型的基类，出于性能原因省略了对 [Object](../object/) 的继承。此类型应在栈上分配，并通过值或引用传递给函数。切勿使用 [System::SmartPtr](../smartptr/) 类来管理此类型的对象。

```cpp
class ValueType
```
