## TimeZone::IsDaylightSavingTime method


确定由指定的 [DateTime](../../datetime/) 对象表示的日期时间值是否落在当前 [TimeZone](../) 对象所表示的时区的夏令时范围内。

```cpp
virtual bool System::TimeZone::IsDaylightSavingTime(DateTime time)=0
```


### ReturnValue

如果由指定的 [DateTime](../../datetime/) 对象表示的日期时间值落在当前 [TimeZone](../) 对象所表示的时区的夏令时范围内，则为 true；否则为 false。
