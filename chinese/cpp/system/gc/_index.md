## GC class


表示一种模拟的垃圾回收，它更像一个实际上不执行任何操作的存根。这是一个没有实例服务的静态类型。无论何种方式都不应创建其实例。

```cpp
class GC : public System::Object
```

| static [GetTotalMemory](./gettotalmemory/)(bool) | 返回当前进程当前已分配的私有内存字节数。 |
