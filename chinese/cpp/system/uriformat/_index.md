## UriFormat enum


指定 URI 的转义方式。

```cpp
enum class UriFormat
```

### 值

| 名称 | 值 | 描述 |
| --- | --- | --- |
| UriEscaped | 1 | 根据 RFC-2396 进行转义。 |
| Unescaped | 2 | 不进行转义。 |
| SafeUnescaped | 3 | 仅对在 URI 中具有保留含义的字符进行转义。 |
