## String::CompareTo method


以“less-equals-more”风格比较两个字符串。使用当前文化。

```cpp
int System::String::CompareTo(const String &str) const
```


| Parameter | Type | 描述 |
| --- | --- | --- |
| str | const String\& | [String](../) 用于与其比较。 |

### ReturnValue

如果此字符串被视为低于 str，则返回负值；如果相等，则返回零；如果大于 str，则返回正值。
