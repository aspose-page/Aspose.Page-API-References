## String::IsNull method


检查字符串是否被视为 null。只有在通过 [String()](../string/) 构造函数构造、移动、复制或从 null 字符串赋值，或调用 [reset()](../reset/) 方法时，[String](../) 才为 null。

```cpp
bool System::String::IsNull() const
```


### ReturnValue

如果字符串被视为 null 则返回 true，否则返回 false。
