## String::end method


返回指向实际字符串缓冲区末尾的指针。永不重新分配任何内容。不能保证缓冲区以空字符结尾。

```cpp
const UChar * System::String::end() const
```


### ReturnValue

指向实际字符串缓冲区之后的第一个元素的指针，无论是否以空字符结尾。
