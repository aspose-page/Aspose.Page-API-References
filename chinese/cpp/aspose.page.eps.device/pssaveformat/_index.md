## PsSaveFormat enum


此枚举包含可用的保存格式选项。它可以是 PS 或 [EPS](../../aspose.page.eps/)。[EPS](../../aspose.page.eps/) 仅用于单页文档，而 PS 文件可以包含任意页数。

```cpp
enum class PsSaveFormat
```

### 值

| 名称 | 值 | 描述 |
| --- | --- | --- |
| PS | 0 | 此选项指示生成的文档必须是 PostScript (PS) 或封装的 PostScript（[EPS](../../aspose.page.eps/)）文件。 |
| EPS | 1 | 此选项指示生成的文档必须是封装的 PostScript（[EPS](../../aspose.page.eps/)）文件。它仅用于单页文档； |
