## TimeSpanStyles enum


指定格式化选项，以自定义将时间间隔的字符串表示转换为 Timespan 对象的方法的字符串解析。

```cpp
enum class TimeSpanStyles : int32_t
```

### 值

| 名称 | 值 | 描述 |
| --- | --- | --- |
| None | 0 | 由 inptut 字符串表示的时间间隔仅在出现负号时才被解释为负的时间间隔。 |
| AssumeNegative | 1 | 由 inptut 字符串表示的时间间隔始终被解释为负的时间间隔。 |
