## Units enumeration

指定尺寸类型。

```js
enum Units
```

### 值

| 名称 | 值 | 描述 |
| ---        | ---   | --- |
| Points | `0` | 点 (1/72 英寸)。 |
| Inches | `1` | 英寸。 |
| Millimeters | `2` | 毫米。 |
| Centimeters | `3` | 厘米。 |
| Percents | `4` | 现有大小的百分比。 |
