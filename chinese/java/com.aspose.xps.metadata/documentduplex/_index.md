描述输出的双面特性。双面功能允许在介质的两面进行打印。每个文档单独进行双面处理。DocumentDuplex 和 JobDuplexAllDocumentsContiguously 互斥。由驱动程序决定这些关键字之间的约束处理方式。 https://docs.microsoft.com/en-us/windows/win32/printdocs/documentduplex
