描述 ScaleOffsetAlignment 功能选项。指定缩放内容的对齐方式。
## 字段

| 字段 | 描述 |
| --- | --- |
| [BottomCenter](#BottomCenter) | 指定缩放应对齐在介质底部边缘的中心。 |
| [BottomLeft](#BottomLeft) | 指定缩放应对齐在媒体的左下角。 |
| [BottomRight](#BottomRight) | 指定缩放应对齐在媒体的右下角。 |
| [Center](#Center) | 指定缩放应居中于媒体。 |
| [LeftCenter](#LeftCenter) | 指定缩放应对齐在媒体左边缘的中心。 |
| [RightCenter](#RightCenter) | 指定缩放应对齐在媒体右边缘的中心。 |
| [TopCenter](#TopCenter) | 指定缩放应对齐在媒体上边缘的中心。 |
| [TopLeft](#TopLeft) | 指定缩放应对齐在媒体的左上角。 |
| [TopRight](#TopRight) | 指定缩放应对齐在媒体的右上角。 |
