public enum PsSaveFormat extends Enum<PsSaveFormat>

此枚举包含可用的保存格式选项。它可以是 PS 或 EPS。EPS 仅用于单页文档，而 PS 文件可以包含任意页数。
## 字段

| 字段 | 描述 |
| --- | --- |
| [EPS](#EPS) | 此选项表示生成的文档必须是封装的 PostScript (EPS) 文件。 |
| [PS](#PS) | 此选项表示生成的文档必须是 PostScript (PS) 或封装的 PostScript (EPS) 文件。 |
