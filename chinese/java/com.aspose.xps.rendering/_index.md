该 **com.aspose.xps.rendering** 包提供将 XPS 文档渲染为其他格式的基础类。


## 类

| 类 | 描述 |
| --- | --- |
| [BmpSaveOptions](../com.aspose.xps.rendering/bmpsaveoptions) | 用于 XPS-as-BMP 保存选项的类。 |
| [ImageSaveOptions](../com.aspose.xps.rendering/imagesaveoptions) | 用于 XPS-as-image 保存选项的基础类。 |
| [JpegSaveOptions](../com.aspose.xps.rendering/jpegsaveoptions) | 用于 XPS-as-JPEG 保存选项的类。 |
| [PdfEncryptionDetails](../com.aspose.xps.rendering/pdfencryptiondetails) | 包含 pdf 加密的详细信息。 |
| [PdfSaveOptions](../com.aspose.xps.rendering/pdfsaveoptions) | 用于 XPS-as-PDF 保存选项的类。 |
| [PngSaveOptions](../com.aspose.xps.rendering/pngsaveoptions) | 用于 XPS-as-PNG 保存选项的类。 |
| [TiffSaveOptions](../com.aspose.xps.rendering/tiffsaveoptions) | 用于 XPS-as-TIFF 保存选项的类。 |

## 接口

| 接口 | 描述 |

## 枚举

| 枚举 | 描述 |
| --- | --- |
| [InterpolationMode](../com.aspose.xps.rendering/interpolationmode) | 指定在对图像进行缩放或旋转时使用的算法。 |
| [PdfEncryptionAlgorithm](../com.aspose.xps.rendering/pdfencryptionalgorithm) | 加密模式枚举。 |
| [PdfImageCompression](../com.aspose.xps.rendering/pdfimagecompression) | 指定在 PDF 文件中应用于图像的压缩类型。 |
| [PdfTextCompression](../com.aspose.xps.rendering/pdftextcompression) | 指定在 PDF 文件中除图像外的所有内容所使用的压缩类型。 |

| [TextRenderingHint](../com.aspose.xps.rendering/textrenderinghint) | 指定文本渲染的质量。 |
