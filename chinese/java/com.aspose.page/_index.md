该 **Aspose.Page** 是 Aspose.Page 库中所有类的根包，这些类要么直接位于其中，如 **Device**，要么通过多个子包间接包含。

| 类 | 描述 |

| [DimensionF](../com.aspose.page/dimensionf) | `Dimension` 类将组件的宽度和高度（单精度）封装在一个对象中。 |
| [Document](../com.aspose.page/document) | 所有封装文档的超类。 |
| [ExternalFontCache](../com.aspose.page/externalfontcache) | 使用此类获取以 Device 接受的形式封装的字体。 |
| [License](../com.aspose.page/license) | 提供对组件授权的方法。 |
| [Metered](../com.aspose.page/metered) | 提供设置计量密钥的方法。 |
| [SaveOptions](../com.aspose.page/saveoptions) | 此类包含管理转换过程所需的选项。 |
| [TransformedStroke](../com.aspose.page/transformedstroke) | 包含变换的笔画。 |
| [UserProperties](../com.aspose.page/userproperties) | 特殊属性类，允许设置和返回类型化属性。 |
