## SaveOptions.JpegQualityLevel property

质量类别指定图像的压缩级别。 可用值为 0 到 100。 指定的数字越小，压缩越高，因此图像质量越低。 0 值导致图像质量最低，而 100 导致图像质量最高。

```csharp
public int JpegQualityLevel { get; set; }
```
