## SaveOptions.SupressErrors property

指定是否必须抑制错误。 如果 true 抑制错误添加到[`Exceptions`](../exceptions/)list. 如果为假，第一个错误将终止程序。

```csharp
public virtual bool SupressErrors { get; set; }
```
