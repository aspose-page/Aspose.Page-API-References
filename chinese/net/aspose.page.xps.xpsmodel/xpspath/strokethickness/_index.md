## XpsPath.StrokeThickness property

返回/设置笔划的粗细，以 有效坐标空间（包括路径的渲染变换）为单位。 笔划绘制在由 Path 元素的数据属性指定的 几何图形的边界之上。一半的 StrokeThickness extends 超出了 Data 属性指定的几何图形，另一半 扩展到了几何图形的内部。

```csharp
public float StrokeThickness { get; set; }
```
