## XpsCanvas class

封装 Canvas 元素特性的类。 此元素将元素组合在一起。例如，Glyphs 和 Path 元素 可以在画布中进行分组，以便被标识为一个单元（作为超链接目标）或 以将组合属性值应用于每个子元素和祖先元素。
