## XpsTileMode enumeration

平铺画笔的 TileMode 属性的有效值。

```csharp
public enum XpsTileMode
```

### 价值观

| 姓名 | 价值 | 描述 |
| --- | --- | --- |
| None | `0` | 在此模式下，仅绘制单个基础图块。剩余区域保持透明。 |
| Tile | `1` | 在此模式下，绘制基础图块并通过重复 base 图块填充剩余区域，这样每个图块的右边缘邻接下一个图块的左边缘，每个图块的 bottom 边缘邻接下一个图块的顶部边缘接下来. |
| FlipX | `2` | 瓦片排列类似于瓦片瓦片模式，但瓦片的交替列是 水平翻转。基础图块按照视口指定的方式定位。瓦片 in 此瓦片左右两列水平翻转。 |
| FlipY | `3` | 瓦片排列类似于瓦片瓦片模式，但瓦片的交替行垂直翻转 。基础图块按照视口指定的方式定位。上面和下面的行是 垂直翻转的。 |
| FlipXY | `4` | 瓦片排列类似于瓦片瓦片模式，但瓦片的交替列 水平翻转，瓦片的交替行垂直翻转。基础图块是 由视口指定的位置。 |
