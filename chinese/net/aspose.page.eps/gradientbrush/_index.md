## GradientBrush class

此类用于封装 LinearGradientBrush 和 PathGradientBrush，可以将 wrap 模式设置为 clamp。 当有人尝试将 WrapMode 属性设置为 WrapMode.Clamp 时，本机渐变画笔总是抛出异常。

```csharp
public class GradientBrush : Brush
```
