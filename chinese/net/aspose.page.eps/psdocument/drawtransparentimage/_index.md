## PsDocument.DrawTransparentImage method

绘制变换后的透明图像。如果图像没有 Alpha 通道，它将被绘制为不透明 image

```csharp
public void DrawTransparentImage(Bitmap image, Matrix transform, int transparencyThreshold)
```

| 范围 | 类型 | 描述 |
| --- | --- | --- |
| image | Bitmap | 要绘制的图像。 |
| transform | Matrix | 转换图像的矩阵。 |
| transparencyThreshold | Int32 | 定义从哪个透明度像素值将被解释为完全透明的阈值。所有低于此阈值的值都将被解释为完全不透明。 |
