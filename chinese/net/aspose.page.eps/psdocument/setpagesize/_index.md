## PsDocument.SetPageSize method

设置页面大小。要在一个文档中创建不同大小的页面，请使用[`SetPageDevice`](../setpagedevice/) 方法就在这个方法之后。

```csharp
public void SetPageSize(float width, float height)
```

| 范围 | 类型 | 描述 |
| --- | --- | --- |
| width | Single | 生成的 PostScript 文件中的页面宽度。 |
| height | Single | 生成的 PostScript 文件中页面的高度。 |
