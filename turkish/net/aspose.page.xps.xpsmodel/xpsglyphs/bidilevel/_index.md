## XpsGlyphs.BidiLevel property

Unicode algoritması çift yönlü iç içe yerleştirme düzeyini belirten değeri döndürür/ayarlar. Çift değerler soldan sağa yerleşimi, tek değerler sağdan sola yerleşimi belirtir. Sağdan sola yerleşimi, çalıştırma kaynağını sağ tarafa yerleştirir pozitif ilerleme genişliklerine sahip (soldaki ilerlemeleri temsil eder) ilk glifin, sonraki gliflerini önceki glifin soluna yerleştirme.

public int BidiLevel { get; set; }
