## XpsGlyphs.BidiLevel property

Unicode algoritması çift yönlü iç içe yerleştirme düzeyini belirten değeri döndürür/ayarlar. Çift değerler soldan sağa düzeni, tek değerler sağdan sola düzeni ifade eder. Sağdan sola düzen çalıştırma başlangıcını sağ tarafa yerleştirir sonraki glifleri önceki glifin soluna yerleştiren pozitif ilerleme genişlikleriyle (sola ilerlemeleri temsil eder) ilk glifin,

```csharp
public int BidiLevel { get; set; }
```
