## XpsDocument.CreateArcSegment method

Yeni bir eliptik yay parçası oluşturur.

```csharp
public XpsArcSegment CreateArcSegment(PointF point, SizeF size, float rotationAngle, 
    bool isLargeArc, XpsSweepDirection sweepDirection, bool isStroked = true)
```

| Parametre | Tip | Tanım |
| --- | --- | --- |
| point | PointF | Eliptik yayın uç noktası. |
| size | SizeF | Bir x,y çifti olarak eliptik yayın x ve y yarıçapı. |
| rotationAngle | Single | Elipsin geçerli koordinat sistemine göre nasıl döndürüldüğünü gösterir. |
| isLargeArc | Boolean | Yayın 180 veya daha büyük bir tarama ile çizilip çizilmediğini belirler. |
| sweepDirection | XpsSweepDirection | Yayın çizildiği yön. |
| isStroked | Boolean | Yolun bu bölümü için konturun çizilip çizilmeyeceğini belirtir. |
