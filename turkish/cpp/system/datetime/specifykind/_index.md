## DateTime::SpecifyKind method

```cpp
static DateTime System::DateTime::SpecifyKind(DateTime value, DateTimeKind kind)
```


| Parameter | Type | Açıklama |
| --- | --- | --- |
| value | DateTime | Tik sayısını kopyalanacak [DateTime](../) nesnesi |
| tür | DateTimeKind | Yeni nesnenin yerel zaman, UTC zamanı veya hiçbiri olarak temsil edilip edilmemesi gerektiğini belirtir. |

### ReturnValue

**value** ile aynı sayıda tik'i ve **kind** tarafından belirtilen [DateTimeKind](../../datetimekind/) değerini temsil eden yeni bir [DateTime](../) nesnesi.
