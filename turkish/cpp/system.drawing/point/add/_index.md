## Point::Add method


Belirtilen [Size](../../size/) nesnesinin genişlik ve yükseklik değerlerini, belirtilen [Point](../) nesnesinin X ve Y koordinat değerlerine karşılık gelecek şekilde ekler.

```cpp
static Point System::Drawing::Point::Add(const Point &point, const Size &size)
```


| Parameter | Type | Açıklama |
| --- | --- | --- |
| nokta | const Point\& | Çevrilecek nokta |
| size | const Size\& | [Size](../../size/) nesnesi, **point**'in koordinat değerlerine eklenmesi gereken değerleri belirler. |

### ReturnValue

Yeni bir [Point](../) nesnesi; X koordinat değeri **point**'in X koordinat değeri ile **size**'ın genişlik değerinin toplamına, Y koordinat değeri ise **point**'in Y koordinat değeri ile **size**'ın yükseklik değerinin toplamına eşittir.
