## PointF::Add(const PointF\&, const Size\&) method


Belirtilen [Size](../../size/) nesnesinin genişlik ve yükseklik değerlerini, belirtilen [PointF](../) nesnesinin X ve Y koordinat değerlerine karşılık gelen şekilde ekler.

```cpp
static PointF System::Drawing::PointF::Add(const PointF &point, const Size &size)
```


| Parameter | Type | Açıklama |
| --- | --- | --- |
| nokta | const PointF\& | Çevrilecek nokta |
| size | const Size\& | [Size](../../size/) nesnesi, **point**'in koordinat değerlerine eklenmesi gereken değerleri belirler. |

### ReturnValue

X koordinat değeri **point**'in X koordinat değeri ile **size**'in genişlik değerinin toplamına eşit, Y koordinat değeri ise **point**'in Y koordinat değeri ile **size**'in yükseklik değerinin toplamına eşit olan yeni bir [PointF](../) nesnesi
