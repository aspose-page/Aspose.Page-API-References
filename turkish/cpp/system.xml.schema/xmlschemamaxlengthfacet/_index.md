## XmlSchemaMaxLengthFacet class

Bu sınıf, **simpleType** öğesinin veri değerinin maksimum uzunluğuna bir kısıtlama belirtmek için kullanılabilir. Uzunluk, **maxLength** öğesinin değerinden küçük olmalıdır.

