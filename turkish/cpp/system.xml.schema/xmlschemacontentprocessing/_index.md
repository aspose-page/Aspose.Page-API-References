## XmlSchemaContentProcessing enum


**any** ve **anyAttribute** öğe ikameleri için doğrulama moduyla ilgili bilgi sağlar.

```cpp
enum class XmlSchemaContentProcessing
```

### Değerler

| Ad | Değer | Açıklama |
| --- | --- | --- |
| None | 0 | Belge öğeleri doğrulanmaz. |
| Atla | 1 | Belge öğeleri iyi biçimlendirilmiş XML içermelidir ve şema tarafından doğrulanmaz. |
| Lax | 2 | İlişkili şema bulunursa, belge öğeleri doğrulanacaktır. Aksi takdirde hata atılmaz. |
| Strict | 3 | Şema işlemcisi, belge öğelerini doğrulamak için belirtilen ad alanı ile ilişkili bir şema bulmalıdır. |
