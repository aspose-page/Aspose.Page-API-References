## XmlSchemaMaxInclusiveFacet class

Bu sınıf, bir **simpleType** öğesinin maksimum değerine bir kısıtlama belirtmek için kullanılabilir. Öğenin değeri, **maxInclusive** öğesinin değeri ile eşit ya da daha küçük olmalıdır.

```cpp
class XmlSchemaMaxInclusiveFacet : public System::Xml::Schema::XmlSchemaFacet
```
