## XmlWriterSettings::get_NewLineChars method


Satır sonları için kullanılacak karakter dizisini döndürür.

```cpp
String System::Xml::XmlWriterSettings::get_NewLineChars()
```


### ReturnValue

Satır sonları için kullanılacak karakter dizisi. Bu, herhangi bir dize değerine ayarlanabilir. Ancak, geçerli bir XML sağlamak için yalnızca boşluk karakterleri, sekmeler, satır başı veya satır beslemeleri gibi geçerli boşluk karakterlerini belirtmelisiniz. Varsayılan değer **\\r\\n** (satır başı, yeni satır) olarak belirlenmiştir.
