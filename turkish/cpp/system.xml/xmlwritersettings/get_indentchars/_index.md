## XmlWriterSettings::get_IndentChars method


Girintileme sırasında kullanılacak karakter dizesini döndürür. Bu ayar, [XmlWriterSettings::set_Indent](../set_indent/) değeri **true** olarak ayarlandığında kullanılır.

```cpp
String System::Xml::XmlWriterSettings::get_IndentChars()
```


### ReturnValue

Girintileme sırasında kullanılacak karakter dizesi. Bu, herhangi bir dize değerine ayarlanabilir. Ancak, geçerli XML sağlamak için yalnızca boşluk karakterleri, sekmeler, satır başı karakterleri veya satır sonları gibi geçerli boşluk karakterlerini belirtmelisiniz. Varsayılan iki boşluktur.
