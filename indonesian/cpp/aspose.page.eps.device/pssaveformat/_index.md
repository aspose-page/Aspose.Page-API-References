## PsSaveFormat enum


Enumerasi ini berisi opsi format penyimpanan yang tersedia. Bisa berupa PS atau [EPS](../../aspose.page.eps/). [EPS](../../aspose.page.eps/) digunakan hanya untuk dokumen satu halaman sementara file PS dapat berisi sejumlah halaman.

```cpp
enum class PsSaveFormat
```

### Nilai

| Nama | Nilai | Deskripsi |
| --- | --- | --- |
| PS | 0 | Opsi ini menunjukkan bahwa dokumen yang dihasilkan harus berupa file PostScript (PS) atau Encapsulated PostScript ([EPS](../../aspose.page.eps/)). |
| EPS | 1 | Opsi ini menunjukkan bahwa dokumen yang dihasilkan harus berupa file Encapsulated PostScript ([EPS](../../aspose.page.eps/)). Ini hanya digunakan untuk dokumen satu halaman;. |
