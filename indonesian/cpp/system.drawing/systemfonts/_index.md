## SystemFonts class


Menyediakan sekumpulan objek [Font](../font/) yang telah dibuat sebelumnya yang mewakili font yang digunakan untuk menampilkan teks dalam elemen tampilan [Windows](../../system.windows/). Ini adalah tipe statis tanpa layanan instance. Anda tidak boleh pernah membuat instance darinya dengan cara apapun.

```cpp
class SystemFonts : public System::Object
```
