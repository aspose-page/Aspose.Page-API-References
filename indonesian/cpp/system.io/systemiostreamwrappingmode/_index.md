## SystemIOStreamWrappingMode enum


Menentukan mode operasi I/O yang akan dilakukan pembungkus pada aliran mirip [System::IO::Stream](../stream/).

```cpp
enum class SystemIOStreamWrappingMode
```

### Nilai

| Nama | Nilai | Deskripsi |
| --- | --- | --- |
| Biner | 0 | Mode yang memungkinkan operasi input untuk mengenkode byte aliran menjadi data char_type dan mendekode data char_type menjadi byte aliran untuk operasi output. |
| Konversi | 1 | Mode yang memungkinkan operasi input untuk mengonversi byte aliran dari tipe uint8_t ke tipe char_type dan sebaliknya untuk operasi output. |
