## STDIOStreamWrappingMode enum


Menentukan mode operasi I/O yang akan dilakukan pembungkus pada aliran mirip std::iostreams.

```cpp
enum class STDIOStreamWrappingMode
```

### Nilai

| Nama | Nilai | Deskripsi |
| --- | --- | --- |
| Biner | 0 | Mode yang memungkinkan operasi input untuk mendekode data aliran tipe char_type menjadi byte, dan mengkodekan byte menjadi data tipe char_type untuk operasi output. |
| Konversi | 1 | Mode yang memungkinkan operasi input untuk mengonversi data aliran dari tipe char_type ke tipe uint8_t dan sebaliknya untuk operasi output. |
