## FileMode enum


Menentukan bagaimana sebuah file harus dibuka.

```cpp
enum class FileMode
```

### Nilai

| Nama | Nilai | Deskripsi |
| --- | --- | --- |
| CreateNew | 1 | Buat file baru. Jika file sudah ada, sebuah pengecualian akan dilempar. |
| Buat | 2 | Buat file baru. Jika file sudah ada, file tersebut akan ditimpa. |
| Open | 3 | Buka file yang sudah ada. Jika file tidak ada, sebuah pengecualian akan dilempar. |
| OpenOrCreate | 4 | Buka file yang sudah ada atau buat file baru jika tidak ada. |
| Potong | 5 | Buka file yang sudah ada dan pangkas sehingga kosong. Jika file tidak ada, sebuah pengecualian akan dilempar. |
| Tambahkan | 6 | Buka file yang sudah ada dan lompat ke akhir file atau buat file baru jika tidak ada. |
