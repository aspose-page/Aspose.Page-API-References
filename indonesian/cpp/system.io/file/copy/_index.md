## File::Copy method


Menyalin file yang ditentukan ke lokasi yang ditentukan. Jika file tujuan sudah ada, sebuah parameter menentukan apakah harus ditimpa.

```cpp
static void System::IO::File::Copy(const String &sourceFileName, const String &destFileName, bool overwrite=false)
```


| Parameter | Tipe | Deskripsi |
| --- | --- | --- |
| sourceFileName | const String\& | Jalur file yang akan disalin |
| destFileName | const String\& | Jalur lokasi baru file yang akan disalin |
| timpa | bool | True jika file tujuan yang ada harus ditimpa, false jika penyalinan harus gagal jika file tujuan sudah ada |
