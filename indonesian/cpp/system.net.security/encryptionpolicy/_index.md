## EncryptionPolicy enum


Menumerasikan kebijakan enkripsi.

```cpp
enum class EncryptionPolicy
```

### Nilai

| Nama | Nilai | Deskripsi |
| --- | --- | --- |
| RequireEncryption | 0 | Memerlukan enkripsi dan tidak pernah mengizinkan cipher 'Null'. |
| AllowNoEncryption | 1 | Lebih memilih menggunakan enkripsi penuh tetapi cipher 'Null' dapat digunakan jika server menyetujui. |
| NoEncryption | 2 | Mengizinkan tanpa enkripsi dan meminta agar cipher 'Null' digunakan jika ujung lainnya dapat menangani cipher 'Null'. |
