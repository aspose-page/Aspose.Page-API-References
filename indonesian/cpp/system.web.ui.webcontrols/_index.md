| [DataGrid](./datagrid/) | Kelas dummy yang memungkinkan penautan kode terjemahan dengan referensi [DataGrid](./datagrid/), tetapi tidak mengeksekusinya. Tidak berisi anggota yang diimplementasikan dengan benar. Objek dari kelas ini hanya boleh dialokasikan menggunakan fungsi [System::MakeObject()](../system/makeobject/). Jangan pernah membuat instance tipe ini di stack atau menggunakan operator new, karena akan menyebabkan kesalahan runtime dan/atau kegagalan asersi. Selalu balut kelas ini dengan pointer [System::SmartPtr](../system/smartptr/) dan gunakan pointer ini untuk meneruskannya ke fungsi sebagai argumen. |
