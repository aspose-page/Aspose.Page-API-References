## Rfc2898DeriveBytes class


Mengimplementasikan derivasi kunci berbasis kata sandi, PBKDF2. Objek dari kelas ini hanya boleh dialokasikan menggunakan fungsi [System::MakeObject()](../../system/makeobject/). Jangan pernah membuat instance tipe ini di stack atau menggunakan operator new, karena akan menyebabkan kesalahan runtime dan/atau kegagalan asersi. Selalu balut kelas ini ke dalam pointer [System::SmartPtr](../../system/smartptr/) dan gunakan pointer tersebut untuk meneruskan ke fungsi sebagai argumen.

class Rfc2898DeriveBytes : public System::Security::Cryptography::DeriveBytes
