## HMACSHA512 class


Kode Pesan Berbasis Hash [Autentikasi](../../system.security.authentication/) yang menggunakan fungsi hash [SHA512](../sha512/). Implementasi sebagian. Objek dari kelas ini hanya boleh dialokasikan menggunakan fungsi [System::MakeObject()](../../system/makeobject/). Jangan pernah membuat instance tipe ini di stack atau menggunakan operator new, karena akan menyebabkan kesalahan runtime dan/atau kegagalan asersi. Selalu bungkus kelas ini ke dalam pointer [System::SmartPtr](../../system/smartptr/) dan gunakan pointer tersebut untuk meneruskan ke fungsi sebagai argumen.

```cpp
class HMACSHA512 : public System::Security::Cryptography::HashAlgorithm
```
