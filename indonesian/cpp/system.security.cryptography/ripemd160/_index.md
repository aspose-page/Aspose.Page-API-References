## RIPEMD160 class


Menghitung hash [RIPEMD160](./) untuk data input. Dalam C++ [RIPEMD160](./) bukan kelas abstrak. Objek dari kelas ini hanya boleh dialokasikan menggunakan fungsi [System::MakeObject()](../../system/makeobject/). Jangan pernah membuat instance tipe ini di stack atau menggunakan operator new, karena akan menyebabkan kesalahan waktu jalan dan/atau kegagalan asersi. Selalu balut kelas ini ke dalam pointer [System::SmartPtr](../../system/smartptr/) dan gunakan pointer tersebut untuk meneruskannya ke fungsi sebagai argumen.

```cpp
class RIPEMD160 : public System::Security::Cryptography::HashAlgorithm
```
