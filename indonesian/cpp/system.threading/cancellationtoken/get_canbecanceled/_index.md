## CancellationToken::get_CanBeCanceled method


Mendapatkan apakah token ini dapat berada dalam keadaan dibatalkan.

```cpp
bool System::Threading::CancellationToken::get_CanBeCanceled() const
```


### ReturnValue

true jika token ini dapat berada dalam keadaan dibatalkan; jika tidak, false.
## Catatan



Token yang dibuat dari [CancellationTokenSource](../../cancellationtokensource/) akan mengembalikan true, sementara token None akan selalu mengembalikan false.
