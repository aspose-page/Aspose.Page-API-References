## SoapParameterStyle enum


Menumerasikan format parameter dalam pesan SOAP.

```cpp
enum class SoapParameterStyle
```

### Nilai

| Nama | Nilai | Deskripsi |
| --- | --- | --- |
| Default | 0 | Jika '[SoapDocumentServiceAttribute](../soapdocumentserviceattribute/)' tidak diterapkan pada kelas, maka nilai default adalah 'Wrapped'. |
| Bare | 1 | Parameter ditempatkan dalam elemen XML yang mengikuti elemen 'Body'. |
