## TypeInfo::IsDefined method

```cpp
bool System::TypeInfo::IsDefined(const TypeInfo &attributeType, bool inherit) const
```


| Parameter | Tipe | Deskripsi |
| --- | --- | --- |
| attributeType | const TypeInfo\& | Tipe atribut khusus yang akan dicari. Pencarian mencakup tipe turunan. |
| inherit | bool | true untuk mencari rantai pewarisan anggota ini guna menemukan atribut; jika tidak, false. Parameter ini diabaikan untuk properti dan event. |

### ReturnValue
