## Array::cbegin method


Mengembalikan iterator ke elemen pertama yang bersifat const pada kontainer. Jika kontainer kosong, iterator yang dikembalikan akan sama dengan [cend()](../cend/).

```cpp
const_iterator System::Array<T>::cbegin() const noexcept
```


### ReturnValue

Iterator yang menunjuk ke elemen pertama yang bersifat const pada kontainer.
