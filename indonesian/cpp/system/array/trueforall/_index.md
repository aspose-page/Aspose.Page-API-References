## Array::TrueForAll method


Menentukan apakah semua elemen dalam array yang ditentukan memenuhi kondisi yang didefinisikan oleh predikat yang ditentukan.

```cpp
static bool System::Array<T>::TrueForAll(System::ArrayPtr<T> arr, System::Predicate<T> match)
```


| Parameter | Tipe | Deskripsi |
| --- | --- | --- |
| arr | System::ArrayPtr\<T\> | Elemen [Array](../) yang akan dicocokkan dengan kondisi |
| cocok | System::Predicate\<T\> | Predikat yang mendefinisikan kondisi untuk mencocokkan elemen array |

### ReturnValue

true jika semua elemen array arr memenuhi kondisi yang didefinisikan oleh predikat match, jika tidak false
