## Array::Resize method



| Parameter | Tipe | Deskripsi |
| --- | --- | --- |
| arr | ArrayPtr\<Type\>\& | [Array](../) untuk diubah ukurannya. Jika **arr** adalah null-pointer, array baru akan dibuat |
| new_size | int | Ukuran baru array, atau ukuran array baru ketika **arr** null |
