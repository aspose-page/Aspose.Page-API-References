## GC class


Mewakili Garbage Collection yang disimulasikan yang berperilaku lebih seperti stub yang pada dasarnya tidak melakukan apa-apa. Ini adalah tipe statis tanpa layanan instance. Anda tidak boleh membuat instance darinya dengan cara apapun.

```cpp
class GC : public System::Object
```
