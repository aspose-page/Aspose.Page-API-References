## SmartPtr::get_shared_count method


Mendapatkan jumlah pointer berbagi yang ada pada objek yang direferensikan, termasuk yang saat ini. Memastikan pointer saat ini berada dalam mode berbagi.

```cpp
int System::SmartPtr<T>::get_shared_count() const
```


### ReturnValue

Jumlah pointer berbagi yang ada ke objek yang direferensikan, jika ada. Jika pointer null, mengembalikan 0.
