## ValueType class


Kelas dasar untuk tipe nilai dengan pewarisan [Object](../object/) yang dipotong demi alasan kinerja. Tipe ini harus dialokasikan pada stack dan diteruskan ke fungsi dengan nilai atau referensi. Jangan pernah menggunakan kelas [System::SmartPtr](../smartptr/) untuk mengelola objek tipe ini.

```cpp
class ValueType
```
