## String::IsNull method


Memeriksa apakah string dianggap null. [String](../) dianggap null hanya jika dibuat melalui konstruktor [String()](../string/), dipindahkan, disalin atau ditetapkan dari string null atau metode [reset()](../reset/) dipanggil.

```cpp
bool System::String::IsNull() const
```


### ReturnValue

true jika string dianggap null, false sebaliknya.
