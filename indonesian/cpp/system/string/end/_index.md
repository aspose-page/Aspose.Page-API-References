## String::end method


Mengembalikan pointer ke akhir buffer string yang sebenarnya. Tidak pernah melakukan alokasi ulang apa pun. Tidak menjamin buffer berakhir dengan null.

```cpp
const UChar * System::String::end() const
```

### ReturnValue

Penunjuk ke elemen pertama setelah buffer string aktual, berakhiran null atau tidak.
