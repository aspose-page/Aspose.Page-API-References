## Char::TryParse method


Mencoba mengonversi string yang terdiri dari satu karakter menjadi karakter UTF-16. Fungsi ini berhasil hanya ketika string masukan tidak null dan memiliki panjang tepat satu karakter.

```cpp
static bool System::Char::TryParse(const System::String &s, char_t &result)
```


| Parameter | Tipe | Deskripsi |
| --- | --- | --- |
| s | const System::String\& | [String](../../string/) untuk mengonversi |
| hasil | char_t\& | Variabel output yang akan berisi hasil konversi jika konversi berhasil |

### ReturnValue

Benar jika konversi berhasil, jika tidak - false
