## XmlSchemaElement::get_IsNillable method


Mengembalikan informasi yang menunjukkan apakah **xsi:nil** dapat muncul dalam data instance. Menunjukkan apakah nilai nil eksplisit dapat diberikan kepada elemen.

```cpp
bool System::Xml::Schema::XmlSchemaElement::get_IsNillable()
```


### ReturnValue

Jika nillable **true**, ini memungkinkan sebuah instance elemen memiliki atribut **nil** yang disetel ke **true**. Atribut **nil** didefinisikan sebagai bagian dari namespace XML [Schema](../../) untuk instance. Nilai default adalah **false**. Opsional.
