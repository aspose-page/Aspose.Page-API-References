## XmlSchemaContentProcessing enum


Memberikan informasi tentang mode validasi pengganti elemen **any** dan **anyAttribute**.

```cpp
enum class XmlSchemaContentProcessing
```

### Nilai

| Nama | Nilai | Deskripsi |
| --- | --- | --- |
| None | 0 | Item dokumen tidak divalidasi. |
| Lewati | 1 | Item dokumen harus terdiri dari XML yang terbentuk dengan baik dan tidak divalidasi oleh skema. |
| Lax | 2 | Jika skema yang terkait ditemukan, item dokumen akan divalidasi. Tidak ada kesalahan yang akan dilemparkan sebaliknya. |
| Strict | 3 | Pemroses skema harus menemukan skema yang terkait dengan ruang nama yang ditunjukkan untuk memvalidasi item dokumen. |
