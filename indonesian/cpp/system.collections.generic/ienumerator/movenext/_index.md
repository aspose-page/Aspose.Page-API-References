## IEnumerator::MoveNext method


Memindahkan enumerator ke elemen berikutnya. Jika tidak ada elemen yang direferensikan sebelumnya, mengatur referensi ke elemen pertama yang tersedia. Jika akhir kontainer tercapai, tidak melakukan apa‑apa.

```cpp
virtual bool System::Collections::Generic::IEnumerator<T>::MoveNext()=0
```


### ReturnValue

True jika ada elemen yang tersedia setelah pemanggilan, false jika tidak.
