## XmlNamespaceScope enum


Mendefinisikan ruang lingkup namespace.

```cpp
enum class XmlNamespaceScope
```

### Nilai

| Nama | Nilai | Deskripsi |
| --- | --- | --- |
| All | 0 | Semua namespace yang didefinisikan dalam ruang lingkup node saat ini. Ini termasuk namespace xmlns:xml yang selalu dideklarasikan secara implisit. Urutan namespace yang dikembalikan tidak ditentukan. |
| ExcludeXml | 1 | Semua namespace yang didefinisikan dalam ruang lingkup node saat ini, kecuali namespace xmlns:xml, yang selalu dideklarasikan secara implisit. Urutan namespace yang dikembalikan tidak ditentukan. |
| Local | 2 | Semua namespace yang didefinisikan secara lokal pada node saat ini. |
