## DtdProcessing enum


Menentukan opsi untuk memproses DTD. Enumerasi [DtdProcessing](./) digunakan oleh kelas [XmlReaderSettings](../xmlreadersettings/).

```cpp
enum class DtdProcessing
```

### Nilai

| Nama | Nilai | Deskripsi |
| --- | --- | --- |
| Prohibit | 0 | Menentukan bahwa ketika DTD ditemui, sebuah [XmlException](../xmlexception/) dilempar dengan pesan yang menyatakan bahwa DTD dilarang. Ini adalah perilaku default. |
| Ignore | 1 | Menyebabkan elemen DOCTYPE diabaikan. Tidak ada pemrosesan DTD yang terjadi, dan DTD/DOCTYPE hilang pada output. |
| Parse | 2 | Digunakan untuk memparsing DTD. |
