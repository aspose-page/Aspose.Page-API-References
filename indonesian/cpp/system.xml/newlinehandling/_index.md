## NewLineHandling enum


Menentukan cara menangani pemutusan baris.

```cpp
enum class NewLineHandling
```

### Nilai

| Nama | Nilai | Deskripsi |
| --- | --- | --- |
| Replace | 0 | Karakter baris baru diganti agar sesuai dengan karakter yang ditentukan dalam nilai [XmlWriterSettings::set_NewLineChars](../xmlwritersettings/set_newlinechars/). |
| Entitize | 1 | Karakter baris baru di-entitas-kan. Pengaturan ini mempertahankan semua karakter ketika output dibaca oleh [XmlReader](../xmlreader/) yang menormalkan. |
| None | 2 | Karakter baris baru tidak diubah. Outputnya sama dengan input. |
