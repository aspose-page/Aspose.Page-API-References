## XmlWriterSettings::get_NamespaceHandling method


Mengembalikan nilai yang menunjukkan apakah [XmlWriter](../../xmlwriter/) harus menghapus deklarasi namespace duplikat saat menulis konten XML. Perilaku default adalah penulis mengeluarkan semua deklarasi namespace yang ada di resolver namespace penulis.

```cpp
System::Xml::NamespaceHandling System::Xml::XmlWriterSettings::get_NamespaceHandling()
```


### ReturnValue

Enum [NamespaceHandling](../../namespacehandling/) yang digunakan untuk menentukan apakah akan menghapus deklarasi namespace duplikat dalam [XmlWriter](../../xmlwriter/).
