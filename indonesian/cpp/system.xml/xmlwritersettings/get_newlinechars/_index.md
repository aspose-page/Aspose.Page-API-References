## XmlWriterSettings::get_NewLineChars method


Mengembalikan string karakter yang digunakan untuk pemisah baris.

```cpp
String System::Xml::XmlWriterSettings::get_NewLineChars()
```


### ReturnValue

String karakter yang digunakan untuk pemisah baris. Ini dapat diatur ke nilai string apa pun. Namun, untuk memastikan XML yang valid, Anda harus menentukan hanya karakter spasi putih yang valid, seperti spasi, tab, carriage return, atau line feed. Defaultnya adalah **\r\n** (carriage return, baris baru).
