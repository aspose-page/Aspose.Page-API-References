## XmlWriterSettings::get_IndentChars method


Mengembalikan string karakter yang digunakan saat mengindentasi. Pengaturan ini digunakan ketika nilai [XmlWriterSettings::set_Indent](../set_indent/) diatur ke **true**.

```cpp
String System::Xml::XmlWriterSettings::get_IndentChars()
```


### ReturnValue

String karakter yang digunakan saat mengindentasi. Ini dapat diatur ke nilai string apa pun. Namun, untuk memastikan XML yang valid, Anda harus menentukan hanya karakter spasi putih yang sah, seperti spasi, tab, carriage return, atau line feed. Defaultnya adalah dua spasi.
