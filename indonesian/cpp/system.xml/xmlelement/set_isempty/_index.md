## XmlElement::set_IsEmpty method


Mengatur format tag elemen.

```cpp
void System::Xml::XmlElement::set_IsEmpty(bool value)
```


| Parameter | Tipe | Deskripsi |
| --- | --- | --- |
| value | bool | Nilai yang akan disetel. Saat menyetel nilai ini, jika disetel ke **true**, anak-anak elemen dihapus dan elemen diserialisasi dalam format tag pendek. Jika disetel ke **false**, nilai diubah (tanpa mempedulikan apakah elemen memiliki konten atau tidak); jika elemen kosong, ia diserialisasi dalam format panjang. |
