## XmlAttribute::get_BaseURI method


Mengembalikan Uniform Resource Identifier (URI) dasar dari node.

```cpp
String System::Xml::XmlAttribute::get_BaseURI() override
```


### ReturnValue

Lokasi dari mana node dimuat atau [String::Empty](../../../system/string/empty/) jika node tidak memiliki URI dasar. Node [Attribute](../../../system/attribute/) memiliki URI dasar yang sama dengan elemen pemiliknya. Jika node atribut tidak memiliki elemen pemilik, get_BaseURI mengembalikan [String::Empty](../../../system/string/empty/).
