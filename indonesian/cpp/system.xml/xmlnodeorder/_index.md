## XmlNodeOrder enum


Menjelaskan urutan dokumen sebuah node dibandingkan dengan node kedua.

```cpp
enum class XmlNodeOrder
```

### Nilai

| Nama | Nilai | Deskripsi |
| --- | --- | --- |
| Before | 0 | Node saat ini dari navigator ini berada sebelum node saat ini dari navigator yang diberikan. |
| Setelah | 1 | Node saat ini dari penjelajah ini berada setelah node saat ini dari penjelajah yang diberikan. |
| Sama | 2 | Kedua penjelajah berada pada node yang sama. |
| Tidak diketahui | 3 | Posisi node tidak dapat ditentukan dalam urutan dokumen, relatif satu sama lain. Hal ini dapat terjadi jika kedua node berada di pohon yang berbeda. |
