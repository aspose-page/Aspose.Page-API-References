## XmlDeclaration::get_Standalone method


Mengembalikan nilai atribut standalone.

```cpp
String System::Xml::XmlDeclaration::get_Standalone()
```


### ReturnValue

Nilai yang valid adalah **yes** jika semua deklarasi entitas yang diperlukan oleh dokumen XML terkandung dalam dokumen atau **no** jika diperlukan definisi tipe dokumen eksternal (DTD). Jika atribut standalone tidak ada dalam deklarasi XML, metode ini mengembalikan [String::Empty](../../../system/string/empty/).
