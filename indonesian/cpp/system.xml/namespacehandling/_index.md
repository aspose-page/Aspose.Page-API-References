## NamespaceHandling enum


Menentukan apakah akan menghapus deklarasi namespace duplikat dalam [XmlWriter](../xmlwriter/).

```cpp
enum class NamespaceHandling
```

### Nilai

| Nama | Nilai | Deskripsi |
| --- | --- | --- |
| Default | 0 | Menentukan bahwa deklarasi namespace duplikat tidak akan dihapus. |
| OmitDuplicates | 1 | Menentukan bahwa deklarasi namespace duplikat akan dihapus. Agar namespace duplikat dihapus, prefiks dan namespace harus cocok. |
