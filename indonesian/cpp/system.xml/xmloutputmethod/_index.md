## XmlOutputMethod enum


Menentukan metode yang digunakan untuk menyerialisasi output [XmlWriter](../xmlwriter/).

```cpp
enum class XmlOutputMethod
```

### Nilai

| Nama | Nilai | Deskripsi |
| --- | --- | --- |
| Xml | 0 | Serialisasi sesuai dengan aturan XML 1.0. |
| Html | 1 | Serialisasi sesuai dengan aturan HTML yang ditentukan oleh XSLT. |
| Text | 2 | Hanya serialisasi blok teks. |
| AutoDetect | 3 | Gunakan aturan XSLT untuk memilih antara metode output [XmlOutputMethod::Xml](./) dan [XmlOutputMethod::Html](./) pada waktu berjalan. |
