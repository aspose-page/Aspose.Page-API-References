## XmlWriter::WriteProcessingInstruction method


Ketika dioverride dalam kelas turunan, menulis instruksi pemrosesan dengan spasi antara nama dan teks sebagai berikut: **<?name text?>**.

```cpp
virtual void System::Xml::XmlWriter::WriteProcessingInstruction(String name, String text)=0
```


| Parameter | Tipe | Deskripsi |
| --- | --- | --- |
| name | String | Nama instruksi pemrosesan. |
| text | String | Teks yang akan dimasukkan ke dalam instruksi pemrosesan. |
## Catatan



Metode ini digunakan untuk membuat deklarasi XML setelah [XmlWriter::WriteStartDocument](../writestartdocument/) sudah dipanggil.
