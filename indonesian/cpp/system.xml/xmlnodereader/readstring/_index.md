### ReturnValue

Isi elemen atau node mirip teks (Ini dapat mencakup CDATA, [Text](../../../system.text/) node, dan sebagainya). Ini dapat berupa string kosong jika pembaca berada pada sesuatu selain elemen atau node teks, atau jika tidak ada lagi konten teks yang dapat dikembalikan dalam konteks saat ini. Catatan: Node teks dapat berupa elemen atau node teks atribut.
