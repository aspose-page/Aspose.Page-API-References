## XmlTextWriter::get_Indentation method


Mengembalikan berapa banyak **IndentChars** yang akan ditulis untuk setiap level dalam hierarki ketika [XmlTextWriter::set_Formatting](../set_formatting/) diatur ke [Formatting::Indented](../../formatting/).

```cpp
int32_t System::Xml::XmlTextWriter::get_Indentation()
```


### ReturnValue

Jumlah **IndentChars** untuk setiap level. Nilai default adalah 2.
