## HttpRequestCreator class


Membuat instance kelas WebRequest. Objek dari kelas ini hanya boleh dialokasikan menggunakan fungsi [System::MakeObject()](../../../system/makeobject/). Jangan pernah membuat instance tipe ini di stack atau menggunakan operator new, karena akan menyebabkan kesalahan runtime dan/atau kegagalan asersi. Selalu balut kelas ini ke dalam pointer [System::SmartPtr](../../../system/smartptr/) dan gunakan pointer ini untuk meneruskannya ke fungsi sebagai argumen.

```cpp
class HttpRequestCreator : public System::Net::IWebRequestCreate
```
