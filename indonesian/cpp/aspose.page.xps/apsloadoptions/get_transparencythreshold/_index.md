## ApsLoadOptions::get_TransparencyThreshold method


Nilai integer dari 0 hingga 255 di bawah mana piksel gambar yang mengandung nilai alfa akan dianggap sepenuhnya transparan. PostScript tidak mendukung transparansi, tetapi dapat menerapkan mask eksplisit di atas gambar berwarna dimana beberapa piksel akan sepenuhnya tidak tembus dan beberapa akan sepenuhnya transparan. Ambang transparansi digunakan untuk menemukan kemiripan terbaik antara asli dan hasil PostScript. Nilai default adalah 255.

```cpp
int32_t Aspose::Page::XPS::ApsLoadOptions::get_TransparencyThreshold() const
```
