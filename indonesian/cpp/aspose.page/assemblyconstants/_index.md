## AssemblyConstants class


Mendefinisikan konstanta yang berpartisipasi dalam pemeriksaan lisensi untuk komponen. Konstanta ini sebelumnya didefinisikan langsung sebagai atribut assembly, tetapi saya memindahkannya ke kelas terpisah karena di .NET Compact Framework saya tidak dapat mengakses atribut assembly. Sekarang kode lisensi ketika dikompilasi untuk .NET Compact Framework menggunakan konstanta ini alih-alih atribut assembly.

```cpp
class AssemblyConstants : public System::Object
```
