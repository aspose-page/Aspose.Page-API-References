## TraceLevel enum


Menentukan pesan apa yang akan dikeluarkan untuk kelas [System.Diagnostics.Debug](../debug/), [System.Diagnostics.Trace](../trace/) dan System.Diagnostics.TraceSwitch.

```cpp
enum class TraceLevel
```

### Nilai

| Nama | Nilai | Deskripsi |
| --- | --- | --- |
| Off | 0 | Tidak mengeluarkan pesan tracing dan debugging. |
| Kesalahan | 1 | Mengeluarkan pesan penanganan error. |
| Warning | 2 | Keluarkan peringatan dan pesan penanganan kesalahan. |
| Info | 3 | Keluarkan pesan informatif, peringatan, dan pesan penanganan kesalahan. |
| Detail | 4 | Keluarkan semua pesan debug dan pelacakan. |
