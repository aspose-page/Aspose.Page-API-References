Menjelaskan output dari beberapa halaman logis ke satu lembar fisik. Semua dokumen dalam pekerjaan digabungkan secara berurutan. JobNUpAllDocumentsContiguously dan DocumentNUp bersifat saling eksklusif. Penentuan penanganan batasan antara kata kunci ini menjadi tanggung jawab driver. https://docs.microsoft.com/en-us/windows/win32/printdocs/jobnupalldocumentscontiguously
