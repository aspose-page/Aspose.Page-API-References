## Bidang

| Bidang | Deskripsi |
| --- | --- |
| [Automatic](#Automatic) | Menentukan maksud untuk output otomatis (memungkinkan klien secara otomatis memilih pengaturan terbaik). |
| [Draft](#Draft) | Menentukan maksud untuk output draf (seimbang untuk teks dan grafik kualitas draf). |
| [Fax](#Fax) | Menentukan maksud untuk output faks (seimbang untuk kualitas faks standar). |
| [High](#High) | Menentukan maksud untuk output kualitas tinggi (seimbang untuk teks dan grafik kualitas tinggi). |
| [Normal](#Normal) | Menentukan maksud untuk output normal (seimbang untuk teks dan grafik berkualitas baik). |
| [Photographic](#Photographic) | Menentukan maksud untuk output fotografi (gambar harus memiliki kualitas tertinggi). |
| [Text](#Text) | Menentukan maksud untuk output hanya teks (grafik mungkin keluar dengan buruk atau tidak sama sekali). |
