## Ruang nama

| Ruang nama | Keterangan |
| --- | --- |
| [Aspose.Page](./aspose.page/) | Itu **Aspose.Halaman** adalah root namespace untuk semua kelas perpustakaan Aspose.Page yang langsung ada di dalamnya **Perangkat** atau secara tidak langsung melalui beberapa subnamespaces. |
| [Aspose.Page.EPS](./aspose.page.eps/) | Itu **Aspose.Page.EPS** adalah root namespace untuk semua kelas yang berhubungan dengan file PS/EPS. |
| [Aspose.Page.EPS.Device](./aspose.page.eps.device/) | Itu **Aspose.Page.EPS.Device** namespace menyediakan kelas perangkat yang memungkinkan dan menyimpan opsi yang dapat digunakan saat mengonversi PS/EPS ke format lain. |
| [Aspose.Page.EPS.XMP](./aspose.page.eps.xmp/) | Itu **Aspose.Page.EPS.XMP** namespace menyediakan kelas untuk membaca, mengubah metadata XMP di file EPS. |
| [Aspose.Page.Font](./aspose.page.font/) | Itu **Aspose.Page.Font** namespace berisi kelas untuk manipulasi font. |
| [Aspose.Page.XPS](./aspose.page.xps/) | Itu **Aspose.Page.XPS** adalah root namespace untuk semua kelas yang berhubungan dengan dokumen XPS. |
| [Aspose.Page.XPS.Presentation](./aspose.page.xps.presentation/) | Itu **Aspose.Page.Xps.Presentation**namespace menyediakan kelas dasar untuk merender dokumen XPS ke format lain. |
| [Aspose.Page.XPS.Presentation.Image](./aspose.page.xps.presentation.image/) | Itu **Aspose.Page.Xps.Presentation.Image** namespace menyediakan kelas untuk merender dokumen XPS ke gambar. |
| [Aspose.Page.XPS.Presentation.Pdf](./aspose.page.xps.presentation.pdf/) | Itu **Aspose.Page.Xps.Presentation.Pdf** namespace menyediakan kelas untuk merender dokumen XPS ke dokumen PDF. |
| [Aspose.Page.XPS.XpsMetadata](./aspose.page.xps.xpsmetadata/) | Itu **Aspose.Page.Xps.XpsMetadata** namespace menyediakan kelas yang mendeskripsikan metadata dokumen XPS. |
| [Aspose.Page.XPS.XpsModel](./aspose.page.xps.xpsmodel/) | Itu **Aspose.Page.Xps.XpsModel**namespace menyediakan kelas yang mendeskripsikan elemen dokumen XPS. |
