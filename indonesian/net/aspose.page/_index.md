Itu **Aspose.Page** adalah root namespace untuk semua kelas perpustakaan Aspose.Page yang langsung ada di dalamnya **Perangkat** atau secara tidak langsung melalui beberapa subnamespaces.

## Kelas

| Kelas | Keterangan |
| --- | --- |
| [BuildVersionInfo](./buildversioninfo/) | Kelas ini memberikan informasi tentang pembuatan produk saat ini. |
| [Device](./device/) | Kelas ini merangkum rendering dokumen ke perangkat abstrak. Rendering dokumen dilakukan halaman demi halaman. |
| [Document](./document/) | Superclass untuk semua dokumen yang dienkapsulasi. |
| [ExternalFontCache](./externalfontcache/) | Gunakan kelas ini untuk mendapatkan enkapsulasi font dalam bentuk yang diterima oleh[`Device`](../aspose.page/device/) . |
| [License](./license/) | Menyediakan metode untuk melisensikan komponen. |
| [Margins](./margins/) | Kelas ini merangkum margin atas, kiri, bawah, dan kanan. |
| [Metered](./metered/) | Menyediakan metode untuk menyetel kunci terukur. |
| [SaveOptions](./saveoptions/) | Kelas ini berisi opsi yang diperlukan untuk mengelola proses konversi. |
| [UserProperties](./userproperties/) | Kelas properti khusus yang memungkinkan properti yang diketik disetel dan dikembalikan. Ini juga memungkinkan pemasangan dua objek properti default untuk dicari jika objek properti ini tidak berisi properti. |
## Antarmuka

| Antarmuka | Keterangan |
| --- | --- |
| [IGlyph](./iglyph/) | Antarmuka ini memberikan akses ke parameter utama mesin terbang. |
| [IMultiPageDevice](./imultipagedevice/) | Antarmuka ini berisi metode untuk memanipulasi perangkat multi-halaman. |
| [IStreamable](./istreamable/) | Antarmuka ini digunakan untuk perangkat yang merender dokumen ke aliran. |
| [ITrFont](./itrfont/) | Antarmuka ini memberikan akses ke parameter utama font. |
## Pencacahan

| Pencacahan | Keterangan |
| --- | --- |
