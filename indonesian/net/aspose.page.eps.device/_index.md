Itu **Aspose.Page.EPS.Device** namespace menyediakan kelas perangkat yang memungkinkan dan menyimpan opsi yang dapat digunakan saat mengonversi PS/EPS ke format lain.

## Kelas

| Kelas | Keterangan |
| --- | --- |
| [ImageDevice](./imagedevice/) | Kelas ini merangkum rendering dokumen ke gambar. |
| [ImageSaveOptions](./imagesaveoptions/) | Kelas ini berisi opsi yang diperlukan untuk mengelola proses penyimpanan gambar. |
| [PdfDevice](./pdfdevice/) | Kelas ini merangkum rendering dokumen ke PDF. |
| [PdfSaveOptions](./pdfsaveoptions/) | Kelas ini berisi input dan output stream dan opsi lain yang diperlukan untuk mengelola proses konversi. |
| [PsSaveOptions](./pssaveoptions/) | Kelas ini berisi opsi-opsi yang diperlukan untuk mengelola proses konversi dokumen ke file PostScript (PS) atau Encapsulated PostScript (EPS). |
