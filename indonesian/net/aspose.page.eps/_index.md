Itu **Aspose.Page.EPS** adalah root namespace untuk semua kelas yang berhubungan dengan file PS/EPS.

## Kelas

| Kelas | Keterangan |
| --- | --- |
| [FontConstants](./fontconstants/) | Kelas ini mendefinisikan sekumpulan konstanta untuk menyimpan font. |
| [GradientBrush](./gradientbrush/) | Kelas ini digunakan untuk mengenkapsulasi LinearGradientBrush dan PathGradientBrush dengan kemungkinan untuk menyetel mode bungkus ke clamp. Kuas gradien asli selalu mengeluarkan pengecualian saat seseorang mencoba menyetel properti WrapMode ke WrapMode.Clamp. |
| [LoadOptions](./loadoptions/) | Kelas dasar untuk opsi pemuatan dokumen. |
| [PageConstants](./pageconstants/) | Kelas ini mendefinisikan sekumpulan konstanta yang mendeskripsikan sebuah halaman. Objek kenyamanan disediakan untuk berbagai margin, orientasi, penskalaan ulang, dan ukuran halaman standar. |
| [PsConverterException](./psconverterexception/) | Kelas ini berisi informasi tentang error yang dilontarkan saat file PS diubah menjadi dokumen PDF. |
| [PsDocument](./psdocument/) | Kelas ini merangkum dokumen PS/EPS. |
| [PsDocumentException](./psdocumentexception/) | Kelas ini berisi informasi tentang error yang dilontarkan saat file PostScript dibuat, diedit, dan disimpan. |
| [PsLoadOptions](./psloadoptions/) | Opsi pemuatan dokumen PS. |
