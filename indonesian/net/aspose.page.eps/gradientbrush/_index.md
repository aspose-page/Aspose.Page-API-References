## GradientBrush class

Kelas ini digunakan untuk mengenkapsulasi LinearGradientBrush dan PathGradientBrush dengan kemungkinan untuk menyetel mode bungkus ke clamp. Kuas gradien asli selalu mengeluarkan pengecualian saat seseorang mencoba menyetel properti WrapMode ke WrapMode.Clamp.

```csharp
public class GradientBrush : Brush
```

## Konstruktor

| Nama | Keterangan |
| --- | --- |
| [GradientBrush](gradientbrush/)(Brush) | Membuat instance baru GradientBrush.  LinearGradientBrush atau PathGradientBrush. Jika selain ArgumentException akan dilempar. |
