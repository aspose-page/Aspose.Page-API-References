## PsDocument.DrawTransparentImage method

Menggambar gambar transparan yang diubah. Jika gambar tidak memiliki saluran Alpha, itu akan digambar sebagai gambar buram

```csharp
public void DrawTransparentImage(Bitmap image, Matrix transform, int transparencyThreshold)
```

| Parameter | Jenis | Keterangan |
| --- | --- | --- |
| image | Bitmap | Gambar untuk menggambar. |
| transform | Matrix | Matriks untuk mengubah gambar. |
| transparencyThreshold | Int32 | Ambang batas yang menentukan dari mana nilai piksel transparansi akan ditafsirkan sepenuhnya transparan. Semua nilai di bawah ambang batas ini akan ditafsirkan sepenuhnya buram. |
