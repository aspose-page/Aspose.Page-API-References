## XpsGlyphs.BidiLevel property

Mengembalikan/mengatur nilai yang menentukan tingkat penyatuan dua arah algoritme Unicode. Nilai genap menyiratkan tata letak kiri ke kanan, nilai ganjil menyiratkan tata letak kanan ke kiri. Tata letak kanan ke kiri menempatkan asal proses di sisi kanan mesin terbang pertama, dengan lebar muka positif (mewakili kemajuan ke kiri) menempatkan mesin terbang berikutnya di sebelah kiri mesin terbang sebelumnya.

```csharp
public int BidiLevel { get; set; }
```
