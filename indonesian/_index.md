## Aspose.Page untuk .NET

{{% alert color="primary" %}}

![Aspose.Page untuk Logo Produk .NET](../home_1.png)


Aspose.Page adalah .NET API dan dibangun untuk memungkinkan pengembang bekerja dengan dokumen XPS dan EPS/PS. Dengan menggunakan API, Anda dapat membuat, mengedit, dan menyimpan dokumen XPS yang sudah ada maupun yang baru. Selanjutnya, Anda dapat mengonversi dokumen XPS dan EPS/PS ke PDF dan Gambar. Aspose.Page untuk .NET diimplementasikan menggunakan C# dan dapat digunakan dengan bahasa .NET apa pun seperti C#, VB.NET dan J# dll. Aspose.Page dapat diintegrasikan dengan aplikasi apa pun baik itu Aplikasi Web ASP.NET atau Windows Aplikasi.

## Aspose.Page untuk C++

{{% alert color="primary" %}}

![Aspose.Page untuk Logo Produk C++](../home_3.png)


Aspose.Page untuk C++ adalah API manipulasi dan konversi file XPS dan PostScript yang andal dan mandiri. Ini memungkinkan Anda untuk membuat, mengedit, dan mengonversi file XPS tanpa memerlukan alat tambahan apa pun. API memungkinkan operasi seperti manipulasi halaman & elemen dokumen, membuat grafik vektor, mengelompokkan bentuk, dan menentukan warna dalam ruang warna yang berbeda. Selain itu, ini memungkinkan pengembang untuk mengonversi file PS dan EPS ke dokumen PDF, gambar raster seperti BMP, EMF, GIF, JPEG, PNG, TIFF, dan banyak lagi.
